이 프로젝트는 "텐서플로2와 머신러닝으로 시작하는 자연어처리" 책의 리소스 집약적 예제들을 Colab 환경에서 실행할 수 있도록 구성한 실습 저장소입니다. 로지스틱 회귀부터 최신 BERT 및 GPT2 모델까지 다양한 자연어 처리 기법을 GPU 가속 환경에서 실습할 수 있습니다. Colab 노트북 파일을 개인 드라이브로 복사한 후 GPU 리소스를 선택해 실행하는 방식으로 진행됩니다.
