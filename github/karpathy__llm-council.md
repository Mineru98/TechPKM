LLM Council은 여러 대형 언어 모델(LLM)을 통합하여 사용자 질의에 대한 다양한 의견을 비교 분석하는 로컬 웹 애플리케이션입니다. OpenRouter를 통해 여러 LLM에 동시에 질의를 전송하고, 각 모델의 응답을 비교/평가한 후 의장 모델이 최종 답변을 생성합니다. 이 프로젝트는 기술적 탐구와 재미로 제작되었으며, "책을 함께 읽는 LLM" 프로젝트에서 영감을 받아 개발되었습니다.  

**핵심 기능**:  
1. 다중 LLM 응답 수집 및 탭 뷰 비교  
2. 익명화된 모델 간 응답 순위 평가  
3. 의장 모델의 통합 최종 응답 생성  
**기술 스택**: FastAPI(Backend), React(Vite 기반 프론트엔드), OpenRouter API 연동  

(※ 프로젝트 유지보수 중단 상태: "vibe 코드"로 분류되며, 사용자 직접 수정 권장)
