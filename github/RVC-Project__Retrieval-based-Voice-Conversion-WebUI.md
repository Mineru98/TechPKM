이 프로젝트는 VITS 기반의 직관적인 웹 인터페이스 음성 변환 프레임워크로, 고품질 음성 변환을 위한 다양한 기능을 제공합니다. 훈련 및 실시간 변환 기능을 지원하며, 적은 데이터로도 우수한 결과를 도출할 수 있으며, AMD/Intel GPU 가속 및 다양한 전처리 기능을 포함합니다. 오픈소스 라이선스(MIT)로 자유롭게 사용 가능합니다. 주요 특징으로는 음색 누출 방지, 모델 융합, UVR5 기반 음성 분리, RMVPE 기반 음고 추출 등이 있습니다.
