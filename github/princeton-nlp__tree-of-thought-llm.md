이 프로젝트는 논문 "Tree of Thoughts: Deliberate Problem Solving with Large Language Models"의 공식 구현체로, LLM을 활용한 체계적인 문제 해결 프레임워크를 제공합니다. BFS/DFS 기반 탐색 알고리즘, 게임 24 문제 해결, 창의적 글쓰기 등 다양한 작업에 적용 가능한 모듈식 구조를 특징으로 합니다. OpenAI API와 연동된 파이썬 패키지 형태로 제공되며, 사용자 정의 작업 추가 기능을 지원합니다.  

핵심 기능으로는 사고 과정 생성/평가/선택 전략, 샘플링 프롬프트 시스템, 실험 재현 스크립트 등을 포함합니다. MIT 라이선스로 배포되며, 연구 논문 인용 시 활용할 수 있습니다.
