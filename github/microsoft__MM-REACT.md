MM-REACT는 ChatGPT와 다양한 비전 전문가 모델을 통합하여 복잡한 영상 이해 작업을 수행하는 멀티모달 추론 시스템 패러다임입니다. 이미지 파일 경로를 입력으로 받아 ChatGPT가 필요에 따라 비전 전문가(예: 유명인 인식, 객체 탐지)의 도움을 받아 추론을 진행하고, 이를 텍스트 형식으로 통합하여 응답합니다. 주요 기술로 Azure OpenAI GPT-4.0, Computer Vision 서비스, Bing 검색 API를 활용하며, Langchain 기반으로 구현되어 있습니다. 실시간 데모와 문서화된 샘플 코드를 제공합니다.
