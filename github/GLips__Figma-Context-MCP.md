이 프로젝트는 Figma 설계 데이터를 AI 코딩 도구(예: Cursor)에 연동하기 위한 Model Context Protocol(MCP) 서버를 제공합니다. Figma 파일, 프레임, 그룹 링크를 통해 AI가 디자인 구조를 정확히 이해하고 한 번에 코드 구현이 가능하도록 핵심 레이아웃 및 스타일 정보를 필터링하여 전달합니다. Figma API 응답을 단순화·변환하여 AI의 정확도와 관련성을 높이는 것이 핵심 기능입니다. JavaScript/Node.js 기반 구현체로, 환경 설정 및 사용 방법은 README 문서에 상세히 기술되어 있습니다.
