midi2musicxml은 MIDI 파일과 가사 텍스트 파일을 MusicXML 형식으로 변환하는 프로젝트입니다. NEUTRINO 악보 생성 소프트웨어와 연동되며, 한자 가사 지원과 GUI/CUI 인터페이스를 제공합니다. 더 이상 유지보수되지 않으며, 대체 프로젝트로 tyouseisientool을 권장합니다.  

> 참고: 본 프로젝트는 Java 11 이상 환경에서 실행 가능하며, GUI 기반 사용 편의성과 다양한 출력 옵션을 특징으로 합니다.
