ChatWithPDF는 PDF 문서와 자연어 대화를 가능하게 하는 파이썬 기반 프로젝트입니다. 스트림릿(Streamlit)으로 구현된 웹 인터페이스에서 LLM(Large Language Model)을 활용해 PDF 내용을 질의·응답할 수 있습니다. 사용자는 개인 API 키를 설정해 문서 분석 기능을 활용할 수 있으며, 기술 문서나 연구 자료 등의 대화형 탐색이 주요 용도입니다.
