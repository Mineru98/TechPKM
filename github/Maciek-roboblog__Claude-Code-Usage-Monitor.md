Claude AI 토큰 사용량을 실시간으로 모니터링하는 터미널 기반 도구입니다. 고급 분석, 머신러닝 기반 예측, 비용 분석, 세션 한계 감지 등의 기능을 제공하며, 사용자 정의 가능한 계획과 다양한 보기 옵션(실시간, 일간, 월간)을 지원합니다. Python 3.9 이상에서 동작하며 MIT 라이선스를 따릅니다.
