클로드 AI의 토크너 사용량을 실시간으로 모니터링하는 터미널 기반 도구입니다. 고급 분석, 머신러닝 기반 예측, 풍부한 UI를 제공하며, 토크너 소비량, 소모율, 비용 분석 및 세션 제한에 대한 지능형 예측을 지원합니다. WCAG 호환 색상 체계, 자동 테마 감지, 사용자 정의 가능한 갱신 주기 등의 기능을 포함합니다.  

프로, 맥스5, 맥스20 구독 플랜 및 사용자 정의 플랜을 지원하며, 사용자 패턴에 기반한 P90 백분위수 분석을 통해 정확한 한계값을 자동으로 감지합니다. 실시간 모니터링, 일일/월간 집계 뷰, 다중 경고 시스템을 제공하여 효율적인 리소스 관리를 가능하게 합니다.
