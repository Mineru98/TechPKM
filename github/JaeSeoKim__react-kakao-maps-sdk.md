# `@woai345/react-kakao-maps-sdk`

> 📦 React wrapper for [Kakao Maps JavaScript API](https://apis.map.daum.net/javascript/v2/)

This package provides a React component for embedding Kakao Maps in React applications.

## 🧩 Features

- Simple React component for Kakao Maps
- TypeScript support
- Customizable map options
- Support for common map controls (zoom, type, etc.)

## 📦 Installation

```bash
npm install @woai345/react-kakao-maps-sdk
```

## 📌 Usage

```tsx
import React from 'react'
import KakaoMap from '@woai345/react-kakao-maps-sdk'

const App = () => {
  return (
    <KakaoMap
      center={{ lat: 37.5665, lng: 126.9780 }}
      style={{ width: '100%', height: '500px' }}
      mapTypeId="ROADMAP"
    />
  )
}
```

## 🧪 Development

This project uses [Vite](https://vitejs.dev/).

```bash
npm run dev
