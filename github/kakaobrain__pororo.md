PORORO는 자연어 처리 및 음성 관련 작업을 위한 딥러닝 모델 플랫폼입니다. 간단한 작업 이름 입력으로 다양한 NLP/음성 처리 작업(예: 개체명 인식, 기계 번역, 감정 분석)을 수행할 수 있으며, 다국어 모델 지원과 다양한 사전 학습 모델을 제공합니다. Python 기반으로 설치 및 사용이 간편하며, 오픈소스 Apache 2.0 라이선스로 배포됩니다.
