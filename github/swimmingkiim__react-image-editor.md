React 기반 웹 이미지 편집기로, 캔버스 기반의 다양한 편집 기능을 제공합니다. 히스토리 관리(실행 취소/재실행), 색상 조정, 탭 관리, 프레임 조작, 객체 편집(이미지/도형/아이콘/텍스트) 및 내보내기 기능을 포함하며, 로컬 스토리지에 색상 목록을 저장합니다. 리액트 프로젝트에서 그래픽 편집 기능이 필요한 경우 적합한 솔루션입니다.
