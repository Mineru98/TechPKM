LLaMA-persona는 대규모 언어 모델에 캐릭터 페르소나를 부여하여 대화형 상호작용을 구현하는 프로젝트입니다. 사용자는 alpaca, vicuna 등의 llama.cpp 호환 모델에 사용자 정의 페르소나를 적용하여 캐릭터 역할을 부여하는 대화형 AI를 만들 수 있습니다. 설정 파일에서 캐릭터의 이름과 설명을 편집하는 방식으로 작동하며, ChatGPT를 활용해 페르소나 설명을 자동 생성할 수 있습니다. 예제는 1년 차 강아지 캐릭터를 통해 인간과 자연스럽게 대화하는 방식을 보여줍니다. Python 3.9 이상에서 동작하며, 로컬 환경에서 실행 가능한 경량화 LLM 애플리케이션입니다.
