이 프로젝트는 Rust 언어로 작성된 WASI-NN 고수준 바인딩과 WasmEdge-TensorFlow 플러그인 예제를 제공합니다. 개발자는 이 예제를 참조하여 고수준 언어로 머신러닝 애플리케이션을 작성하고 WebAssembly로 컴파일한 후 WASI-NN을 지원하는 런타임(예: WasmEdge)에서 실행할 수 있습니다. OpenVINO, PyTorch, TensorFlow-Lite 백엔드와 호환되며, MobileNet 등의 딥러닝 모델을 활용하는 예제를 포함합니다. Apache 2.0 라이선스로 배포됩니다.
