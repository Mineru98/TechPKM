이 프로젝트는 Rust 언어를 사용한 고수준 wasi-nn 바인딩과 WasmEdge-TensorFlow 플러그인 예제를 제공합니다. 개발자는 고수준 언어로 머신러닝 애플리케이션을 작성한 후 WebAssembly로 컴파일하고 WasmEdge와 같은 런타임에서 실행할 수 있습니다. OpenVINO, PyTorch, TensorFlow-Lite 백엔드와 WasmEdge 플러그인을 지원하는 WASI-NN 기능을 활용하는 방법을 보여줍니다.  

```bash
# 주요 기술 스택 및 구성 요소
- WASI-NN 바인딩 (Rust)
- WasmEdge 런타임
- OpenVINO/PyTorch/TensorFlow 백엔드
- WebAssembly 머신러닝 애플리케이션
```
