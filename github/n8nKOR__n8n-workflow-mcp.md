n8n 워크플로우 검증 및 관리를 위한 MCP(Model Context Protocol) 서버입니다. 다국어(한국어, 영어, 일본어, 베트남어, 대만어, 태국어)를 지원하며, 워크플로우 문법 검증, n8n 노드 검색, 커뮤니티 워크플로우 템플릿 검색 기능을 제공합니다. Go 언어로 개발된 이 서버는 독립 실행형(Standalone) 또는 파일 시스템 모드로 운영 가능하며, 검색 가중치 조정 및 다국어 설정 기능을 포함합니다.
