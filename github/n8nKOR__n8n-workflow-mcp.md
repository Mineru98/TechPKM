n8n 워크플로우 검증 및 관리를 위한 MCP (Model Context Protocol) 서버입니다. 다국어 지원(한국어, 영어, 일본어 등)과 워크플로우 문법 검증, 커뮤니티 템플릿 검색 기능을 제공하며, 검색 가중치 조정 및 플랫폼별 빌드 옵션을 통해 유연하게 사용 가능합니다. Go 언어로 구현된 서버형 도구로, 개발 및 운영 환경에 최적화된 배포 방식을 지원합니다.
