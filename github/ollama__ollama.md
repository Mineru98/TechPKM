Ollama는 로컬 환경에서 대규모 언어 모델(LLM)을 쉽게 실행할 수 있는 오픈소스 프레임워크입니다. macOS, Windows, Linux, Docker 등 다양한 플랫폼에서 지원되며, GGUF 포맷 모델 임포트, 커스텀 프롬프트 설정, REST API 통합 등의 기능을 제공합니다. 7B부터 400B 파라미터 모델까지 다양한 크기의 모델을 지원하며, RAM 요구 사항에 따라 유연하게 선택할 수 있습니다. 커뮤니티 통합 및 라이브러리 지원을 통해 개발자와 일반 사용자 모두에게 접근성을 높였습니다.
