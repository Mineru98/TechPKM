Ollama는 로컬 환경에서 대규모 언어 모델을 실행하고 관리할 수 있는 오픈소스 프레임워크입니다. macOS, Windows, Linux 등 다양한 플랫폼에서 설치 가능하며, Gemma, Llama, Phi 등 다양한 모델의 다운로드 및 실행 기능을 제공합니다. REST API와 CLI를 통해 모델을 제어할 수 있으며, 사용자 정의 모델 생성 및 커스터마이징 기능을 지원합니다. 로컬에서 실행되므로 데이터 프라이버시를 유지하면서 LLM 기반 애플리케이션을 개발할 수 있습니다.
