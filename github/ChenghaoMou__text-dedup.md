이 프로젝트는 다양한 텍스트 중복 제거 알고리즘을 제공하는 오픈소스 도구로, MinHash, SimHash, Suffix Array, Bloom Filter 등의 기술을 활용하여 근사적 또는 정확한 중복을 식별합니다. TOML 설정 파일을 통해 알고리즘 파라미터를 쉽게 커스터마이징할 수 있으며, 학술 데이터셋 및 뉴스 데이터셋에서 검증된 벤치마크 결과를 포함합니다. Apache 2.0 라이선스로 배포되며, 대규모 텍스트 데이터 전처리 작업에 최적화된 솔루션입니다.
