이 프로젝트는 다양한 텍스트 중복 제거 알고리즘을 제공하는 Python 기반 도구입니다. MinHash, SimHash, SuffixArray, Bloom Filter 등의 알고리즘을 TOML 설정 파일로 유연하게 구성할 수 있으며, 학술 문서 및 뉴스 데이터셋 등에서의 성능 벤치마크 결과를 포함합니다. 데이터 전처리와 중복 탐지에 특화된 오픈소스 솔루션입니다.
