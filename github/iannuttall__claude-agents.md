이 프로젝트는 Claude Code를 위한 커스텀 에이전트들을 제공하는 저장소로, 코드 리팩토링, 콘텐츠 작성, 프론트엔드 설계, 보안 감사 등 다양한 개발 작업을 지원합니다. 사용자는 프로젝트 단위 또는 글로벌 환경에 맞춰 에이전트 파일을 설치하여 Claude Code에서 자동으로 활용할 수 있습니다. 각 에이전트는 특정 업무 영역에 최적화된 기능을 제공합니다.
