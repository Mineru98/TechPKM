LLamaNET은 .NET 7 환경에서 LLM(Large Language Model) 추론을 지원하는 라이브러리로, llamacpp 라이브러리를 기반으로 구축되었습니다. 이 프로젝트는 C# 개발자가 로컬에서 LLM 모델을 로드하고 텍스트 생성을 수행할 수 있도록 설계되었으며, TopSampler, MirostatSampler 등 다양한 샘플링 전략을 제공합니다. 모델 파일 로드, 추론 세션 관리, 토큰 생성 등 핵심 LLM 기능을 .NET 생태계에서 활용할 수 있도록 구현되었습니다.
