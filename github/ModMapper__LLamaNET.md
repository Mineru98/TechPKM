LLamaNET은 .NET 7 환경에서 llama.cpp 라이브러리를 활용한 대규모 언어 모델(LLM) 추론을 위한 프레임워크입니다. C# 코드로 LLM 모델을 로드하고 텍스트 생성 추론을 수행할 수 있으며, 다양한 샘플링 기법(TopK, TopP, Mirostat 등)을 지원합니다. .NET 기반 AI 애플리케이션을 위한 핵심 NLP 도구로 설계되었습니다.
