Bleve는 Go로 작성된 현대적인 인덱싱 및 검색 라이브러리로, JSON 또는 Go 데이터 구조를 색인하고 다양한 검색 기능을 제공합니다. 텍스트, 숫자, 날짜, 지리공간 데이터 등 다양한 데이터 유형을 지원하며, TF-IDF 및 BM25 같은 검색 모델과 하이브리드 검색(RRF/RSF)을 특징으로 합니다. CLI 도구를 통해 인덱싱 및 검색 작업을 편리하게 수행할 수 있으며, 30개 이상의 언어 분석기와 동의어 검색, 계층적 검색 등의 고급 기능을 포함합니다.  

Apache 2.0 라이선스 하에 공개되어 자유롭게 사용 가능한 오픈소스 프로젝트입니다.
