Bleve는 Go로 작성된 현대적인 인덱싱 및 검색 라이브러리로, JSON 또는 Go 데이터 구조를 인덱싱하고 다양한 검색 기능을 제공합니다. 텍스트, 숫자, 날짜, 지리적 위치 등 다양한 데이터 유형을 지원하며, 퍼지 검색, 벡터 유사도 검색, 동의어를 활용한 검색 등 고급 기능을 포함합니다. 커스터마이즈 가능한 텍스트 분석기와 하이브리드 검색(정확 일치 + 의미 기반)을 통해 유연한 검색 솔루션을 구축할 수 있습니다.

주요 특징:
- 17개 언어 텍스트 분석 지원
- RRF/RSF 기반 하이브리드 검색
- TF-IDF/BM25 스코어링 모델
- 쿼리 타임 부스팅 및 하이라이트 기능
- 페이징 및 패싯 지원

CLI 도구를 통해 인덱스 생성, 쿼리 실행, 분석기 테스트 등 다양한 작업을 수행할 수 있습니다. Apache 2.0 라이선스 하에 배포됩니다.
