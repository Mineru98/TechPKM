DeepL 번역기의 공식 API 키 없이도 웹 기반 번역을 지원하는 Python 기반 CLI 도구입니다. 터미널에서 직접 또는 파일을 통해 텍스트를 번역할 수 있으며, Playwright-Python을 활용해 브라우저에서 DeepL 웹 인터페이스를 자동화하는 방식으로 동작합니다. 번역 언어 쌍, 타임아웃 설정, 비동기 번역 등 다양한 기능을 제공합니다.  

MIT 라이선스로 배포되는 오픈소스 프로젝트입니다.
