이 프로젝트는 DeepL API 키 없이도 웹 기반의 번역을 지원하는 Python 커맨드라인 도구입니다. Playwright를 활용해 DeepL 웹 인터페이스를 스크래핑하여 텍스트 또는 파일 단위로 다국어 번역을 수행하며, 비동기 번역 기능도 제공합니다. MIT 라이선스로 배포됩니다.
