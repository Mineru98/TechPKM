이 프로젝트는 메타의 Threads 서비스를 위한 데스크톱 웹 클라이언트를 제공합니다. 공식 모바일 앱의 제한된 사용자 경험을 개선하기 위해 개발되었으며, NextJS와 Tailwind CSS를 기반으로 구현되었습니다. Docker를 통해 간단하게 실행할 수 있으며, 로그인 후 localStorage에 토큰을 저장해 개인 피드를 이용할 수 있습니다. GPLv3 라이선스로 배포되는 순수 취미 프로젝트입니다.
