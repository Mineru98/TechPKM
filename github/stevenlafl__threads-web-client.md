이 프로젝트는 Threads 앱을 위한 데스크탑 웹 클라이언트로, Meta의 공식 구현이 지연되는 동안 사용자 경험을 개선하기 위해 개발되었습니다. NextJS와 Tailwind CSS를 기반으로 구현되었으며 Docker를 통해 쉽게 실행할 수 있습니다. 모바일보다 편리한 데스크탑 환경에서 Threads 피드를 보고 상호작용할 수 있는 기능을 제공합니다. 현재 기본 피드 표시, 게시물/답글 생성, 좋아요, 리포스트 등의 기능을 지원하며 GPLv3 라이선스로 배포되는 취미 프로젝트입니다.
