OpenClaw는 사용자가 직접 장치에서 실행하는 개인용 AI 어시스턴트입니다. WhatsApp, Telegram, Slack 등 다양한 채널에서 작동하며, macOS/iOS/Android에서 음성 인식 및 Canvas 인터페이스를 지원합니다. 로컬에서 빠르게 실행되며 항상 활성화된 상태로 유지되는 것이 특징으로, 사용자 지정 가능한 에이전트, 스킬 시스템, 보안 강화 기능을 제공합니다. Node.js 기반으로 구현되었으며, npm/bun/pnpm으로 설치할 수 있습니다. Anthroic Claude 및 OpenAI 모델과 통합되어 강력한 AI 기능을 제공합니다.
