GUI-Actor는 GUI 에이전트를 위한 좌표 없는 시각적 그라운딩 기술을 제안하는 프로젝트입니다. 인간의 자연스러운 상호작용 방식을 모방하여 정확한 좌표 계산 없이 UI 요소를 인식/선택하는 방식을 도입했으며, Qwen2-VL/VL-2.5 기반의 다양한 모델 크기(2B/3B/7B)에서 SOTA 성능을 달성했습니다. 특히 ScreenSpot-Pro 벤치마크에서 기존 72B 모델 대비 우수한 성능을 입증했으며, 그라운딩 베리파이어 모듈을 통해 후보 영역 평가/선택 기능을 추가했습니다.  

주요 특징:  
1. **액션 어텐션 헤드**: 인간처럼 시각적 요소에 직접 주목하는 좌표 없는 그라운딩  
2. **멀티 스케일 데이터 처리**: 다양한 해상도/레이아웃에 대한 일반화 능력  
3. **검증 기반 성능 향상**: 후보 영역 평가를 통한 정확도 개선(Verifier)  
4. **Qwen2-VL 호환성**: 기존 VLM 백본과의 통합 용이성  

Hugging Face에서 모델 체크포인트, 프로젝트 페이지, arXiv 논문 등을 제공합니다.
