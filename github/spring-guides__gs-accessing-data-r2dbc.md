이 프로젝트는 Spring Data R2DBC를 사용하여 리액티브 방식으로 관계형 데이터베이스(H2)와 상호작용하는 Java 애플리케이션을 구축하는 방법을 설명합니다. `Customer` POJO를 메모리 기반 데이터베이스에 저장하고 조회하는 간단한 애플리케이션을 구현하며, Spring Boot와 R2DBC의 자동화된 리포지토리 기능을 활용합니다. 주요 내용은 엔티티 정의, 쿼리 생성, 리액티브 데이터 처리 방법 등을 포함합니다.
