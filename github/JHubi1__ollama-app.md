이 프로젝트는 Ollama 서버와 연동되는 현대적 클라이언트 애플리케이션으로, 로컬 네트워크에서 개인 정보를 보호하면서 LLM(대형 언어 모델)을 사용할 수 있도록 설계되었습니다. Flutter로 개발되어 모바일 및 데스크톱 플랫폼(실험적)에서 작동하며, 직관적인 인터페이스와 모델 선택, 채팅, 음성 모드 등의 기능을 제공합니다. Ollama 서버와의 API 연동을 통해 로컬에서 AI 모델을 실행 및 관리할 수 있습니다.
