OpenCodeInterpreter는 코드 생성과 실행, 반복적인 개선 기능을 통합한 오픈소스 코드 생성 시스템입니다. GPT-4 Code Interpreter와 같은 프로프라이어터리 시스템과의 격차를 해소하기 위해 설계되었으며, 실행 피드백과 인간 피드백을 통한 동적 코드 리파인먼트 기능을 제공합니다. 다양한 벤치마크(HumanEval, MBPP)에서 검증된 성능을 보이며, Hugging Face를 통해 모델, 데이터셋, 데모를 공개하고 있습니다. 사용자는 로컬에서 코드를 생성/실행하고 피드백을 통해 코드를 조정할 수 있으며, 대화형 인터페이스로도 활용할 수 있습니다.
