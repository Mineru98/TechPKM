OpenCodeInterpreter는 코드 생성과 실행, 반복적 개선 기능을 통합하여 GPT-4 Code Interpreter와 같은 프로프라이어터리 시스템과의 격차를 줄이기 위한 오픈소스 코드 생성 시스템 모음입니다. 실행 피드백을 통한 모델 성능 향상을 목표로 하며, 다양한 크기의 모델 시리즈(예: 1.3B~70B 파라미터)와 HumanEval/MBPP 벤치마크에서의 성능 비교를 제공합니다. HuggingFace에 공개된 모델, 데이터셋, 데모 구현체를 통해 연구 및 실험에 활용할 수 있습니다.
