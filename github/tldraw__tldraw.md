tldraw는 React 기반 무한 캔버스 경험을 구축하기 위한 오픈소스 라이브러리입니다. 디지털 화이트보드 서비스 tldraw.com의 핵심 소프트웨어로, npm을 통해 설치 가능하며 React 프로젝트에 쉽게 통합할 수 있습니다. 상업적/비상업적 프로젝트에서 사용 가능하지만, 캔버스에 "Made with tldraw" 워터마크를 유지해야 하며, 워터마크 제거를 위해서는 비즈니스 라이선스 구매가 필요합니다. TypeScript로 작성되었으며, 지역 개발 환경과 컨텍스트 문서(CONTEXT.md)를 통해 개발 편의성을 제공합니다.
