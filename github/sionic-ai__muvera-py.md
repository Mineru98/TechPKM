이 프로젝트는 고정차원인코딩(FDE) 알고리즘을 Python으로 구현한 것으로, 원본 C++ 구현과의 완벽한 호환성을 유지하면서 접근성을 높였습니다. FDE는 문서를 수백 개의 벡터로 표현하는 현대 검색 시스템(ColBERT 스타일)에서 발생하는 성능 문제를 해결하며, 다중 벡터를 유사도 관계를 유지한 채 단일 고정 크기 벡터로 변환합니다. 이를 통해 기존 Chamfer 유사도 계산을 근사하면서도 검색 속도를 획기적으로 개선합니다. 주요 기술로는 SimHash 기반 공간 분할, 차원 축소, 반복 파티셔닝 등을 사용하며, 검색 및 평가 파이프라인을 포함합니다.
