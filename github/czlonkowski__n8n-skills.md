이 프로젝트는 n8n-mcp MCP 서버를 사용하여 완벽한 n8n 워크플로우를 구축하기 위한 7가지 보완적 Claude Code 스킬을 제공합니다. 주요 목적은 AI 어시스턴트가 n8n 워크플로우 생성 시 발생하는 일반적인 문제(검증 오류, 잘못된 도구 사용, 노드 구성 오류 등)를 해결하도록 교육하는 것입니다. n8n 표현식 문법, MCP 도구 사용법, 검증된 워크플로우 패턴, 검증 오류 해석, 노드 구성 방법 등 실용적인 지식을 포함하며, JavaScript 및 Python 코드 노드 작성 가이드도 제공합니다.  

> 📌 **핵심 기능**  
> - n8n 표현식 문법 및 웹훅 데이터 접근 가이드  
> - n8n-mcp 도구 효과적 사용법 (노드 검색, 검증, 템플릿 관리)  
> - 실제 사례 기반 5가지 워크플로우 패턴 (웹훅, HTTP API, DB, AI, 예약 작업)  
> - 검증 오류 카탈로그 및 해결 전략  
> - JavaScript/Python 코드 노드 작성 시 주의사항 및 생산 테스트 패턴  

이 스킬들은 상호 보완적으로 작동하여 복잡한 워크플로우 생성 및 디버깅을 지원합니다.
