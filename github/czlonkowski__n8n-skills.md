n8n-mcp MCP 서버와 함께 사용할 수 있는 7가지 Claude Code 기술 세트를 제공하는 프로젝트입니다. 이 기술들은 AI 어시스턴트가 효율적인 n8n 워크플로우를 구축할 수 있도록 표현력, MCP 도구 사용법, 검증된 워크플로우 패턴, 검증 오류 수정, 노드 구성 지침 등을 제공합니다. 각 기술은 특정 사용 사례에 자동으로 활성화되며, 525개 이상의 n8n 노드와 2,653개 이상의 워크플로우 템플릿을 지원합니다.  

MIT 라이선스로 배포되며, n8n-mcp 서버와 호환됩니다. 설치 방법은 Claude Code 플러그인 설치, 마켓플레이스 추가, 수동 설치 등이 있습니다. Python이 주 언어로 표시되어 있지만, 실제 구현은 JavaScript 중심으로 이루어져 있습니다. 주요 기술에는 n8n 표현력, MCP 도구 전문가, 워크플로우 패턴, 검증 전문가, 노드 구성, Code JavaScript/Python 등이 포함됩니다.
