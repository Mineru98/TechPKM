`skopeo`는 컨테이너 이미지와 레지스트리에서 다양한 작업을 수행하는 명령줄 유틸리티입니다. 루트 권한이나 데몬 없이 OCI 및 Docker v2 이미지를 처리할 수 있으며, 레지스트리 간 이미지 복사, 원격 이미지 검사, 이미지 삭제, 오프라인 배포를 위한 레지스트리 동기화 등의 기능을 제공합니다. Docker Hub, Quay, 프라이빗 레지스트리 및 로컬 저장소와 호환됩니다.
