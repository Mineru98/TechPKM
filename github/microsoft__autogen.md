AutoGen은 자율적으로 작동하거나 인간과 협업할 수 있는 멀티 에이전트 AI 애플리케이션을 구축하기 위한 프레임워크입니다. 이 프로젝트는 핵심 메시지 전달, 이벤트 기반 에이전트, 확장 가능한 개발자 도구(AutoGen Studio, 벤치마크 도구)를 제공하며, 코드 실행 없이도 멀티 에이전트 워크플로우를 프로토타이핑할 수 있는 GUI를 포함합니다. Python 3.10+ 기반으로 OpenAI 및 AzureOpenAI와 같은 다양한 LLM 클라이언트를 지원합니다.  

> 주요 특징:  
> - 3계층 설계(코어 API, AgentChat API, 확장 API)  
> - 웹 브라우징, 코드 실행, 파일 처리 등 복합 작업 지원  
> - 커뮤니티 주도 생태계(주간 오피스 아워, Discord, GitHub Discussions)  
> - .NET 및 Python 크로스 플랫폼 지원
