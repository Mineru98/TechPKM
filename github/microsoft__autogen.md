AutoGen은 자율적으로 작동하거나 인간과 협력하여 작업을 수행할 수 있는 멀티 에이전트 AI 애플리케이션을 구축하기 위한 프레임워크입니다. Python 3.10 이상에서 작동하며, OpenAI API와 같은 언어 모델 클라이언트와의 통합, 코드 실행, 웹 브라우징 등의 기능을 지원합니다. Core API, AgentChat API, Extensions API의 계층적 설계를 통해 다양한 수준의 추상화에서 개발이 가능하며, AutoGen Studio를 통해 코드 없이도 멀티 에이전트 워크플로우를 프로토타입하고 실행할 수 있습니다. Microsoft에서 개발한 오픈소스 프로젝트로 활발한 커뮤니티 지원을 제공합니다.
