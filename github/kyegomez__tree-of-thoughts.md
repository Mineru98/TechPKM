이 프로젝트는 Tree of Thoughts(ToT) 알고리즘을 구현한 파이썬 라이브러리로, LLM(대형 언어 모델)의 추론 능력을 최대 70%까지 향상시키는 방법을 제공합니다. DFS/BFS 기반 탐색 알고리즘을 통해 여러 에이전트가 협업하며 문제를 해결하는 구조로 설계되었습니다. 24 게임 해결, 복잡한 질문 분석 등 논리적 사고가 필요한 작업에 특화된 오픈소스 도구입니다. Apache 라이선스로 배포되며, OpenAI API와 연동 가능한 유연한 아키텍처를 갖추고 있습니다.
