SearXNG는 사용자 추적을 하지 않는 메타검색엔진 프로젝트입니다. 여러 검색엔진의 결과를 집계하여 제공하며, 프라이버시를 중시하는 대체 검색 솔루션으로 개발되었습니다. Python 기반으로 구현되었고, 자체 호스팅이 가능하며 커뮤니티 기반의 적극적인 기여를 장려합니다. 오픈소스 라이선스로 배포되어 자유롭게 사용 및 수정이 가능합니다.
