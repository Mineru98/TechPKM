SearXNG는 사용자 추적이나 프로파일링 없이 작동하는 메타검색엔진 프로젝트입니다. 여러 검색엔진의 결과를 집계하여 개인정보 보호를 중시하는 검색 서비스를 제공합니다. 오픈소스 기반으로 자체 호스팅이 가능하며, 커뮤니티 주도로 활발히 개발되고 있습니다. GNU AGPL-3.0 라이선스로 배포됩니다.
