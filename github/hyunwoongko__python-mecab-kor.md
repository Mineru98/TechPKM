python-mecab-kor은 기존 python-mecab-ko의 유지보수를 개선한 한국어 형태소 분석기 래퍼 패키지입니다. 설치 문제를 해결하고 패키지 이름을 `kor`로 변경하여 관리하며, Linux(Ubuntu/CentOS/Amazon), macOS에서 테스트되었습니다. 명사 추출, 품사 태깅, 구문 분석 기능을 제공하며, 공백 보존 옵션을 통해 정밀한 텍스트 처리가 가능합니다.  

**참고**: Windows 환경은 지원하지 않습니다.
