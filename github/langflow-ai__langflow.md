Langflow은 AI 기반 에이전트 및 워크플로우를 구축하고 배포하기 위한 강력한 플랫폼입니다. 시각적 저작 환경과 내장된 API 및 MCP 서버를 제공하여 모든 워크플로우를 애플리케이션에 통합 가능한 도구로 변환합니다. 주요 LLM 및 벡터 데이터베이스와 호환되며, 멀티 에이전트 오케스트레이션, 관찰 가능성, 엔터프라이즈급 보안 등의 기능을 지원합니다. 사용자는 데스크톱 애플리케이션, Python 패키지, Docker를 통해 설치할 수 있으며 다양한 배포 옵션이 제공됩니다.
