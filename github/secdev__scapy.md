Scapy는 Python 기반의 강력한 패킷 조작 프로그램 및 라이브러리로, 네트워크 프로토콜의 패킷을 생성, 디코딩, 전송, 캡처할 수 있습니다. 네트워크 스캐닝, 추적 경로 분석, 공격 시뮬레이션, 패킷 분석 등의 작업에 활용되며, 기존 도구(nmap, tcpdump 등)를 대체할 수 있는 다기능성을 제공합니다. Linux, macOS, BSD, Windows 등 다양한 플랫폼에서 작동합니다.  

Scapy는 대화형 쉘 또는 라이브러리로 사용 가능하며, 기본적인 네트워크 상호작용부터 복잡한 프로토콜 조작까지 지원합니다. GPL v2 라이선스로 배포되며, 네트워크 보안 테스트 및 연구 개발에 널리 활용됩니다.
