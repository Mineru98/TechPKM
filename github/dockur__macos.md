Docker 컨테이너 내에서 macOS를 실행할 수 있는 오픈소스 프로젝트입니다. KVM 가속, 웹 기반 뷰어, 자동 다운로드 기능을 제공하며, Docker Compose, CLI, Kubernetes, GitHub Codespaces를 통해 간편하게 배포할 수 있습니다. 사용자는 디스크 유틸리티로 설정을 구성하고 macOS를 설치한 후 애플리케이션처럼 사용할 수 있습니다. Apple 하드웨어에서만 실행이 허용된다는 점을 반드시 유의해야 합니다.
