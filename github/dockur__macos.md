이 프로젝트는 macOS를 Docker 컨테이너 내부에서 실행할 수 있도록 구현한 오픈소스 솔루션입니다. KVM 가속, 웹 기반 뷰어, 자동 다운로드 등의 기능을 제공하며, macOS 설치 및 실행을 위한 간결한 구성 가이드를 포함합니다. 사용자는 Docker Compose, CLI, Kubernetes 등 다양한 방식으로 컨테이너를 배포할 수 있습니다. macOS 버전 선택, 저장 공간 및 리소스 할당, 네트워크 설정 등 사용자 정의 옵션을 지원합니다. Apple 하드웨어에서만 사용이 허용되며, EULA 준수를 강조합니다.
