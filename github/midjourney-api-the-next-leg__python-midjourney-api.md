The Next Leg(TNL)은 Midjourney API와 연동하여 AI 생성 이미지를 생성하는 기능을 제공하는 파이썬 모듈입니다. 프롬프트 또는 기존 이미지를 기반으로 한 이미지 생성, 이미지 설명, 웹훅 연동, 계정 설정 관리 등의 다양한 작업을 간단한 인터페이스로 수행할 수 있습니다. Midjourney의 버튼 및 슬래시 명령어 사용도 지원하며, 개발자들은 pip를 통해 쉽게 설치하여 프로젝트에 통합할 수 있습니다.
