The Next Leg(TNL)은 Midjourney API와 상호작용하여 AI 생성 이미지를 만드는 기능을 제공하는 Python 모듈입니다. 프롬프트 또는 이미지 URL을 기반으로 이미지 생성, 이미지 설명, 버튼/슬래시 명령 사용, 계정 설정 관리 등 다양한 작업을 수행할 수 있는 간단한 인터페이스를 제공합니다. 주요 기능으로는 이미지 생성(imagine, img2img), 이미지 설명(describe), 시드 추출, 계정 정보 조회 등이 있습니다.  

**핵심 특징**:  
1. Midjourney API와의 통합을 위한 Python 래퍼  
2. 웹훅 오버라이드 및 참조 문자열 지원  
3. 다양한 이미지 조작 기능(생성, 변환, 분석)  
4. 계정 설정 및 실시간 진행 상황 추적 가능
