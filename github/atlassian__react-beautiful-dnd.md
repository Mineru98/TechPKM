react-beautiful-dnd는 리액트 애플리케이션에서 아름답고 접근성 있는 드래그 앤 드롭 기능을 구현하기 위한 라이브러리입니다. 아카이빙된 프로젝트로, 수직/수평 목록 간 이동, 가상 목록 지원, 키보드/스크린 리더 접근성을 제공하며, 애니메이션과 성능 최적화가 특징입니다. npm에서 더 이상 유지보수되지 않으며, 대체 솔루션으로 [Pragmatic drag and drop](https://github.com/atlassian/pragmatic-drop-and-drag)을 추천하고 있습니다.
