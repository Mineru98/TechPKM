Surya는 90개 이상의 언어에서 OCR(광학 문자 인식)을 지원하는 문서 분석 도구킷입니다. 텍스트 라인 감지, 레이아웃 분석(표, 이미지, 헤더 감지), 읽기 순서 추적, 표 인식, LaTeX OCR 기능을 제공합니다. 클라우드 서비스와 비교해 우수한 성능을 보이며 다양한 문서 형식에 적용 가능합니다. 오픈소스 라이선스로 제공되며, 연구 및 개인 사용에 무료입니다.  

이 프로젝트는 효율적인 문서 처리를 위한 다양한 모델과 API를 포함하며, Python 기반으로 설치 및 사용이 간편합니다. GPU/CPU 환경에서 모두 작동하며, 배치 크기 조정을 통한 성능 최적화가 가능합니다.  

핵심 기능:  
- 90+ 언어 OCR 및 라인 수준 텍스트 감지  
- 표 인식 및 LaTeX 수식 인식  
- 레이아웃 분석(텍스트, 표, 이미지, 헤더 구분)  
- 읽기 순서 추적 및 커스텀 모델 파인튜닝 지원  
- Streamlit 기반 대화형 GUI 제공  

성능:  
- Tesseract, Google Cloud Vision 대비 높은 정확도  
- GPU 활용 시 빠른 처리 속도  
- 배치 크기 조정을 통한 VRAM 효율성 관리  

주요 용도:  
- 학술 논문, 신문 기사, 프레젠테이션, 스캔 문서 등 다양한 문서 처리  
- 표 데이터 추출 및 LaTeX 수식 변환  
- 레이아웃 기반 텍스트 구조 분석
