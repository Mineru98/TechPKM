Scikit-LLM은 ChatGPT와 같은 대형언어모델(LLM)을 scikit-learn 생태계에 통합하는 Python 라이브러리입니다. 기존 머신러닝 워크플로우에 LLM 기반 텍스트 분석 기능을 추가하여 제로샷 분류 등의 고급 NLP 작업을 가능하게 합니다. GPT-4 등 다양한 모델 지원 및 scikit-learn 호환 API를 제공하는 것이 핵심 특징입니다.  

> 📌 **기능 요약**: LLM과 전통적 ML 파이프라인의 융합, 코드 예제를 통한 쉬운 시작, OpenAI 인증 기반의 모델 접근.
