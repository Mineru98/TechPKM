이 프로젝트는 YouTube의 프라이빗 API인 "InnerTube"를 활용하는 JavaScript 클라이언트입니다. Node.js, Deno 및 최신 브라우저에서 동영사, 댓글, 실시간 채팅, 스트리밍 데이터 등에 프로그래밍 방식으로 접근할 수 있도록 지원합니다. MIT 라이선스로 배포되며 공식 YouTube 서비스와 무관함을 명시합니다. 주요 기능과 설치 방법, 사용 문서는 공식 가이드에서 제공됩니다.
