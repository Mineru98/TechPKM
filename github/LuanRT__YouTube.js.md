YouTube.js는 YouTube의 비공개 API인 "InnerTube"를 위한 JavaScript 클라이언트입니다. 비디오, 댓글, 라이브 채팅, 스트리밍 데이터 등 YouTube 기능을 프로그래밍 방식으로 접근할 수 있으며, Node.js, Deno, 최신 브라우저에서 완벽하게 작동합니다. 오픈소스 프로젝트로 MIT 라이선스로 배포되며, 커뮤니티 기여를 적극 환영합니다. 공식 문서와 API 가이드는 [ytjs.dev](https://ytjs.dev)에서 제공됩니다.
