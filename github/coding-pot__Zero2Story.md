Zero2Story는 일반 사용자가 PaLM API, Stable Diffusion, MusicGen을 활용해 자신만의 스토리를 창작할 수 있게 하는 프레임워크입니다. 배경 설정, 캐릭터 생성, 대화형 스토리 진행의 3단계로 구성되며, 이미지/음악 생성 및 사용자 선택에 따른 분기형 스토리텔링을 지원합니다. NeurIPS 2023 워크숍에서 소개된 생성형 AI 기반 창의적 도구입니다.
