한국어 언어모델 다분야 사고력 벤치마크 프로젝트로, 모델의 논리 추론 및 사고력을 평가하는 데이터셋과 평가 코드를 제공합니다. 리더보드 운영 중단 발표와 함께 향후 개선된 버전(LogicKor Hard) 출시 예정이며, PR 및 Self-Report 방식의 평가 결과를 받습니다. Python 기반의 오픈소스 프로젝트로 코드 포맷팅 도구와 평가 파이프라인(generator/evaluator/score)을 포함합니다.
