**View Claude Code**는 Claude Code 에이전트, 스킬, 명령을 대화형 그래프로 시각화하는 도구입니다. 로컬 및 글로벌 범위 스캔을 통해 에이전트-스킬 관계를 명확히 보여주며, UI에서 직접 실행 가능한 기능을 제공합니다. 복잡한 설정 없이도 `npx viewcc` 명령으로 즉시 사용할 수 있는 직관적이고 강력한 시각화 솔루션입니다.

이 도구는 계층적 레이아웃, 색상 코드 노드, 실시간 실행 기능을 통해 개발자들이 Claude Code 프로젝트를 더 효과적으로 분석 및 관리할 수 있도록 설계되었습니다. Node.js 18+ 환경에서 작동하며 MIT 라이선스로 배포됩니다.
