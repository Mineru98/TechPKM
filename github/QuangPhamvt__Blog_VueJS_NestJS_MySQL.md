이 프로젝트는 Vue 3, NestJS, MySQL을 활용한 블로그 시스템으로, 회사 인턴십에서 개발된 개인 웹 블로그 코드베이스입니다. 프론트엔드는 Vue 3 + Vite + Sass/Scss로 구현되었으며, 백엔드는 NestJS와 TypeORM을 통해 RESTful API와 인증 시스템(액세스 토큰/리프레시 토큰)을 제공합니다. 주요 기능으로는 게시글 관리, 카테고리 생성, 사용자 인증(회원가입/로그인/로그아웃) 등이 포함됩니다.  

(요약: Vue 3 + NestJS 기반의 블로그 시스템으로, 사용자 인증과 게시글 관리 기능을 제공하는 인턴십 프로젝트)
