이 프로젝트는 Oobabooga의 Text Generation Web UI용 실험적인 장기 기억(LTM) 확장입니다. 챗봇이 대화를 장기적으로 "기억"할 수 있도록 하며, 의미 기반 검색을 사용해 과거 대화를 필터링하여 컨텍스트에 주입합니다. 다중 캐릭터 지원, 메모리 길이/유사도 설정 등 다양한 구성 옵션을 제공하지만, 현재는 개발 중단 상태이며 Ubuntu 22.04에서 테스트되었습니다. 주요 기술 스택으로 Python, zarr, SQLite, SentenceTransformers 등을 사용합니다.
