이 프로젝트는 oobabooga의 "Text Generation Web UI"에 장기 메모리(LTM) 기능을 추가하는 실험적 확장입니다. 채팅봇이 대화 내용을 장기적으로 기억하고 이를 후속 대화에 반영할 수 있도록 설계되었습니다. 주요 기능은 의미 기반 검색, 캐릭터별 독립 메모리 관리, 사용자 정의 가능한 메모리 형식 등입니다. 현재는 Ubuntu에서 테스트되었으며, Windows에서도 제한적으로 호환되지만 더 이상 활발히 개발되지 않고 있습니다. 메모리 데이터를 CSV 형식으로 내보내는 가이드를 제공하여 마이그레이션을 권장합니다.
