이 프로젝트는 Go 프로그램의 정적 분석을 위한 다양한 도구와 패키지를 제공하는 모듈입니다. 주요 기능으로는 `gopls` LSP 서버를 통한 IDE 지원, 코드 포맷팅(`goimports`), 호출 그래프 분석(`callgraph`), SSA(Static Single Assignment) 중간 표현(`go/ssa`) 등이 포함됩니다. Go 개발자를 위한 정적 분석 프레임워크와 개발자 도구에 중점을 둔 오픈소스 프로젝트입니다. Go 프로젝트의 분석, 리팩토링, IDE 통합을 지원하는 핵심 라이브러리와 CLI 도구들을 제공합니다.
