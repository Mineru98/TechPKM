Mistral-7B 모델의 실행 및 파인튜닝을 위한 최소 코드를 제공하는 프로젝트입니다. 모델 다운로드, Docker 기반 배포, 대화형 실행, LoRA 기반 메모리 효율적인 파인튜닝 기능을 지원하며, JSONL 형식의 데이터셋 처리로 사전학습 및 대화형 학습이 가능합니다. vLLM을 활용한 서빙과 멀티GPU 학습(DDP)을 포함한 실용적인 워크플로우를 제공합니다.
