jetson-stats는 NVIDIA Jetson 시리즈(Thor, Orin, Xavier, Nano, TX)의 모니터링 및 제어를 위한 Python 패키지입니다. 터미널에서 실행 가능한 `jtop` 인터페이스를 제공하거나 Python 스크립트 내 임포트가 가능하며, 하드웨어 정보, CPU/GPU/메모리 사용량, 팬 속도 등을 실시간으로 확인할 수 있습니다. Docker 컨테이너 환경에서도 사용 가능하며, 관리자 권한 없이도 작동하여 다양한 개발 시나리오에 유연하게 적용됩니다. 공식 문서 및 커뮤니티 지원을 통해 활용 방법을 쉽게 익힐 수 있습니다.
