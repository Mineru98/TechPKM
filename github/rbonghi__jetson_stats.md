**jetson-stats**는 NVIDIA Jetson 시리즈(Thor, Orin, Xavier, Nano, TX 등)의 하드웨어 모니터링 및 시스템 제어를 위한 Python 패키지입니다. `jtop` 명령어로 터미널에서 실시간 CPU, GPU, 메모리, 팬 속도 등을 확인하거나 Python 라이브러리 형태로 프로젝트에 통합할 수 있습니다. Docker 환경에서도 사용 가능하며, 슈퍼유저 권한 없이 작동하는 것이 특징입니다. 다양한 Jetpack 버전과 하드웨어 구성을 지원합니다.
