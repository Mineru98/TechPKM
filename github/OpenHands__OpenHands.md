OpenHands는 AI 기반 소프트웨어 개발을 위한 오픈소스 프로젝트입니다. Python SDK, CLI, 로컬 GUI 및 클라우드 버전을 제공하며, 다양한 LLM과 통합해 개발자 생산성을 높이는 에이전트 기반 기술을 구현합니다. 엔터프라이즈 솔루션을 포함한 확장 가능한 아키텍처와 커뮤니티 지원을 강점으로 합니다.
