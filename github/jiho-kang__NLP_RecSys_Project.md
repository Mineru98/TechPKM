이 프로젝트는 한국어 BERT 모델인 KoBERT를 활용해 정책 텍스트에서 태그를 추출하고, 개인 프로필과 정책 데이터를 매칭하는 추천 시스템을 구축하는 것을 목표로 합니다. 사용자 데이터(2만 건)와 정책 데이터(8.8만 건)를 활용해 키워드 기반 태깅 모델과 유사도/와이드앤딥 기반 추천 모델을 개발하였습니다. 태깅 정확도와 추천 성능을 개선하기 위한 다양한 방법론이 시도되었으며, 향후 모델 성능 향상과 실무 적용을 위한 개선 방향이 제시되었습니다.
