이 프로젝트는 Google Directions 서비스에서 추출한 GPX 파일을 기반으로 가상의 차량 이동 시뮬레이션을 구현하고, Socket.IO를 통해 실시간 위치 데이터를 클라이언트에 전송하는 데모 애플리케이션입니다. 웹, React Native, iOS(Swift), Android(Kotlin) 등 다양한 플랫폼에서 차량 이동 경로를 시각화할 수 있습니다. Uber/Grab과 유사한 프로토타입 개발에 유용한 참조 아키텍처를 제공합니다. GPX 파일 추가만으로 새로운 이동 경로를 쉽게 확장할 수 있습니다.
