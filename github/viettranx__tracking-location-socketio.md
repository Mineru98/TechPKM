이 프로젝트는 Socket.IO를 사용해 GPX 파일 기반 가상 위치 데이터를 클라이언트에 전송하여 지도 상에 차량 이동을 시각화하는 데모 앱을 구현하는 데 목적을 두고 있습니다. Uber/Grab 같은 프로토타입 앱 개발에 유용하며, 웹과 React Native(iOS/Android), 네이티브(iOS Swift/Android Kotlin) 클라이언트 데모를 포함합니다. GPX 파일을 통해 다양한 이동 경로를 시뮬레이션할 수 있습니다.
