이 프로젝트는 웹 기술 기반의 오픈소스 HWP 뷰어 및 파서 라이브러리입니다. 한글과컴퓨터의 한/글 문서(.hwp)를 웹 환경에서 처리할 수 있도록 개발되었으며, npm/yarn을 통해 쉽게 설치할 수 있습니다. 주요 기술 스택으로 JavaScript를 사용하며, hwplib, cfb-js 등의 라이브러리를 활용해 구현되었습니다. 브라우저 확장 프로그램 및 문서 변환 솔루션에 활용 가능한 것이 특징입니다. Apache 2.0 라이선스로 배포됩니다.
