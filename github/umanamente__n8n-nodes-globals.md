이 프로젝트는 n8n 워크플로우 내에서 전역적으로 사용 가능한 상수를 관리할 수 있는 커뮤니티 노드입니다. 키-값 쌍 또는 JSON 형식으로 상수를 정의하고 저장한 후 모든 워크플로우에서 재사용할 수 있도록 지원합니다. n8n의 Credential 시스템을 활용해 안전하게 상수를 저장 및 호출하는 기능을 제공합니다.
