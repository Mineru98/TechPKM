이 프로젝트는 n8n 워크플로우에서 재사용 가능한 전역 상수를 저장하고 관리하기 위한 커뮤니티 확장 노드입니다. 키-값 쌍 또는 JSON 형식으로 상수를 정의하고, 다양한 워크플로우에서 일관된 값으로 참조할 수 있도록 지원합니다. n8n 인증 시스템을 활용해 안전하게 전역 변수를 저장/호출하는 것이 핵심 기능입니다. 자동화 워크플로우에서 반복적인 설정 값을 중앙 집중식으로 관리할 때 유용합니다.
