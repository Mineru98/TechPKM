NeuralGCM은 날씨와 기후 시뮬레이션을 위한 하이브리드(ML/물리) 대기 모델 구축을 지원하는 Python 라이브러리입니다. 연구 논문에서 소개된 이 프로젝트는 머신러닝과 물리 기반 모델을 결합해 기상 예측 정확도를 향상시키는 데 목적을 두고 있습니다. 주요 특징으로는 아파치 라이선스 2.0 하에 공개된 코드와 크리에이티브 커먼즈 라이선스의 학습된 모델 가중치가 포함됩니다. 기후과학 및 대기역학 분야에서 데이터 기반 모델링 접근을 혁신하는 것을 목표로 합니다.
