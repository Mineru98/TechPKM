Sentence Transformers는 문장 임베딩, 검색, 재순위화를 위한 프레임워크입니다. 사전 훈련된 모델을 활용해 의미론적 유사성 계산, 의미 검색, 문장 변환 탐색 등 다양한 NLP 작업을 수행할 수 있으며, 사용자 정의 모델 훈련 기능도 제공합니다. PyTorch와 Hugging Face Transformers 기반으로 구축되었으며, 15,000개 이상의 사전 훈련 모델과 20개 이상의 손실 함수를 지원합니다. 주요 활용 분야는 의미론적 텍스트 유사성 분석, 밀집/희소 임베딩 생성, 검색 시스템 구축 등입니다.
