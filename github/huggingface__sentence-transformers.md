이 프로젝트는 문장 임베딩, 검색, 재순위화 작업을 위한 프레임워크로, 사전 훈련된 모델 활용과 사용자 정의 모델 학습을 지원합니다. 15,000개 이상의 사전 훈련 모델을 제공하며, 의미 검색, 텍스트 유사도 계산, 패러프레이즈 마이닝 등 다양한 응용 사례에 적용할 수 있습니다. PyTorch 기반으로 GPU 가속을 지원하며, Hugging Face와 통합되어 있습니다.
