ThinkGPT는 대규모 언어 모델(LLM)을 위한 "체인 오브 사고" 구현을 목표로 하는 Python 라이브러리입니다. 제한된 컨텍스트 환경에서 장기 기억과 지식 압축을 통해 추론 능력을 향상시키고, 코드 기반에 지능형 결정 기능을 추가하는 데 중점을 둡니다. 주요 기능으로는 기억, 자기 개선, 지식 압축, 추론, 자연어 조건 처리 등이 있으며, DocArray 기반의 직관적인 API로 쉽게 활용할 수 있습니다.  

이 프로젝트는 GPT 모델의 단일 추론 한계를 극복하고, 고차원 사고 패턴을 지원하는 것을 목표로 합니다. 코드 자동 수정, 언어 규칙 추상화, 자연스러운 조건 선택 등 다양한 사용 사례를 제공하며, 생성형 에이전트 개발에 적합합니다.
