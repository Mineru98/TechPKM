이 프로젝트는 WebGPU 기술로 브라우저에서 직접 이미지 배경을 제거할 수 있는 도구입니다. RMBG V1.4 모델과 transformer.js를 활용해 웹 기반 배경 제거를 구현했으며, Vite와 React 기반 프론트엔드로 개발되었습니다. 크롬 플래그 설정이나 CPU 모드 전환을 통해 호환성 문제를 해결할 수 있으며, 오픈소스 커뮤니티의 기여를 적극 환영합니다.
