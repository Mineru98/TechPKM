OmniParser는 GUI 스크린샷을 구조화된 UI 요소로 변환하는 컴퓨터 비전 기반 파싱 도구입니다. LLM(대형 언어 모델)이 화면 요소를 정확히 인식하고 액션을 생성할 수 있도록 지원하며, 윈도우 에이전트 아레나에서 최고 성능을 기록했습니다. V2 버전에서는 아이콘 감지 정확도 향상과 상호작용 가능 요소 예측 기능을 추가했습니다.  

이 프로젝트는 HuggingFace에서 제공하는 사전 학습 모델과 데모 환경을 통해 쉽게 활용할 수 있으며, MIT/AGPL 라이선스 하에 배포됩니다. GUI 자동화 및 에이전트 개발에 특화된 기술 스택을 보유하고 있습니다.
