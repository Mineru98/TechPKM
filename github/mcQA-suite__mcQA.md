mcQA는 언어 모델을 활용하여 다중 선택 질문에 답변하는 Python 라이브러리입니다. RACE, SWAG, ARC 등의 데이터셋을 지원하며, CSV 형식의 데이터를 학습해 모델 훈련 및 예측이 가능합니다. BERT 기반의 모델 구현을 제공하며, PyTorch와 Huggingface Transformers 프레임워크에 의존합니다. 현재 개발 중이며, Apache-2.0 라이선스로 배포됩니다.
