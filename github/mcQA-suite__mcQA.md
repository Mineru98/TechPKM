**mcQA**는 언어 모델을 활용해 다중 선택 질문에 답변을 생성하는 라이브러리입니다. RACE, SWAG, ARC 등의 데이터셋을 지원하며, 사용자 정의 CSV 형식의 데이터로 모델 학습 및 예측을 수행할 수 있습니다. BERT 기반 모델을 활용하여 질문-답변 태스크를 처리하며, PyTorch와 Hugging Face Transformers를 기반으로 구현되었습니다. Apache-2.0 라이선스로 배포되는 오픈소스 프로젝트입니다.
