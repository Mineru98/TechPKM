Matter.js는 웹 기반 2D 강체 물리 엔진으로, 다양한 모양과 충돌 처리, 중력, 마찰 등의 물리 시뮬레이션 기능을 제공합니다. 게임 개발, 인터랙티브 웹 콘텐츠 제작에 활용되며, 플러그인 시스템과 확장 가능한 구조로 사용자 정의가 가능합니다. 브라우저 및 Node.js 환경에서 작동하며, 캐주얼한 데모와 실제 프로젝트에 적용할 수 있는 예제들을 제공합니다.
