Switcher는 macOS 환경에서 키보드 입력 전환, 마우스 휠 방향 변경, 사용자 정의 단축키 매핑을 제공하는 유틸리티 애플리케이션입니다. SF Symbols 설치가 필요하며, Command + Q 더블 클릭으로 애플리케이션을 종료할 수 있는 기능을 포함합니다. 주로 macOS 사용성을 개선하기 위한 목적으로 개발되었으며, 독립적인 애플리케이션 키 매핑과 마우스 휠 방향 전환 기능을 지원합니다. GitHub 릴리스를 통해 다운로드할 수 있으며, 한국어 및 영어 문서를 제공합니다.
