Devika는 고급 AI 소프트웨어 엔지니어로, 고수준의 인간 지시를 이해하고 단계별로 분해하며 관련 정보를 조사하고 코드를 작성하는 기능을 제공합니다. Devika는 대규모 언어 모델, 계획 및 추론 알고리즘, 웹 브라우징 기능을 활용하여 최소한의 인간 개입으로 복잡한 코딩 작업을 수행할 수 있도록 설계되었습니다. 이 프로젝트는 Cognition AI의 Devin에 대한 오픈소스 대안으로서 SWE-bench 벤치마크에서 동등한 점수를 목표로 합니다.

주요 기능으로는 Claude 3, GPT-4, Gemini, Mistral 등 다양한 모델 지원, 자연어 상호작용을 통한 채팅 인터페이스, 프로젝트 기반 관리 시스템 등이 포함됩니다. Python 3.10-3.11, Node.js 18, bun이 필요하며, 웹 검색 및 코드 생성을 위한 API 키 설정이 요구됩니다. 현재 초기 개발 단계로 기능 구현 및 개선이 진행 중이며, 커뮤니티 기여를 적극 환영합니다.
