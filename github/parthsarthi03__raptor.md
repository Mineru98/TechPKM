RAPTOR은 문서를 재귀적인 트리 구조로 구성하여 대규모 텍스트에서 효율적이고 문맥 인식 가능한 정보 검색을 지원하는 새로운 접근 방식을 제시합니다. 기존 언어 모델의 한계를 극복하기 위해 설계된 이 프로젝트는 계층적 추상화와 검색 증강을 결합하여 질의응답 성능을 향상시킵니다. 주요 특징으로는 사용자 정의 요약/질의응답 모델 통합, 트리 구조 저장/로드 기능, 다양한 임베딩 모델 지원이 포함됩니다.  

이 프로젝트는 MIT 라이선스로 공개된 오픈소스이며, 학술 연구 시 인용이 권장됩니다. 기술 구현과 사용법은 공식 논문과 데모 노트북을 참조할 수 있습니다.
