RAPTOR은 문서를 재귀적 트리 구조로 조직화하여 효율적이고 문맥 인식적인 정보 검색을 지원하는 새로운 검색 증강 언어 모델 접근법입니다. 기존 언어 모델의 한계를 해결하며, 대규모 텍스트 처리 시 계층적 추상화 및 유연한 모델 통합을 제공합니다. 사용자는 OpenAI API 또는 커스텀 모델(요약, QA, 임베딩)을 활용해 문서를 인덱싱하고 질문에 답변할 수 있습니다.  

주요 기능으로는 문서 추가, 질문 응답, 트리 저장/로드, 다양한 모델 통합 확장성이 포함됩니다. MIT 라이선스로 배포되며, 연구 논문 구현 및 상업적 활용이 가능합니다.
