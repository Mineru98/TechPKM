이 프로젝트는 네이버 오픈 API와 연동되는 MCP(Model Context Protocol) 서버를 구현합니다. 블로그, 뉴스, 책, 이미지, 쇼핑 등 다양한 네이버 서비스 검색을 지원하며, Claude Desktop 또는 Cursor 같은 AI 도구와 통합되어 사용됩니다. Python 기반의 MIT 라이선스 오픈소스 프로젝트로, 사용 전 네이버 개발자 계정이 필요합니다.  

### 📌 핵심 기능  
- 블로그/뉴스/책/이미지/쇼핑 등 13가지 검색 도구 제공  
- 맞춤법 검사 및 성인 콘텐츠 필터링 기능 포함  
- FastMCP CLI 또는 직접 설치로 서버 구성 가능  
- MCP 프로토콜을 통해 AI 에이전트와 즉시 연동  

> 설치 시 네이버 클라이언트 ID 및 시크릿 키 필수
