Naver OpenAPI와 Model Context Protocol(MCP)을 연동한 서버 구현 프로젝트입니다. 네이버 블로그, 뉴스, 책, 이미지 등 다양한 서비스를 검색할 수 있는 도구를 제공하며, FastMCP를 통해 서버로 통합됩니다. Python 기반 서버로 Claude Desktop 또는 Cursor와 같은 AI 도구에 쉽게 설치 및 실행할 수 있습니다. MIT 라이선스로 공개된 오픈소스 프로젝트입니다.
