EasyLLMFeaturePorter는 대규모 언어 모델(LLM)의 특징을 단일 클릭으로 다른 모델에 전이하기 위한 도구입니다. 베이스 모델과 컨텍스트 확장 모델, 파인튜닝 모델 간의 차이를 분석하여 핵심 특징을 추출하고, 활성화 비율을 계산한 후 최적의 방식으로 대상 모델에 적용합니다. 이 프로젝트는 모델 확장 과정을 간소화하여 효율적인 LLM 튜닝을 목표로 합니다. 주요 방법론은 모델 간 차이 계산, 활성화 비율 분석, 선택적 특징 전이로 구성됩니다.
