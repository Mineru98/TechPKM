이 프로젝트는 AWS Lightsail 인스턴스에 n8n, Nginx Proxy Manager, OpenWebUI를 한 번에 설치하기 위한 자동화 스크립트를 제공합니다. Ubuntu 환경에서 Docker 기반의 통합 설치 및 관리를 지원하며, 웹훅 자동 설정 및 프록시 관리가 가능합니다. 사용자는 단일 스크립트 실행으로 전체 서비스 스택을 배포하고 관리할 수 있습니다.
