SkyPilot은 다양한 AI 인프라에서 워크로드를 실행, 관리 및 확장하는 시스템입니다. AI 팀은 간단한 인터페이스로 클라우드 또는 자체 인프라에서 작업을 실행할 수 있으며, 인프라 팀은 통합된 제어 평면을 통해 고급 스케줄링 및 확장 기능을 활용할 수 있습니다. Kubernetes, Slurm, AWS, GCP 등 20개 이상의 인프라를 지원하며, 자동 비용 절감(Spot 인스턴스), 리소스 공유, 작업 큐 관리 기능을 제공합니다. Python API와 YAML 기반 작업 정의로 유연한 배포가 가능합니다.
