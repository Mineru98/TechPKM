wttr.in은 터미널에서 사용할 수 있는 강력한 날씨 예보 서비스입니다. curl, 브라우저나 그래픽 뷰어를 통해 ANSI, HTML, PNG 등 다양한 형식의 날씨 정보를 제공하며, 전 세계 54개 언어로 지원됩니다. 사용자가 직접 위치를 지정하거나 IP 기반 자동 위치 감지가 가능하며, 특수 위치(공항, 랜드마크 등) 검색 및 다양한 출력 형식(단일 줄, 데이터 기반, 지도 보기 등)을 지원합니다. 터미널 통합, 이모지 표시, 커스텀 포맷 등 다양한 기능을 통해 개발자와 일반 사용자 모두에게 유용합니다.
