Buck은 페이스북에서 개발한 레거시 빌드 시스템입니다. 현재는 더 이상 유지보수되지 않으며, 후속 프로젝트인 [Buck2](https://github.com/facebook/buck2)로 대체되었습니다. 이 README는 역사적 참조용으로 보존되었으며, Java 8/11 기반으로 Ant를 사용한 초기 빌드 프로세스와 빌드 규칙, 명령어 인터페이스 등의 기능을 포함합니다. Apache License 2.0으로 배포됩니다.
