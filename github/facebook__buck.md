이 프로젝트는 페이스북에서 개발한 레거시 빌드 시스템인 Buck에 대한 저장소입니다. 현재는 더 이상 유지되지 않으며, 공식적으로는 [Buck2](https://github.com/facebook/buck2)로 대체되었습니다. Buck은 빠른 빌드 속도와 효율적인 프로젝트 관리를 목적으로 설계되었으며, Java 8/11 기반으로 동작합니다. 문서는 http://buck.build/에서 확인할 수 있으며, 초기 설치는 Ant를 통한 부트스트랩 과정이 필요합니다.  

> 참고: 이 README 내용은 역사적 참고용으로 남아 있으며, 현재 활성화된 개발 버전은 아닙니다.
