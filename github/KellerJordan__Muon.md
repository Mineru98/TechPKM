Muon은 신경망의 은닉층 가중치를 최적화하기 위한 옵티마이저로, 임베딩이나 분류기 헤드 등의 다른 파라미터는 기존 AdamW로 최적화하는 것이 권장됩니다. 이 프로젝트는 Muon의 구현체를 제공하며, 학습 속도 향상과 대규모 배치 처리에 효과적이라는 벤치마크 결과를 포함합니다. 핵심 하이퍼파라미터 튜닝은 학습률과 가중치 감쇠에 집중되며, 모델 크기 확장 시 별도의 재조정이 필요 없는 muP 스케일링을 지원합니다.  

> 요약: 신경망 은닉층 전용 최적화 알고리즘(Muon)과 구현체 제공. AdamW 대비 학습 속도 및 효율성 개선 효과를 검증한 벤치마크와 사용 사례 포함. Python 기반 PyTorch 호환 구현체 제공.
