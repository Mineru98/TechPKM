ARM Cortex-M 계열 MCU에서 동작하는 Non-RealTime OS로, 개발자가 개인 프로젝트로 진행 중입니다. 주로 펌웨어/하드웨어 개발 경험을 바탕으로 코드 정리와 기능 확장을 지속하고 있으며, Crossworks for ARM 및 STM32 CubeIDE를 공식 지원합니다. 이순신 장군의 이니셜에서 이름을 딴 이 OS는 실용적인 적용 가능성을 목표로 개발 중입니다.
