Deep-Live-Cam 2.0.1c는 단일 이미지와 한 번의 클릭으로 실시간 얼굴 교체와 영상 딥페이크를 가능하게 하는 오픈소스 프로젝트입니다. 웹캠, 영화, 라이브 방송 등에서 다양한 얼굴을 실시간으로 적용할 수 있으며, 윤리적 사용을 강조하는 내장 필터링 시스템을 포함합니다. Python 기반으로 CPU/GPU 가속을 지원하며, Windows 및 Mac Silicon 환경에서 사용 가능합니다.  

핵심 기능으로는 ▲실시간 얼굴 교체 ▲다중 얼굴 매핑 ▲마우스 마스크 보존 ▲영상/이미지 변환 등 AI 기반 미디어 제작 도구를 제공합니다. 사용자는 3단계(얼굴 선택 → 카메라 선택 → 라이브 실행)로 간편하게 접근할 수 있습니다.
