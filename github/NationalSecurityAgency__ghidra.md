Ghidra는 미국 국가안보국(NSA)에서 개발한 오픈소스 소프트웨어 리버스 엔지니어링(SRE) 프레임워크입니다. Windows, macOS, Linux 등 다양한 플랫폼에서 바이너리 분석, 디스어셈블리, 디컴파일, 그래프 생성, 스크립팅 등의 기능을 제공하며, 확장 가능한 연구 플랫폼과 협업 도구를 목표로 설계되었습니다. Java 또는 Python으로 사용자 확장 기능을 개발할 수 있으며, 보안 분석 및 취약점 연구에 활용됩니다.  

(※ 요약: 3문장 / 태그는 기술 스택과 용도 중심으로 구성 / 별칭은 프로젝트 명칭 변형과 주요 기능 반영)
