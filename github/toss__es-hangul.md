`es-hangul`은 한글을 쉽게 다룰 수 있도록 도와주는 JavaScript 라이브러리입니다. 초성 추출, 조사 붙이기 등의 한글 작업을 편리하게 수행할 수 있는 모던한 API를 제공하며, ECMAScript Modules을 지원하여 브라우저 환경에서 효율적으로 사용할 수 있습니다. 토스에서 개발했으며 MIT 라이선스로 배포됩니다.
