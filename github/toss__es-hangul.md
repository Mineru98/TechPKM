`es-hangul`은 한국어 문자열을 편리하게 처리할 수 있도록 돕는 JavaScript 라이브러리입니다. 초성 추출, 조사 붙이기 등 한글 관련 작업을 모던한 API로 제공하며, ECMAScript Modules을 지원해 브라우저 환경에서 효율적인 코드 사용이 가능합니다. 토스에서 개발하고 MIT 라이선스로 배포되는 이 라이브러리는 한국어 자연어 처리에 특화된 기능을 제공합니다.
