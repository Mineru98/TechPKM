PyCaret은 Python 기반의 오픈소스 로우코드 머신러닝 라이브러리로, 복잡한 머신러닝 워크플로우를 자동화합니다. 기존 라이브러리 대비 코드 작성량을 획기적으로 줄여 실험 주기를 가속화하고 생산성을 향상시키는 것이 핵심 목적입니다. 분류, 회귀, 시계열 분석, 클러스터링, 이상 탐지 등 다양한 모듈을 제공하며, scikit-learn, XGBoost 등 주요 라이브러리를 래핑하여 사용합니다. 데이터 과학자, 시민 데이터 과학자, 학생 등 다양한 사용자에게 적합하며, GPU 가속 및 Intel 최적화도 지원합니다.
