이 프로젝트는 Go로 작성된 자체 포함형 HTTP/HTTPS 프록시 서버입니다. 간단한 배포와 실행이 가능하도록 설계되었으며, 기본 인증, 로깅, SOCKS5 터널링 등 다양한 기능을 지원합니다. 사용자는 포트, 인증서, 로그 설정을 커스터마이즈할 수 있어 유연한 네트워크 프록싱이 가능합니다.  

- **주요 기능**: HTTP/HTTPS 양방향 프록시, 기본 인증, 요청 로깅, SOCKS5 터널링  
- **특징**: 단일 바이너리 배포, 커스터마이즈 가능한 로그 설정(glog), 플랫폼 독립 실행  
- **사용 사례**: 개발/테스트 환경에서의 간단한 프록시 서버, 네트워크 디버깅 도구  

> 태그: `Go`, `네트워크`, `프록시`, `인증`, `로깅`
