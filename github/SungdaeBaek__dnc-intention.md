dnc-intention은 monologg의 KoELECTRA 모델을 활용하여 한국어 텍스트에서 의도를 인식하는 프로그램입니다. LGPL-3.0 라이선스로 배포되며, 기술 문서는 공식 홈페이지에서 제공됩니다. 한국어 자연어 처리 분야에서 의도 분류 작업에 특화된 솔루션을 제공합니다.
