zoxide는 자주 사용하는 디렉토리를 학습해 빠르게 이동할 수 있는 **스마터한 `cd` 명령어로**, z 및 autojump에서 영감을 받은 Rust 기반 CLI 도구입니다. 다양한 쉘 환경에서 작동하며, 디렉토리 사용 빈도를 기반으로 순위를 매겨 `z` 명령어로 빠르게 접근할 수 있도록 설계되었습니다. Linux, macOS, Windows 등 주요 플랫폼을 지원하며, `cd` 대신 사용하거나 기존 디렉토리 점프 도구(autojump 등)의 데이터를 가져오는 기능도 제공합니다.
