Lexical은 뛰어난 신뢰성, 접근성, 성능을 제공하는 확장 가능한 텍스트 에디터 프레임워크입니다. 프레임워크에 구애받지 않는 코어 기술을 기반으로 하며, 공식 React 바인딩을 포함한 다양한 기능을 지원합니다. 플러그인 기반의 아키텍처로 사용자 정의가 용이하며, 실시간 협업 편집, 불변 상태 모델, JSON/Markdown/HTML 직렬화 등의 핵심 기능을 갖추고 있습니다.  

이 프로젝트는 크로스 브라우저 호환성을 갖추고 있으며, WCAG 준수 및 TypeScript 기반의 타입 안전성을 제공합니다. 개발자 친화적인 문서와 예제가 풍부하여 빠른 시작과 확장이 가능합니다.
