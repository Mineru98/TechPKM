렉시컬은 뛰어난 신뢰성, 접근성, 성능을 제공하는 확장 가능한 텍스트 에디터 프레임워크입니다. React 공식 바인딩을 포함한 모든 UI 프레임워크와 호환되며, 플러그인 기반 아키텍처로 실시간 협업, JSON/마크다운/HTML 변환, 테이블·리스트·코드 블록 지원 등 풍부한 기능을 제공합니다. 프레임워크에 구애받지 않는 코어 라이브러리로 타입스크립트 기반으로 개발되었습니다.
