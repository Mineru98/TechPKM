GoCD는 빌드-테스트-릴리스 주기를 자동화하여 제품의 지속적 전달을 지원하는 CI/CD 서버입니다. Java와 TypeScript를 주 언어로 사용하며 Spring Framework, SparkJava, MithrilJS 등의 기술을 기반으로 구축되었습니다. 사용자는 직관적인 파이프라인 구성, Docker 지원, 확장 가능한 아키텍처를 통해 효율적인 배포 워크플로우를 설계할 수 있습니다. 오픈소스 프로젝트로 Apache 2.0 라이선스로 배포되며 Thoughtworks의 후원을 받고 있습니다.
