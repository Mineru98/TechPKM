WireGuard VPN을 Docker 컨테이너로 배포하기 위한 리눅스서버 공식 이미지입니다. 서버 및 클라이언트 모드 지원, 자동 구성 생성, 다중 아키텍처 호환성을 특징으로 하며, 상태 기반 암호화를 활용한 빠르고 안전한 VPN 솔루션을 제공합니다. Docker 환경 변수를 통해 피처 수, 서브넷, DNS 설정 등을 유연하게 관리할 수 있습니다. 리눅스서버의 정기적인 업데이트와 보안 패치가 적용됩니다.
