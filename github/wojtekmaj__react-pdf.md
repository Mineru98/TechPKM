제공된 입력 정보에는 `packages/react-pdf/README.md` 파일의 내용이 포함되어 있지 않습니다. README 파일의 내용을 제공해 주시면, 해당 프로젝트의 목적, 기술 스택, 핵심 기능 등을 분석하여 Obsidian에 적합한 메타데이터 문서를 생성해 드리겠습니다.

현재 입력된 정보만으로는 필수 항목(요약, 태그, 별칭, 언어 등)을 추론할 수 없습니다. README 텍스트를 제공해 주시면 즉시 분석을 시작하겠습니다. 

예시 형식:
```
# react-pdf

React를 위한 PDF 렌더링 라이브러리입니다. 웹 애플리케이션에서 PDF 파일을 표시하고 상호작용하는 기능을 제공합니다.

## 주요 기능
- PDF 문서 렌더링
- 페이지 단위 확대/축소
- 텍스트 선택 및 검색
``` 

이와 같은 README 내용을 입력하시면 바로 메타데이터를 생성해 드리겠습니다.
