React 기반 PDF 문서 렌더링 라이브러리입니다. 웹 애플리케이션에서 PDF 파일을 직접 표시할 수 있는 컴포넌트를 제공하며, TypeScript와 호환됩니다. 사용자 친화적인 API와 커스터마이징 옵션으로 다양한 문서 뷰어 요구사항에 대응할 수 있습니다. 주요 기능으로는 텍스트 선택, 링크 클릭, 줌 제어 등이 포함됩니다.
