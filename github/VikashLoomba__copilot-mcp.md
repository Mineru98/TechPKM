이 프로젝트는 VS Code 확장을 통해 오픈소스 MCP 서버를 탐색하고 설치하며, GitHub Copilot Chat에 해당 도구들을 노출할 수 있게 해주는 도구입니다. 로컬 또는 원격(Cloud MCP) 서버 관리와 실시간 상태 모니터링, 자동 재연결 기능을 제공하며, Copilot에 표준화된 도구를 추가하는 데 목적이 있습니다. VS Code Marketplace에서 설치 가능하며, MCP 클라이언트 생태계의 일부입니다.
