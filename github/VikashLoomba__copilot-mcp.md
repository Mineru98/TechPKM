이 프로젝트는 VS Code 확장을 통해 오픈 소스 MCP(Model Context Protocol) 서버를 발견하고 설치하며, GitHub Copilot Chat에 도구를 직접 노출할 수 있는 기능을 제공합니다. 로컬 또는 원격(Cloud MCP) 환경에서 MCP 서버를 관리하는 직관적인 UI와 자동 재연결, 실시간 상태 모니터링 등의 핵심 기능을 포함합니다. VS Code Marketplace에서 설치 가능하며, 로컬 설정 없이도 OAuth 기반 원격 서버 사용이 가능합니다.  

> ✨ 주요 기능: MCP 서버 검색/관리, Copilot Chat 연동, 원격 Cloud MCP 지원, 실시간 서버 상태 모니터링.  
> 📦 기술 스택: VSCode 확장 기반 JavaScript 구현.  
> 🌟 특징: 로컬/원격 유연한 작업 환경, 모델 맥락 프로토콜(MCP) 생태계 통합.
