이 프로젝트는 마인크래프트 게임 내에서 자연어로 명령을 받아 자원을 채굴하고 구조물을 건설하며 전투를 수행하는 자율 AI 에이전트를 구현합니다. 멀티에이전트 협업 시스템을 통해 복잡한 작업(예: 성 건설)을 자동으로 분할하고 병렬 처리하며, OpenAI/Groq/Gemini LLM을 활용해 자연어 명령을 해석과 실행을 위한 구조화된 액션 시퀀스로 변환합니다. Forge 기반의 마인크래프트 모드로 개발되었으며, 실시간 게임 플레이에 최적화된 틱 기반 실행 모델과 직접 액션 실행 프레임워크를 채택했습니다.  

핵심 기능에는 자원 채굴, 자율적 건설, 전투 대응, 자원 수집, 협업 작업 분배 및 충돌 방지가 포함됩니다. 사용자는 `/steve spawn [이름]` 명령어로 에이전트를 소환하고 `K` 키를 눌러 자연어로 명령을 입력할 수 있습니다. 설정 파일(`steve-common.toml`)에서 LLM 제공업체와 API 키를 지정할 수 있으며, Groq/GPT-4/Gemini 모델을 지원합니다.
