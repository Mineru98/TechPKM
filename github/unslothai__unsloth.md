Unsloth는 대규모 언어 모델(LLM)을 2배 빠르게, VRAM 사용량을 70% 이상 줄여 미세 조정할 수 있는 Python 기반 라이브러리입니다. 다양한 모델(GPT, Qwen, Llama, Gemma 등)을 지원하며, 강화 학습(RL), 텍스트 음성 변환(TTS), 멀티모달 비전 훈련 등을 포함한 다양한 기능을 제공합니다. 초보자 친화적인 Colab 노트북과 포괄적인 문서를 통해 쉽게 시작할 수 있으며, Windows, Linux, Docker 환경에서 호환됩니다. Hugging Face, vLLM, llama.cpp 등과의 통합도 지원합니다.
