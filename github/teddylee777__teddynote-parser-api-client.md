TeddyNote Parser API를 사용하는 Python 클라이언트 라이브러리로, PDF 문서를 텍스트 및 이미지로 변환하고 구조화합니다. UPSTAGE와 OpenAI API를 연동하여 언어별 문서 처리가 가능하며, Docker 기반 서버 배포와 CLI/코드 기반 사용 방식을 지원합니다. MIT 라이선스로 배포되는 오픈소스 프로젝트입니다.
