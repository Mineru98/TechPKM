테디노트 파서 클라이언트는 PDF 문서를 구조화된 데이터로 변환하는 Python 클라이언트 라이브러리입니다. UPSTAGE와 OpenAI API를 연동하여 텍스트 추출, 이미지 처리, 다국어 지원을 제공하며, Docker 기반 API 서버와 연동해 대용량 문서 처리를 지원합니다. Jupyter Notebook 예제와 CLI 도구를 포함해 개발자 친화적인 인터페이스를 제공하는 것이 특징입니다. MIT 라이선스로 배포됩니다.
