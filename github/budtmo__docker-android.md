Docker-Android는 Android 애플리케이션 개발, 테스트, 빌드 및 클라우드 통합을 위한 도커 이미지입니다. 다양한 안드로이드 버전과 기기 프로필을 지원하며, VNC, ADB 연결, 로그 공유, Genymotion 클라우드 연동 등의 기능을 제공합니다. 컨테이너 환경에서 에뮬레이터 제어와 지속적 통합/배포(CI/CD) 파이프라인 구축에 최적화되어 있습니다.  

태그: Android 9.0~14.0, Genymotion 지원, 테스트 프레임워크(Espresso, Appium) 호환.  
기능: 프록시 설정, 헤드리스 모드, 언어 변경, 루트 권한(Pro 버전 한정) 등 확장 기능 별도 제공.
