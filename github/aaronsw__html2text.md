html2text는 HTML 페이지를 깔끔하고 읽기 쉬운 ASCII 텍스트(또는 유효한 Markdown)로 변환하는 Python 스크립트입니다. 링크, 이미지, 목록 등의 요소를 처리하며, Google 문서 변환 옵션도 제공합니다. CLI 또는 Python 라이브러리 형태로 사용할 수 있으며, GPLv3 라이선스로 배포됩니다. 주요 기능은 HTML 콘텐츠를 구조화된 텍스트로 변환하는 것입니다.
