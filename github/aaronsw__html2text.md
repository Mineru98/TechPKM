html2text는 HTML 페이지를 가독성 좋은 일반 ASCII 텍스트 또는 유효한 Markdown 형식으로 변환하는 Python 스크립트입니다. 명령줄 또는 Python 라이브러리 형태로 사용할 수 있으며, 링크/이미지 무시, Google 문서 변환 최적화, 목록 형식 조정 등의 기능을 제공합니다. GPLv3 라이선스로 배포되며, Travis CI를 통한 지속적 통합 테스트를 지원합니다.  

요약: HTML을 Markdown 또는 ASCII 텍스트로 변환하는 CLI 도구 및 Python 라이브러리. 링크/이미지 처리, Google 문서 변환 최적화, 줄 바꿈 조정 등의 기능 제공.
