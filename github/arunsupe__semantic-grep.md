w2vgrep은 단어 임베딩을 활용한 의미 기반 텍스트 검색 커맨드라인 도구입니다. 기존 grep과 유사한 사용 경험을 제공하면서도 단순한 문자열 매칭을 넘어 의미적으로 유사한 단어를 검색할 수 있으며, 다중 언어 지원과 컨텍스트 표시, 유사도 점수 계산 등의 기능을 포함합니다. 텍스트 파일이나 표준 입력에서 작동하며 JSON 설정 파일 또는 명령줄 옵션으로 구성 가능합니다.
