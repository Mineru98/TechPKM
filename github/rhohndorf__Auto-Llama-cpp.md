이 프로젝트는 Auto-GPT를 포크하여 llama.cpp를 통해 로컬에서 실행 가능한 자율적인 라마 모델을 지원하는 실험적인 구현입니다. 작은 로컬 모델로도 마법 같은 결과를 생성할 수 있지만, GPT-4 수준의 성능을 기대해서는 안 됩니다. 주요 목적은 자체 프롬프팅과 JSON 형식 응답을 통한 자율적인 작업 수행을 탐구하는 것입니다. 현재 속도 문제와 컨텍스트 창 크기 한계가 있으나, 다양한 언어 모델 지원을 통해 지속적인 개발이 진행 중입니다.
