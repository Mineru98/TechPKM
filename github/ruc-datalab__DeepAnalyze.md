DeepAnalyze는 자율적인 데이터 과학 작업을 수행하는 최초의 에이전트형 대형 언어 모델(LLM)입니다. 데이터 준비, 분석, 모델링, 시각화, 보고서 생성 등 전체 데이터 과학 파이프라인을 자동으로 처리하며, 구조화된 데이터부터 비정형 데이터까지 다양한 소스를 분석하여 분석가 수준의 연구 보고서를 생성합니다. 모델, 코드, 훈련 데이터, 데모가 완전히 오픈소스화되어 개인 배포 또는 확장이 가능합니다.
