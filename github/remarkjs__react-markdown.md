react-markdown은 React 환경에서 Markdown을 안전하게 렌더링할 수 있는 컴포넌트입니다. 기본적으로 XSS 공격을 방지하고, 사용자 정의 컴포넌트를 지원하여 HTML 요소를 대체할 수 있습니다. CommonMark 및 GFM(기호 플러그인 사용 시)을 완벽히 준수하며, remark 및 rehype 플러그인을 통해 확장 가능합니다. GitHub Flavored Markdown, 수학 공식, 구문 강조 등의 기능을 추가할 수 있고, TypeScript 타입 정의를 제공합니다.
