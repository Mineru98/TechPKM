이 프로젝트는 Dify 플랫폼에 플러그인을 개발하기 위한 Python SDK를 제공합니다. 플러그인의 메타데이터 버전 관리 및 호환성 체계를 정의하며, OAuth, 멀티모달 출력, 데이터소스 등 다양한 기능을 지원합니다. 개발자는 명세된 매니페스트 버전을 통해 플러그인의 하위/상위 호환성을 명시적으로 관리할 수 있습니다.  

> 요약: Dify 플러그인 개발을 위한 Python SDK 및 버전 호환성 가이드라인 제공
