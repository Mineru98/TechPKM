에이전트2에이전트(A2A) 프로토콜은 서로 다른 프레임워크로 구현된 AI 에이전트들이 내부 상태를 노출하지 않으면서 효과적으로 통신하고 협업할 수 있도록 하는 오픈소스 표준입니다. 이 프로토콜은 에이전트 발견, 상호작용 방식 협상(텍스트/폼/미디어), 장기 작업 협업 등을 지원하며, JSON-RPC 2.0 over HTTP(S)를 기반으로 표준화된 통신을 제공합니다. 특히 보안, 인증, 관측 가능성을 고려한 엔터프라이즈급 설계로 다양한 산업 분야에 적용 가능합니다.  

프로젝트 소개 영상, 공식 문서 링크, Python/Go/JS/Java/.NET SDK 제공, 샘플 구현체 등 개발자 친화적 도구를 지원합니다. 리눅스 재단 산하의 오픈소스 프로젝트로 Apache 2.0 라이선스로 배포되며, 커뮤니티의 기여를 적극 권장하고 있습니다.
