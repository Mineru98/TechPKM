오픈 러버블은 AI와 채팅으로 React 앱을 즉시 생성할 수 있는 오픈소스 프로젝트입니다. Firecrawl 팀이 개발한 이 프로젝트는 다양한 LLM 제공업체와 연동해 프런트엔드 프로토타이핑을 간소화합니다. Vercel 또는 E2B 샌드박스 환경에서 실행 가능하며, MIT 라이선스로 배포됩니다. 주요 기능은 AI 기반 코드 생성 및 클라우드 연동 워크플로우 지원입니다.
