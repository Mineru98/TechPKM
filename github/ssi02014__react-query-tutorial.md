이 프로젝트는 TanStack Query(React) v5의 주요 개념과 활용 방법을 정리한 튜토리얼 저장소입니다. 서버 상태 관리, 캐싱, 데이터 동기화 등 리액트 애플리케이션 개발에 필요한 핵심 기능을 설명하며, v4와의 마이그레이션 가이드도 포함되어 있습니다. 타입스크립트 적용 사례와 ESLint 플러그인 사용법 등 실용적인 내용을 다루며, 지속적인 업데이트로 최신 정보를 제공합니다. 쿼리 무효화, 낙관적 업데이트, Suspense 통합 등 고급 패턴도 학습할 수 있습니다.
