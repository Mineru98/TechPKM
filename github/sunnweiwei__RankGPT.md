RankGPT는 정보검색(IR) 분야에서 대규모 언어 모델(LLM)을 재순위화 에이전트로 활용하는 프로젝트입니다. ChatGPT 및 GPT-4와 같은 생성형 모델을 활용해 검색 결과의 관련성을 평가하고 순위의 정확도를 향상시키는 방법을 연구합니다. MS MARCO, TREC-DL, BEIR 등의 벤치마크에서 실험적으로 검증되었으며, 슬라이딩 윈도우 전략과 Instruction Distillation 기술을 통해 효율적인 순위 조정을 구현합니다. EMNLP 2023에서 최우수 논문상을 수상한 연구 기반입니다.
