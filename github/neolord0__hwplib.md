hwplib은 한컴오피스의 HWP 파일 형식을 읽고 쓰기 위한 Java 라이브러리입니다. Apache-POI 기반으로 개발되었으며, 텍스트 추출, 필드 조작, 표 삽입 및 수정, 이미지 추가 등의 기능을 제공합니다. Apache-2.0 라이선스로 배포되며, 메이븐 저장소를 통해 쉽게 통합 가능합니다. 현재 암호화된 파일 처리, PDF/이미지 변환 기능은 지원하지 않습니다.
