이 프로젝트는 PDF 문서를 다양한 애니메이션 효과가 포함된 동영상 프리젠테이션으로 변환하는 도구입니다. React 기반의 Remotion 프레임워크를 활용해 카드 스택, 줌 인, 페이지 전환 등 동적 시각 효과를 구현하며, 사용자 정의 가능한 제목 시스템, 정보 표시줄, 배경 음악 통합 기능을 제공합니다. JSON 설정 파일을 통해 영상 구성 요소를 유연하게 제어하거나 Claude AI 기술과 연동해 자동 변환을 수행할 수 있습니다.

핵심 기능으로는 다중 장면 유형 지원(스택/포커스/전환/팬), 스마트 애니메이션 효과(호흡 효과/바운스/페이드), PDF 페이지 별 커스터마이징 가능한 메타데이터, 동적 영상 길이 계산 기능이 포함됩니다. MIT 라이선스로 배포되며, 기술 스택은 Remotion, React, pdf.js, Zod 등을 기반으로 합니다.
