`typia`는 TypeScript 타입 기반의 초고속 런타임 검증, JSON 스키마 생성, LLM 함수 호출 스키마, 프로토콜 버퍼 변환, 랜덤 데이터 생성 기능을 제공하는 라이브러리입니다. 순수 TypeScript 타입을 단일 라인으로 입력받아 컴파일 단계에서 최적화된 런타임 함수로 변환하며, 기존 검증 도구보다 20,000배, JSON 변환 도구보다 200배 빠른 성능을 자랑합니다. 타입 안전성을 보장하면서도 추가적인 스키마 정의 없이 TypeScript 타입만으로 개발 효율성을 극대화합니다.
