이 프로젝트는 MNIST 숫자 이미지를 10차원 임베딩으로 변환하는 Siamese Network 모델을 구현합니다. "Image similarity using Deep CNN and Curriculum Learning" 논문의 SimNet 손실 함수를 적용해 유사도 기반 임베딩 학습을 진행하며, 이미지 비교 및 유사도 분석을 위한 딥러닝 연구를 목표로 합니다. 주요 구성 요소로는 `simnet.py`와 관련 논문 참조 자료가 포함됩니다.
