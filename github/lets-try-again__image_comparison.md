이 프로젝트는 MNIST 숫자 데이터셋의 10차원 임베딩을 학습하기 위해 Siamese Network 모델을 구현합니다. "Image similarity using Deep CNN and Curriculum Learning" 논문의 SimNet 손실 함수를 적용하여 이미지 유사도 기반 임베딩 학습을 수행합니다. 딥러닝을 활용한 숫자 이미지의 유사도 측정이 핵심 목적이며, 교육 전략(cruriculum learning) 요소가 포함된 모델 아키텍처를 특징으로 합니다.
