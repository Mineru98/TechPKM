이 프로젝트는 Arxiv에 업로드된 최신 AI/ML 논문들을 한국어로 빠르게 번역하여 제공하는 것을 목표로 합니다. Nougat OCR 라이브러리를 활용해 PDF에서 텍스트를 추출하고, 자체 번역 모델로 한글화하여 웹에서 쉽게 접근할 수 있도록 HTML 형태로 변환합니다. 주로 대규모 언어 모델(LLM) 관련 연구 동향을 다루며, 기술 스택에는 파이썬, OCR, 번역 모델, 마크다운/HTML 변환 도구가 포함됩니다. 번역 정확도는 상용 번역 도구의 중간 수준이며, 향후 이미지 포함 번역 기능을 확장할 예정입니다.
