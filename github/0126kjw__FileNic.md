파일닉은 macOS와 윈도우 간 한글 파일명의 자소분리 호환 문제를 해결하는 macOS 전용 도구입니다. 드래그 앤 드롭으로 간편하게 파일/폴더명을 변환하여 양쪽 운영체제에서 정상적으로 표시되도록 합니다. 로컬에서 처리되며 서버 통신이 없어 개인정보 유출 위험이 전혀 없습니다. 메뉴바에서 즉시 사용 가능하며 폴더 변환 시 하위 파일 처리 여부를 선택할 수 있는 기능이 있습니다.
