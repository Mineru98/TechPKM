`plotly.py`는 파이썬을 위한 인터랙티브하고 오픈소스 기반의 브라우저 기반 그래프 라이브러리입니다. Plotly.js를 기반으로 30종 이상의 차트 유형(3D 그래프, 과학 차트, 통계 차트, 지도, 금융 차트 등)을 지원하며, MIT 라이선스로 배포됩니다. Jupyter 노트북, 독립 HTML 파일, Dash 애플리케이션 등에 통합 가능하며, `pip install plotly`로 쉽게 설치할 수 있습니다.  

핵심 특징으로는 Kaleido를 통한 정적 이미지 내보내기, 지리적 데이터를 위한 `plotly-geo` 확장 지원, 그리고 Jupyter 위젯 통합 기능이 포함됩니다. 공식 문서는 [plotly.com/python/](https://plotly.com/python/)에서 확인할 수 있습니다.
