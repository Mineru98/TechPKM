plotly.py는 Python용 오픈소스 대화형 그래프 라이브러리로, Plotly.js 기반으로 30종 이상의 차트 유형(과학 차트, 3D 그래프, 통계 차트 등)을 지원합니다. Jupyter 노트북, Dash 애플리케이션, 독립 HTML 파일 등에서 사용 가능하며 MIT 라이선스를 따릅니다.  

`pip install plotly`로 간편하게 설치할 수 있으며, `plotly.express` 모듈을 통해 선언적 방식으로 시각화를 생성할 수 있습니다. 정적 이미지 내보내기, 지리적 데이터 지원 등 확장 기능도 제공됩니다.
