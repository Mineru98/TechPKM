Brotli는 LZ77 알고리즘 변형, 허프만 코딩, 2차 컨텍스트 모델링을 결합한 범용 무손실 압축 알고리즘입니다. Deflate와 유사한 속도를 유지하면서 더 높은 압축률을 제공하며, 웹 전송 최적화 등에 활용됩니다. MIT 라이선스로 배포되는 오픈소스 프로젝트로, C/C++ 기반의 핵심 구현체와 Python 바인딩을 포함합니다. 압축 스트림 포맷은 RFC 7932로 표준화되었습니다.
