LangChain은 LLM(대형 언어 모델) 기반 애플리케이션 및 에이전트 구축을 위한 파이썬 프레임워크입니다. 모델, 임베딩, 벡터 저장소 등의 상호 운용 가능한 구성 요소를 연결하여 AI 애플리케이션 개발을 간소화하고, 기술 진화에 대비한 유연한 설계를 제공합니다. 주요 사용 사례로는 실시간 데이터 증강, 모델 교체 가능성, 빠른 프로토타이핑, 프로덕션 준비 기능, 활발한 커뮤니티 생태계가 포함됩니다.  

```bash  
pip install langchain  
```  

고급 커스터마이징이나 에이전트 오케스트레이션이 필요한 경우 [LangGraph](https://docs.langchain.com/oss/python/langgraph/overview)를 참고하세요.
