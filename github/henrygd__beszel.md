Beszel은 도커 통계, 과거 데이터 추적, 경고 기능을 포함한 경량 서버 모니터링 플랫폼입니다. 사용자 친화적인 웹 인터페이스, 간단한 구성, 즉시 사용 가능한 기능을 제공하며, 자동 백업, 멀티 사용자 지원, OAuth 인증, API 접근 등을 특징으로 합니다. PocketBase 기반의 허브와 각 시스템 모니터링을 위한 에이전트로 구성되어 있습니다.
