Beszel은 Docker 통계, 역사적 데이터, 경고 기능을 포함한 경량 서버 모니터링 플랫폼입니다. 직관적인 웹 인터페이스, 간단한 설정, 즉시 사용 가능한 특징을 가지며, 자동 백업, 멀티유저 지원, OAuth 인증 및 API 접근성을 제공합니다. 호스트 시스템과 컨테이너의 CPU, 메모리, 네트워크 사용량, 디스크 I/O, 온도, 배터리 상태 등 다양한 메트릭을 실시간으로 모니터링할 수 있습니다. PocketBase 기반의 허브와 각 시스템에 설치되는 에이전트로 구성된 아키텍처를 사용합니다.
