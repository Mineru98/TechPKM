테슬라 REST API를 Node.js 환경에서 사용할 수 있도록 구현한 클라이언트 라이브러리입니다. 테슬라 공식 앱이 사용하는 비공차 API를 통해 차량 상태 모니터링 및 원격 제어 기능을 제공합니다. 충전 상태, 기후 제어, 차량 위치 확인, 도어 잠금 등 다양한 기능을 지원하며, 스트리밍 데이터 수집 및 시각화 도구도 포함되어 있습니다.  

> ⚠️ 이 프로젝트는 테슬라 모터스 공식 제품이 아니며, 사용 시 발생하는 책임은 전적으로 사용자에게 있습니다. 서버 과부하 방지를 위해 API 호출 빈도를 적절히 조절해야 합니다.
