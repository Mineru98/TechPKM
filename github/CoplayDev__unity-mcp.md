**MCP for Unity**는 Unity 에디터를 LLM(대형 언어 모델) 기반 AI 어시스턴트(Claude, Cursor 등)와 연결하는 오픈소스 프로젝트입니다. Model Context Protocol(MCP)을 활용해 자연어 명령으로 에셋 관리, 씬 제어, 스크립트 편집, 작업 자동화 등 Unity 개발 워크플로우를 혁신합니다. Python 3.10+ 및 Unity 2021.3 LTS 이상 환경에서 동작하며, 다양한 설치 옵션(패키지 관리자, Asset Store 등)을 지원합니다. 주요 기능으로는 배치 작업, 여러 Unity 인스턴스 관리, Roslyn 기반 스크립트 검증 등이 있습니다. MIT 라이선스로 배포되며 Coplay에서 후원 및 유지보수합니다.
