**Unity 애플리케이션을 LLM(Large Language Model)과 연결하여 개발 워크플로우를 혁신하는 프로젝트**입니다. Model Context Protocol(MCP)을 통해 Claude, Cursor, VS Code Copilot 등 다양한 AI 도구와 Unity 에디터를 연동하여 자산 관리, 장면 제어, 스크립트 편집, 작업 자동화 등을 자연어로 수행할 수 있게 합니다. Python 3.10+ 및 Unity 2021.3 LTS 이상이 필요하며, MIT 라이선스로 배포됩니다.

핵심 기능으로는 자연어 제어, 강력한 도구 세트(자산/장면/스크립트 관리), 확장성, 다중 Unity 인스턴스 지원이 포함됩니다. `batch_execute` 도구를 사용하면 여러 작업을 일괄 처리하여 성능을 극대화할 수 있습니다. Coplay에서 스폰서링하는 오픈소스 프로젝트로, 연구 논문 인용 정보도 제공됩니다.
