Vanna 2.0은 자연어 질문을 SQL 쿼리로 변환하고 데이터베이스 인사이트(테이블, 차트, 요약)를 실시간으로 제공하는 오픈소스 프로젝트입니다. 엔터프라이즈급 보안(행 수준 권한, 감사 로그)과 사용자 인식 기능을 통해 다중 테넌트 SaaS 환경에 적합하며, `<vanna-chat>` 웹 컴포넌트를 통해 기존 웹 애플리케이션에 쉽게 통합할 수 있습니다. 다양한 LLM(OpenAI, Anthropic 등)과 데이터베이스(SQLite, PostgreSQL 등)를 지원합니다.  

핵심 기능:  
1. 자연어 → SQL 자동 변환  
2. 실시간 스트리밍 응답(진행 상태, 테이블, 시각화 차트)  
3. 사용자 권한 기반 쿼리 필터링  
4. 기존 인증 시스템(JWT, 쿠키)과 통합  
5. 커스텀 도구 확장을 통한 비즈니스 로직 통합
