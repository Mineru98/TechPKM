Vanna 2.0은 자연어 질문을 SQL 쿼리로 변환하고 데이터 인사이트를 제공하는 오픈소스 도구입니다. 사용자 권한 기반 쿼리 필터링, 스트리밍 응답, 엔터프라이즈급 보안 기능을 탑재했으며, 웹 컴포넌트를 통해 다양한 플랫폼에 통합 가능합니다. PostgreSQL, Snowflake 등 다양한 데이터베이스와 LLM 모델을 지원하며, 사용자 정의 도구와 확장 기능을 제공합니다.  

핵심 기능:  
1. **보안 강화** - 행 수준 보안, 감사 로그, 사용자별 권한 관리  
2. **실시간 스트리밍** - 테이블, 차트, SQL 결과를 웹 UI에 즉시 표시  
3. **통합 용이성** - 기존 인증 시스템(JWT/쿠키)과 호환되는 `<vanna-chat>` 컴포넌트  
4. **확장성** - 사용자 정의 도구 및 LLM 미들웨어 지원  
5. **다양한 지원** - FastAPI, React, Vue 등 프레임워크와의 통합  

사용 사례: 데이터 분석 대시보드, 다중 테넌트 SaaS, 보안 요구 사항이 있는 엔터프라이즈 환경 등에 적합합니다.
