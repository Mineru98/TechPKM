이 프로젝트는 한국 기상청 API를 활용한 MCP 서버로, 한국 지역의 단기 날씨 예보 정보를 AI 에이전트에 제공합니다. 행정구역(시/구/동) 단위의 정확한 기상 데이터를 그리드 좌표로 조회하고, 온도, 강수 확률, 습도, 바람 정보 등을 구조화된 텍스트 형식으로 반환합니다. LLM과의 연동 최적화가 가능하며 Smithery를 통한 간편한 설정이 지원됩니다.
