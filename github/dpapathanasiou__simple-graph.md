이 프로젝트는 SQLite를 기반으로 한 간단한 그래프 데이터베이스로, 노드(JSON 객체)와 엣지(노드 간 연결) 구조를 지원합니다. 소셜 네트워크, 추천 시스템, 태스크 관리 등 다양한 애플리케이션에 적용 가능하며, CRUD 연산과 경로 탐색 기능을 제공합니다. Python, Go, Julia 등 다양한 언어에서 사용 가능한 무료 라이브러리와 상용 RESTful API 서비스를 함께 제공합니다.
