Zero는 사용자가 직접 호스팅할 수 있는 오픈소스 AI 기반 이메일 솔루션으로, Gmail 등 외부 서비스와 통합되는 통합 수신함을 제공합니다. 데이터 프라이버시 보호를 최우선으로 하며, AI 에이전트와 커스터마이징 가능한 UI를 통해 현대적인 이메일 경험을 구현합니다. Next.js, Node.js, PostgreSQL 등의 최신 기술 스택을 기반으로 개발되었습니다.
