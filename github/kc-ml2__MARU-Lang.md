**MARU**는 기업 환경에서 효과적으로 작동하도록 설계된 오픈소스 RAG 검색 증강 생성 챗봇 엔진입니다. 기업 데이터와의 원활한 통합을 핵심 원칙으로 삼아 문서 관리 시스템과 인프라를 고려한 설계로, 기존 기업 데이터와의 통합 및 사용자 경험 최적화에 중점을 둡니다. 설치 후 인그레스트 파이프라인과 채팅 파이프라인을 통해 문서 처리부터 질의 응답까지의 전체 RAG 워크플로우를 구축할 수 있으며, 모든 구성 요소를 커스터마이징할 수 있습니다.  

MARU는 PDF, DOCX 등 다양한 파일 형식의 문서 파싱, 권한 기반 필터링, 검색 결과 재순위 지정, 사용자 정의 에이전트 실행 등 엔터프라이즈급 기능을 제공하며, 모듈식 설계로 확장성이 뛰어납니다. 웹 UI 통합 및 REST API를 통해 기존 시스템에 쉽게 통합할 수 있으며, MIT 라이선스로 배포되어 커뮤니티 기여를 장려합니다.
