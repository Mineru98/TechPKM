@marp-team/marp-cli는 마크다운 파일을 프레젠테이션 형식으로 변환하는 명령줄 인터페이스입니다. Marp 및 Marpit 프레임워크를 기반으로 하여 HTML, PDF, PowerPoint(PPTX), 이미지(PNG/JPEG) 등 다양한 형식으로 변환할 수 있습니다. 브라우저 기반 변환(Chrome, Edge, Firefox 필요)과 실시간 미리보기, 서버 모드 등의 기능을 제공하며, 테마 및 엔진 설정을 통해 커스터마이징이 가능합니다.

주요 기능:
- 마크다운 → HTML/PDF/PPTX/이미지 변환
- 실시간 파일 감시 및 자동 재변환 (Watch 모드)
- HTTP 서버 모드 지원
- 내장된 여러 템플릿 및 테마 제공
- 커스텀 엔진 및 플러그인 확장 가능

변환 옵션에는 PDF 북마크, PPTX 편집 가능 모드, 이미지 스케일 조정 등이 포함됩니다. Docker 컨테이너 이미지와 Standalone 바이너리 형태로도 제공되며, npm, Homebrew, Scoop 등 다양한 설치 방식을 지원합니다.
