Observer AI는 사용자의 디지털 환경을 관찰, 기록 및 반응하는 로컬 마이크로 에이전트 시스템입니다. 화면, 카메라, 마이크 등의 센서 데이터를 수집하고 로컬 LLM 모델로 분석한 후 이메일, 텔레그램, 디스코드 등의 도구로 지능적으로 반응합니다. 모든 데이터는 사용자 장치에서 처리되며 100% 프라이버시가 보장됩니다. 데스크톱 앱 형태로 제공되며 Ollama, vLLM 등 오픈소스 LLM과 호환됩니다. 사용자는 간단한 코드 탭에서 자바스크립트 로직을 추가하여 맞춤형 에이전트 동작을 구현할 수 있습니다.
