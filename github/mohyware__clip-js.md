이 프로젝트는 nextjs, remotion, 웹 어셈블리 포트의 ffmpeg를 기반으로 제작된 온라인 비디오 편집 도구입니다. 실시간 프리뷰, 인터랙티브 타임라인 편집, 다양한 미디어 포맷 지원 및 고급 요소 제어 기능을 통해 사용자가 손쉽게 영상을 편집하고 1080p까지 고품질 렌더링이 가능합니다. 키보드 단축키와 사용자 친화적인 편집 유틸리티를 제공하여 효율적인 작업 환경을 구축했습니다. Docker 지원 및 설치 가이드도 포함되어 있습니다.
