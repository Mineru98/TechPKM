LAS Book ERP 시스템은 도서 관리를 위한 웹 기반 ERP 솔루션입니다. FastAPI와 Docker를 활용해 백엔드와 프론트엔드를 통합한 배포 환경을 제공하며, 도서 관리 워크플로우 전 과정을 디지털화합니다. 정적 웹 서버와 API 문서 자동 생성 기능을 포함하는 것이 특징입니다. 주요 기술 스택으로는 Python, Vite, TypeScript를 사용합니다.
