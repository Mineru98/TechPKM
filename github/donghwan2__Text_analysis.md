이 프로젝트는 화장품 리뷰 데이터를 기반으로 감성 분석을 수행하는 한국어 자연어처리 연구입니다. 글로우픽 플랫폼에서 수집한 고객 후기를 Okt 형태소 분석기와 konlpy를 활용해 긍정/부정 감성을 예측하며, 이모티콘 처리와 오타 수정 등 한글 NLP 특화 기법을 적용합니다. 분석 결과는 화장품 추천 모델 개발에 기여할 수 있는 의미 있는 인사이트를 제공합니다.  

### 요약  
1. **목적**: 화장품 리뷰 감성 예측을 통한 제품 추천 모델 기반 마련  
2. **기술**: konlpy/Okt 형태소 분석, 이모티콘/하트/오타 전처리  
3. **의의**: 화장품 업계의 리뷰 데이터 활용 중요성 강조  
4. **확장성**: 화장품 추천 시스템으로의 적용 가능성 탐구  

> 💡 한글 NLP 특화 전처리(이모티콘, 오타 처리)가 핵심 포인트입니다.
