이 프로젝트는 MidJourney의 비공식 API를 사용하는 Node.js 클라이언트 라이브러리입니다. Discord 봇과 연동해 텍스트 프롬프트로 AI 이미지를 생성하거나 변형, 확대, 리믹스 등 다양한 기능을 지원합니다. TypeScript로 작성된 이 도구는 개발자들이 MidJourney 기능을 애플리케이션에 쉽게 통합할 수 있도록 설계되었습니다. 주요 기능으로는 얼굴 교체, 니지 봇 지원, 커스텀 줌, 리믹스 모드 등이 있으며, 다양한 예제 코드와 상세한 사용 설명서가 제공됩니다.
