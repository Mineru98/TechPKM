이 프로젝트는 비공식 MidJourney API를 사용할 수 있는 Node.js 클라이언트 라이브러리입니다. Discord 봇 연동을 통해 AI 이미지 생성, 변형, 확대 등 다양한 기능을 제공하며, Remix 모드, Niji 봇 지원, 커스텀 줌 등 고급 기능을 포함합니다. Discord 토큰 기반 인증 시스템과 웹소켓 통신을 활용해 실시간 메시지 처리가 가능합니다. TypeScript로 작성되었으며, NPM 패키지로 쉽게 설치할 수 있습니다.
