이 프로젝트는 [Zeo++ 라이브러리](http://zeoplusplus.org)를 Python에서 사용할 수 있도록 하는 래퍼입니다. Apple Silicon을 포함한 여러 플랫폼용 사전 빌드 바이너리를 제공하며, pip를 통한 간편한 설치가 가능합니다. 과학 계산 분야에서 원자 구조 분석에 활용되는 Zeo++ 및 Voro++ 라이브러리와 통합되어 있습니다. 소스 빌드 시 플랫폼별 컴파일 요구사항이 존재하며, Cython을 이용한 바인딩 재생성 기능도 지원합니다. Apache 2.0 라이선스로 배포됩니다.
