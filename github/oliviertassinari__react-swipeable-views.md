이 프로젝트는 React 기반의 스와이프 가능한 뷰 컴포넌트를 제공합니다. 모바일 환경에서 터치 제스처를 통해 슬라이드를 전환할 수 있으며, 웹과 네이티브(실험적) 버전을 모두 지원합니다. 가볍고 빠른 초기 렌더링과 지연 로딩 기능을 특징으로 하며, 다양한 데모 및 문서를 통해 사용 방법을 확인할 수 있습니다.

라이브러리는 Material-UI, Doctolib, Uber 등 다양한 기업에서 활용되고 있으며, MIT 라이선스로 배포됩니다. 웹과 React Native 환경에서 각각 최적화된 구현을 제공합니다.
