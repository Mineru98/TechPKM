OpenAI에서 개발한 Codex CLI는 로컬에서 실행되는 코딩 에이전트 도구로, 개발자가 코드 편집 없이 터미널에서 AI 기반 코드 분석 및 생성 기능을 활용할 수 있도록 설계되었습니다. ChatGPT 구독 플랜과 연동되며 npm 또는 Homebrew를 통해 간편하게 설치 가능합니다. 주요 특징으로는 로컬 실행, IDE 통합 옵션, 다양한 플랫폼 지원(macOS/Linux)이 포함됩니다. Apache-2.0 라이선스로 배포되며 공식 문서를 통해 상세한 사용법을 확인할 수 있습니다.
