Codex CLI는 OpenAI에서 제공하는 로컬 실행형 AI 코딩 에이전트입니다. 코드 편집기 외부에서 터미널에서 직접 코드 생성 및 보조 기능을 사용할 수 있으며, ChatGPT 구독 플랜 또는 API 키를 통해 인증이 가능합니다. macOS와 Linux를 지원하는 크로스플랫폼 도구로, 패키지 관리자(npm/Homebrew) 또는 GitHub 릴리스를 통해 설치할 수 있습니다. 주요 기술 스택은 Shell 기반 CLI 환경을 중심으로 구현되었습니다.
