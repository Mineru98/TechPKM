AWS S3를 벡터 저장소로 활용하는 비공식 Rust 기반 CLI 도구. RAG(Retrieval-Augmented Generation) 기능을 내장하여 문서 임베딩 생성, 벡터 저장, 유사도 검색 및 자연어 쿼리를 지원합니다. S3 버킷/인덱스/벡터 관리 및 정책 설정 기능을 제공하며, all-MiniLM-L6-v2 모델을 활용한 문서 분할과 임베딩 처리를 수행합니다. 대화형 모드, 배치 처리, 다양한 거리 메트릭(코사인/유클리드) 지원을 특징으로 합니다.
