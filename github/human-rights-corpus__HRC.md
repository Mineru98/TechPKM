이 프로젝트는 대화형 생성 모델 학습을 위한 인권 관련 코퍼스를 구축하는 것을 목적으로 합니다. 대한민국 국가인권위원회의 결정례와 상담사례를 기반으로 텍스트를 재구성하고 GPT-3.5-turbo 모델을 활용하여 학습 데이터를 생성합니다. 인권 관련 법적 사례와 상담 내용을 자연어 처리 가능한 구조로 변환하여 AI 모델의 생성 능력 향상을 지원합니다. 주요 기술 스택은 Python 기반 자연어 처리 도구이며, JSONL 형식의 데이터셋으로 구성됩니다.
