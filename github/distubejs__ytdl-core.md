DisTube에서 유지하는 `ytdl-core` 포크 버전으로, YouTube 동영상 다운로드를 위한 Node.js 라이브러리입니다. 원본 저장소보다 빠르게 버그 수정과 기능 추가를 반영하며, 쿠키/프록시 지원, IP 회전, 다운로드 제한 해결 등의 기능을 제공합니다. YouTube의 지역 제한, 프라이빗 콘텐츠 등 특정 조건의 동영상도 조건에 따라 다운로드할 수 있습니다.  

> [!NOTE]  
> 이 저장소는 더 이상 유지보수되지 않으며, 대안으로 [youtubei.js](https://github.com/LuanRT/YouTube.js) 사용을 권장합니다.
