@distube/ytdl-core은 DisTube 팀이 개발한 YouTube 비디오 다운로드 라이브러리로, 원본 ytdl-core의 버그 수정과 기능 추가를 신속하게 반영합니다. 쿠키 인증, 프록시 지원, IP 로테이션 등의 고급 기능을 제공하며, 지역 제한 또는 프리미엄 콘텐츠에 접근할 수 있는 확장성을 특징으로 합니다. YouTube.js 기반 솔루션으로 전환되는 과정에서 더 이상 유지보수되지 않을 예정이며, 최신 버전 사용을 권장합니다.
