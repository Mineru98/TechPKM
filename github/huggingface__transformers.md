이 프로젝트는 텍스트, 컴퓨터 비전, 오디오, 비디오 및 멀티모달 모델을 위한 최첨단 사전 훈련 모델을 제공하는 Python 라이브러리입니다. 추론 및 훈련을 위한 모델 정의 프레임워크로, 다양한 프레임워크 간 호환성을 보장하며 생태계 전반에서 모델 정의를 표준화하는 것을 목표로 합니다. 100만 개 이상의 모델 체크포인트를 허깅페이스 허브에서 사용할 수 있으며, 간단한 API로 사전 훈련 모델을 빠르게 활용할 수 있습니다.
