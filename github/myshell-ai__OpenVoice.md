OpenVoice는 정확한 톤 복제, 유연한 음성 스타일 제어, 제로샷 크로스링구얼 보이스크로닝 기능을 제공하는 음성 합성 프로젝트입니다. V2에서는 향상된 오디오 품질, 네이티브 다국어 지원(영어, 스페인어, 프랑스어, 중국어, 일본어, 한국어), MIT 라이선스로 상업적 사용이 가능합니다. 연구 논문과 웹사이트, 실제 플랫폼 적용 사례를 포함하며, Coqui-TTS와 VITS 프로젝트 기반 구현이 특징입니다.
