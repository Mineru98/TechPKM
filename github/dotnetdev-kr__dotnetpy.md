DotNetPy는 .NET 애플리케이션에서 Python 코드를 직접 실행할 수 있게 해주는 라이브러리입니다. C# 코드 내에서 Python 로직을 인라인으로 정의하고 실행할 수 있으며, 최소한의 설정으로 Python 스크립트 실행 및 데이터 흐름 제어가 가능합니다. 파일 기반 앱(.NET 10+) 및 Native AOT를 지원하며, uv를 통한 Python 환경 관리가 특징입니다. 실험 단계 프로젝트로, 프로덕션 사용 전 충분한 테스트가 필요합니다. Python.NET과 비교해 더 가벼운 의존성과 현대적인 .NET 시나리오를 지원하지만, 안정성 면에서 Python.NET을 권장합니다.
