DotNetPy는 C# 애플리케이션에서 Python 코드를 직접 실행할 수 있게 하는 .NET 라이브러리입니다. 실험 단계에 있는 이 프로젝트는 최소한의 설정으로 Python 스크립트와 표현식을 실행할 수 있는 직관적인 API를 제공하며, 파일 기반 앱(.NET 10+) 및 Native AOT를 지원합니다. uv를 통한 선언적 환경 관리, 데이터 마샬링, 변수 관리 등의 기능을 포함하며, 보안상의 고려사항이 명시되어 있습니다.  

> ⚠️ 실험적 단계이며, 프로덕션 환경에서는 [Python.NET](https://github.com/pythonnet/pythonnet) 사용을 권장합니다.
