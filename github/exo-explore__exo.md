exo는 일상적인 장치들을 연결하여 집에서 AI 클러스터를 구축할 수 있는 오픈소스 프로젝트입니다. 여러 장치를 통합해 단일 장치에 비해 더 큰 모델을 실행할 수 있으며, RDMA over Thunderbolt 5 기술로 지연 시간을 최대 99% 감소시킵니다. macOS를 중심으로 GPU 가속을 지원하며, 자동 장치 발견, 토폴로지 인식 자동 병렬 처리, 텐서 분할 등의 기능을 제공합니다. MLX를 백엔드로 사용하여 분산 추론을 수행합니다. Linux는 현재 CPU 전용 실행 가능하지만 GPU 지원이 개발 중입니다. 사용자 친화적인 대시보드와 OpenAI 호환 API를 제공합니다.
