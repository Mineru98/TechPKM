ShortGPT는 AI 기반 동영상 및 숏폼 콘텐츠 자동화를 위한 오픈소스 프레임워크입니다. 스크립트 생성부터 영상 편집, 음성 합성, 자막 생성, 외부 자산 수집까지 전 과정을 자동화하며, YouTube 및 TikTok 콘텐츠 제작에 최적화되어 있습니다. Python 기반으로 MoviePy, OpenAI, ElevenLabs 등 다양한 기술을 통합하여 다국어 지원 및 커스터마이징 기능을 제공합니다.  

> 핵심 기능: LLM 기반 편집 언어 지원, 다국어 영상 번역/더빙, Web API 연동 영상 수집, Docker/Colab 배포 지원
