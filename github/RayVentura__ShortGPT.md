ShortGPT는 AI를 활용한 동영상 및 숏폼 콘텐츠 자동화를 위한 프레임워크입니다. 영상 제작, 영상 소스 수집, 음성 합성, 편집 작업을 간소화하며, 유튜브 및 TikTok 자동화 등 다양한 활용 사례를 제공합니다. LLM 기반 편집 언어와 통합 문서 기능을 지원하며, 다중 언어 음성 더빙 및 캡션 생성이 가능합니다. Docker 기반 로컬 실행과 Google Colab을 통한 간편한 사용 옵션을 제공합니다.  

### 핵심 기능  
- **자동화 편집 프레임워크**: LLM 지향 비디오 편집 언어 지원  
- **멀티 언어 음성 합성**: 30개 이상 언어 지원 (ElevenLabs, EdgeTTS)  
- **영상 소스 수집**: Pexels API 및 Bing Image 연동을 통한 자동 자산 확보  
- **지속성 관리**: TinyDB를 활용한 편집 변수 장기 저장  
- **모듈식 아키텍처**: 쇼트폼/장편 영상/번역 엔진 등 사용 사례에 따른 유연한 구성  

### 기술 스택  
- **Python** 기반의 MoviePy, OpenAI, ElevenLabs, Microsoft EdgeTTS, Pexels API 통합  
- Gradio를 활용한 웹 인터페이스 제공
