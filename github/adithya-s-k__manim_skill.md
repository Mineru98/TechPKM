이 프로젝트는 **Manim Community Edition**과 **ManimGL**을 활용한 수학 애니메이션 제작의 모범 사례, 패턴, 예제를 종합적으로 제공하는 레포지토리입니다. 두 프레임워크의 차이점, 설치 방법, 테스트 방법, AI 에이전트와의 통합 가이드를 포함하며, 교육 콘텐츠 제작자나 개발자 대상으로 실용적인 코드 예제와 문제 해결 방법을 제공합니다.  

Manim Community Edition은 안정성과 문서화 우수성으로 프로덕션 환경에 적합하며, ManimGL은 OpenGL 기반의 3D 렌더링과 인터랙티브 개발에 특화되어 있습니다. 각 버전에 대한 CLI 명령어, 임포트 방식, 주요 기능 차이를 명확히 비교하여 호환성 문제를 방지합니다.  

AI 에이전트 스킬(skills.sh)과의 통합을 통해 ManimCE 또는 ManimGL 작업 시 자동으로 베스트 프랙티스가 적용되도록 지원합니다. 예를 들어 `from manim import *` 또는 `manim` CLI 사용 시 관련 스킬이 활성화됩니다.  

테스트 스위트와 기여 가이드라인을 포함하여 코드 예제의 신뢰성을 보장하며, LaTeX, FFmpeg 등 필수 사전 요구 사항 설치 방법도 상세히 안내합니다.
