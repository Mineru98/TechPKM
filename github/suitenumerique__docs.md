Docs는 실시간 협업을 통해 개인 노트를 지식으로 전환할 수 있는 오픈소스 문서 편집기입니다. 프랑스 및 독일 정부 주도로 개발된 이 프로젝트는 팀 기반 문서 작성, 오프라인 편집, AI 기반 작업(재구성, 요약 등), 다양한 형식(ODT, DOCX, PDF)으로의 내보내기 기능을 제공하며, Kubernetes, Docker Compose 등을 통해 자체 호스팅이 가능합니다. 현재 MIT 라이선스로 배포되며, 공공 및 민간 분야에서 활용되도록 설계되었습니다. 주요 기술 스택은 Django Rest Framework, Next.js, BlockNote.js 등으로 구성되어 있습니다.
