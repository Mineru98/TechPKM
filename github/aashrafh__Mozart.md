이 프로젝트는 악보를 기계가 읽을 수 있는 형식으로 변환하는 광학 음악 인식(OMR) 시스템을 개발합니다. 이미지 기반 악보를 텍스트 표현으로 변환해 MIDI 파일이나 오디오 파일 생성이 가능하도록 하는 것이 목적이며, 노이즈 필터링, 분할, 악보선 감지/제거, 심볼 감지/인식 등 단계별 처리 방식을 사용합니다. Python 기반의 OpenCV, NumPy, scikit-learn 등의 기술을 활용해 구현되었습니다.
