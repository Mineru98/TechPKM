**KoSpeech**는 PyTorch 기반의 오픈소스 한국어 엔드투엔드 자동 음성 인식(ASR) 툴킷입니다. Deep Speech 2, LAS, Transformer, Conformer 등 다양한 모델을 지원하며, KsponSpeech 코퍼스에 대한 전처리 방법과 모델 구현을 제공합니다. 연구자와 개발자가 한국어 음성 인식 모델을 학습, 평가, 추론할 수 있는 모듈식 구조를 갖추고 있으며, Hydra 프레임워크를 활용한 구성 관리 시스템을 포함합니다. Apache 2.0 라이선스로 배포됩니다.
