이 프로젝트는 인스타그램 스레드(Threads) 데이터를 분석하기 위한 오픈소스 파이썬 API입니다. 스레드 게시물/계정 데이터를 자동으로 수집하고 분석하는 기능을 제공하며, 마케팅 및 소셜 미디어 연구 목적으로 설계되었습니다. `python-threads-api` 패키지를 통해 간단한 코드로 스레드 플랫폼 데이터 추출이 가능합니다.  

> 요약: 인스타그램 스레드 데이터 수집·분석을 위한 파이썬 API 도구 (2023년 10월 릴리스)
