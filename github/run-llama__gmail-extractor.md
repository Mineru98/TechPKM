이 프로젝트는 LLM을 활용해 사용자의 GMail 계정에서 항공편 일정을 자동으로 추출하는 시스템을 구축하기 위한 코드 저장소입니다. GMail API를 통해 이메일을 검색하고, LLM을 사용해 일정 정보를 분류하거나 Python 코드 생성을 통해 효율적인 필터링 로직을 개발하는 두 가지 접근법을 실험했습니다. 주요 목적은 대량의 이메일 데이터에서 여행 목적지를 자동으로 식별하는 것입니다. 현재 스팸 제외 최적화, 로컬 모델 적용, 코드 결합 기능 개선 등의 발전 계획이 있습니다.
