본 프로그램은 오픈톡방에서 공유되는 개발 관련 뉴스 및 논문을 체계적으로 아카이빙하고 공유하기 위한 주간 프로그램입니다. 참여자들이 GitHub 이슈 쓰레드에 자료를 제출하면, 주말에 간단한 설명과 토론을 진행하는 방식으로 운영됩니다. 정보를 효과적으로 보관하고 참여 중심의 활발한 지식 교류를 목적으로 합니다. 추가적으로 GitHub 이슈 내용을 마크다운 형식으로 추출하는 크롬 확장 프로그램도 제공합니다.
