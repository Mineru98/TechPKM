본 프로그램은 오픈톡방에서 공유되는 개발 관련 뉴스 및 논문 정보를 GitHub Issue를 통해 체계적으로 아카이빙하고, 매주 일요일 진행되는 모임에서 간단한 설명과 토론을 통해 지식을 공유하는 것을 목표로 합니다. 참여자는 뉴스/논문을 요약해 제출하고, 모더레이터와 함께 주제별 토론을 진행하며, 크롬 익스텐션을 활용해 GitHub 이슈 내용을 마크다운 형식으로 쉽게 추출할 수 있습니다. 정보 보관과 참여 중심의 지식 공유 문화를 지향하는 간단하고 실용적인 프로그램입니다.
