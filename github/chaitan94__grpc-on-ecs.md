이 프로젝트는 AWS ECS에서 EC2 런치 타입을 사용하여 gRPC 서비스를 실행하는 샘플 코드를 제공합니다. 블로그 포스트에 설명된 대로 gRPC 서비스를 ECS 환경에 배포하는 방법을 보여줍니다. 주요 목적은 EC2 기반 컨테이너 오케스트레이션에서 gRPC 통신 구현을 시연하는 것입니다. Python 기반 구현이며 클라우드 네이티브 서비스 배포에 적합합니다.
