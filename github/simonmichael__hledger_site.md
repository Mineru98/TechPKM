이 프로젝트는 hledger.org 웹사이트의 소스 파일을 관리하는 리포지토리입니다. hledger 관련 기술 문서, 매뉴얼, 개발자 문서를 포함하며, mdbook을 사용해 정적 웹사이트로 빌드됩니다. 메인 리포지토리의 개발 문서 및 사용자 매뉴얼을 심볼릭 링크로 통합해 웹사이트 콘텐츠를 구성합니다. Debian/Ubuntu 환경에서 빌드 도구 설치와 빌드 프로세스를 지원하는 Makefile이 제공됩니다.
