GraphRAG는 비정형 텍스트에서 의미 있는 구조화된 데이터를 추출하기 위한 데이터 파이프라인 및 변환 스위트입니다. LLM(대형 언어 모델)이 개인 데이터에 대해 추론 능력을 향상시킬 수 있도록 지식 그래프 메모리 구조를 활용하는 방법론을 제시합니다. 주요 목적은 비공개 데이터 분석 및 발견을 용이하게 하는 것이며, 프롬프트 튜닝과 버전 관리 가이드 등을 제공합니다. 프로젝트 코드는 데모용으로 제공되며 공식 지원 제품은 아닙니다.  

⚠️ *주의: 인덱싱은 비용이 많이 드는 작업이므로 문서를 충분히 읽고 소규모로 시작하는 것이 좋습니다.*
