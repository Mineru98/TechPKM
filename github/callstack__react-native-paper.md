React Native Paper는 Google의 Material Design 가이드라인을 준수하는 크로스플랫폼 UI 키트로, 커스터마이징 가능한 프로덕션 레디 컴포넌트를 제공합니다. iOS와 Android 모두에서 플랫폼 적응 가이드라인을 따르며, 완전한 테밍 지원 기능을 포함하고 있습니다. Expo로 실행되는 예제 앱과 문서 사이트에서 컴포넌트 사용법을 확인할 수 있습니다.
