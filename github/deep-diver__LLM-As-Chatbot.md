이 프로젝트는 다양한 오픈 소스 LLM(대형 언어 모델)을 채팅봇 서비스로 활용하기 위한 프레임워크입니다. Gradio UI와 Discord 봇을 통해 Hugging Face 허브를 포함한 다양한 모델을 대화형으로 사용할 수 있으며, 인터넷 검색 기능이 통합되어 실시간 정보 수집이 가능합니다. 모델별 프롬프트 형식과 컨텍스트 관리를 지원하는 `Ping Pong` 라이브러리와 `GradioChat` UI를 기반으로 구축되었습니다.  

Jarvislabs.ai 및 dstack과 연동되어 클라우드 GPU 환경에서 쉽게 배포할 수 있으며, 로컬/원격 모델 다운로드, 다양한 실행 모드, 동시 작업 처리 등의 기능을 제공합니다. 현재 40개 이상의 오픈 소스 모델을 지원하며, Falcon, Vicuna, LLaMA, StableLM 등 주요 기반 모델을 포함합니다.
