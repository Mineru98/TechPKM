**Supertonic**은 ONNX 런타임을 기반으로 하는 극저지연 온디바이스 텍스트-음성 변환 시스템입니다. 클라우드 의존 없이 개인 기기(스마트폰, 라즈베리 파이 등)에서 초당 12,164자 처리 속도로 실시간 음성을 생성하며, 66M 파라미터의 초경량 아키텍처를 갖춘 것이 특징입니다. 금융/기술 용어, 날짜, 전화번호 등 복잡한 텍스트 표현을 사전 처리 없이 자연스럽게 처리하며, Python, Java, C++, Swift 등 다양한 언어 SDK를 지원합니다. 현재 영어, 한국어, 스페인어, 포르투갈어, 프랑스어를 지원하며, Hugging Face에서 사전 학습 모델과 데모 환경을 제공합니다. MIT 라이선스 기반의 오픈소스 프로젝트로, 개인 정보 보호가 필요한 모든 온디바이스 애플리케이션에 적합합니다.
