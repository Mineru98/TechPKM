Perplexica는 개인 정보 보호에 중점을 둔 AI 답변 엔진으로, 사용자 하드웨어에서 완전히 실행됩니다. 인터넷에서 수집한 지식과 Ollama 기반 로컬 LLM, OpenAI/Claude/Groq 등의 클라우드 제공자를 결합하여 정확한 답변을 제공하며, 검색 기록을 철저히 비공개로 유지합니다. Docker를 통한 간편한 설치가 가능하며, 웹 검색, 파일 업로드 분석, 위젯 기능 등 다양한 검색 모드를 지원합니다. MIT 라이선스로 배포되는 오픈소스 프로젝트입니다.
