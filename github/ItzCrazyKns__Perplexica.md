Perplexica는 사용자 하드웨어에서 완전히 실행되는 **프라이버시 중심 AI 답변 엔진**입니다. 로컬 LLM(Ollama) 및 클라우드 제공자(OpenAI, Claude, Groq 등)와 연동해 정확한 답변을 인용 소스와 함께 제공하며, 검색 활동을 완전히 비공개로 유지합니다. 웹 검색, 문서 분석, 이미지/동영상 검색, 파일 업로드 등 다양한 기능을 지원하며, 사용자가 소스를 직접 선택하고 검색 모드를 조절할 수 있습니다. Docker를 통한 간편한 설치와 SearxNG 기반 검색 엔진 통합이 특징입니다.
