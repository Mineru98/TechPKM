도다리는 로컬 환경에서 AI 기반 한영/영한 번역을 제공하는 Gradio 기반 웹서비스입니다. 일반인도 쉽게 사용할 수 있도록 설계되었으며, txt, epub, srt 파일 형식을 지원하고 번역문과 원문 대조 기능을 제공합니다. NHNDQ 모델과 EEVE-Korean-Instruct 모델을 활용한 일반/고급 번역 옵션을 선택할 수 있으며, 파일 드래그 앤 드롭 방식으로 간편한 번역 실행이 가능합니다. 번역 속도와 품질 면에서 구글 번역 대비 우수하며, 오프라인 환경에서도 제한 없이 사용 가능한 것이 특징입니다.
