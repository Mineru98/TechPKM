도다리는 로컬 환경에서 작동하는 AI 기반 한영/영한 번역 서비스입니다. 일반인도 쉽게 사용할 수 있도록 설계되었으며, 텍스트(TXT), 전자책(EPUB), 자막(SRT) 파일 형식을 지원합니다. 번역 결과물은 원본과 번역문을 함께 제공하는 "번역문(원문)" 파일과 순수 번역문 파일로 출력되어 비교 분석이 용이합니다. Gradio 기반 웹 인터페이스로 제공되며, 드래그 앤 드롭 방식의 간편한 사용법을 특징으로 합니다. 고성능 GPU 환경에서 초고속 번역이 가능하며, 일반 번역과 고급 번역(EEVE 모델) 두 가지 모드를 지원합니다.
