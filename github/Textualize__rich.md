Rich는 터미널에서 풍부한 텍스트와 아름다운 포맷팅을 제공하는 파이썬 라이브러리입니다. Rich를 사용하면 색상, 스타일, 표, 진행률 표시줄, 마크다운, 구문 강조 소스 코드, 추적백 등을 간단히 렌더링할 수 있습니다. 리눅스와 macOS, 윈도우를 지원하며 Python 3.8 이상에서 작동합니다. Jupyter 노트북과도 호환되며, 터미널 출력에 생동감을 더하는 다양한 기능을 제공합니다.
