Promptfoo는 LLM 애플리케이션 테스트를 위한 개발자 친화적인 로컬 도구입니다. 프롬프트와 모델의 신뢰성 및 보안 검증을 위해 자동화된 평가와 레드 팀링 기능을 제공하며, 다양한 모델 간의 비교 테스트와 CI/CD 통합이 가능합니다. npm 패키지로 제공되며, 개발자 중심의 빠른 피드백, 로컬 실행, 오픈 소스 생태계가 핵심 특징입니다. 프롬프트 보안 취약점 검출과 팀 협업을 위한 결과 공유 기능을 포함합니다.
