이 프로젝트는 $100 예산으로 단일 GPU 노드(8XH100)에서 실행 가능한 미니 챗GPT 구현체를 제공합니다. 토크나이저, 사전 학습, 파인튜닝, 평가, 추론, 웹 기반 인터페이스 등 전체 LLM 파이프라인을 포함하며, 해킹 가능한 경량 코드베이스로 설계되었습니다. 사용자는 전체 학습 과정을 직접 실행하거나 사전 학습된 모델을 웹 UI에서 테스트할 수 있습니다.  

주요 특징은 빠른 학습 스크립트(speedrun.sh), 모델 성능 평가 리포트, 다양한 커스터마이징 옵션(성격 주입, 능력 추가) 등이며, GPT-2 수준의 성능을 목표로 합니다. 학습 비용은 약 $100~$1,000 범위로, 고사양 LLM 대비 접근성을 강조했습니다.
