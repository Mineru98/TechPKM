nanochat은 단일 8XH100 노드에서 실행할 수 있는 경량 LLM 구현체로, 토크나이저부터 사전학습/미세조정/추론/웹 서빙까지 전체 파이프라인을 포함합니다. $100 예산으로 4시간 내에 학습 가능한 초소형 ChatGPT 클론을 목표로 하며, 34층 트랜스포머 기반의 22억 파라미터 모델을 제공합니다. 사용자는 직접 학습시킨 모델을 웹 UI나 CLI로 테스트할 수 있으며, 커스터마이징을 통해 모델 성격과 기능을 개조할 수 있습니다. 교육 및 연구 목적의 접근성 높은 오픈소스 프로젝트입니다.
