이 프로젝트는 LangChain 워크플로우를 PlantUML 형식의 활동 다이어그램과 시퀀스 다이어그램으로 시각화하는 도구입니다. LangChain 이벤트 콜백을 구독하여 자동으로 UML 다이어그램을 생성하며, 복잡한 LLM 애플리케이션 흐름을 쉽게 분석하고 문서화할 수 있도록 지원합니다. Python 패키지로 제공되며, PyPI를 통해 간편하게 설치할 수 있습니다.  

핵심 기능으로는 활동 다이어그램 생성, 시퀀스 다이어그램 생성, 다이어그램 노트 길이/폭 커스터마이징, 그리고 PlantUML 파일 저장 기능이 포함됩니다. 생성된 UML 파일은 PlantUML 엔진을 사용해 PNG 이미지 형태로 변환할 수 있습니다.
