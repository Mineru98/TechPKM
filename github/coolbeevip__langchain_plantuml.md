이 프로젝트는 LangChain 워크플로우를 PlantUML 형식의 활동 다이어그램(Activity Diagram) 및 시퀀스 다이어그램(Sequence Diagram)으로 시각화하는 도구입니다. LangChain 애플리케이션의 이벤트를 구독하여 분석 및 문서화가 용이한 UML 형식으로 변환하며, 복잡한 LLM 워크플로우를 그래픽으로 표현할 수 있습니다. 주요 기능은 PlantUML 콘텐츠 생성, PNG 파일 내보내기, 사용자 정의 노트 길이/폭 설정입니다. Python 환경에서 LangChain과의 통합을 지원합니다.
