OpenELM은 CarperAI에서 개발한 오픈소스 라이브러리로, 코드 및 자연어 영역에서 언어 모델을 활용한 진화 검색을 가능하게 합니다. 주요 기능은 품질-다양성 알고리즘(MAP-Elites 등)과 언어 모델 기반 변이/재조합 연산자 지원이며, 로컬/클라우드 기반 오픈소스 모델 및 OpenAI 등 상용 API와 통합됩니다. 소다시뮬레이션, 이미지 생성, 프로그래밍 퍼즐, 프롬프트 최적화 등 다양한 실험 환경을 제공하며, 안전한 코드 생성을 위한 샌드박싱 시스템도 포함됩니다.  

(요약: 4문장 구성, 프로젝트 목적/주요 기능/기술 통합/사용 사례 강조)
