이 프로젝트는 벤치마크 웹사이트에서 CPU, GPU, 저장장치, RAM의 최신 성능 점수와 순위를 자동으로 추출하여 CSV/XLSX/XLS 형식으로 저장하는 도구입니다. 사용자가 지정한 하드웨어 항목에 따라 데이터를 필터링할 수 있으며, 다양한 파일 형식으로 출력 가능합니다. 웹 크롤링과 데이터 변환 기능을 결합한 자동화 도구로, 개발자 및 시스템 성능 분석가에게 유용합니다. Python 기반으로 구현되었으며 PyInstaller로 패키징 가능합니다.
