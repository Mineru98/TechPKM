Convoviz는 ChatGPT 대화 기록을 구조화된 마크다운 파일과 데이터 시각화(워드 클라우드, 그래프 등)로 변환하는 도구입니다. Obsidian과 같은 노트 앱에서 활용하거나, 대화 내용을 체계적으로 분석/보관할 수 있도록 설계되었습니다. YAML 헤더, 인라인 이미지 지원, 선택적 출력 기능 등을 제공하며, Python 기반으로 간단한 명령어 입력으로 실행 가능합니다.  

* 프로젝트 요약: ChatGPT 대화 기록의 마크다운 변환 및 시각화 도구  
* 핵심 기능: 마크다운 생성, 워드 클라우드, 대화 사용 통계 그래프  
* 사용 목적: 대화 데이터 분석, 백업, 지식 관리 시스템 통합(obsidian 등)
