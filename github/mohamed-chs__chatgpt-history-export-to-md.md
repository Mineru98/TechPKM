Convoviz는 ChatGPT 대화 기록을 구조화된 Markdown 파일로 변환하고, 워드 클라우드 및 사용 통계 그래프를 생성하는 도구입니다. OpenAI에서 내보낸 ZIP 파일을 분석해 읽기 쉬운 문서로 변환하며, Obsidian 등 Markdown 기반 노트 앱과 호환됩니다. YAML 헤더, 인라인 이미지 지원, 선택적 출력 생성 기능을 제공합니다. Python으로 제작되었으며, 대화형 명령줄 인터페이스와 시각화 기능을 포함합니다.
