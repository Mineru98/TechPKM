네이버 영화 리뷰를 기반으로 한 한국어 감성 분석 데이터셋입니다. 20만 개의 리뷰(긍정 10만 개, 부정 10만 개)로 구성되며, 140자 이내의 짧은 텍스트로 되어 있어 자연어 처리 및 머신러닝 모델 평가에 적합합니다. 훈련용(15만 개)과 테스트용(5만 개) 데이터가 분리되어 제공됩니다.
