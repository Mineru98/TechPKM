이 프로젝트는 Python에서 Zstandard 압축 라이브러리를 사용할 수 있도록 C 확장 및 CFFI 인터페이스를 제공합니다. 주요 목표는 Python의 사용성과 안전성을 유지하면서도 C API의 모든 기능과 유연성을 노출하는 것입니다. 성능 저하 없이 Pythonic 인터페이스를 구현하는 데 중점을 두고 있습니다. 프로젝트 문서는 ReadTheDocs에서 제공됩니다.
