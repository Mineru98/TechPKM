이 프로젝트는 ChatKit을 활용한 시나리오 기반 데모 예제들을 수집합니다. 각 예제는 FastAPI 백엔드와 Vite + React 프론트엔드를 결합하여 ChatKit Python SDK로 커스텀 백엔드를 구현하고 ChatKit.js 클라이언트 측과 연동합니다. 가상 고양이 관리, 항공사 고객 지원, 뉴스룸 보조, 지하철 노선 계획 등 다양한 사용 사례를 포함하며, 위젯 기능, 데이터 연동, 실시간 상태 업데이트 등의 고급 기능을 시연합니다.
