이 프로젝트는 OpenAI ChatKit SDK를 활용한 다양한 시나리오 기반 고급 샘플 애플리케이션을 제공합니다. 각 예제는 FastAPI 백엔드와 Vite + React 프론트엔드로 구성되며, AI 기반 상호작용 인터페이스 구현에 필요한 핵심 기술(툴 호출, 위젯, 실시간 상태 동기화 등)을 보여줍니다. 주요 시나리오로는 가상 고양이 관리, 항공사 고객 지원, 뉴스룸 보조, 지하철 경로 계획 등이 포함됩니다. ChatKit의 확장성을 입증하는 기술 데모 리포지토리입니다.
