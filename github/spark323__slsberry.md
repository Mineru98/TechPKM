slsberry는 AWS Lambda 기반 Serverless 개발을 효율적으로 지원하는 프레임워크입니다. 소스 코드 내 apiSpec 정의를 기반으로 Serverless Framework yml 파일과 OpenAPI 3.0, Notion 문서를 자동 생성합니다. 템플릿 기반 구성, 다양한 이벤트 소스 지원, 환경별 배포 관리, TypeScript 완벽 지원 등을 특징으로 합니다. Serverless 애플리케이션의 개발-배포-문서화 워크플로우를 간소화합니다.
