이 저장소는 O'Reilly 출판사의 책 *Practical MLOps*의 코드 샘플을 제공하는 공식 리포지토리입니다. 머신러닝 모델의 운영화(MLOps) 전 과정을 다루며, 클라우드 기반 배포, 모니터링, 자동화 등 프로덕션 환경에서의 ML 시스템 구축 방법을 실습 예제와 함께 설명합니다. AWS, GCP 등 주요 클라우드 플랫폼과 컨테이너 기술, AutoML, 로깅 시스템 등 현대적인 MLOps 패턴을 포함합니다.  

핵심 주제는 다음과 같습니다:  
1. MLOps 기초와 다중 클라우드 배포 전략  
2. 모델 모니터링 및 로깅 시스템 구현  
3. AWS/GCP 기반 ML 인프라 운영  
4. AutoML 도구와 CLI 개발 기법  
5. 실제 산업 사례 연구  

코드는 Python 기반이며, Docker, Kubernetes, GitHub Actions 등 DevOps 도구와의 통합 사례를 포함합니다.
