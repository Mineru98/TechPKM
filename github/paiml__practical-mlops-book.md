이 프로젝트는 O'Reilly 출판사의 책 *Practical MLOps*의 코드 샘플 저장소입니다. 머신러닝 모델을 프로덕션 환경에 배포하고 운영하는 기술(MLOps)을 다루며, AWS/Azure/GCP 클라우드 환경에서의 구현, 모니터링, 자동 배포, CLI 도구 개발 등 실용적인 사례를 제공합니다. 각 장별로 분리된 코드 예제를 통해 MLOps의 전 과정을 학습할 수 있습니다.  

핵심 주제:  
1. MLOps 기본 개념 및 다중 클라우드 배포  
2. 로깅, 모니터링 시스템 구축  
3. AutoML 및 모델 설명 가능성  
4. 컨테이너/서버리스 기반 배포  
5. 실제 사례 연구 및 커뮤니티 레시피
