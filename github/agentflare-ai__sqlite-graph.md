sqlite-graph는 SQLite에 그래프 데이터베이스 기능을 추가하는 확장 라이브러리로, Cypher 쿼리 언어 지원을 통해 노드와 엣지의 저장 및 분석이 가능합니다. 기존 SQLite 기반 애플리케이션에 그래프 기능을 통합하거나 프로토타이핑, 임베디드 환경에 적합하며, 별도의 의존성을 가지지 않는 순수 C99로 구현되었습니다. 현재는 알파 버전으로 테스트 중이며, 향후 버전에서는 복잡한 쿼리 및 집계 기능 확장이 예정되어 있습니다.
