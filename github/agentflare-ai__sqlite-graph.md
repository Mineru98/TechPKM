sqlite-graph는 SQLite를 확장하여 그래프 데이터베이스 기능을 추가하는 오픈소스 프로젝트입니다. Cypher 쿼리 언어 지원을 통해 노드와 관계 기반 데이터를 저장하고 조회할 수 있으며, 그래프 알고리즘(연결성 검증, 중심성 분석 등)을 적용할 수 있습니다. 기존 SQLite 애플리케이션에 그래프 기능을 추가하거나 프로토타입 개발에 적합하며, 순수 C99로 구현되어 외부 의존성이 없습니다. 리눅스, macOS, WSL 환경에서 사용 가능하며, 현재 알파 버전(0.1.0)으로 개발 중입니다.
