"모두를 위한 메타러닝: PyTorch를 활용한 Few-shot 학습 모델과 빠른 강화학습 에이전트 만들기" 책의 코드 레포지토리입니다. 메타러닝(Meta-SL)과 메타강화학습(Meta-RL) 알고리즘을 구현하고 실험할 수 있는 PyTorch 기반 프로젝트를 제공합니다. Python 3.8.8 환경에서 Anaconda를 통해 개발 환경을 구성하며, Torchmeta와 텐서보드를 활용한 학습 및 결과 분석이 가능합니다.  

이 레포지토리는 Few-shot 학습 모델과 강화학습 에이전트 구현을 위한 실용적인 튜토리얼과 코드 예제를 포함합니다. Meta-SL(MAML, Prototypical Networks 등)과 Meta-RL(RL², PEARL 등) 알고리즘을 Jupyter Notebook 또는 CLI로 실행할 수 있도록 구성되어 있습니다.
