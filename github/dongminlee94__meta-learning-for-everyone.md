"모두를 위한 메타러닝: PyTorch를 활용한 Few-shot 학습 모델과 빠른 강화학습 에이전트 만들기" 책의 공식 코드 저장소입니다. PyTorch 기반으로 Few-shot 학습과 메타 강화학습(Meta-RL) 알고리즘을 구현하는 튜토리얼 및 실습 환경을 제공합니다. Meta-SL(메타-감독학습)과 Meta-RL(메타-강화학습) 두 가지 주요 접근법을 포함하며, Jupyter Notebook과 CLI를 통해 실습할 수 있도록 구성되었습니다. Python 3.8.8, PyTorch 1.9.1 환경에서 개발되었으며 코드 품질 관리(black, isort, flake8 등)를 위한 도구들이 적용된 오픈소스 프로젝트입니다. Apache 2.0 라이선스로 배포됩니다.
