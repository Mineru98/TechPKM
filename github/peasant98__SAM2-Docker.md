이 프로젝트는 Meta의 Segment Anything 2(SAM2) 모델을 Docker 컨테이너로 패키징한 것입니다. NVIDIA GPU 환경에서 CUDA 12.1/12.6 기반으로 실행되며, 이미지 분할 마스크 생성 및 GUI 기반 시각화를 지원합니다. ROS Noetic 버전이 별도로 제공되며, `image_predictor.py` 예제를 통해 모델 기능을 테스트할 수 있습니다.  

태그: Docker 이미지 분할 기술, GUI 통합, CUDA 최적화, SAM2 모델 도커라이징, ROS 호환성.
