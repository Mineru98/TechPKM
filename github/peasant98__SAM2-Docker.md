이 프로젝트는 메타의 Segment Anything 2 모델을 도커 환경에서 실행할 수 있도록 구현한 것입니다. NVIDIA GPU 기반 시스템에서 GUI 지원 및 이미지/마스크 표시 기능을 제공하며, ROS Noetic 통합 버전도 별도로 제공됩니다. CUDA 12.1 및 12.6 호환 이미지를 통해 다양한 하드웨어 환경에서 컴퓨터 비전 작업을 수행할 수 있습니다.  

요약: Segment Anything 2 모델을 도커 컨테이너로 패키징하여 GPU 환경에서 쉽게 실행할 수 있도록 지원하는 프로젝트입니다. 이미지 세그먼테이션 작업을 위한 GUI 인터페이스와 ROS 통합 기능을 포함하며, CUDA 버전별 호환성을 고려한 이미지를 제공합니다.
