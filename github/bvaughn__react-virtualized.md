React-virtualized는 대량의 목록 및 표 형식 데이터를 효율적으로 렌더링하기 위한 React 컴포넌트 라이브러리입니다. 성능 최적화를 위해 보이는 영역만 렌더링하는 가상화 기술을 적용하며, 테이블, 리스트, 스크롤 동기화 등 다양한 UI 요소를 지원합니다. 사용자 정의 스타일링, 동적 크기 조정, 무한 로딩 등 고급 기능을 제공하여 대규모 데이터 처리에 적합합니다.  

*참고: 더 가벼운 대안으로는 `react-window`을 고려할 수 있습니다.*
