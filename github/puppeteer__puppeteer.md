Puppeteer는 Chrome 또는 Firefox 브라우저를 제어할 수 있는 JavaScript 라이브러리입니다. DevTools Protocol 또는 WebDriver BiDi를 사용하여 헤드리스(UI 없음) 환경에서 웹 페이지를 자동화, 테스팅, 스크래핑하는 데 주로 활용됩니다. 기본 설치 시 호환되는 Chrome 버전이 자동 다운로드되며, 코어 라이브러리로도 설치 가능합니다. 브라우저 자동화 및 디버깅 서버와 통합하여 사용할 수 있습니다.
