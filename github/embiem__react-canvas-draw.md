React 기반 캔버스 드로잉 컴포넌트로, 간단한 사용법과 강력한 커스터마이징 옵션을 제공합니다. 실시간 드로잉, 저장/불러오기, 지우기, 되돌리기 등의 기능을 지원하며, 그리드 표시 및 확대/축소 기능을 포함한 다양한 사용자 정의 프로퍼티를 제공합니다. 주로 디지털 드로잉, 서명 수집, 간단한 그래픽 편집 도구 구현에 활용됩니다.  

> 💡 **주요 기능**: 실시간 드로잉, LZ 압축 저장, 데이터 URL 내보내기, 확대/축소, 그리드 시스템, 다중 undo 지원  
> 📦 **적용 분야**: 리액트 애플리케이션 내 그래픽 인터페이스, 디지털 서명, 시각적 메모 도구 등
