이 프로젝트는 언어 에이전트가 자기 반성과 강화 학습을 통해 성능을 개선하는 "Reflexion" 방법론을 구현한 저장소입니다. NeurIPS 2023에 발표된 논문의 공식 코드로, HotPotQA 추론 태스크와 AlfWorld 의사결정 환경에서 실험을 수행할 수 있습니다. 다양한 반사 전략과 에이전트 유형(ReAct, CoT 등)을 지원하며, GPT-4 기반 실험을 위한 로그와 데모를 제공합니다.  

주요 특징으로는 반복 학습을 통한 언어 모델의 성능 향상, 실험 재현을 위한 노트북 및 스크립트, OpenAI API 통합이 포함됩니다. 프로그래밍/추론/의사결정 태스크별 로그 데이터를 별도로 관리하며, API 비용 문제로 인해 사전 실행된 결과 로그를 제공합니다.
