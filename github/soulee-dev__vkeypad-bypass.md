금융 및 정부 웹사이트에서 사용하는 가상키보드(vKeypad)의 보안을 우회하기 위한 Python 라이브러리입니다. 키 배열 무작위 변경 상황에서도 이미지 유사도 분석을 통해 키 입력을 인식하며, Figma 기반 레이아웃 추출 도구(`vkeypad-studio`)와 연동해 사용 편의성을 높였습니다. 셀레늄(Selenium) 등과 연동해 웹 자동화 구현에 활용할 수 있습니다.
