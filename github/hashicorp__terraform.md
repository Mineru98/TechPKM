Terraform은 인프라를 안전하고 효율적으로 구축, 변경 및 버전 관리하는 도구입니다. 코드 기반의 인프라 정의, 실행 계획 생성, 리소스 간 의존성 그래프 관리 등의 기능을 제공하며, 클라우드 서비스 및 맞춤형 솔루션 통합이 가능합니다. HashiCorp에서 개발했으며, 오픈소스 라이선스로 배포됩니다.

### 핵심 기능
- **인프라스트럭처 as 코드**: 고수준 구성 문법으로 인프라 정의 및 버전 관리
- **실행 계획**: 적용 전 변경 사항 사전 확인 가능
- **리소스 그래프**: 의존성 분석을 통한 효율적 인프라 생성
- **변경 자동화**: 복잡한 인프라 변경을 최소한의 사용자 개입으로 수행

※ 언어 표기: Terraform의 주 구성 언어는 HCL(HashiCorp Configuration Language)이며, 코어 구현에는 Go 언어 사용됨. 프로젝트 특성상 HCL을 주요 언어로 표기함.
