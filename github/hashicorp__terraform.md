테라폼은 안전하고 효율적으로 인프라를 구축, 변경, 버전 관리하는 도구입니다. 기존 서비스 제공업체와 사용자 정의 솔루션을 관리할 수 있으며, HCL(Hashicorp Configuration Language) 기반의 인프라 코드화, 실행 계획, 리소스 그래프, 변경 자동화 등의 핵심 기능을 제공합니다. 오픈소스 프로젝트로, 플러그인 방식의 프로바이더를 통해 확장 가능한 클라우드 프로비저닝 솔루션을 구현합니다.
