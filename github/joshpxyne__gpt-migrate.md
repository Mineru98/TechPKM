GPT-Migrate는 대규모 언어 모델(LLM)을 활용해 코드베이스를 한 프레임워크/언어에서 다른 프레임워크/언어로 자동 마이그레이션하는 오픈소스 프로젝트입니다. 기존 코드를 분석해 타겟 언어 의존성 파악, 코드 재구성, 유닛 테스트 생성 및 디버깅까지 전 과정을 자동화합니다. 현재는 파이썬과 자바스크립트 간 변환이 가장 안정적이며, 도커 환경과 GPT-4를 기반으로 동작합니다. MIT 라이선스로 배포되며, 복잡한 마이그레이션 작업을 지원하는 실험적 단계의 도구입니다.
