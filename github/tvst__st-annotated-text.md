이 프로젝트는 Streamlit 애플리케이션에서 어노테이션(주석)이 표시된 텍스트를 렌더링하기 위한 컴포넌트 라이브러리입니다. 사용자가 텍스트에서 특정 단어나 구문에 스타일 및 메타데이터를 적용할 수 있도록 지원하며, 다양한 사용자 정의 옵션(색상, 배경색 등)을 제공합니다. 자연어 처리(NLP) 결과 시각화 또는 텍스트 기반 분석에 적합합니다. Streamlit 생태계에 통합되어 쉽게 사용할 수 있습니다.
