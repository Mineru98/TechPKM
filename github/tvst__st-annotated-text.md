Streamlit 앱에서 주석이 달린 텍스트를 표시하기 위한 간단한 컴포넌트입니다. 텍스트 조각에 메타데이터(품사, 태그 등)를 연결하여 시각적 강조와 상호작용 기능을 제공합니다. 주석 색상, 배경 등 커스터마이징이 가능하며 HTML 요소 통합도 지원합니다.  

핵심 기능:  
- 텍스트/튜플 혼합 입력 처리  
- CSS 스타일링 지원  
- 계층적 리스트 구조 허용  
- 데모 앱에서 다양한 활용 사례 제공  

NLP 분석 결과 시각화, 문서 주석 시스템 구축 등에 활용 가능합니다.
