칸(Kan)은 오픈소스 기반의 프로젝트 관리 플랫폼으로, Trello의 대안으로 팀 협업을 지원하는 웹 애플리케이션입니다. 보드 기반 작업 관리, 멤버 초대, 카드 라벨 필터링, 활동 기록 추적 등의 기능을 제공하며, Docker를 통한 자체 호스팅이 가능합니다. Next.js, tRPC, Tailwind CSS 등의 기술 스택으로 구축되었으며, Trello 데이터 마이그레이션과 자체 템플릿 기능을 지원합니다.  

> 📌 핵심 기능: 보드 가시성 제어, 팀 협업, Trello 데이터 변환, 라벨 기반 필터링, 활동 로그 추적, 재사용 가능한 템플릿  

프로젝트는 Docker Compose 설정으로 간편하게 배포할 수 있으며, PostgreSQL 데이터베이스와 연동됩니다. AGPLv3 라이선스로 배포되며, 공식 문서와 Discord 커뮤니티를 통해 지원을 제공합니다.
