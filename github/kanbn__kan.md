칸(Kan)은 오픈소스 기반의 프로젝트 관리 도구로, 트렐로의 대안으로 개발된 협업 솔루션입니다. 보드 기반 작업 관리, 팀 협업 기능, 트렐로 데이터 마이그레이션 등을 지원하며, Docker를 통한 셀프호스팅이 가능합니다. Next.js, tRPC, Drizzle ORM 등 최신 웹 기술을 활용하여 개발되었습니다.  

### 핵심 기능
- **보드 가시성 제어**: 접근 권한 관리  
- **트렐로 임포트**: 기존 데이터 쉽게 이전  
- **라벨/필터**: 작업 카드 효율적 정리  
- **템플릿**: 재사용 가능한 보드 생성  
- **댓글과 활동 로그**: 작업 이력 추적  

### 기술 스택
- 프론트엔드: Next.js, Tailwind CSS  
- 백엔드: tRPC, Drizzle ORM  
- 인증: Better Auth  
- 데이터베이스: PostgreSQL  
- 배포: Docker Compose  

### 주요 특징
- AGPLv3 라이선스의 오픈소스  
- 이메일, OAuth, 파일 업로드 등 확장 기능 지원  
- 화이트 라벨링 및 자체 호스팅 최적화  

> GitHub 저장소에서 Docker Compose 설정, 환경 변수 가이드, 개발 문서 등을 확인할 수 있습니다.
