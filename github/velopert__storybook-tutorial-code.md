이 프로젝트는 TypeScript, Storybook, Rollup을 활용한 React UI 키트 개발 샘플입니다. UI 컴포넌트 설계부터 라이브러리 패키징까지 단계별 학습을 제공하는 튜토리얼용 저장소로, 버튼·아이콘·다이얼로그 등 핵심 컴포넌트 구현을 포함합니다. 각 기능별 태그가 세분화되어 있어 단계별 코드 확인이 가능합니다.
