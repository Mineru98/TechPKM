Adorable은 채팅 인터페이스를 통해 웹사이트와 앱을 생성할 수 있는 오픈소스 AI 에이전트입니다. Lovable의 오픈소스 버전으로, 코드 생성, 패치 기반 편집, Git 통합, 프리뷰 기능을 제공합니다. Node.js, PostgreSQL, Redis, Anthropic AI 등의 기술을 활용하며, 개발자들이 AI 기반 앱 빌더를 구축할 수 있도록 설계되었습니다. 프리스타일 가이드와 연동되어 AI 코드 어시스턴트를 효과적으로 활용할 수 있습니다.
