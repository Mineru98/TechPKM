Adorable은 채팅 인터페이스를 통해 웹사이트와 앱을 만들 수 있는 오픈소스 AI 에이전트 프로젝트입니다. Lovable의 오픈소스 버전으로, 패치 기반 코드 편집, Git 통합, AI 코드 어시스턴트 기능을 제공합니다. Node.js와 PostgreSQL 기반 시스템에 Anthropic 및 Freestyle API를 연동하여 개발 환경을 구성하며, 개발자와 비개발자 모두가 코드 생성 및 수정 작업을 수행할 수 있도록 설계되었습니다. 오픈소스 생태계 확장과 AI 기반 앱 빌더 개발을 위한 핵심 도구로 활용될 수 있습니다.
