이 프로젝트는 Annotation-AI 내부 사용을 위한 파이썬 프로젝트 템플릿입니다. 개발 환경 구성, 코드 포맷팅, 린팅, 유닛 테스트 등 표준화된 개발 워크플로우를 제공합니다. GitHub Actions를 통한 CI 통합과 브랜치 보호 규칙을 적용하여 코드 품질을 관리합니다. 설정 파일 기반의 일관된 개발 환경을 주요 특징으로 합니다.
