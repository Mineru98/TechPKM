Tesseract OCR는 오픈소스 OCR(광학 문자 인식) 엔진으로, 이미지 내 텍스트를 인식하는 데 사용됩니다. 100개 이상의 언어를 지원하며, LSTM 기반 신경망 엔진과 레거시 OCR 엔진을 모두 제공합니다. 다양한 이미지 형식과 출력 형식(텍스트, PDF, hOCR 등)을 지원하며, 커스텀 언어 학습이 가능합니다.  

이 프로젝트는 명령줄 프로그램과 C/C++ API를 제공하므로 개발자 및 연구자 모두에게 유용합니다. GUI는 별도로 제공되지 않으며, 외부 이미지 처리 도구를 통해 인식 품질을 개선할 수 있습니다. Apache 2.0 라이선스로 배포됩니다.
