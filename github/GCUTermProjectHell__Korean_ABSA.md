이 프로젝트는 국립국어원 인공 지능 언어 능력 평가 대회에 참가한 Team GCU_텀프지옥의 한국어 속성 기반 감성 분석(ABSA) 모델 구현입니다. 쇼핑 리뷰 데이터에서 속성 범주와 감성(긍정/부정/중립)을 추출하는 과제를 수행하며, F1 점수로 성능을 평가합니다. Electra와 T5 기반 모델을 앙상블하여 속성 분류 정확도를 향상시켰습니다. 추가로 네이버 쇼핑 크롤링 데이터와 국립국어원 말뭉치를 활용해 데이터 불균형 문제를 보완하였습니다.
