이 프로젝트는 국립국어원 주관 2022 인공지능 언어 능력 평가에서 쇼핑 리뷰 기반 **속성 기반 감성 분석**(ABSA) 모델을 구현한 팀 GCU_텀프지옥의 솔루션입니다. 한국어 쇼핑 리뷰 문장에서 속성 범주(예: "제춤 전체#인지도")와 감성(긍정/부정/중립)을 동시에 추출하는 모델을 개발하였으며, **Electra**와 **T5** 기반 모델을 앙상블로 결합해 F1 점수 최적화를 목표로 합니다. 데이터 불균형 문제 해결을 위해 네이버 쇼핑 후기 크롤링 및 금칙 데이터 추가 등의 전처리 기법을 적용했습니다. Docker 환경에서의 추론 실행 및 최종 결과 파일 생성이 가능합니다.
