ComfyDeploy는 ComfyUI를 서버리스 환경에 배포할 수 있는 오픈소스 클라우드 플랫폼입니다. 프론트엔드와 백엔드 전체를 포함하여 팀이 AI 워크플로우를 실험하고 관리할 수 있도록 지원하며, 현재는 기존 고객 지원과 새로운 아이디어 탐색을 병행 중입니다. ComfyOrg와의 경쟁보다는 공식 솔루션 추천을 우선시하며, 사용자는 Pay-as-you-go 방식이나 자체 호스팅을 선택할 수 있습니다. 주요 기술 스택은 Modal, Vercel, Clerk, Upstash, AWS S3 등을 기반으로 구축되었습니다.
