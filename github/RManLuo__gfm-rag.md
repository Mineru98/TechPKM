GFM-RAG은 그래프 신경망 기반의 검색 증강 생성(RAG) 파이프라인으로, 지식 그래프의 관계를 추론하여 질문에 대한 문서 검색을 수행합니다. 문서로부터 지식 그래프 인덱스를 구축하고, 사전 훈련된 그래프 파운데이션 모델(GFM) 리트리버를 활용하여 LLM 생성을 위한 관련 문서를 검색합니다. GFM-RAG은 대규모 훈련을 거친 모델로, 미세 조정 없이도 새로운 데이터셋에 직접 적용 가능하며, 해석 가능성과 효율성을 강점으로 합니다.
