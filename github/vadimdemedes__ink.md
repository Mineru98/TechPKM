React의 컴포넌트 기반 UI 개발 경험을 터미널 애플리케이션에 적용한 라이브러리입니다. Yoga 레이아웃 엔진을 활용해 Flexbox 스타일의 레이아웃을 지원하며, React 생태계와 완벽히 호환됩니다. CLI 애플리케이션에서 텍스트 스타일링, 레이아웃 구성, 사용자 입력 처리 등을 리액트 방식으로 구현할 수 있습니다. npm 주간 다운로드 수 40만 이상의 인기 있는 프로젝트로, Shopify CLI, Gatsby, Prisma 등 다양한 프로덕션 환경에서 사용 중입니다.
