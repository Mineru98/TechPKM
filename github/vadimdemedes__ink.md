React 개발 경험을 터미널 CLI 애플리케이션으로 확장하는 라이브러리입니다. 브라우저 React와 유사한 컴포넌트 기반 UI 구축 방식을 제공하며, Yoga 레이아웃 엔진을 사용해 터미널에서도 CSS Flexbox 스타일의 레이아웃을 구현할 수 있습니다. `Text`, `Box` 등 기본 컴포넌트와 상태 관리를 위한 React 훅을 지원하며, 다양한 오픈소스 프로젝트에서 사용되고 있습니다. 터미널 환경에 최적화된 UI 개발을 단순화합니다.
