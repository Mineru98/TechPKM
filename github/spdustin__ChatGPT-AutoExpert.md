ChatGPT AutoExpert는 GPT-4 및 GPT-3.5-Turbo 모델의 성능을 전문가 수준으로 향상시키기 위한 커스텀 지시문 세트입니다. "표준 에디션"과 "개발자 에디션"으로 구성되어 있으며, 각각 비코딩 작업과 고급 데이터 분석이 필요한 코딩 작업에 특화되어 있습니다. 주요 기능으로는 질문 자동 개선, 슬래시 명령 지원, 최적의 전문가 역할 자동 선택, 명시적 추론 및 리소스 링크 생성 등이 포함됩니다. 개발자 에디션은 Python 패키지 설치, 작업 저장, 파일/심볼 트리 관리 등 고급 기능을 제공하며, Jupyter 기반 코드 샌드박스와 연동됩니다.  

(요약: GPT 모델의 전문가 수준 응답을 위한 커스텀 지시문 세트. 표준/비코딩용과 개발자용 에디션 제공. 질문 개선, 코드 생성, 슬래시 명령 등 고급 기능 포함.)
