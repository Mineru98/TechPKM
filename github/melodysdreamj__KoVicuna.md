KoVicuna은 영어 Vicuna 모델과 동일한 방식으로 학습된 한국어 특화 언어 모델입니다. 62만 개의 ShareGPT 대화문을 DeepL로 번역해 학습 데이터로 사용했으며, A100 GPU 8개로 15시간 동안 학습되었습니다. 한국어 질의응답 및 생성 작업에 특화된 오픈소스 모델로, 허깅페이스와 GGML 형식으로 공개되어 다양한 환경에서 활용 가능합니다.
