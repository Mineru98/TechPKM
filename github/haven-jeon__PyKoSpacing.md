한국어 텍스트 분석을 위한 자동 띄어쓰기 Python 패키지입니다. 딥러닝 모델 기반으로 온라인/SNS 텍스트를 포함한 다양한 환경에서 높은 정확도를 보이며, 사용자 정의 규칙을 통해 복합어 처리를 최적화할 수 있습니다. 1억 개 이상의 뉴스 기사로 학습된 모델로 세종 말뭉치 97.1%, 문학식 말뭉치 94.3%의 정확도를 달성했습니다.  

핵심 기능:  
1. `Spacing` 클래스를 통한 인터페이스 제공  
2. 사용자 정의 규칙 적용 기능 (CSV 파일 지원)  
3. 영문/기호 처리 옵션 (`ignore` 파라미터)  
4. 커맨드 라인 실행 가능
