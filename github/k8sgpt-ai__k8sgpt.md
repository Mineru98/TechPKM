`k8sgpt`는 쿠버네티스 클러스터를 스캔하고 문제를 진단하며 영어로 해결책을 제시하는 도구입니다. 내장된 SRE 경험을 활용해 AI로 보강된 정확한 정보를 제공하며, OpenAI, Azure, Cohere, Amazon Bedrock, Google Gemini 등 다양한 AI 백엔드와 즉시 통합됩니다. CLI 및 MCP(Model Context Protocol) 서버를 통해 로컬 및 클라우드 환경에서 유연하게 사용할 수 있으며, 커스텀 분석기와 원격 캐싱 기능을 지원합니다.  

주요 기능:  
- 기본 분석기(20+ 종류) 및 커스텀 분석기 확장 지원  
- 보안 강화를 위한 데이터 익명화 기능  
- AWS S3/Azure Blob/Google Cloud Storage 원격 캐싱  
- Claude Desktop 등 MCP 호환 도구와의 통합  
- 다양한 설치 옵션(Homebrew, RPM/DEB, Windows 바이너리)  

용도: 쿠버네티스 클러스터 상태 점검, 리소스 문제 진단, 자동 문서화, AI 기반 문제 해결 가이드 생성.
