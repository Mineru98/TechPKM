PyObjC 프로젝트는 Python과 Objective-C 간의 상호 운용성을 제공하는 브릿지로, macOS 환경에서 Python을 통해 Objective-C 프레임워크에 접근할 수 있게 합니다. 이 저장소는 'pyobjc' 메타 패키지와 핵심 브릿지 구현체인 'pyobjc-core', 그리고 다양한 macOS 프레임워크를 위한 래퍼 패키지들을 포함합니다. Python 3.9 이상에서 사용 가능하며, PyPI를 통한 설치보다 공식 설치 스크립트 사용을 권장합니다.
