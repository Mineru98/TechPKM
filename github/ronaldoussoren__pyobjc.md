이 프로젝트는 Python과 Objective-C 간의 상호작용을 가능하게 하는 PyObjC 브릿지를 제공합니다. 주요 구성 요소로 'pyobjc' 메타 패키지, 'pyobjc-core' 브릿지 구현, 그리고 macOS 프레임워크 래퍼 패키지들('pyobjc-framework-*')을 포함합니다. Python 3.9 이상에서 사용 가능하며, pip를 통한 설치 또는 설치 스크립트를 지원합니다. PyPy/Jython/IronPython은 지원하지 않습니다.
