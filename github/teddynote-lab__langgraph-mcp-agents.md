이 프로젝트는 LangChain-MCP-Adapters를 활용하여 Model Context Protocol(MCP) 도구와 통합되는 LangGraph ReAct 에이전트를 배포하기 위한 사용자 친화적 인터페이스를 제공합니다. Streamlit 기반의 웹 UI를 통해 MCP 도구를 실시간으로 추가/수정할 수 있으며, 에이전트 응답 및 도구 호출 스트리밍, 대화 기록 추적 기능을 지원합니다. MCP 호스트-클라이언트-서버 아키텍처를 기반으로 다양한 데이터 소스와 API에 접근할 수 있는 AI 에이전트 구축에 최적화되어 있습니다. Docker와 직접 설치 방식을 모두 지원합니다.
