Diffusion Bee는 Intel/M1 Mac에서 로컬로 스테이블 디퓨전을 실행하는 GUI 애플리케이션입니다. 데이터 프라이버시 보장, 원클릭 설치, 모델 다운로드 기능을 제공하며, 인페인팅/아웃페인팅, ControlNet, LoRA 등 고급 생성 기능을 지원합니다. 사용자 친화적인 인터페이스로 기술적 지식 없이도 AI 이미지를 생성할 수 있습니다.  

> 주요 특징:  
> - 로컬 실행으로 데이터 보안 강화  
> - SD 1.x, 2.x, XL 및 다양한 확장 모델 지원  
> - 이미지/텍스트 기반 생성 및 편집 기능  
> - M1/M2 칩 최적화 및 맥OS 호환
