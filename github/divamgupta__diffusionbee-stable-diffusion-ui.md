Diffusion Bee는 인텔/M1 맥에서 스테이블 디퓨전을 로컬로 실행할 수 있는 사용자 친화적 GUI 애플리케이션입니다. 데이터 프라이버시 보호, 원클릭 설치, 다양한 모델 지원(SD 1.x, SD 2.x, SD XL 등)을 제공하며, 이미지 생성/편집(인페인팅, 아웃페인팅), 업스케일링, 프롬프트 최적화 등의 기능을 포함합니다. 기술 지식 없이도 쉽게 사용할 수 있는 MacOS 전용 도구로, 모든 처리가 로컬에서 이루어집니다.
