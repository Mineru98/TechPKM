Leptos는 Rust로 작성된 풀스택 웹 프레임워크로, 미세한 반응성 시스템을 기반으로 선언형 사용자 인터페이스를 구축합니다. 클라이언트 측 렌더링, 서버 측 렌더링, 또는 스트리밍 HTML과 하이드레이션을 결합한 다양한 모드를 지원하며, 서버 함수를 통해 클라이언트-서버 간 통합 개발 환경을 제공합니다. 웹 표준 기반의 라우터와 성능 최적화로 현대적인 웹 애플리케이션 개발에 적합합니다.
