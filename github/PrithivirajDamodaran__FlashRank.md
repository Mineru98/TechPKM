FlashRank는 검색 및 정보 검색 파이프라인에 재순위화 기능을 추가하기 위한 초경량·고속 파이썬 라이브러리입니다. 최신 대규모 언어 모델(LLM)과 크로스 인코더 기반으로 구현되었으며, CPU 환경에서도 Torch 또는 Transformers 의존성 없이 작동합니다. 4MB에서 4GB까지의 다양한 모델 크기를 지원하며, 서버리스 환경에서 효율적인 재순위화를 위한 최적화된 설계를 특징으로 합니다. 특히 MS-MARCO 데이터셋으로 미세 조정된 모델들을 활용해 검색 결과 품질을 향상시키는 데 중점을 둡니다.
