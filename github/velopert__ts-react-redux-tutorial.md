타입스크립트를 사용하는 리액트 프로젝트에서 리덕스를 프로처럼 사용하는 방법을 단계별로 학습하는 튜토리얼 프로젝트입니다. 카운터, 투두리스트, 비동기 작업 등 다양한 예제와 미들웨어(thunk, saga) 활용법을 10개의 브랜치로 구성하여 체계적으로 설명합니다. 리덕스 상태 관리, 액션 타입 정의, 리듀서 구현, 비동기 처리 등 핵심 개념을 실습할 수 있습니다.
