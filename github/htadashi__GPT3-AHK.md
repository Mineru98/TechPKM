GPT3-AHK는 컴퓨터 내 모든 입력 필드에서 GPT3 및 기타 LLM 모델을 사용할 수 있게 해주는 AutoHotKey 스크립트입니다. 사용자가 선택한 텍스트를 기반으로 GPT3 모델을 호출하여 자동 완성 또는 수정을 수행하며, 커스텀 모델 연동도 지원합니다. 단축키로 빠르게 텍스트를 생성/수정할 수 있어 생산성 향상에 도움이 됩니다.  

(요약: AutoHotKey 기반의 GPT3 연동 스크립트로, 텍스트 자동 생성 및 수정 기능을 제공하며 커스텀 모델 설정이 가능합니다.)
