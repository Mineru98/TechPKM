이 프로젝트는 AutoHotKey 스크립트로, 컴퓨터의 모든 입력 필드에서 GPT3 및 기타 LLM(대형 언어 모델)을 사용할 수 있게 합니다. OpenAI API와 연동되어 텍스트 완성, 수정 기능을 제공하며, 사용자 정의 모델 서버와도 호환됩니다. `Win+o` 또는 `Win+Shift+o` 단축키로 간편하게 활용할 수 있으며, 시스템 트레이 메뉴에서 모델을 전환할 수 있습니다.  

주요 기능은 선택된 텍스트 자동 완성, GPT3 기반 텍스트 수정, 다양한 LLM 모델 지원, 사용자 정의 모델 엔드포인트 연결 등입니다.
