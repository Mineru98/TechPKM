이미지를 ASCII 아트로 변환하는 파이썬 프로젝트입니다. 입력 이미지/비디오를 다양한 언어(영어, 독일어, 프랑스어, 한국어 등)의 텍스트 또는 이미지 형식(흑백/컬러, 배경 반전 옵션 포함)으로 변환하며, 비디오 처리 기능도 지원합니다. OpenCV, PIL, NumPy를 활용해 구현되었습니다.  

핵심 기능:  
1. 이미지를 텍스트(.txt) 또는 이미지 형식(.png, .jpg)으로 변환  
2. 비디오를 ASCII 비디오(.avi, .mp4)로 변환  
3. 다국어 문자셋 지원 및 흑백/컬러 출력 옵션  
4. 단순 문자/복합 문자 두 가지 출력 모드 제공
