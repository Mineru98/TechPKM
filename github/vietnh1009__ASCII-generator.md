이 프로젝트는 이미지를 다양한 언어(영어, 독일어, 프랑스어, 한국어, 일본어, 러시아어 등)의 ASCII 아트로 변환하는 Python 기반 도구입니다. 이미지/비디오 입력에 대해 텍스트(.txt) 또는 이미지(.png, .jpg 등)/비디오(.avi, .mp4 등) 형식의 출력을 생성하며, 흑백 반전, 그레이스케일/컬러 모드 선택이 가능합니다. OpenCV, PIL, NumPy를 활용해 복잡한 문자 또는 단순 문자로 표현할 수 있어 예술적 변환에 적합합니다.
