Capacitor는 JavaScript와 웹 기술을 활용해 iOS, Android, 웹 등 다양한 플랫폼에서 네이티브 앱을 단일 코드베이스로 개발할 수 있는 프레임워크입니다. 웹 앱에 네이티브 SDK 접근 기능과 커스텀 플러그인 개발 기능을 제공하며, 프로그레시브 웹 앱(PWA) 지원을 통해 앱 스토어와 모바일 웹 동시 배포가 가능합니다. 현대적 도구링과 모듈식 플러그인 아키텍처를 특징으로 하며, Ionic Framework와의 통합이 권장됩니다.  

> 💡 **핵심 기능**: 웹 앱의 네이티브 기능 확장, 크로스플랫폼 API 제공, 커스텀 네이티브 플러그인 개발 지원, PWA 호환성.  
> 💡 **주요 특징**: Cordova 대비 현대적인 접근 방식, 소스 아티팩트 기반 네이티브 프로젝트 관리, 커뮤니티 기반 플러그인 생태계.
