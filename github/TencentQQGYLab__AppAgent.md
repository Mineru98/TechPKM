AppAgent는 스마트폰 애플리케이션에서 인간처럼 상호작용(탭, 스와이프 등)할 수 있는 LLM 기반 멀티모달 에이전트 프레임워크입니다. 시스템 백엔드 접근 없이 다양한 앱에 적용 가능하며, 자율 탐색 또는 인간 시연 관찰을 통해 새로운 앱을 학습합니다. GPT-4V 또는 Qwen-VL 모델을 활용해 복잡한 작업을 실행할 수 있는 지식 기반을 생성하며, Android 기기에서 직접 작동합니다.  

```plaintext
[CHI 2025] AppAgent 프로젝트 요약: 스마트폰 앱 자동화 에이전트 프레임워크
```
