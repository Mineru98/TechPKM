Velog-server는 개발자를 위한 한국어 지원 블로그 플랫폼의 백엔드 시스템입니다. Node.js와 TypeScript 기반으로 구축되었으며 Apollo GraphQL을 활용한 API 서버와 PostgreSQL 데이터베이스, 다양한 AWS 서비스(람다, SES 등)를 통한 서버리스 아키텍처를 특징으로 합니다. 프론트엔드와의 분리를 통해 백엔드 기능(마크다운 편집, 검색, 인증 등)에 집중하고 있습니다.
