MinIO는 GNU AGPL v3.0 라이선스로 배포되는 고성능 S3 호환 객체 저장 솔루션입니다. AI/ML, 분석 및 데이터 집약적 워크로드를 위해 설계되었으며, 대규모 데이터 파이프라인에 최적화되어 있습니다. 현재 유지보수 모드로 운영되며, 커뮤니티용으로는 소스 코드만 제공되며, 상용 버전인 AIStor Free/Enterprise로 대체되었습니다.  

핵심 특징으로는 기존 S3 도구와의 호환성, 대규모 데이터 처리 최적화, 높은 성능과 확장성이 있으며, Go 언어로 개발되었습니다. Go 1.24 이상 환경이 필요하며, 소스 빌드 또는 Docker 이미지로 배포할 수 있습니다.
