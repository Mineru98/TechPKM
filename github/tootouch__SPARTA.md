SPARTA는 한국어 질문을 SQL 쿼리로 변환하는 딥러닝 모델을 구현한 교육 프로젝트입니다. 고려대학교 DSBA 연구실 수업에서 진행된 이 프로젝트는 WikiSQL 데이터셋을 기반으로 4가지 번역 방법을 적용하여 한국어 텍스트-투-SQL 모델을 개발하고, SQLova/HydraNet/BRIDGE 아키텍처를 활용한 실험을 수행했습니다. 모델은 논리적 정확성(65.8%)과 실행 정확성(74.3%)을 달성하며 테이블 인식 자연어 처리 기술의 가능성을 탐구합니다.
