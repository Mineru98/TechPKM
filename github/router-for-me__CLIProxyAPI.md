CLIProxyAPI는 OpenAI, Gemini, Claude, Qwen, iFlow와 호환되는 API 인터페이스를 제공하는 프록시 서버입니다. 로컬 또는 멀티 계정 CLI 환경에서 AI 모델을 사용할 수 있도록 지원하며, OAuth 인증을 통해 다양한 AI 코딩 도구와의 통합을 가능하게 합니다. 스트리밍/논스트리밍 응답, 함수 호출, 멀티모달 입력, 자동 모델 폴백 등의 기능을 제공합니다.  

주요 특징:  
- CLI 기반 AI 모델 접근을 위한 OpenAI/Gemini/Claude 호환 API  
- 멀티 계정 로드 밸런싱 및 모델 매핑 기능  
- Amp CLI 및 IDE 확장 지원  
- 재사용 가능한 Go SDK 제공  
- 패키코드, 큐벤스 등의 서드파티 서비스와 연동 가능  

MIT 라이선스로 배포되며, 오픈소스 생태계 확장을 위한 기여가 활발히 이루어지고 있습니다.
