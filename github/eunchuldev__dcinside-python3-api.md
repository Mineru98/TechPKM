이 프로젝트는 비공식으로 제공되는 Python 3용 비동기 DC인사이드 API 라이브러리입니다. 디시인사이드 갤러리 게시물 및 댓글 크롤링, 이미지 다운로드, 글/댓글 작성 및 삭제 기능을 제공하며, aiohttp와 lxml을 기반으로 구현되었습니다. 비동기 방식으로 효율적인 데이터 수집이 가능하며, 간단한 인터페이스로 DC인사이드 콘텐츠 관리가 가능합니다.
