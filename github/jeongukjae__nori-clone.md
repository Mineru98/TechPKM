이 프로젝트는 Apache Lucene의 Java 기반 한국어 형태소 분석기 'nori'를 C++로 재구현한 독립 실행형 라이브러리입니다. 다양한 언어(C++/Python/Golang)에서 고성능 한국어 형태소 분석이 가능하도록 이식성과 사용성을 개선했으며, 기존 노리와 유사한 알고리즘과 사전 구축 방식을 채택했습니다. 레거시/최신 두 가지 정규화 옵션을 지원하는 사전과 벤치마크 도구를 제공합니다.
