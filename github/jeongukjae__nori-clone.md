nori-clone은 Apache Lucene의 Java 기반 한국어 형태소 분석기 'nori'를 C++로 재구현한 독립 실행형 프로젝트입니다. Python 및 Golang 바인딩을 제공하여 다양한 언어 환경에서 고성능 한국어 형태소 분석이 가능하도록 설계되었습니다. 사전 정규화 옵션과 레거시/최신 두 가지 사전 체계를 지원하며, Elasticsearch의 nori와 유사한 알고리즘을 구현하였습니다.  

이 프로젝트는 Lucene에 강하게 결합된 원본 nori의 한계를 극복하고, 크로스플랫폼 환경에서의 활용성과 이식성을 높이기 위해 개발되었습니다. 성능 벤치마크 및 사전 비교 도구를 통해 원본 nori와의 차이점을 명확히 확인할 수 있습니다.
