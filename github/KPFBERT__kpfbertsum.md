이 프로젝트는 한국어 뉴스 기사 요약에 특화된 KpfBertSum 모델을 구현한 것으로, 한국언론진흥재단의 kpfBERT 사전학습 모델과 ETRI AI-HUB의 문서요약 데이터셋을 활용하였다. Bert 기반 인코더를 사용하여 추출 요약을 수행하며, 특히 뉴스 기사 텍스트 처리에 최적화되어 있다. PRESUMM 모델을 참조하여 개발되었으며, 한국어 NLP 작업을 위한 엔드투엔드 요약 솔루션을 제공한다.
