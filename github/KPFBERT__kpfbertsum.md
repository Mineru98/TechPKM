KpfBertSum은 한국어 뉴스 기사 요약을 위해 kpfBERT 사전학습 모델을 활용한 추출형 요약 모델이다. 한국언론진흥재단의 뉴스 코퍼스로 학습되었으며, PRESUMM 논문(EMNLP 2019)의 방법론과 ETRI의 AI-HUB 데이터셋(비플라이소프트)을 기반으로 구현되었다. 주요 특징은 한국어 특화 요약, 뉴스 도메인 최적화, BERT 기반 문장 중요도 예측 기능을 포함한다.  

> 핵심 기술: kpfBERT 파인튜닝, 추출 요약, 텍스트 랭킹  
> 관련 리소스: [kpfBERT](https://github.com/KPFBERT/kpfbert), [PreSumm](https://github.com/nlpyang/PreSumm)
