리틀FS는 마이크로컨트롤러용으로 설계된 소형 파일 시스템입니다. 전원 손실 시 안정성을 보장하며, 동적 웨어 레벨링과 플래시 메모리 최적화를 제공합니다. POSIX와 유사한 파일 연산을 지원하면서도 RAM/ROM 사용량을 엄격하게 제한합니다. 임베디드 시스템에서 파일 관리를 위한 경량 솔루션으로 설계되었습니다.
