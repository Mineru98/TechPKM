한굴라이즈는 다양한 언어의 외래어를 한글 표기 규칙에 맞게 자동으로 변환하는 도구입니다. 현재는 아제르바이잔어, 독일어, 이탈리아어 등 30여 개 언어를 지원하며, 각 언어의 표기 규칙을 기반으로 권장 한글 표기를 제공합니다. Go 언어로 구현되어 있으며 프로그래머와 언어학자의 협력으로 개발되었습니다.
