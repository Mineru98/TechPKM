한글라이즈는 다양한 언어의 외래어를 한글 표기 규칙에 따라 변환하는 오픈소스 도구입니다. 주로 프로그래밍 방식으로 외국어 단어를 한국어 발음으로 자동 변환하며, 개발 중인 프로젝트입니다. 이탈리아어 "Cappuccino"를 한글 "카푸치노"로 변환하는 예시처럼, 각 언어의 표기 규칙을 적용해 표준적인 한글 표기를 제공합니다.  

> 📌 **요약**: 외국어 → 한글 표기 변환 도구 (개발 중, 20+ 언어 지원)
