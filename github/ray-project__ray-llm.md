이 프로젝트는 Ray 기반 대규모 언어 모델(LLM)의 배포를 간소화하기 위한 API를 제공합니다. 현재는 레이 메인 저장소로 통합되어 `ray.serve.llm` 및 `ray.data.llm`으로 관리되며, 해당 레포지토리는 보관 상태입니다. 레이 공식 문서에서 최신 API 정보를 확인할 수 있습니다.
