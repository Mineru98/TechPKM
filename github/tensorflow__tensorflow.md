TensorFlow는 머신러닝을 위한 엔드투엔드 오픈소스 플랫폼입니다. 연구자와 개발자가 머신러닝 연구의 최첨단 기술을 추진하고, ML 기반 애플리케이션을 쉽게 구축 및 배포할 수 있는 포괄적인 도구, 라이브러리, 커뮤니티 리소스를 제공합니다. Python 및 C++ API를 주로 지원하며, 다양한 설치 옵션과 활발한 오픈소스 생태계를 갖추고 있습니다. Google에서 개발된 이 프레임워크는 GPU/CPU 지원, 튜토리얼, 모델 최적화 도구 등 풍부한 기능을 제공합니다.
