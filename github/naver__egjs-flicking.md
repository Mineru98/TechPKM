@egjs/flicking은 Angular, React, Vue 등 다양한 자바스크립트 프레임워크에서 사용 가능한 캐러셀 라이브러리입니다. 매일 3천만 명이 사용하는 신뢰성, 유연성, 확장성을 갖춘 컴포넌트로, 자동 재생, 패럴랙스 효과 등 즉시 사용 가능한 플러그인을 제공합니다. SSR(서버 사이드 렌더링) 호환성과 IE9+ 지원을 포함하며, 상태 복원 및 데스크톱/모바일 환경 최적화 기능을 갖추었습니다. TypeScript 기반으로 개발되어 타입 안정성과 유지보수성을 보장합니다.
