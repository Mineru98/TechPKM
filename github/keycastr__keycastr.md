KeyCastr는 macOS에서 키 입력과 마우스 클릭을 시각적으로 표시하는 오픈 소스 유틸리티입니다. 스크린캐스트 제작, 발표, 협업 시 키보드 입력을 효과적으로 공유할 수 있도록 설계되었으며, 명령어 키/수정 키/모든 키스트로크 표시 옵션과 마우스 클릭 시각화 기능을 제공합니다. 사용자는 화면 상의 위치를 자유롭게 조정할 수 있으며, BSD 3-Clause 라이선스로 배포됩니다.
