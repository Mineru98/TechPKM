KeyCastr는 macOS에서 키 입력과 마우스 클릭을 시각적으로 보여주는 오픈소스 도구입니다. 스크린캐스트 제작, 발표, 협업 시 키 입력 과정을 공유할 수 있으며, 필요에 따라 명령어 키, 수정 키, 모든 키 입력 또는 마우스 클릭을 선택적으로 표시할 수 있습니다. 또한 사용자가 직접 커스텀 비주얼라이저를 개발할 수 있는 확장성을 제공합니다.
