Penpot은 디자인 및 코드 협업을 위한 최초의 오픈소스 디자인 도구입니다. 디자이너는 SVG, CSS, HTML, JSON과 같은 오픈 표준을 활용해 프로토타이핑과 디자인 시스템을 구축할 수 있으며, 개발자는 즉시 사용 가능한 코드를 통해 효율적인 워크플로우를 구현할 수 있습니다. 브라우저 기반 또는 자체 호스팅 방식으로 접근 가능하며, MPL-2.0 라이선스로 배포됩니다. 디자인 토큰, 컴포넌트 시스템, CSS 그리드 레이아웃 등 현대적인 기능을 지원하여 디자인-개발 간 원활한 협업을 촉진합니다.
