펜팟(Penpot)은 디자이너와 개발자 간의 협업을 위한 최초의 오픈소스 디자인 도구입니다. SVG, CSS, HTML, JSON 등 개방형 표준을 기반으로 웹 또는 자체 호스팅 환경에서 작동하며, 디자인 시스템 구축을 위한 디자인 토큰, 컴포넌트, 변형 기능을 제공합니다. 2.0 버전에서는 CSS 그리드 레이아웃, UI 재설계, 플러그인 시스템 등 혁신적인 기능을 도입하여 디자인-코드 워크플로우를 최적화했습니다.  

디자이너는 시각적 디자인을 생성하고 개발자는 즉시 사용 가능한 코드를 활용하면서 협업할 수 있으며, 오픈소스 라이선스(MPL-2.0)로 자유롭게 사용 및 확장할 수 있습니다. 커뮤니티 기반 지원과 다양한 통합 기능을 통해 개발 생태계에 통합됩니다.
