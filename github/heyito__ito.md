Ito는 컴퓨터 내 모든 애플리케이션에서 자연어 음성 받아쓰기를 지원하는 지능형 음성 보조 도구입니다. macOS와 Windows를 지원하며, 실시간 전사, 커스텀 사전, 문맥 인식, 다중 언어 지원 등 고급 AI 기반 기능을 제공합니다. 사용자가 트리거 키를 누른 상태에서 말하면 자동으로 텍스트 필드에 입력되는 방식으로 작동합니다. 현재는 더 이상 유지보수되지 않는 프로젝트입니다.
