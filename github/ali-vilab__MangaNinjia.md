MangaNinja는 라인 아트 색칠화를 위한 참조 기반 딥러닝 모델입니다. CVPR 2025에 채택된 이 프로젝트는 애니메이션 산업에서 색칠 작업 효율성을 높이기 위해 설계되었으며, 참조 이미지와 라인 아트를 정밀하게 정렬하여 일관된 색칠 결과를 제공합니다. 사용자는 점 제어 기능을 통해 더 복잡한 작업도 수행할 수 있습니다. 주요 기술로는 스테이블디퓨전 기반의 생성 모델과 CLIP, ControlNet 아키텍처가 활용되었습니다.
