이 프로젝트는 *"MangaNinja: Line Art Colorization with Precise Reference Following"* 논문의 공식 구현체로, 참조 이미지를 기반으로 라인아트 채색 작업을 자동화하는 딥러닝 모델입니다. 사용자가 점 제어를 통해 복잡한 편집도 가능하며, 애니메이션 산업에서의 채색 작업 효율화를 목표로 합니다. CVPR 2025에 채택된 기술로, Gradio 기반의 GUI를 통해 직관적인 사용이 가능합니다.  

(요약: 참조 이미지를 정확히 따라 라인아트를 컬러화하는 AI 모델 구현체. 점 제어 기능과 Gradio 인터페이스를 통해 사용자 친화적인 애니메이션 채색 솔루션 제공)
