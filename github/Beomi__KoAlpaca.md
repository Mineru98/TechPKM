KoAlpaca는 Stanford Alpaca 모델을 한국어로 학습시킨 오픈소스 언어 모델입니다. Polyglot-ko 및 LLaMA 기반 모델을 지원하며, 한국어 텍스트 생성 및 이해 작업에 최적화되었습니다. QLoRA와 풀 파인튜닝 학습 방법을 포함하여 다양한 학습 코드와 웹UI 데모를 제공합니다. HuggingFace에 공개된 모델을 활용해 한국어 명령어 기반 태스크를 처리할 수 있습니다.
