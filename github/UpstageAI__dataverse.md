Dataverse는 Python 기반의 오픈소스 ETL(추출, 변환, 적재) 파이프라인 프로젝트로, 데이터 과학자와 개발자를 위한 표준화된 데이터 처리 및 관리 솔루션을 제공합니다. Spark를 쉽게 활용할 수 있도록 구성 기반의 블록 조합 방식을 채택했으며, AWS EMR과의 연동을 통해 대규모 데이터 처리도 지원합니다. 특히 LLM 훈련을 위한 고품질 데이터 생성에 최적화되어 있으며, Hugging Face 데이터셋 로딩부터 중복 제거, 샘플링, Parquet 저장까지 전 과정을 직관적으로 구성할 수 있습니다. Upstage Solar의 데이터 인프라 기반으로 활용 중입니다.
