Dataverse는 파이썬 기반의 오픈소스 ETL 파이프라인으로, 데이터 과학자와 개발자를 위한 사용자 친화적인 데이터 처리 및 관리 솔루션을 제공합니다. Spark 전문 지식이 없어도 구성 기반으로 블록 형태의 ETL 함수를 조합할 수 있으며, AWS EMR 연동과 고품질 LLM 학습 데이터 생성 기능을 지원합니다.  

> 핵심 기능:  
> - **블록 기반** ETL 구성 (퍼즐 조각처럼 조립)  
> - **설정 파일 기반** Spark 및 ETL 프로세스 관리  
> - HuggingFace, Parquet 등 다양한 데이터 소스 연동  
> - 중복 제거, 샘플링, PII 처리 등 50여 개의 내장 함수 제공  

Upstage의 Solar LLM 학습 데이터 전처리에 실제로 활용 중이며, 확장성을 통해 커스텀 함수 추가가 가능합니다.
