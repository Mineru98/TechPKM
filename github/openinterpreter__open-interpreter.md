**Open Interpreter**는 LLM(대형 언어 모델)이 로컬 환경에서 Python, JavaScript, 쉘 등의 코드를 실행할 수 있도록 하는 프로젝트입니다. 터미널에서 ChatGPT와 유사한 인터페이스로 대화하며 컴퓨터의 일반적인 기능(사진/동영상 편집, 브라우저 제어, 데이터 분석 등)에 자연어 기반 접근이 가능합니다. OpenAI의 Code Interpreter와 달리 로컬 환경에서 실행되어 인터넷 접근, 파일 크기 제한 없음, 사용자 정의 패키지 활용 등의 유연성을 제공합니다. 실행 전 사용자 확인 절차를 거쳐 안전성을 강화했습니다.
