BERTopic은 BERT 변환기와 c-TF-IDF를 활용해 밀집 클러스터를 생성하는 토픽 모델링 기법입니다. 다양한 토픽 모델링 기법(지도 학습, 비지도 학습, 준지도 학습 등)을 지원하며, Python 환경에서 쉽게 사용할 수 있습니다. 문서 데이터의 숨겨진 주제를 해석하고 시각화하는 데 특화되어 있으며, GPU 가속 및 다중 언어 처리 기능을 포함합니다.  

BERTopic은 `bertopic` 파이 패키지를 통해 설치 가능하며, 사용자 정의 임베딩 모델, 시각화 도구, 토픽 병합/축소 기능 등을 제공합니다. arXiv 논문 및 공식 문서를 참조해 보다 심층적인 정보를 확인할 수 있습니다.
