PandasAI는 Python 라이브러리로, 자연어로 데이터에 질문하거나 분석을 수행할 수 있는 기능을 제공합니다. 비기술 사용자도 데이터에 쉽게 접근할 수 있도록 하며, 기술 사용자는 데이터 처리 시간과 노력을 절약할 수 있습니다. LLM(예: GPT-4.1-mini)과 연동해 평균 수익 계산, 차트 생성, 여러 데이터프레임 간 비교 등 복잡한 분석을 지원합니다. Docker 샌드박스 환경에서 안전하게 실행할 수 있는 기능도 포함되어 있습니다. MIT 라이선스로 배포됩니다.
