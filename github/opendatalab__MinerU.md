MinerU는 PDF 문서를 기계가 읽을 수 있는 형식(마크다운, JSON 등)으로 변환하는 오픈소스 도구입니다. 과학 문헌의 기호 변환 문제 해결을 중점으로 개발되었으며, 복잡한 레이아웃에서도 텍스트 순서 유지, 표/수식 추출, 다국어 OCR 지원 등의 기능을 제공합니다. GPU 가속 및 CPU 전용 환경을 모두 지원하며, Windows/Linux/macOS 플랫폼에서 사용 가능합니다.  

MinerU는 헤더/푸터 제거, 표/이미지/각주 추출, LaTeX 수식 변환, 스캔 PDF 자동 감지 등의 핵심 기능을 갖추고 있으며, 기술 논문, 연구 자료 등 다양한 문서의 구조화된 데이터 추출에 최적화되어 있습니다.
