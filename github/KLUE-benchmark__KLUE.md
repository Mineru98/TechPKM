KLUE는 한국어 자연어 처리(NLP) 분야의 발전을 평가하기 위한 벤치마크 데이터셋과 프레임워크입니다. 8개의 다양한 한국어 이해 과제(토픽 분류, 의미적 유사도 분석, 개체명 인식 등)를 포함하며, KLUE-BERT 및 KLUE-RoBERTa와 같은 사전 훈련 모델과 평가 지표, 파인튜닝 레시피를 제공합니다. 공개된 데이터와 모델을 통해 한국어 NLP 모델의 공정한 비교와 연구를 촉진하며, 윤리적 문제 완화를 고려한 설계 원칙이 적용되었습니다.
