KLUE는 한국어 자연어 처리(NLP) 분야의 발전을 위해 설계된 벤치마크입니다. 8가지 다양한 작업(주제 분류, 문장 유사도 분석, 자연어 추론 등)과 데이터셋, 평가 지표, 파인튜닝 레시피를 제공하며, KLUE-BERT 및 KLUE-RoBERTa 등의 사전 학습 모델을 공개해 연구의 공정한 비교와 진전을 촉진합니다. 모든 자료는 Creative Commons 라이선스로 자유롭게 활용 가능하며, AI 윤리 문제를 고려한 설계 원칙을 따릅니다.  

참고: 주요 언어 모델은 Python 기반의 HuggingFace 플랫폼에서 제공됩니다.
