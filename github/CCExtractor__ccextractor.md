CCExtractor는 전 세계 TV 녹화에서 자막을 추출하는 도구로, 실시간 자막 추출, 번역, DVD 자막 추출 및 자막 변환 기능을 제공합니다. 명령줄 도구나 Windows GUI를 통해 사용할 수 있으며, C로 작성된 코어 기능을 기반으로 다양한 플랫폼에서 지원됩니다.  

이 프로젝트는 청각 장애인이나 언어 학습자에게 유용한 자막 생성을 목표로 하며, Windows 패키지 관리자(WinGet, Chocolatey, Scoop)를 통해 쉽게 설치할 수 있습니다. 소스 코드 빌드 시 WSL 또는 Linux autotools 방식을 지원합니다.
