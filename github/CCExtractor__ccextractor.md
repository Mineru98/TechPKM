CCExtractor는 전 세계 TV 녹화 파일에서 자막을 추출하는 오픈소스 도구입니다. 명령줄 또는 Windows GUI를 통해 자막 생성, 실시간 추출, 번역 및 DVD 폐쇄자막 변환 기능을 제공하며, 주로 C 언어로 개발되었습니다. 다양한 운영체제와 패키지 관리자를 지원하여 접근성을 높였으며, 자막이 필요한 학습자나 청각 장애인에게 유용한 솔루션을 제공합니다. GNU GPL-2.0 라이선스로 배포됩니다.
