AirSim은 드론, 자동차 등 자율 시스템을 위한 고성능 시뮬레이션 플랫폼으로, 언리얼 엔진과 Unity 기반으로 개발되었습니다. AI 연구 및 실험을 위해 딥러닝, 컴퓨터 비전, 강화학습 알고리즘 테스트에 최적화된 환경을 제공하며, PX4/ArduPilot 등 실제 비행 컨트롤러와 호환되는 소프트웨어/하드웨어 인더루프 시뮬레이션을 지원합니다. 마이크로소프트 리서치가 2017년 출시한 오픈소스 프로젝트로, 2022년 이후 아카이브되며 후속 프로젝트인 Project AirSim으로 전환되었습니다.  

> **프로젝트 상태**: 아카이브 예정 (2023년 기준)  
> **라이선스**: MIT License  
> **특징**: 다중 플랫폼 지원, 물리/시각 정확도 시뮬레이션, API 기반 제어, 컴퓨터 비전 모드, 날씨 효과, 학습 데이터 수집 기능 제공.
