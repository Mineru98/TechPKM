이 프로젝트는 oobabooga/text-generation-webui의 도커화된 배포 솔루션입니다. Nvidia/AMD/Intel GPU 및 CPU 환경을 지원하며, 기본 구성과 확장 기능이 없는 경량 버전을 제공합니다. 사용자 정의 가능한 docker-compose 설정과 확장 가능한 확장 기능 관리를 통해 효율적인 배포와 업데이트가 가능합니다. 주요 사용 사례는 로컬 텍스트 생성 모델 호스팅 및 다양한 GPU 플랫폼에서의 추론 가속화입니다.
