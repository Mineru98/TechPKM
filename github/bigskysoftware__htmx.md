htmx는 HTML에서 직접 AJAX, CSS 전환, 웹소켓 및 서버 전송 이벤트를 활용할 수 있도록 하는 경량 자바스크립트 라이브러리입니다. 단순한 HTML 속성 추가로 현대적인 사용자 인터페이스를 구축할 수 있으며, 의존성 없이 확장 가능한 구조를 제공합니다. 기존 HTML의 한계를 극복하여 하이퍼텍스트의 잠재력을 완성하는 것이 목표입니다. 위키피디아의 "HATEOAS" 개념과 REST 아키텍처 철학을 기반으로 설계되었습니다.
