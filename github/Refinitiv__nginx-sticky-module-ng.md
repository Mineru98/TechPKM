이 프로젝트는 Nginx 서버에서 동일한 클라이언트를 항상 동일한 백엔드 서버로 라우팅하기 위한 스티키 쿠키 모듈을 제공합니다. IP 기반 해시의 단점을 보완하여 쿠키를 통해 세션 지속성을 구현하며, 로드 밸런싱 환경에서 브라우저별 고유 식별이 필요한 경우 유용합니다. 모듈은 다양한 해시 알고리즘과 쿠키 설정 옵션을 지원하며, 보안 및 성능 관련 플래그를 제공합니다. Nginx 소스 컴파일 시 추가 모듈로 포함되어야 합니다.
