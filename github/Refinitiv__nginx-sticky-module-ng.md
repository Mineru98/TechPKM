이 프로젝트는 Nginx 서버에 세션 지속성을 제공하는 모듈을 구현합니다. 동일한 클라이언트가 항상 같은 백엔드 서버로 라우팅될 수 있도록 스티키 쿠키를 생성 및 관리합니다. IP 해시 방식의 한계를 보완하기 위해 쿠키 기반 라우팅 메커니즘을 사용하며, 로드 밸런싱 시스템에서 프록시 환경에도 안정적인 세션 유지가 가능합니다. 보안보다는 정상 사용자의 일관된 백엔드 연결을 목표로 합니다.  

> 요약 핵심:  
1) 쿠키 기반 세션 지속성 모듈  
2) IP 해시 방식의 프록시 문제 해결  
3) 다양한 해시 알고리즘 및 보안 옵션 지원  
4) Nginx 소스 컴파일 필요
