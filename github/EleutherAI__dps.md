DPS(Data Processing System)는 RDD 기반 아파치 스파크 작업을 처리하기 위한 프레임워크로, JSON 데이터 샘플링, 중복 제거, 한국어 정제 등의 작업을 수행합니다. YAML 설정 파일을 통해 파라미터를 관리하며, 사용자는 새로운 작업 모듈을 쉽게 추가할 수 있습니다. 파이썬 3.8 환경에서 실행되며, 스파크 애플리케이션 배포 및 테스트를 위한 포괄적인 가이드를 제공합니다.
