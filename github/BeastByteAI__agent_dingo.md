Agent Dingo는 LLM 기반 파이프라인 및 에이전트 구축을 위한 마이크로프레임워크입니다. 간결한 설계로 프로덕션 수준의 LLM 애플리케이션 개발을 지원하며, 유연한 파이프라인 구성과 프로세스 제어 기능을 제공합니다. Python 기반으로 OpenAI 등 외부 모델과의 통합을 지원하며, 예제 문서는 공식 문서에서 확인할 수 있습니다.
