LLM-Aided OCR 프로젝트는 광학 문자 인식(OCR) 결과의 품질을 향상시키기 위해 대규모 언어 모델(LLM)과 자연어 처리 기술을 결합한 시스템입니다. PDF 문서를 이미지로 변환한 후 Tesseract OCR로 텍스트를 추출하고, LLM을 활용해 오류를 수정하며 마크다운 형식으로 포맷팅합니다. 로컬/클라우드 기반 LLM 지원, 비동기 처리, GPU 가속 등의 기능을 제공하여 정확하고 가독성 있는 문서를 생성합니다.  

이 프로젝트는 OCR 오류 수정, 헤더/페이지 번호 억제, 중복 내용 제거 등의 기능을 통해 원본 스캔 문서의 품질을 크게 향상시킵니다. OpenAI, Anthropic API 또는 로컬 LLM을 유연하게 사용할 수 있으며, 토큰 관리 및 품질 평가 시스템을 내장하고 있습니다. 과학 논문, 역사적 문서, 비즈니스 리포트 등 다양한 분야의 문서 처리에 적합합니다.
