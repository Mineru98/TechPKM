이 프로젝트는 OpenCode 편집기를 위한 확장 플러그인으로, AI 에이전트 기반 코드 자동화 및 생산성 향상 도구를 제공합니다. 주요 기능으로 Sisyphus 에이전트를 중심으로 한 다중 모델 오케스트레이션, LSP/AST 도구 통합, 자동화된 배경 작업 실행, 커스텀 MCP(모델 호출 프로토콜) 지원 등을 포함하며, "ultrawork" 키워드로 모든 고급 기능을 자동 활성화할 수 있습니다. 오픈소스 개발자 도구 생태계를 혁신적으로 개선하는 것을 목표로 합니다.
