이 프로젝트는 GPT 모델의 "Pro Mode" 기능을 구현한 것으로, 사용자 프롬프트를 기반으로 다중 생성을 합성하여 향상된 답변을 제공합니다. Colab 노트북과 API 서버를 통해 접근 가능하며, `num_gens` 매개변수에 따라 단일/토너먼트 모드 생성이 가능합니다. 주요 사용 사례는 AI 모델의 응답 품질을 개선하는 프롬프트 엔지니어링입니다. Python 기반 FastAPI로 구현되어 로컬에서 직접 실행하거나 클라우드 환경에서 사용할 수 있습니다.
