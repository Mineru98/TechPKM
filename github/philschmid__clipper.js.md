Clipper는 웹 페이지의 콘텐츠를 클리핑하고 Markdown으로 변환하는 Node.js 기반 CLI 도구입니다. Mozilla의 Readability 라이브러리와 Turndown을 활용해 웹 콘텐츠를 분석하고 추출하며, 터미널에서 직접 사용할 수 있는 간단한 아카이빙 및 노트 작성 솔루션입니다. 브라우저 확장 프로그램 없이도 URL 또는 HTML 파일에서 콘텐츠를 추출할 수 있으며, JSON 형식으로 출력하거나 웹 크롤링 기능도 지원합니다. PDF를 HTML로 변환한 후 Markdown으로 처리하는 등의 확장 사용 사례도 가능합니다.
