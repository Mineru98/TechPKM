이 프로젝트는 한국어 개체명 인식(NER)을 위해 BERT와 CRF를 결합한 PyTorch 구현체입니다. KoBERT 사전 학습 모델과 CRF 레이어를 활용하여 인물, 지명, 기관명, 날짜 등 다양한 개체명을 인식하며, 한국어 자연어 처리 작업에 최적화된 모델을 제공합니다. PyTorch v1.2 및 Python 3.x 기반으로 개발되었습니다.
