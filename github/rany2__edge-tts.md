`edge-tts`는 Microsoft Edge의 온라인 텍스트-음성 변환(TTS) 서비스를 Python 코드 또는 커맨드 라인에서 사용할 수 있게 해주는 프로젝트입니다. 간단한 커맨드로 음성을 생성하거나 Python 모듈로 직접 활용할 수 있으며, 다양한 언어/음성 옵션, 속도/볼륨/피치 조절이 가능합니다. 음성 파일 생성 및 자막 추출 기능도 지원합니다.

태그: 기술 스택(Python), 사용 목적(TTS), 문제 영역(Microsoft Edge 음성 서비스)  
에일리어스: 주요 명령어 및 서비스 특징을 반영한 대안 검색어 제공
