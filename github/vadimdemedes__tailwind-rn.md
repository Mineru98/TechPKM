이 프로젝트는 React Native 환경에서 Tailwind CSS를 사용할 수 있게 해주는 라이브러리입니다. Tailwind의 출력을 기반으로 모든 스타일을 생성하므로 항상 최신 상태를 유지하며, JIT 모드, 반응형 브레이크포인트, 다크 모드, 사용자 정의 설정 등을 지원합니다. 사용자는 Tailwind의 유틸리티 클래스를 React Native 구성 요소에 직접 적용할 수 있어 일관된 스타일링이 가능합니다.
