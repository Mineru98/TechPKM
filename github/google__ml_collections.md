ML Collections는 머신러닝 실험 및 모델 설정을 위한 Python 컬렉션 라이브러리입니다. `ConfigDict`와 `FrozenConfigDict` 클래스를 통해 중첩된 요소에 대한 점(.) 접근과 타입 안정성, 지연 계산, 불변성 등의 기능을 제공합니다. 실험 구성 관리, 설정 검증, YAML 형식의 가독성 있는 출력이 주요 특징입니다. Google에서 개발되었으며, PyPI와 PyTest를 통해 배포 및 테스트됩니다.
