ML Collections는 머신러닝 실험을 위한 설정 관리를 목적으로 설계된 Python 컬렉션 라이브러리입니다. 주요 기능인 `ConfigDict`와 `FrozenConfigDict`는 점(`.`) 접근 방식, 타입 안전성, 지연 계산, 불변성 등의 특징을 제공하며, 설정의 유연한 조작과 명확한 관리를 지원합니다. 이 라이브러리는 YAML 형식의 가독성 있는 출력을 생성하고, 참조 기반 필드 공유 및 명령줄 오버라이딩 기능을 포함합니다. Google에서 개발되었으며, 실험 구성 및 모델 설정 표현에 주로 사용됩니다.
