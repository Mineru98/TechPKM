이 프로젝트는 합격 자기소개서 데이터를 분석하여 취준생들에게 키워드 추천, 단어 추천, 중요 문장 추출, 소제목 작성 지원 등의 기능을 제공합니다. LDA, LSTM, GPT2 등 다양한 머신러닝/딥러닝 모델을 활용하여 자기소개서 작성 과정을 체계적으로 지원하는 것이 핵심 목적입니다. 크롤링부터 전처리, 모델링까지 전 과정을 포함하며 SKT-AI의 KoGPT2와 같은 한국어 특화 모델을 활용했습니다.
