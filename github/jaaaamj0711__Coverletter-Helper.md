취업을 준비하는 취준생들을 위한 자기소개서 작성 지원 프로젝트입니다. 합격 자기소개서를 분석하여 키워드 추천, 막힌 단어 대체, 중요 문장 추출, 소제목 생성 등의 기능을 제공합니다. LDA, RNN, LSTM, GPT-2 등 다양한 자연어 처리 모델을 활용해 취업 트렌드에 맞는 자기소개서 작성을 지원합니다.  

이 프로젝트는 크롤링, 전처리, 키워드 분석, 문장 생성 등의 기능을 포함하며, KoGPT2 및 FastText와 같은 한국어 특화 모델을 활용합니다. 취업을 준비하는 학생들이 자기소개서 작성에 어려움을 겪는 문제를 해결하기 위한 데이터 기반 솔루션입니다.
