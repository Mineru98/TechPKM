Claude Composer는 Claude Code의 사용성을 향상시키는 CLI 도구입니다. 자동 확인 수락, 도구 관리, 시스템 알림 등의 기능을 제공하여 작업 흐름을 개선합니다. 프로젝트 초기화, YOLO 모드(자동 수락), 도구셋 구성 등을 지원하며, Node.js 환경에서 실행됩니다. 현재는 유지보수가 중단되었으나, 포킹 또는 지속적인 사용이 가능합니다.
