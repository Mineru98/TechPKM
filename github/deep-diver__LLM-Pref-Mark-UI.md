LLM-Pref-Mark-UI는 생성 텍스트에 대한 인간 피드백 선호도를 표시하기 위한 Gradio UI를 제공하는 프로젝트입니다. RLHF(인간 피드백 기반 강화학습)를 위한 보상 모델 훈련에 활용될 수 있으며, 기본 버전과 고급 버전의 두 가지 인터페이스를 포함합니다. 기본 버전은 Flan Alpaca 모델을 데모로 사용하며, 사용자 정의 모델 적용 및 선호도 기록 처리 기능을 제공합니다. 고급 버전은 확장된 UI 구조를 제시합니다. 주요 기술적 요소는 Python, Gradio, 생성 모델 평가입니다.
