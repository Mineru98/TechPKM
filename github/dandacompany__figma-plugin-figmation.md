Figmation은 Figma 디자인 워크플로우를 외부 자동화 시스템에서 제어할 수 있도록 하는 React 기반 Figma 플러그인입니다. n8n 워크플로우와 WebSocket을 통해 연결되어 AI 에이전트(Claude/GPT/Gemini 등)의 자연어 명령을 Figma API 작업으로 변환 및 실행합니다. 55개 이상의 Figma API 명령어를 지원하며, 실시간 통신, 채널 기반 프로젝트 분리, 자동 재연결 기능을 갖춘 안정적인 디자인 자동화 솔루션입니다. 주요 기술 스택으로 Vite + Plugma, React 18, TypeScript를 사용합니다.
