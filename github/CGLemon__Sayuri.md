**Sayuri**는 GTP 호환 바둑 엔진으로, 심층 합성곱 신경망(DCNN)과 몬테카를로 트리 검색(MCTS)을 기반으로 합니다. Leela Zero와 KataGo에서 영감을 받아 개발되었으며, AlphaZero 스타일 알고리즘을 통해 인간 전략 없이 바둑을 학습합니다. 다양한 규칙 세트(Japanese/Territory), 콤미 설정, 보드 크기를 지원하며, 효율적인 강화 학습 시스템을 특징으로 합니다.  

단일 GPU로도 3개월 만에 훈련이 완료될 정도로 계산 효율성이 뛰어나며, C++ 및 Python 버전을 제공합니다. Sabaki 또는 GoGui와 같은 GTP 기반 GUI 도구와 함께 사용할 수 있습니다.
