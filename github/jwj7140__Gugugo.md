Gugugo는 한국어-영어 및 한국어-일본어 양방향 번역을 위한 대규모 언어 모델 프로젝트로, Polyglot-ko와 Llama-2-ko를 기반으로 개발되었습니다. QLoRA 기술을 활용해 다양한 버전을 출시하며, AIHUB 및 HuggingFace 데이터셋을 기반으로 장문 번역 성능을 지속적으로 개선하고 있습니다. 현재 영어-한국어 번역에서는 BLEU 점수가 측정 중이며, 일본어-한국어 번역에서는 27.83점을 기록했습니다. 웹 인터페이스를 통해 모델을 직접 사용할 수 있습니다.
