이 프로젝트는 한컴 한글의 HWP 파일을 XML 기반 HWPX 형식으로 변환하는 Java 라이브러리입니다. 개인 및 상업적 사용이 가능한 Apache-2.0 라이선스로 배포되며, 문서 구조 분석을 통해 정확한 변환 기능을 제공합니다. hwplib/hwpxlib 라이브러리를 기반으로 개발되어 표 스타일, 이미지 처리 등 핵심 기능을 지원합니다.  

(참고: README에 명시된 최신 커밋 날짜(2026.1.20)는 프로젝트 활성 상태를 반영하나, 실제 유효성은 확인 필요)
