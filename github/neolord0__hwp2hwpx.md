이 프로젝트는 한글 워드프로세서 파일 형식인 HWP를 차세대 형식인 HWPX로 변환하는 Java 라이브러리입니다. hwplib과 hwpxlib을 기반으로 개발되었으며, 주로 문서 마이그레이션이나 호환성 개선 목적으로 사용됩니다. Apache-2.0 라이선스로 자유롭게 활용할 수 있지만, 상용 지원 요청은 별도 협의가 필요합니다. 주요 기능으로는 표 셀의 배경색 설정 오류 수정, 포함된 이미지 파일명 생성 로직 개선 등이 있습니다.
