n8n-MCP는 n8n 워크플로우 자동화 플랫폼과 AI 모델 간의 연결을 가능하게 하는 Model Context Protocol(MCP) 서버입니다. 1,084개 n8n 노드(537개 코어 + 547개 커뮤니티)의 구조화된 문서, 속성, 작업에 대한 접근을 제공하며, Claude 등의 AI 어시스턴트가 n8n 노드 정보를 효과적으로 이해하고 활용할 수 있도록 지원합니다. 빠른 배포(호스팅 또는 자체 호스팅 옵션)와 실시간 검증 기능을 통해 AI 기반 워크플로우 생성을 최적화합니다. 주요 기능으로는 노드 속성 검색, 템플릿 기반 워크플로우 생성, AI 에이전트 검증, 실시간 실행 테스트 등이 포함됩니다.
