n8n-MCP는 n8n 워크플로우 자동화 플랫폼과 AI 모델 간의 연결을 제공하는 MCP(Model Context Protocol) 서버입니다. 1,084개의 n8n 노드(코어 537개 + 커뮤니티 547개)에 대한 포괄적인 문서, 속성, 작업 정보를 제공하여 Claude 등의 AI 어시스턴트가 n8n 노드를 효과적으로 이해하고 활용할 수 있도록 지원합니다. 사전 구축된 데이터베이스와 다양한 배포 옵션(로컬, Docker, 클라우드)을 통해 AI 기반 워크플로우 생성 및 관리를 간소화합니다. AI 도구 검증, 실시간 검증, 템플릿 기반 자동화를 통해 AI 생성 워크플로우의 신뢰성을 확보합니다.
