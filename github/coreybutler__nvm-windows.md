Windows 환경에서 여러 Node.js 버전을 관리할 수 있는 도구입니다. Microsoft, npm, Google에서 권장하는 Windows용 Node.js 버전 관리자로, 심볼릭 링크를 활용하여 시스템 재부팅 없이 즉시 버전을 전환할 수 있습니다. 관리자 권한으로 실행해야 하며, 기존 Node.js 설치 제거 후 설치를 권장합니다. 주요 기능으로는 특정 버전 설치/전환, 글로벌 모듈 관리, 미러 설정 등이 있습니다. Go 언어로 작성되어 Windows에 최적화되었습니다.
