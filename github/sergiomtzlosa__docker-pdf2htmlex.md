이 프로젝트는 PDF 파일을 고품질 HTML 문서로 변환하는 Docker 이미지를 제공합니다. pdf2htmlEX 도구를 컨테이너화하여 로컬 환경에서 간편하게 PDF를 HTML로 변환할 수 있도록 지원하며, 오프라인 빌드 및 설치 옵션도 포함되어 있습니다. 주요 사용 사례는 학술 논문, 보고서 등의 디지털 변환 작업이며, 단일 HTML 파일 또는 독립형 변환 환경을 구성할 수 있습니다. Docker 명령어 또는 별칭(alias)을 통해 쉽게 실행할 수 있는 것이 특징입니다.
