KR-WordRank는 한국어 텍스트에서 비지도 방식으로 키워드와 핵심 문장을 추출하는 Python 라이브러리입니다. Substring 그래프와 PageRank 기반 알고리즘을 활용하여 키워드를 추출하며, 추출된 키워드를 기반으로 핵심 문장을 선택할 수 있습니다. 영화 평론 데이터를 활용한 키워드 추출 및 시각화 예제가 포함되어 있습니다. 한국어 텍스트 분석에 특화된 기능을 제공합니다.
