이 프로젝트는 스레즈(threads.net) 플랫폼의 비공식 파이썬 API 래퍼로, 교육적 목적으로 역공학을 통해 개발되었습니다. 2023년 9월 19일 기준 메타 플랫폼의 법적 통보로 개발이 중단되었으며, 현재는 유지보수가 이루어지지 않습니다. 스레즈와 인스타그램 연동 기능을 연구 목적으로 구현했으나, 서비스 약관 위반 가능성으로 인해 사용에 주의가 필요합니다. 문서화는 [PRESERVED.md](https://github.com/dmytrostriletskyi/threads-net/blob/main/PRESERVED.md)에 보존되어 있습니다.
