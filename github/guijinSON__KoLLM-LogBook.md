KoLLM-LogBook은 15개 분야 100개의 프롬프트로 다양한 언어 모델의 한국어 생성 능력을 비교하는 프로젝트입니다. Finance, Medical, Programming 등 분야별 모델 답변을 시각적으로 대조하며, VLLM 패키지로 생성된 결과물을 제공합니다. 원본 LLM-Logbook의 한국어 버전으로, 모델별 지식 표현력과 사고 과정을 체계적으로 분석할 수 있는 벤치마크 데이터셋 역할을 수행합니다. Streamlit 기반 리포트 페이지에서 직접 비교 결과를 확인할 수 있습니다.
