**KoLLM-LogBook**은 15개 분야(금융, 수학, 의료 등)에 걸쳐 100개의 프롬프트를 활용해 다양한 언어 모델의 답변을 비교 평가하는 프로젝트입니다. 한국어 생성 능력을 중심으로 모델의 지식, 사고 과정, 창의성을 체계적으로 분석하며, VLLM 패키지로 생성된 결과를 포함합니다. 오픈소스 LLM-Logbook의 한국어 버전으로, 언어 모델 간 성능 차이를 직관적으로 확인할 수 있는 벤치마크 자료를 제공합니다.
