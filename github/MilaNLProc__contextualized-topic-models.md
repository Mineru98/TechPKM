Contextualized Topic Models(CTM)는 사전 훈련된 언어 모델(예: BERT)을 활용한 토픽 모델링 라이브러리입니다. CombinedTM과 ZeroShotTM 두 가지 모델을 제공하며, 각각 문맥 임베딩과 BoW 결합, 제로샷/다국어 토픽 모델링을 지원합니다. Kitty 서브모듈로 인간 참여 분류도 가능하며, SBERT를 기반으로 다양한 언어 모델에 적용 가능합니다. 논문 기반의 기술 구현과 Colab 튜토리얼, 다국어 처리 기능을 포함합니다.
