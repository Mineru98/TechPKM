Dify는 LLM 애플리케이션 개발을 위한 오픈소스 플랫폼으로, 직관적인 인터페이스를 통해 에이전트 기반 AI 워크플로우, RAG 파이프라인, 모델 관리, 관측 가능한 기능 등을 제공합니다. 사용자는 프로토타입부터 프로덕션까지 빠르게 진행할 수 있으며, 클라우드 서비스와 자체 호스팅 옵션을 모두 지원합니다. 주요 기능으로는 시각적 워크플로우 제작, 다양한 LLM 지원, 프롬프트 IDE, 문서 처리 RAG 파이프라인, 에이전트 도구 통합, LLMOps 모니터링, API 기반 백엔드 서비스 등이 포함됩니다. Docker 또는 소스 코드 기반 설치 방식을 선택하여 빠르게 시작할 수 있습니다.
