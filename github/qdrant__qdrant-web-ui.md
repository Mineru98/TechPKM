이 프로젝트는 Qdrant 벡터 검색 엔진을 위한 셀프 호스팅 웹 UI로, 컬렉션 관리 및 시각화 기능을 제공합니다. Qdrant 자체 또는 독립 애플리케이션으로 배포 가능하며, 별도의 추가 서비스 없이 직관적인 인터페이스를 제공하는 것이 주목적입니다. React, MUI, Axios 기술 스택을 사용하여 개발되었습니다.
