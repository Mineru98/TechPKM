Qdrant 벡터 검색 엔진을 위한 자체 호스팅 웹 UI입니다. 주요 목적은 컬렉션의 시각적 확인 및 관리를 제공하는 것이며, Kibana와 유사한 기능을 추가 서비스 없이 지원합니다. React, MUI, Axios 기술 스택으로 개발되어 Qdrant와 통합하거나 독립 실행형으로 사용할 수 있습니다.
