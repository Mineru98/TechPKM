Sapling SCM은 수백만 개의 파일과 커밋을 처리할 수 있는 확장성을 갖춘 Git 호환 소스 컨트롤 시스템입니다. 사용자 친화적인 인터페이스와 강력한 기능을 제공하며, 대규모 저장소에서도 빠른 개발자 경험을 보장합니다. 주요 구성 요소로는 CLI 도구, Mononoke 서버, EdenFS 가상 파일 시스템이 포함됩니다. 메타 내부에서 사용 중이지만 외부 지원은 아직 제공되지 않습니다.
