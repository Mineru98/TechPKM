이 프로젝트는 PDF, HTML, Word 문서 등 다양한 형식의 비정형 데이터(주로 문서)를 구조화하기 위한 오픈소스 라이브러리를 제공합니다. LLM(대형 언어 모델) 기반 애플리케이션의 데이터 처리 워크플로우 최적화를 목표로 하며, 모듈식 설계와 커넥터를 통해 다양한 플랫폼에서의 데이터 수집과 전처리를 간소화합니다. 문서 분할(partitioning), 파일 유형 감지, 텍스트/이미지 추출 등의 기능을 지원하며, Docker 컨테이너 또는 Python 패키지로 사용할 수 있습니다.  

주요 특징:  
1. **다양한 문서 형식 지원** (PDF, HTML, DOCX, PPTX, EPUB 등)  
2. **구조화된 출력 생성** (텍스트, 표, 이미지 등 요소 분할)  
3. **LLM 파이프라인 통합 용이성**  
4. **커스터마이징 가능한 모듈식 아키텍처**  
5. **로컬 개발 및 컨테이너 환경 지원**  

사용 사례: 연구 논문 분석, 계약서 처리, 대량의 스캔 문서 디지털화 등에 적합합니다.
