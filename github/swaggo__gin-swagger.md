gin-swagger는 Go 언어로 작성된 Gin 웹 프레임워크용 미들웨어로, Swagger 2.0 기반의 RESTful API 문서를 자동 생성하는 프로젝트입니다. 주석에 기반한 API 설명과 Swagger UI를 통해 직관적인 문서화를 제공하며, 프로젝트 구조에 따라 유연하게 문서를 생성할 수 있습니다. 핵심 기능은 주석 파싱, 문서 생성, Swagger UI 서빙입니다.
