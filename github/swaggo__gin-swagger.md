gin-swagger는 Gin 웹 프레임워크에서 Swagger 2.0 기반의 RESTful API 문서를 자동 생성하기 위한 미들웨어입니다. Go 코드 주석 기반으로 API 사양을 파싱하여 Swagger UI를 제공하며, 복잡한 프로젝트 구조와 다중 API 설정도 지원합니다. `swag` 도구와 연동하여 문서 생성과 관리를 간소화합니다. 주요 기능으로는 문서 확장 깊이 설정, OAuth2 인증 지원, 인스턴스별 문서 분리 등이 있습니다. Gin 기반 프로젝트에서 API 문서화를 빠르게 적용할 수 있는 솔루션입니다.
