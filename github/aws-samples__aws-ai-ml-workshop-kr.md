AWS AI/ML 워크샵 프로젝트는 한국어 사용자를 위한 생성형 인공지능 및 머신러닝 예제 리포지토리입니다. AWS Generative AI 응용 사례, SageMaker 기반 모델 훈련, AWS Neuron(인퍼런시아/트레이니엄) 활용 예제를 포함하며, 각 기술별 실습 환경을 제공합니다. MIT 라이선스 하에 공개되어 커뮤니티 기여를 환영합니다. 주요 구성 요소는 생성형 AI 모델 배포/파인튜닝, SageMaker 입문/하이퍼파라미터 최적화, Neuron 하드웨어 가속 기술 탐구로 분류됩니다.
