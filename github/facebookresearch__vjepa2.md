V-JEPA 2는 인터넷 규모의 비디오 데이터를 활용한 자기지도 학습 비디오 인코더로, 동작 이해 및 인간 행동 예측 작업에서 최첨단 성능을 달성합니다. V-JEPA 2-AC는 소량의 로봇 데이터만으로 미세 조정되어 새로운 환경에서 로봇 조작 작업을 수행할 수 있는 세계 모델입니다. 주요 기능으로는 비디오 예측, 로봇 계획, 다중 작업 평가(EK100, SSv2, MVP 등)가 포함됩니다. PyTorch 및 Hugging Face를 통해 모델 체크포인트와 평가 도구를 제공합니다.
