이 프로젝트는 1비트 가중치 양자화를 적용한 "BitNet" 트랜스포머 모델을 허깅페이스 트랜스포머스로 구현한 것입니다. Llama(2) 아키텍처를 기반으로 하여 1비트 연산과 메모리 효율성을 달성하며, 특히 대규모 언어 모델의 경량화에 초점을 맞추고 있습니다. 위키텍스트-103 데이터셋으로 훈련할 수 있으며, 다양한 비트 구성(16비트, 8비트, 1비트)을 지원합니다. GPU 메모리 사용량을 크게 줄이면서도 모델 성능을 유지하는 것이 핵심 목표입니다.
