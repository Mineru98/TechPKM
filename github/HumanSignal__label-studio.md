라벨 스튜디오는 오픈소스 데이터 라벨링 도구로, 이미지, 텍스트, 오디오, 비디오, 시계열 데이터 등 다양한 데이터 유형을 직관적인 UI로 라벨링하고 다양한 ML 모델 형식으로 내보낼 수 있습니다. 로컬 설치 또는 클라우드 배포가 가능하며, 사용자 정의 템플릿과 머신러닝 모델 통합 기능을 제공하여 정확한 ML 모델 구축을 지원합니다. 멀티유저 프로젝트 관리, 클라우드 스토리지 연동, REST API 통합이 주요 특징입니다.
