이 프로젝트는 Claude Code 및 Codex AI 코딩 도구를 모바일 및 웹에서도 안전하게 사용할 수 있게 하는 클라이언트 도구입니다. 컴퓨터 CLI에서 실행 중인 AI 세션을 모바일 기기로 원격 제어할 수 있으며, 모든 통신은 엔드투엔드 암호화로 보호됩니다. iOS/Android 앱, 웹 인터페이스, CLI 및 백엔드 서버로 구성된 오픈소스 프로젝트입니다. 사용자는 키보드 한 번 입력으로 기기 간 전환 또는 푸시 알림을 통해 AI 작업 상태를 확인할 수 있습니다.
