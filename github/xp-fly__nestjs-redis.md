이 프로젝트는 NestJs 애플리케이션에서 Redis를 사용할 수 있도록 하는 ioredis 기반의 모듈입니다. Redis 연결 설정과 여러 클라이언트 관리를 위한 유연한 API를 제공하며, 동기/비동기 방식의 설정 옵션을 지원합니다. 데코레이터를 활용한 Redis 클라이언트 주입 기능이 특징입니다.
