Vibe Kanban은 AI 코딩 에이전트(예: Claude Code, Gemini CLI 등)의 작업을 효율적으로 관리하고 조율하기 위한 웹 기반 도구입니다. 여러 에이전트를 병렬/순차적으로 실행하거나 작업 상태 추적, 코드 리뷰, 원격 프로젝트 접근 등의 기능을 제공합니다. 개발자와 팀이 AI 에이전트의 작업을 중앙에서 구성하고 모니터링할 수 있도록 설계되었습니다. Rust와 Node.js로 구현된 오픈소스 프로젝트이며, 자체 호스팅 및 커스텀 도메인 설정이 가능합니다.
