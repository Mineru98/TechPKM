바이브 칸반은 Claude Code, Gemini CLI 등 다양한 AI 코딩 에이전트를 통합 관리하는 오픈소스 개발자 도구입니다. 주요 기능으로 다중 에이전트 병렬/순차 작업 오케스트레이션, 작업 상태 추적, 원격 서버 프로젝트 관리, 중앙 집중식 MCP 구성 관리 등을 제공합니다. 로컬 또는 원격 환경에서 개발 워크플로우 효율성을 높이기 위해 설계되었으며, VSCode와의 원격 SSH 통합도 지원합니다. Rust와 Node.js 기반으로 구축되었으며, 개발자 도구의 현대적인 관리 방식을 제시합니다.
