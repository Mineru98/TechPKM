LMQL은 대규모 언어 모델(LLM)과 연동하기 위한 Python 확장 프로그래밍 언어입니다. 기존 Python 문법에 LLM 호출 기능을 통합해 자연어 프롬프트와 제어 로직을 결합한 프로그래밍이 가능하며, 제약 조건과 고급 디코딩 알고리즘을 지원합니다. OpenAI, HuggingFace 모델과의 호환성과 비동기 API를 통해 효율적인 LLM 활용이 가능하며, JSON 스키마 보장, 알고리즘 프롬프팅 등 다양한 응용 사례를 제공합니다.  

> 요약: Python 기반 LLM 통합 언어 (제너레이션 제어, 제약 조건, 멀티모델 지원)
