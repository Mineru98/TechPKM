LMQL은 Python을 확장한 프로그래밍 언어로, 대화형 대규모 언어 모델(LLM)을 코드 수준에서 직접 제어할 수 있는 환경을 제공합니다. 전통적인 알고리즘과 LLM 호출을 결합한 프로그램 작성이 가능하며, 제약 조건 지정 및 고급 디코딩 기법(예: 빔 서치)을 통해 모델 출력을 정교하게 조절할 수 있습니다. OpenAI API, 🤗 Transformers 등 다양한 모델과 호환되며, 비동기 API와 대화형 IDE를 지원합니다.  

> 📌 **핵심 특징**: Python 문법 기반, 조건 제어 흐름, 로그it 마스킹 제약 조건, 실시간 스트리밍 출력, LangChain/LlamaIndex 통합, 로컬/클라우드 모델 지원.
