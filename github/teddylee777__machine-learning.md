이 프로젝트는 머신러닝과 딥러닝을 스스로 학습하기 위한 개인 스터디 저장소입니다. 파이썬, 데이터 분석(Pandas/Numpy), 시각화(Matplotlib/Seaborn), 수학/통계 기초, 머신러닝 알고리즘, 딥러닝 프레임워크(PyTorch/TensorFlow), 최적화/강화학습, 자연어 처리, 컴퓨터 비전 등 다양한 주제의 학습 자료와 동영상 강의 링크를 체계적으로 정리했습니다. 온라인 공유 자료와 GitHub 저장소를 참고하여 스터디할 수 있도록 구성되었으며, TensorFlow 자격증, 빅데이터 분석기사, LLM(Large Language Model) 등 최신 기술 트렌드도 포함합니다. 오픈소스 프로젝트와 캐글/데이콘 대회 자료도 제공되어 실무 적용에 도움을 줍니다.
