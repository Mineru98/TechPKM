이 저장소는 머신러닝과 딥러닝을 독학으로 학습하는 사람들을 위한 종합 가이드입니다. Python을 기반으로 데이터 분석, 시각화, 수학/통계 기초, 머신러닝(선형회귀, SVM, KNN, 결정트리 등), 딥러닝(CNN, RNN, GAN 등) 및 관련 최신 기술(LLM, LangChain, AutoML 등)에 대한 체계적인 학습 자료를 제공합니다. 캐글/데이콘 대회 준비, 논문 리뷰, 오픈소스 라이브러리 활용법, 현업 데이터 사이언티스트 인터뷰 등 실무 중심의 학습 경로를 구성했습니다. 초보자부터 고급자까지 단계별 학습이 가능하도록 유튜브 강의, 블로그, 깃헙 저장소, 온라인 강의 링크를 체계적으로 분류했습니다.
