이 프로젝트는 Jina AI와 FastAPI를 활용하여 LangChain 애플리케이션을 프로덕션 환경에 빠르게 배포할 수 있도록 지원하는 도구입니다. Jina AI 클라우드나 자체 인프라에 REST/WebSocket API, Slack 봇, FastAPI 앱을 배포할 수 있으며, 서버리스 아키텍처, 자동 확장, 보안 기능 등을 제공합니다. AutoGPT, BabyAGI, pandas-ai 등의 LLM 기반 애플리케이션을 한 번의 명령으로 서비스로 실행할 수 있습니다.  

> **중요 공지**: 이 저장소는 더 이상 유지보수되지 않습니다.
