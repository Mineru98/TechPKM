**langchain-serve**는 LangChain 애플리케이션을 Jina AI Cloud에 빠르게 배포할 수 있도록 하는 오픈소스 도구입니다. 단일 명령어로 REST/Websocket API, LLM 기반 슬랙 봇, FastAPI 패키지를 클라우드 또는 온프레미스 환경에서 구축할 수 있으며, 서버리스 아키텍처와 자동 확장 기능을 지원합니다. 데이터 프라이버시를 위해 자체 인프라 배포도 가능합니다.  

> **중요**: 이 저장소는 더 이상 유지보수되지 않습니다.
