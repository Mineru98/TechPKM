**n8n 워크플로우 빌더 MCP 서버**는 AI 어시스턴트(예: Claude Desktop, ChatGPT)를 n8n 인스턴스와 연결하여 자연어 명령으로 워크플로우를 생성, 관리, 실행할 수 있는 MCP(Model Context Protocol) 서버입니다. n8n의 공식 API와 인증을 사용하여 워크플로우의 전체 생명주기(생성, 실행, 활성화/비활성화, 삭제)를 AI 어시스턴트로 제어할 수 있습니다. Node.js v18 이상과 n8n API 키가 필요하며, Smithery.ai 호스팅 또는 로컬 NPX 설치로 간편하게 사용할 수 있습니다. 주요 기능으로는 워크플로우 태그 관리, 실행 기록 조회, 보안 감사 보고서 생성 등이 포함됩니다.
