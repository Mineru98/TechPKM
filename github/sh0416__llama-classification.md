이 프로젝트는 LLaMA 모델을 활용한 기본 텍스트 분류 코드베이스를 제공합니다. AG News 데이터셋에서 직접 확률 비교, 채널 기반 조건부 확률, 생성 기반 방법 등 다양한 분류 기법을 실험하며, 보정 기법을 통한 성능 향상도 포함합니다. GPU 환경에서 구현되었으며 실험 결과(정확도, 추론 시간)와 향후 개선 과제를 명시합니다.  

(요약: LLaMA를 활용한 텍스트 분류 시스템으로, 다양한 추론 방법론과 실험 결과를 포함한 코드베이스. 자연어 처리 연구 및 적용 참고용으로 구성됨)
