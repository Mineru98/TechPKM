ReadabiliPy는 Mozilla의 Readability.js Node.js 패키지를 Python에서 사용할 수 있도록 래핑한 라이브러리입니다. 웹 페이지에서 기사 본문을 추출하는 기능을 제공하며, 순수 Python 기반 추출 방식도 포함되어 있습니다. 명령줄 도구와 Python 라이브러리 형태로 사용 가능하며, 추출된 기사의 제목, 저자, HTML/텍스트 콘텐츠 등을 반환합니다. Node.js 설치가 필요하지 않은 Python 전용 파서도 지원합니다.  

이 프로젝트는 웹 스크래핑 및 텍스트 분석 작업에 최적화되어 있으며, MIT 라이선스로 배포됩니다. 주요 기능으로 텍스트 정규화(NFKC), 콘텐츠 해시 생성, 노드 인덱스 추적 등을 제공합니다.
