ReadabiliPy는 Mozilla의 Readability.js를 Python으로 래핑한 패키지로, HTML 문서에서 기사 내용을 추출하는 기능을 제공합니다. Node.js 기반 Readability.js와 순수 Python 기반 파서를 모두 지원하며, CLI 또는 라이브러리로 사용할 수 있습니다. 추출된 내용은 제목, 작성자, 정제된 HTML, 평문 내용 등으로 제공되며, Unicode 정규화 및 SHA-256 해시 생성 기능도 포함합니다. 학술 연구 및 콘텐츠 분석에 적합한 도구입니다.
