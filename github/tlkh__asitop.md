`asitop`는 Apple Silicon(M1) 기반 Mac 전용 성능 모니터링 CLI 도구로, CPU/GPU 사용량, 메모리 정보, 전력 소모량 등을 실시간으로 표시합니다. `powermetrics` 유틸리티를 기반으로 구축되어 있으며, macOS Monterey 환경에서만 작동합니다. 터미널에서 간편하게 실행할 수 있으며, 최소한의 시스템 리소스를 사용하는 것이 특징입니다.  

**※ 주의**: 루트 권한(`sudo`)이 필요하며, 하드웨어 정보에 대한 일부 추정치를 사용합니다. 공식 지원 없이 개발된 도구로, 사용 시 주의가 필요합니다.
