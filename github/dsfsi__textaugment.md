TextAugment은 자연어처리(NLP) 애플리케이션을 위한 텍스트 증강 기능을 제공하는 Python 라이브러리입니다. 이 프로젝트는 Word2vec, Fasttext, WordNet, 번역 기반 증강 및 EDA(쉬운 데이터 증강) 기법을 활용하여 합성 텍스트 데이터를 생성함으로써 모델 성능을 개선하는 데 중점을 둡니다. PyTorch, TensorFlow, Scikit-learn 등 다양한 머신러닝 프레임워크와 호환되며, 특히 짧은 텍스트 분류 작업에 최적화되어 있습니다. 논문 ["Improving short text classification through global augmentation methods"](https://arxiv.org/abs/1907.03752)에서 제안된 방법론을 구현하였으며, MIT 라이선스로 배포됩니다.
