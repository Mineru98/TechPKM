TextAugment는 자연어 처리(NLP) 애플리케이션을 위한 텍스트 증강 기능을 제공하는 Python 라이브러리입니다. Word2Vec, FastText, WordNet, 번역 기반 증강, EDA(Easy Data Augmentation), AEDA(An Easier Data Augmentation), Mixup 등 다양한 증강 기법을 지원하며, 간단한 인터페이스로 모델 성능 향상을 위한 합성 데이터 생성이 가능합니다. 이 라이브러리는 PyTorch, TensorFlow, Scikit-learn 등 다양한 머신러닝 프레임워크와 호환됩니다.
