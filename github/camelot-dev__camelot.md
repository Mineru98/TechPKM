Camelot는 PDF 문서에서 표를 추출하기 위한 Python 라이브러리입니다. 텍스트 기반 PDF에서 간단한 코드로 표를 추출하고, 정확도, 공백 비율 등의 지표를 기반으로 결과를 평가할 수 있습니다. 추출된 표는 pandas DataFrame으로 제공되며 CSV, JSON, 엑셀 등 다양한 형식으로 내보낼 수 있습니다. 스캔된 문서가 아닌 텍스트 선택 가능한 PDF에 최적화되어 있습니다.
