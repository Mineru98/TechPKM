Camelot는 PDF 문서에서 테이블을 추출하는 Python 라이브러리입니다. 텍스트 기반 PDF에서 간단한 코드로 테이블을 추출하고, 정확도, 공백 비율 등의 메트릭을 제공하며, pandas DataFrame으로 변환하거나 CSV/JSON/Excel 등 다양한 형식으로 내보낼 수 있습니다. CLI 인터페이스와 고급 설정 옵션을 지원하며, 데이터 분석 워크플로우에 최적화되어 있습니다. 스캔된 문서에는 작동하지 않으며, conda 또는 pip로 설치할 수 있습니다.
