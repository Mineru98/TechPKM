이 프로젝트는 Claude.ai, Claude Code, Claude API에서 생산성을 향상시키기 위한 실용적인 커스텀 워크플로우 모음입니다. 문서 처리, 개발 도구, 데이터 분석, 비즈니스, 커뮤니케이션, 창의성, 생산성, 협업, 보안 등 다양한 분야의 기술을 제공하며, 1000개 이상의 앱과 연동하여 실제 작업(이메일 전송, 이슈 생성, 슬랙 게시 등)을 수행할 수 있습니다. 각 기술은 특정 작업을 반복적이고 표준화된 방식으로 실행하도록 설계되었으며, 모든 Claude 플랫폼에서 일관되게 작동합니다.  

- **기술 분야**: 문서 처리, 개발 도구, 데이터 분석, 비즈니스, 커뮤니케이션, 창의성, 생산성, 협업, 보안  
- **주요 특징**: 플랫폼 간 호환성, 실제 앱 연동, 커스터마이징 가능 워크플로우  
- **사용 방법**: 플러그인 설치, API 통합, 로컬 기술 파일 적용 등을 통해 활용 가능
