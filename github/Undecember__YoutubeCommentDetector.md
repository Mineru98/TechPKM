이 프로젝트는 특정 유튜브 동영상의 댓글을 분석하여 일정 시간 이상 새로운 댓글이 등록되지 않은 간격을 감지합니다. Google API를 활용해 최근 댓글을 수집하고, 인접 댓글 간 1분 이상 간격이 발생하는지 확인하며, 결과를 이메일로 보고하는 기능을 제공합니다. Python 기반 자동화 도구로, 주로 이벤트 영상 모니터링에 활용됩니다.
