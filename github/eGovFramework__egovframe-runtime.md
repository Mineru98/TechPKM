전자정부 표준프레임워크 실행환경 4.3.0은 공공사업 개발을 위한 공통 모듈과 핵심 서비스를 제공하는 Java 기반의 엔터프라이즈 개발 플랫폼입니다. Spring Boot 2.7.18 및 Spring Framework 5.3.37을 지원하며, 화면 처리부터 데이터 접근, 배치 처리, 보안 등 다양한 계층에서 재사용 가능한 라이브러리 세트를 제공합니다. 핵심 서비스와 선택적 서비스로 구분되어 있으며, 다양한 데이터베이스 및 비동기 처리 기능을 포함합니다.  

이 프로젝트는 전자정부 표준프레임워크의 공식 오픈소스 프로젝트로, Apache 2.0 라이선스로 배포됩니다.
