표준프레임워크 실행환경 4.3.0은 전자정부 사업에서 업무 프로그램 개발 시 필요한 공통 기능을 패턴화한 라이브러리 모음입니다. 화면 처리, 데이터 접근, 배치 처리, 보안 등 핵심 서비스를 제공하며 Spring Boot 2.7.18, Spring Framework 5.3.37, Spring Security 5.8.13을 지원합니다. 핵심 서비스는 기본 적용되며 선택 서비스는 사업별로 유연하게 적용할 수 있는 모듈식 구조를 가지고 있습니다. Apache 2.0 라이선스로 배포되는 오픈소스 프로젝트입니다.
