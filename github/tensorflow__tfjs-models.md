이 프로젝트는 웹 기반 애플리케이션에서 바로 사용 가능한 사전 학습된 TensorFlow.js 모델들을 제공합니다. 이미지 분류(MobileNet), 실시간 포즈 감지, 음성 명령 인식, 텍스트 임베딩 등 다양한 도메인에서 활용 가능한 모델들을 NPM 패키지로 배포하며, 비전문가도 쉽게 사용할 수 있는 API를 지향합니다. 전이 학습이나 독립적인 모델 사용 시나리오에 모두 적합합니다.
