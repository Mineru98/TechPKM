**dots.ocr**은 단일 비전-언어 모델(VLM)로 문서 레이아웃 검출과 텍스트 인식을 통합한 멀티링구얼 문서 파싱 모델입니다. 1.7B 파라미터의 경량화된 LLM 기반으로 구축되어 OmniDocBench 벤치마크에서 텍스트, 표, 읽기 순서 분야에서 SOTA 성능을 달성하며, 저자원 언어에서도 우수한 성능을 보입니다. 복잡한 다중 모델 파이프라인 없이 단일 프롬프트 변경만으로 레이아웃 검출과 텍스트 인식이 가능한 간결한 아키텍처가 특징이며, 빠른 추론 속도와 효율적인 성능을 제공합니다. 공식 데모 및 라이브 데모를 통해 실제 문서 처리 성능을 확인할 수 있습니다.
