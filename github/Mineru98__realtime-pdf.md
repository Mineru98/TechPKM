이 프로젝트는 PDF.js와 WebSocket을 활용해 실시간으로 PDF를 공유하고 동기화하는 웹 애플리케이션입니다. 호스트가 PDF 파일을 업로드하고 페이지를 제어하면, 참가자 모드 사용자는 실시간으로 동일한 페이지를 따라가며 벡터 그래픽 기반의 선명한 화질로 PDF를 확인할 수 있습니다. Docker 배포와 로컬 개발 환경을 모두 지원하며, Vanilla JavaScript 기반 프론트엔드와 Node.js 백엔드로 구성되었습니다.
