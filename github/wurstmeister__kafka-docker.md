이 프로젝트는 Apache Kafka를 Docker에서 실행할 수 있도록 하는 공식 도커 이미지를 제공합니다. Kafka 클러스터를 빠르게 배포하고 관리할 수 있는 Docker Compose 구성 파일과 다양한 환경 변수 기반 설정 옵션을 포함합니다. 주요 기능은 브로커 ID 관리, 자동 토픽 생성, 네트워크 구성, JMX 모니터링 지원 등을 포함합니다. Kafka의 다양한 버전과 Scala 버전을 지원하는 태그를 제공하며, Docker Hub에서 직접 사용 가능합니다.
