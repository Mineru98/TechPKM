(주)미디어그룹사람과숲과 (주)마커의 연구 컨소시엄에서 개발된 오픈소스 LLM 프로젝트입니다. 2023년 12월 기준 Open LLM 리더보드 1위를 기록한 **Sakura-SOLAR-Instruct** 모델을 중심으로, 모델 병합(merging) 및 DPO(직접 선호 최적화) 기반 파인튜닝 기술을 공유합니다. 다양한 태스크에 특화된 파생 모델(v1~v2)과 학습 하이퍼파라미터, 데이터셋 정보를 제공합니다.
