Gemini CLI는 터미널에서 직접 Gemini 모델의 강력한 기능을 활용할 수 있는 오픈소스 AI 에이전트입니다. 개발자를 위해 설계된 이 도구는 코드 분석, 자동 생성, 작업 자동화 및 Google Search 연동 등의 기능을 제공하며, 확장 가능한 MCP(모델 컨텍스트 프로토콜)를 지원합니다. 무료 티어(60회/분, 1,000회/일)와 다양한 인증 방식(Google 계정, API 키, Vertex AI)을 통해 유연하게 사용할 수 있습니다.  

Gemini CLI는 코드베이스 이해, PDF/이미지 기반 앱 생성, GitHub 워크플로 통합, 대화형 세션 저장 및 재개 등 다양한 개발자 시나리오를 지원하며, Apache 2.0 라이선스로 배포됩니다.
