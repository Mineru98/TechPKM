Gemini CLI는 터미널 환경에서 Gemini AI의 강력한 기능을 활용할 수 있는 오픈소스 도구입니다. 개인 및 조직용 구글 계정을 통해 무료 티어(분당 60회, 일일 1,000회 요청) 및 Gemini 3 모델(1M 토큰 컨텍스트 창)을 지원하며, 코드 분석/생성, 자동화 작업, 검색 기반 정보 활용 등의 기능을 제공합니다. 확장 가능한 MCP 프로토콜과 GitHub 통합(코드 리뷰, 이슈 처리)을 통해 개발자 워크플로우를 혁신합니다. Node.js 기반이며 npm, Homebrew로 설치 가능합니다. Apache 2.0 라이선스 적용.
