LlamaBot은 Jupyter 노트북에서 LLM 실험을 간소화하고 Python 앱에 LLM 통합을 용이하게 하는 Pythonic 인터페이스를 제공하는 프로젝트입니다. LiteLLM에서 지원하는 모든 모델을 호환하며, Ollama, LMStudio, OpenAI, Mistral 등 다양한 로컬/클라우드 기반 모델과 연동할 수 있습니다. 단순 챗봇(SimpleBot), 도구 실행(ToolBot), 구조화된 출력(StructuredBot), 문서 검색(QueryBot) 등 다양한 사용 사례에 적합한 봇 유형을 지원합니다. Markdown 형식의 README는 설치 방법, 모델 접근 옵션, 코드 예제, 확장 기능 등을 상세히 설명합니다.
