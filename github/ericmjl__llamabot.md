LlamaBot은 Jupyter 노트북 및 Python 애플리케이션에서 대규모 언어 모델(LLM)을 쉽게 실험하고 활용할 수 있도록 설계된 파이썬 인터페이스 라이브러리입니다. LiteLLM에서 지원하는 모든 모델을 포함하며, Ollama, LMStudio, OpenAI, Mistral 등의 다양한 로컬/클라우드 모델과 호환됩니다. SimpleBot, QueryBot, StructuredBot 등 다양한 봇 유형을 제공하여 채팅, 문서 검색, 구조화된 데이터 추출 등의 작업을 지원합니다.  

**핵심 특징**:  
- 다양한 LLM 모델 통합 (LiteLLM 기반)  
- Jupyter 친화적인 인터페이스  
- 채팅 메모리, 도구 호출, 구조화된 출력 등 고급 기능  
- 로컬 모델(Ollama/LMStudio) 및 API 기반 모델 지원  
- 이미지 생성 및 분석 기능 포함  

**사용 사례**:  
- LLM 실험 및 프롬프트 엔지니어링  
- 데이터 분석 자동화  
- 문서 기반 질문 응답 시스템  
- 코드 개발 지원 및 디버깅  
- 구조화된 데이터 추출 및 검증
