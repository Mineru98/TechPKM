TTS WebUI는 다양한 텍스트-음성 변환(TTS) 모델을 지원하는 오픈소스 웹 인터페이스입니다. Gradio 및 React 기반 UI를 통해 Bark, Tortoise, StyleTTS2 등 다양한 모델을 활용할 수 있으며, 확장 기능 및 통합 기능을 통해 사용자 정의가 가능합니다. Docker 설치, 수동 설치, 인스톨러 등 다양한 설치 옵션과 Silly Tavern, OpenWebUI 등과의 호환성을 제공합니다.
