Claude Code 요청을 다양한 모델로 라우팅하고 사용자 정의할 수 있는 강력한 도구입니다. 여러 모델 제공업체(OpenRouter, DeepSeek, Ollama 등)를 지원하며, 변환기를 통해 요청/응답을 커스터마이징할 수 있습니다. CLI 및 UI 기반 모델 관리, GitHub Actions 통합, 프리셋 시스템을 통해 유연한 설정이 가능합니다. 사용 사례에 따라 기본/배경/사고/장문 처리 모델로 라우팅할 수 있으며, 동적 모델 전환 및 사용자 정의 라우팅 로직도 지원합니다.
