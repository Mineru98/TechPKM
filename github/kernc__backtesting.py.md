이 프로젝트는 Python을 사용하여 트레이딩 전략을 백테스트하는 도구입니다. 간단한 API와 빠른 실행 속도, 상세한 결과 분석 및 시각화 기능을 제공하며, 다양한 금융 상품에 적용 가능합니다. 내장 최적화 기능과 재사용 가능한 전략 라이브러리를 통해 금융 데이터 기반 전략 개발을 지원합니다.  

**요약**: 금융 시간봉 데이터를 활용한 트레이딩 전략 백테스트 프레임워크. SMA 크로스오버 등 기본 전략 템플릿과 성능 지표(수익률, 샤프 비율, 최대 낙폭 등) 자동 계산 기능 포함. 시각화 및 파라미터 최적화 지원.
