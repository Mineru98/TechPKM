이 프로젝트는 Python을 사용하여 거래 전략을 백테스팅하는 오픈소스 라이브러리입니다. 금융 시장의 캔들스틱 데이터를 기반으로 사용자 정의 트레이딩 전략을 테스트할 수 있으며, 상세한 성과 분석과 시각화 기능을 제공합니다. 간단한 API와 빠른 실행 속도가 특징이며, 다양한 금융 상품(주식, 암호화폐 등)에 적용 가능합니다.  

주요 기능으로는 내장 전략 최적화 도구, 지표 라이브러리, 인터랙티브 차트 생성 등이 포함됩니다. SMA 크로스오버와 같은 전략 예제를 제공하여 초보자도 쉽게 시작할 수 있습니다.
