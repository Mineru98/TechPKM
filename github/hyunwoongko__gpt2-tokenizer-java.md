이 프로젝트는 GPT2 모델의 토크나이저를 Java로 구현한 라이브러리입니다. `encoder.json`과 `vocab.bpe` 파일을 사용하여 텍스트를 토큰으로 인코딩/디코딩할 수 있으며, Spring Boot 및 Google API 클라이언트 등의 의존성을 필요로 합니다. 자연어 처리(NLP) 작업에 활용되며, Apache License 2.0으로 배포됩니다.
