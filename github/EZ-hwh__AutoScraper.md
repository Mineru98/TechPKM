AutoScraper는 웹 스크래퍼를 자동으로 생성하는 진보적 이해 웹 에이전트를 구현한 연구 프로젝트입니다. 논문 *"AutoScraper: A Progressive Understanding Web Agent for Web Scraper Generation"*의 공식 코드로, LLM(대형 언어 모델)을 활용해 웹 사이트 구조를 이해하고 스크래퍼를 생성합니다. SWDE 데이터셋 기반 평가 및 실험을 지원하며, 향후 실제 웹 사이트 적용을 목표로 개발 중입니다.
