AutoScraper는 웹 스크래퍼 생성을 위한 점진적 이해 웹 에이전트를 제안하는 연구 프로젝트의 공식 구현체입니다. SWDE 데이터셋과 ChatGPT/GPT4 모델을 활용하여 웹 스크래퍼를 자동 생성 및 평가하는 기능을 제공하며, 학술 논문(*"AutoScraper: A Progressive Understanding Web Agent for Web Scraper Generation"*)의 실험 결과를 재현할 수 있도록 설계되었습니다. 현재 실제 웹사이트 적응 및 데모 웹사이트 구축이 진행 중입니다.
