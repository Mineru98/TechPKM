Chatbot Ollama은 Ollama 기반의 오픈소스 채팅 인터페이스입니다. [chatbot-ui](https://github.com/mckaywrigley/chatbot-ui) 프로젝트를 기반으로 하여, 문서 업로드(PDF/TXT/MD), 이미지 첨부, 코드 블록 기능 등을 강화한 것이 특징입니다. 다크/라이트 모드 지원과 실시간 스트리밍 기능을 통해 사용자 친화적인 대화 환경을 제공하며, Docker를 통해 간편하게 배포할 수 있습니다.  

> 📌 주요 기능: 모델 프롬프트에 문서 내용 자동 포함, 이미지 인식 지원, 키보드 단축키, 오류 핸들링, 대화 복사 기능 등 포함.  
> 🛠 기술 스택: JavaScript/TypeScript 기반 Next.js 애플리케이션.
