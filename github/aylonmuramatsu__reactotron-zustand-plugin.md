이 프로젝트는 Zustand 상태 관리 라이브러리의 상태를 Reactotron에서 모니터링할 수 있도록 하는 플러그인입니다. React Native 애플리케이션 개발 중 Zustand 스토어의 상태 변화를 실시간으로 디버깅할 수 있으며, 여러 스토어를 등록하여 개별적으로 또는 전체 상태를 관찰할 수 있습니다. 간단한 설정으로 Reactotron에 통합되어 개발 효율성을 높여줍니다.
