이 프로젝트는 Zustand 상태 관리를 Reactotron에서 모니터링할 수 있게 해주는 플러그인입니다. React Native 프로젝트에서 Zustand 스토어의 상태를 실시간으로 추적 및 디버깅할 수 있으며, 여러 스토어를 등록하여 개별 또는 전체 상태를 관찰할 수 있습니다. README에는 설치 방법과 Reactotron 설정 방법, 그리고 상태 관찰 시 필요한 구성이 포함되어 있습니다. 간단한 구성으로 개발자는 상태 변화를 효율적으로 모니터링할 수 있습니다.
