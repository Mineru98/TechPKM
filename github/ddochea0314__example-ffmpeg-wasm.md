이 프로젝트는 WebAssembly 기반 ffmpeg.wasm을 React 환경에서 활용하는 예제 코드를 정리한 저장소입니다. 브라우저에서 미디어 파일을 변환하는 기능 구현을 보여주며, 최신 보안 정책에 따른 CORS 헤더 설정 요구사항을 포함합니다. MIT 라이선스로 공개된 예제 코드이지만, 실제 ffmpeg.wasm 패키지의 라이선스는 별도 적용됩니다. 웹 기반 미디어 처리 개발에 참고용으로 활용할 수 있습니다.
