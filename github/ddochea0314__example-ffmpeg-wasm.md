이 프로젝트는 WebAssembly 기반 `ffmpeg.wasm` 패키지를 React 환경에서 사용하는 예제 코드를 정리한 저장소입니다. 브라우저에서 SharedArrayBuffer 사용을 위한 보안 헤더 설정과 함께 미디어 처리 기술의 실제 적용 사례를 보여줍니다. WASM을 활용한 효율적인 비디오/오디오 변환 구현에 초점을 맞춘 실험적인 예제 저장소입니다.

> **주의**: `@ffmpeg/core` 및 wasm 파일은 MIT 라이선스가 적용되지 않으며, 이 예제는 MIT 라이선스로 배포되었으나 ffmpeg 원본 라이선스는 별도 확인이 필요합니다.
