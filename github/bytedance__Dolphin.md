Dolphin-v2는 다양한 문서 유형(디지털/사진)을 처리하는 범용 문서 이미지 파싱 모델로, 문서 유형 인식 2단계 아키텍처와 확장 가능한 앵커 프롬프팅을 통해 복잡한 문서 요소(텍스트, 수식, 표, 코드 블록 등)를 효과적으로 분석합니다. 경량이면서도 효율적인 병렬 파싱 메커니즘을 갖춘 이 모델은 페이지 및 요소 수준의 파싱 작업에서 우수한 성능을 발휘합니다. 주요 기능은 분석-파싱 접근 방식, 자연스러운 읽기 순서 생성, 다양한 문서 요소 지원을 포함하며, Hugging Face Transformers와 호환됩니다.
