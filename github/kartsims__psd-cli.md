PSD 파일을 PNG로 변환하고 텍스트를 추출하는 CLI 기반 도구입니다. PSD.js 라이브러리를 활용하여 별도의 종속성 없이 NodeJS 환경에서 동작하며, 단일/다중 파일 처리와 미리보기 기능을 제공합니다. 텍스트 추출은 실험적이며, 향후 개선을 위한 피드백 요청이 가능합니다.
