이 프로젝트는 플러터 앱에서 로컬 알림을 구현하는 방법을 설명하는 튜토리얼 리포지토리입니다. 단일 알림, 주기적 알림, 예약 알림 및 알림 취소 기능을 포함하며, iOS와 안드로이드 호환성을 위해 AppDelegate 수정이 필요합니다. 비디오 강의와 함께 제공되며, 앱 종료 상태에서도 알림 클릭 시 페이로드 기반 페이지 이동이 가능합니다.  

(요약: 3문장 / 핵심 기능, 플랫폼, 추가 정보 포함)
