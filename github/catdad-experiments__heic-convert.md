이 프로젝트는 HEIC/HEIF 포맷의 이미지를 JPEG 또는 PNG 형식으로 변환하는 Node.js 라이브러리입니다. 주요 기능은 HEIC 파일 내 단일 이미지 또는 다중 이미지 변환을 지원하며, 품질 설정 옵션을 제공합니다. 브라우저 환경에서도 사용 가능하도록 별도 빌드 옵션을 제공합니다.  

> ✅ 핵심 특징: HEIC→JPEG/PNG 변환, 멀티이미지 지원, 품질 조정 기능, Node.js 및 브라우저 호환성.
