@egjs/infinitegrid는 다양한 크기의 카드 요소로 구성된 그리드를 구현할 수 있는 JavaScript 라이브러리입니다. 무한 스크롤 방식으로 DOM 수를 관리하여 성능을 보장하며, React, Vue, Angular, Svelte 등의 주요 프레임워크를 지원합니다. `data-size-group` 속성을 통해 크기 그룹을 지정하거나 `isEqualSize` 옵션으로 동일 크기 아이템 처리를 최적화할 수 있습니다. TypeScript 기반으로 작성되어 타입 안정성을 제공합니다.
