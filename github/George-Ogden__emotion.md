이 프로젝트는 Yolo-V5와 RepVGG 모델을 결합하여 얼굴 표정을 감지하고 8가지 기본 감정(분노, 경멸, 혐오, 두려움, 행복, 중립, 슬픔, 놀람)을 분류하는 컴퓨터 비전 솔루션입니다. WIDER FACE 데이터셋으로 훈련된 얼굴 감지 모델과 AffectNet 데이터셋 기반의 감정 분류 모델을 연동하여 실시간 감정 분석 기능을 제공합니다. 사용자는 다양한 추론 파라미터(이미지 크기, 신뢰도 임계값 등)를 조정하여 맞춤형 분석이 가능하며, 결과 시각화 및 저장 기능을 포함합니다.
