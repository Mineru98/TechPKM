이 프로젝트는 Yolo-V5와 RepVGG 모델을 결합해 얼굴 표정을 감지하고 8가지 기본 감정(분노, 경멸, 혐오, 공포, 행복, 중립, 슬픔, 놀람)을 분류하는 컴퓨터 비전 솔루션입니다. WIDER FACE 데이터셋으로 얼굴 검출을 훈련하고, AffectNet 데이터셋으로 감정 분류를 수행하며, 실시간 영상 또는 이미지 분석이 가능합니다. 사용자는 다양한 파라미터를 조정해 모델 성능을 최적화할 수 있습니다.
