**marimo**는 재현 가능하고 git 친화적이며 스크립트나 앱으로 배포할 수 있는 반응형 Python 노트북입니다. 셀 실행 시 종속 셀을 자동 실행하거나 스테일 상태로 표시하여 코드와 출력의 일관성을 유지하며, SQL 지원, 대화형 UI, 데이터 분석 기능을 포함한 종합적인 도구입니다. VS Code 확장 및 AI 코드 생성 기능을 제공하여 데이터 작업과 협업을 효율적으로 지원합니다.  

이 프로젝트는 Jupyter, Streamlit 등 기존 도구의 한계를 해결하기 위해 설계되었으며, 순수 Python 파일로 저장되고 CLI, 웹 앱, 브라우저 내 WASM 실행 등 다양한 환경에서 작동합니다. NumFOCUS 소속 프로젝트로 Python 생태계의 핵심 도구 중 하나입니다.
