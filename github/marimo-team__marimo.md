마리모는 재현 가능하고 git 친화적인 반응형 파이썬 노트북입니다. SQL 지원, 대화형 UI 요소, 웹 앱 배포 기능을 갖추었으며, 데이터 작업 및 실험에 최적화된 환경을 제공합니다. 전통적인 노트북의 한계를 해결하며 코드-출력 일관성을 자동으로 유지합니다.  

마리모는 파이썬 스크립트로 저장되어 텍스트 편집기에서 직접 편집 가능하며, CLI 매개변수화, 대화형 웹 앱 실행, AI 기반 코드 생성 등 다양한 기능을 지원합니다. 데이터 분석, 시각화, 머신러닝 등 데이터 과학 워크플로우에 특화된 도구로 설계되었습니다.
