Lottie는 Adobe After Effects에서 제작한 애니메이션을 JSON 형식으로 변환해 웹, 모바일 환경에서 네이티브로 렌더링하는 라이브러리입니다. 디자이너가 직접 제작한 애니메이션을 개발자가 별도 작업 없이 실시간으로 구현할 수 있게 지원하며, SVG/Canvas/HTML 렌더러를 통해 크로스플랫폼 호환성을 제공합니다. Bodymovin 플러그인으로 애니메이션을 내보낸 후 간단한 JavaScript API로 제어할 수 있습니다.  

(요약: Lottie는 AE 애니메이션을 JSON으로 변환해 웹/모바일에서 네이티브 렌더링하는 라이브러리. 디자이너-개발자 협업 효율화 및 실시간 애니메이션 구현 지원)
