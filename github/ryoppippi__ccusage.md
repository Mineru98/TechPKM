# Output

제공된 README 파일(`./apps/ccusage/README.md`)의 내용이 없기 때문에, 요청하신 메타데이터 문서를 생성할 수 없습니다.  
README 파일의 내용을 공유해 주시면, 해당 내용을 기반으로 Obsidian 관리에 최적화된 마크다운 메타데이터를 생성해 드리겠습니다.

필요한 형식은 다음과 같습니다:
```markdown
---
Language: {main_code_language}
tags:
 - {tag_name_1}
 - {tag_name_2}
aliases:
 - {alias_1}
 - {alias_2}
url: {github_url}
---
{summary}
```
