이 프로젝트는 ReactJS와 Tensorflow를 활용한 필기체 숫자 인식 시스템을 구현한 간단한 Tensorflow Serving 예제입니다. MLP 모델과 Docker 기반 서빙 환경을 통해 숫자 이미지 인식 기능을 제공하며, Nginx를 활용한 UI 통합이 특징입니다. `docker-compose up --build` 명령어로 전체 시스템을 실행할 수 있습니다.
