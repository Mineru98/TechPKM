CryptoGPT는 언어 학습 모델(LLM)이 블록체인에서 독립적인 금융 활동을 수행할 수 있도록 하는 실험적 프로젝트입니다. OpenAI API와 이더리움 기반 체인(EVM)을 연동하여 LLM이 지갑 관리, 트랜잭션 실행, 토큰 래핑 등의 작업을 처리할 수 있게 합니다. 현재는 개념 검증(PoC) 단계로, Next.js와 LangChain.js를 기반으로 구현되었으며, 실제 자산 노출 시 보안 주의가 필요합니다.  

> ⚠️ 주의: 테스트 환경에서만 사용해야 하며, 프라이빗 키 노출 시 자산 손실 위험이 있습니다. 주요 기술 스택은 Next.js, TailwindCSS, Vercel 서버리스 함수, LangChain.js입니다.
