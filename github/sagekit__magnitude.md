Magnitude는 Vision AI를 활용해 자연어로 브라우저를 제어하는 오픈소스 프로젝트입니다. 웹 인터페이스의 시각적 이해를 기반으로 탐색, 상호작용, 데이터 추출 및 검증 기능을 제공하며, API 없이도 애플리케이션 통합을 가능하게 합니다. 높은 수준의 작업부터 세부적인 마우스/키보드 제어까지 유연한 추상화 계층을 지원하며, Claude Sonnet 4 또는 Qwen-2.5VL과 같은 시각적 기반 LLM을 활용합니다. 웹 테스트 자동화, 데이터 추출, 브라우저 에이전트 구축 등에 활용할 수 있는 확장 가능한 도구입니다.
