Magnitude는 Vision AI를 활용해 웹 인터페이스를 이해하고 제어하는 도구를 제공합니다. 자연어 명령으로 브라우저 탐색, 상호작용, 데이터 추출 및 테스트 검증을 수행할 수 있으며, 복잡한 UI 환경에 대한 높은 일반화 능력을 갖춘 것이 특징입니다. Vision-first 아키텍처로 DOM 구조에 의존하지 않으며, 다양한 추상화 수준의 자동화 작업(세부 액션/워크플로우)을 지원합니다.  

주요 기능으로는 **"Navigate"(인터페이스 이해 및 경로 계획)**, **"Interact"(정밀한 마우스/키보드 조작)**, **"Extract"(구조화된 데이터 추출)**, **"Verify"(시각적 검증 테스트)**가 포함됩니다. Claude Sonnet 4 또는 Qwen-2.5VL 72B와 같은 대형 시각-언어 모델을 기반으로 동작합니다.
