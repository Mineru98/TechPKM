oh-my-claudecode는 Claude Code 사용자를 위한 다중 에이전트 오케스트레이션 플러그인입니다. 복잡한 작업을 자동화하고, 자연어 인텐트를 기반으로 전문가 에이전트들을 병렬화하며, 데이터 분석부터 계획 수립까지 다양한 기능을 제공합니다. "학습 곡선 제로"를 표방하며, 사용자는 명령어나 키워드 없이도 직관적인 사용이 가능합니다.  

주요 기능으로는 지속성 모드(`ralph`), 병렬 실행(`ulw`), 계획 인터뷰(`plan`), 자율 실행(`autopilot`) 등이 있으며, 데이터 분석 시 Python REPL 지속성과 구조화된 출력 시스템을 지원합니다. 또한 MCP 서버를 통해 외부 도구(문서 검색, 파일 시스템 접근 등)를 통합할 수 있습니다.  

Claude Code의 상태 표시줄(HUD)과 연동되어 실시간 작업 진행 상황을 시각화하며, 기존 2.x 명령어도 호환됩니다. MIT 라이선스로 배포됩니다.
