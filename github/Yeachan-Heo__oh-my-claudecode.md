**oh-my-claudecode**는 Claude Code 사용을 간소화하기 위한 멀티 에이전트 오케스트레이션 도구입니다. 복잡한 작업을 자동으로 분할하고, 자연어 의도를 감지하여 전문가 에이전트들을 병렬 처리하며, 데이터 분석부터 연구 보고서 생성까지 통합된 워크플로우를 제공합니다. "설치 후 즉시 사용" 철학을 기반으로 학습 곡선 없이 최대 생산성을 달성하는 것을 목표로 합니다.

### 주요 특징
- **에이전트 자동 활성화**: "plan this" 또는 "don't stop until done" 같은 자연어 명령으로 자동 작업 수행
- **연구 프로토콜**: 데이터 분석용 과학자 에이전트 계층(`scientist-low/standard/high`)과 Python REPL 지속성 지원
- **MCP 서버 확장**: Context7, Exa, 파일시스템, GitHub API 등 외부 도구 통합
- **HUD 상태 표시**: 실시간 작업 진행률 및 에이전트 상태 시각화

### 사용 사례
- 데이터베이스 마이그레이션 자동화
- 복잡한 데이터 분석 및 통계 검증
- UI/UX 디자인 자동화 지원
- 연구 목표 달성을 위한 병렬 에이전트 오케스트레이션

### 학습 곡선 최소화
기존 명령어(`/ralph`, `/ultrawork`)는 호환되며, 모든 기능이 자연어 명령으로 대체 가능합니다. "ralph", "plan", "autopilot" 같은 키워드 조합으로 정밀 제어 시 사용 가능합니다.
