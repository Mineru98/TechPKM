`llama3.np`은 Llama 3 모델을 순수 NumPy로 구현한 프로젝트입니다. Andrej Karpathy의 훈련된 `stories15M` 모델을 기반으로 정확한 구현을 목표로 하며, CUDA 없이 CPU 환경에서 LLM 인퍼런스를 실험할 수 있도록 설계되었습니다. 주요 기술 문서와 C/CUDA 구현 링크가 제공되며, MIT 라이선스로 배포됩니다. NumPy를 활용한 모델 구현과 토큰 생성 기능을 중심으로 한 연구/교육용 프로젝트에 적합합니다.
