상업적 이용이 가능한 라이센스를 가진 한국어 최적화 LLM(Large Language Model) 프로젝트입니다. llama-3.1 아키텍처를 기반으로 70B 파라미터 규모로 구현되었으며, 법률 분야에서 '정의'와 같은 추상적 개념을 분석하고 보편적 정의 입증 방법을 탐구하는 데 활용됩니다. Python API와 CURL 인터페이스를 통해 모델 접근이 가능하며, 한국어 텍스트 생성 및 질의응답에 특화된 특징을 가집니다.
