Belt는 thoughtbot에서 개발한 React Native 앱 개발 시작을 위한 CLI 도구로, ESLint, Prettier, TypeScript, Jest 등 검증된 도구 체인을 제공합니다. 내부적으로 사용 중인 이 도구는 Expo 기반 앱 생성 시 반복적인 설정 작업을 자동화하며, React Navigation, Tanstack Query 등의 핵심 기능을 기본으로 포함합니다. 아직 초기 개발 단계로 API 변경에 유의해야 하며, 오픈소스 라이선스로 배포됩니다.
