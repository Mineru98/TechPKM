이 프로젝트는 React 환경에서 A/B 테스트를 쉽게 구현할 수 있도록 도와주는 라이브러리입니다. `<Experiment />`와 `<Variant />` 컴포넌트를 사용해 실험 그룹을 정의하고, 사용자마다 무작위로 선택된 변형을 표시할 수 있습니다. 로컬 스토리지에 선택된 변형을 저장하며, Mixpanel이나 Segment.com 같은 분석 도구와의 통합도 지원합니다. 서버 사이드 렌더링, 가중치 기반 변형 선택, 실시간 디버깅 등 다양한 고급 기능을 제공합니다.
