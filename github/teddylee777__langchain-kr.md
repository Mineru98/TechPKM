이 프로젝트는 LangChain 공식 문서와 실용 예제를 기반으로 한 한국어 튜토리얼로, LangChain을 효과적으로 활용하는 방법을 체계적으로 설명합니다. 위키독스 전자책, 유튜브 강의, 블로그 글을 통해 다양한 학습 자료를 제공하며, LLM(Large Language Model) 애플리케이션 개발, RAG(Retrieval-Augmented Generation), 에이전트 구현 등의 주제를 다룹니다. Apache License 2.0으로 배포되며, 상업적 사용 시 저작권자와 사전 협의가 필요합니다.
