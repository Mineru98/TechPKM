이 프로젝트는 YouTube 동영상의 자막(제목)을 추출하거나 번역할 수 있는 Python API입니다. Selenium과 같은 헤드리스 브라우저 없이도 자막 데이터를 가져올 수 있으며, 자동 생성된 자막도 지원합니다. CLI(명령줄 인터페이스)를 통한 사용이나 기존 애플리케이션에 통합하여 활용할 수 있습니다. 주요 기능으로는 특정 언어 자막 추출, 자막 번역, HTTP 요청 프록시 설정, 다양한 포맷터 지원(텍스트, JSON, WebVTT, SRT 등)이 포함됩니다. MIT 라이선스로 배포되며, PyPI를 통해 설치할 수 있습니다.
