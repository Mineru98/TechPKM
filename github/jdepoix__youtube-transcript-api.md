이 프로젝트는 YouTube 동영상의 자막/자막을 추출하는 Python API입니다. 헤드리스 브라우저 없이 작동하며, 자동 생성 자막 지원, 다국어 번역 기능을 제공합니다. CLI와 API 형태로 사용 가능하며, 프록시 설정을 통한 IP 차단 우회 기능도 포함되어 있습니다.  

주요 특징:  
1. 비디오 ID 기반 자막 추출 및 언어 우선순위 지정  
2. 수동 생성 자막과 자동 생성 자막 필터링  
3. HTML 포맷 보존 옵션  
4. Webshare 프록시 통합 및 커스텀 프록시 지원  
5. JSON, WebVTT 등 다양한 형식 변환 포맷터 제공  
6. Python 3.6+ 호환  

MIT 라이선스로 배포되며, 클라우드 서비스에서의 IP 차단 문제를 해결하기 위한 프록시 솔루션을 내장하고 있습니다.
