이 프로젝트는 AutoGEN과 MemGPT를 로컬 LLM(Large Language Model)과 통합하여 AI 에이전트 시스템을 구축하는 방법을 보여줍니다. Runpods를 활용해 하드웨어 인프라를 구성하고, 두 프레임워크의 장점을 결합한 멀티에이전트 환경을 구현합니다. 파이썬 기반의 코드 예제를 통해 API 연동 및 에이전트 전환 로직을 학습할 수 있습니다.  

(요약: AutoGEN과 MemGPT를 로컬 LLM과 통합하는 튜토리얼 프로젝트로, Runpods를 활용한 AI 에이전트 시스템 구축을 다룹니다. AI 프레임워크 간 협업과 로컬 LLM 활용 방법을 설명합니다.)
