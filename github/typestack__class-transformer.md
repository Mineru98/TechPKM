class-transformer는 TypeScript 클래스와 일반 객체(plain object) 간의 변환을 용이하게 하는 라이브러리입니다. ES6+ 환경에서 JSON 데이터를 클래스 인스턴스로 변환하거나, 반대로 클래스 객체를 일반 객체로 직렬화하여 API 응답 등을 제어할 수 있습니다. `@Type`, `@Expose`, `@Transform` 등의 데코레이터를 통해 중첩 객체, 속성 노출, 데이터 변환 등을 세밀하게 관리할 수 있으며, 타입 안전성과 버전 관리 기능도 지원합니다. 프론트엔드와 백엔드 모두에서 활용 가능한 유연한 도구입니다.
