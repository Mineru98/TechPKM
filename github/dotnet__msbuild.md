Microsoft.Build(MSBuild)는 애플리케이션 빌드를 위한 플랫폼으로, XML 스키마를 통해 프로젝트 파일을 제어하여 소프트웨어를 처리하는 엔진입니다. Visual Studio에서 사용되며, msbuild.exe를 통해 Visual Studio 없이도 빌드가 가능합니다. Windows, macOS, Linux 등 다양한 환경에서 작동하며, .NET Core 및 .NET Framework를 지원합니다. 로컬라이제이션, 기여 가이드라인, 다양한 빌드 컴포넌트(MSBuild.exe, Microsoft.Build API 등)를 제공합니다. MIT 라이선스로 배포됩니다.
