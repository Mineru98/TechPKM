Microsoft Build Engine(MSBuild)은 애플리케이션 빌드를 위한 플랫폼으로, XML 기반 프로젝트 파일을 사용해 소프트웨어 빌드 프로세스를 제어합니다. Visual Studio와 통합되지만 독립적으로 실행 가능하며, 다양한 환경에서 프로젝트 빌드 및 관리가 가능합니다. 주요 구성 요소로는 명령줄 도구, API 라이브러리, 태스크 프레임워크 등이 포함됩니다. MIT 라이선스로 배포되며, Windows, Unix 계열 시스템을 지원합니다.
