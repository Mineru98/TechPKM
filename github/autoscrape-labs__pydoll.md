Pydoll은 현대적인 봇 회피 및 고성능 스크래핑을 위해 설계된 비동기 네이티브 웹 자동화 프레임워크입니다. Chrome DevTools 프로토콜(CDP)에 직접 연결하여 `navigator.webdriver` 탐지 없이 자연스러운 사용자 행동을 시뮬레이션하며, 강력한 타이핑 시스템과 네트워크 제어 기능을 제공합니다. 하이브리드 자동화(UI + API), 인간형 상호작용, 브라우저 지문 제어 등 전문가급 자동화 기능을 지원합니다.  

주요 특징:  
- **봇 회피 설계**: 현실적인 키 입력/스크롤 시뮬레이션  
- **비동기 & 타이핑**: `asyncio` 기반 성능 최적화 및 IDE 자동완성 지원  
- **네트워크 제어**: 트래픽 모니터링/차단 및 API 요청 상속 기능  
- **확장성**: 멀티 탭/컨텍스트 관리 및 원격 연결 지원  
- **안정화**: `@retry` 데코레이터로 실패 복구 자동화  

파이썬 3.10+ 환경에서 `pip install pydoll-python`으로 설치 가능합니다.
