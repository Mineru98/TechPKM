Continue는 개발자가 반복적인 코딩 작업을 AI 에이전트에게 위임하여 생산성을 높이는 오픈소스 플랫폼입니다. PR 리뷰, 스케줄 기반 작업, 이벤트 트리거 등의 워크플로우를 자동화하고, CLI, IDE, 클라우드 환경에서 실시간으로 에이전트 작업을 제어할 수 있습니다. 주요 기술 스택은 JavaScript 기반이며, VS Code, JetBrains 등과의 통합으로 개발 흐름을 간소화합니다. Apache 2.0 라이선스 하에 제공됩니다.
