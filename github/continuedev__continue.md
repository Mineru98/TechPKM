Continue는 개발자가 반복적인 코딩 작업을 자동화하고 흥미로운 부분에 집중할 수 있도록 지원하는 Continuous AI 플랫폼입니다. 클라우드, CLI, IDE 에이전트(Visual Studio Code 및 JetBrains)를 통해 PR 검토, 예약 기반 작업, 실시간 워크플로우 실행 등을 자동화하며, 터미널 인터페이스(TUI)나 헤드리스 모드로 유연한 사용이 가능한 개발자 생산성 도구입니다. Apache 2.0 라이선스로 배포되며, GitHub 프로젝트와 Discord 커뮤니티를 통해 지속적으로 발전하고 있습니다.
