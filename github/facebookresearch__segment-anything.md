**Segment Anything Model (SAM)**은 이미지 및 비디오에서 객체 마스크를 생성하는 파운데이션 모델입니다. 점, 박스 등의 프롬프트를 입력받아 고정밀 분할을 수행하며, 1.1억 개의 마스크가 포함된 SA-1B 데이터셋으로 학습되었습니다. 제로샷 방식으로 다양한 세그멘테이션 작업에 적용 가능하며, 비디오 확장 버전인 SAM 2는 실시간 처리 및 메모리 스트리밍 기술을 지원합니다. 주요 사용 사례로는 이미지/비디오 내 객체 추출, 마스크 생성, ONNX 형식 변환, 웹 데모 통합 등이 포함됩니다.
