asdf는 단일 CLI 도구로 여러 프로그래밍 언어 런타임 버전을 프로젝트 단위로 관리할 수 있는 확장 가능한 버전 관리 시스템입니다. 기존 언어별 관리 도구(gvm, nvm 등)의 기능을 통합하며, 플러그인을 통해 새로운 언어를 지원합니다. 각 프로젝트별로 `.tool-versions` 파일을 사용해 버전을 지정하고, 디렉터리 이동 시 자동으로 런타임을 전환하는 것이 핵심 특징입니다. Bash, Zsh 등 주요 쉘 자동 완성 기능도 제공합니다.
