asdf는 여러 프로그래밍 언어의 런타임 버전을 프로젝트별로 관리할 수 있는 CLI 도구입니다. `gvm`, `nvm`, `rbenv`, `pyenv` 등의 기능을 통합하며 플러그인을 통해 확장 가능합니다. 간단한 인터페이스로 일관된 명령어 사용이 가능하고, 프로젝트별 `.tool-versions` 설정 파일을 지원합니다. Shell 자동 완성 및 기존 설정 파일 호환성을 제공합니다.
