이 프로젝트는 YOLOv3의 최소한의 PyTorch 구현체로, 훈련, 추론 및 평가 기능을 제공합니다. YOLOv4와 YOLOv7 가중치도 호환되며, COCO 데이터셋에서 검증 시 논문 대비 경쟁력 있는 성능을 보입니다. 사용자 정의 데이터셋 훈련, TensorBoard 지원, CLI 도구(`yolo-detect`, `yolo-train`, `yolo-test`) 및 Python API를 통해 유연하게 활용할 수 있습니다.
