Qdrant 데이터베이스 컬렉션을 위한 범용 프론트엔드 도구로, 트랜스포머.js를 통해 브라우저에서 직접 모델을 로드하여 벡터 검색을 수행합니다. 사용자는 Qdrant 서버 URL, 대상 컬렉션, 모델 크기(양자화/정확도)를 선택하여 검색 쿼리를 실행할 수 있으며, 북마크를 통해 설정을 저장할 수 있습니다. 정적 페이지로 제공되어 별도의 서버 인프라 없이 배포 가능합니다.
