Remotion은 React를 기반으로 프로그래밍 방식으로 비디오를 생성하기 위한 오픈소스 프레임워크입니다. 웹 기술(CSS, Canvas, SVG 등)과 프로그래밍 기법(변수, 함수, API)을 활용하여 맞춤형 동영상을 제작할 수 있으며, React의 재사용 가능한 컴포넌트와 빠른 리로딩 기능을 지원합니다. 주요 사용 사례로는 코드 기반 영상 제작과 GitHub Unwrapped와 같은 개인화된 연말 요약 영상이 있습니다. Node.js 환경에서 `npx create-video@latest` 명령어로 시작할 수 있으며, 기업 사용 시 라이선스 요구사항을 확인해야 합니다.
