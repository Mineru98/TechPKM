이 프로젝트는 2025년 국립국어원 주최 '인공지능의 한국어 능력 평가' 경진대회의 '한국어 어문 규범 기반 생성(RAG)' 과제에 대한 베이스라인 모델을 재현한 코드 저장소입니다. Qwen3-8B, HyperCLOVAX Text 1.5B 등의 모델을 활용해 한국어 문장 선택 및 이유 생성 작업을 평가하며, EM(Exact Match), ROUGE, BERTScore 등의 메트릭을 사용합니다. 주요 기능으로는 JSON 형식의 입력 데이터 기반 추론 실행과 모델 성능 평가 스크립트 제공이 포함됩니다.
