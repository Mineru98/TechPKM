**Fast Instagram Scraper**는 Tor 네트워크를 통해 인스타그램 해시태그 및 위치 기반 게시물의 메타데이터와 이미지를 고속으로 수집하는 오픈소스 도구입니다. 로그인이나 API 키 없이도 동작하며, 멀티쓰레딩을 지원해 대량의 데이터를 효율적으로 수집할 수 있습니다. 연구 목적의 데이터 분석에 최적화되어 있으며, CSV 또는 JSON 형식으로 결과를 저장합니다. 현재 Torpy의 유지보수 중단으로 인해 일부 기능이 작동하지 않으나, 커뮤니티 기여를 통해 개선이 이루어지고 있습니다.
