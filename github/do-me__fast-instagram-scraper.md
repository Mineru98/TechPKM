이 프로젝트는 Tor 네트워크를 통해 인스타그램 해시태그 및 위치 ID의 게시물 메타데이터와 이미지를 빠르게 수집하는 도구입니다. 로그인이나 API 키 없이 대량의 데이터를 수집할 수 있으며, 멀티쓰레딩과 병렬 처리를 지원하여 효율적인 크롤링이 가능합니다. 연구 목적으로 설계되었으며, Instagram API 변경 사항으로 인해 일부 기능이 불안정할 수 있습니다. CSV 또는 JSON 형식으로 저장되며, 지리공간 데이터 분석을 위한 전처리 가이드도 제공됩니다.
