이 프로젝트는 Spring Cloud를 활용한 마이크로서비스 아키텍처(MSA) 구현을 위한 샘플 프로젝트입니다. Eureka Server, Config Server, Gateway Server 및 테스트용 더미 API 서버로 구성된 멀티 모듈 Gradle 프로젝트입니다. 스프링 기반 분산 시스템의 핵심 컴포넌트들을 로컬 환경에서 실습할 수 있도록 설계되었습니다. Java 17, Spring Boot, Lombok 등의 기술 스택을 사용하며, 각 모듈별 독립적인 실행 및 설정이 가능합니다.
