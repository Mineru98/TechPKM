이 프로젝트는 Spring Cloud를 활용한 마이크로서비스 아키텍처 구현을 보여주는 멀티모듈 Gradle 샘플입니다. Eureka 서버, Config 서버, 게이트웨이 서버, 더미 API 서버로 구성되어 서비스 등록/발견, 외부 설정 관리, API 게이트웨이 기능을 시연할 수 있습니다. Java 17, Gradle, Lombok을 기반으로 로컬 개발 환경에서 Spring Cloud 기반 마이크로서비스 아키텍처를 빠르게 학습하거나 검증할 수 있도록 설계되었습니다.
