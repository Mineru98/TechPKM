Kiwi는 빠른 속도와 높은 정확도를 지향하는 한국어 형태소 분석기 라이브러리입니다. 세종 품사 태그 체계를 기반으로 하여 웹 텍스트에서 약 87%, 문어 텍스트에서 약 94%의 정확도를 제공하며, 오타 교정 기능과 멀티스레딩 지원 기능을 포함합니다. C++로 구현되어 다양한 프로그래밍 언어에서 사용 가능하며, 소형/중형/대형 모델을 선택하여 활용할 수 있습니다. 오픈소스(LGPL v3)로 공개되어 있으며, 웹 데모 페이지와 여러 언어별 래퍼를 제공합니다.
