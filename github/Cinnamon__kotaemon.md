kotaemon은 문서 기반 질의응답(RAG)을 위한 오픈소스 챗봇 UI 프로젝트입니다. 최종 사용자와 개발자 모두를 위해 설계되었으며, 다양한 LLM(연쇄형/로컬) 지원, 사용자 친화적인 인터페이스, 확장 가능한 하이브리드 검색 파이프라인을 특징으로 합니다. 멀티모달 문서 처리, 그래프 기반 인덱싱, 실시간 인용 생성 기능을 통해 정확한 정보 검색이 가능하며, Gradio 기반의 커스터마이징과 Docker/로컬 설치 옵션을 제공합니다.
