kotaemon은 문서를 기반으로 질문과 답변을 수행할 수 있는 오픈소스 RAG(Retrieval-Augmented Generation) UI 도구입니다. 최종 사용자와 개발자 모두를 고려해 설계되었으며, 다양한 LLM(OpenAI, Azure, Ollama 등)과 로컬 모델을 지원하고 하이브리드(풀텍스트 + 벡터) 검색 파이프라인을 통해 높은 검색 품질을 제공합니다. 사용자 친화적인 웹 인터페이스를 통해 문서 관리, 멀티모달 QA, 인용 지원, 그래프 기반 검색 등 고급 기능을 활용할 수 있습니다. Docker와 로컬 설치 옵션을 모두 지원하며, 커스터마이징이 용이한 구조로 개발자에게 유연성을 제공합니다.
