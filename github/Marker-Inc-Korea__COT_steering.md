이 프로젝트는 **프롬프트 없이 체인 오브 사고(CoT) 추론을 구현하는 디코딩 전략**을 제시합니다. 기존 구현에서 발생한 오류를 수정하여 원 논문의 방법론을 충실히 재현했으며, **스티어링 토큰**을 도입해 모델의 추론 경로를 명시적으로 제어합니다. 토큰 수준에서 제약을 가해 검색 공간을 좁히는 방식으로, 모델의 사전 학습된 편향을 극복하고 체계적인 추론 경로를 유도합니다. 33B 파라미터 모델로 84점(한국 수능 언어 영역)의 성능을 달성하며, 대규모 모델 대비 효율적인 추론 가능성을 입증했습니다. 

핵심 기술 스택은 Python과 vLLM을 기반으로 하며, 설치 및 실행 가이드, 평가 결과, 환경 호환성 주의사항 등이 상세히 문서화되어 있습니다.
