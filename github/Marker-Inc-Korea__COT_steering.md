**CoT Steering**은 프롬프트 없이 **Chain-of-Thought(CoT)** 추론을 구현하는 디코딩 전략입니다. 이 방법은 모델의 내부 추론 경로를 탐색하기 위해 **top-k 토큰 샘플링**을 활용하며, 기존 구현의 오류를 수정한 충실한 재현을 기반으로 합니다. 추가적으로 **스티어링 토큰**을 도입해 모델의 추론 궤적을 명시적으로 제어함으로써 검색 공간을 효율적으로 좁히고 구조화된 CoT 경로를 유도합니다. 2025년 한국 수능 한국어 영역에서 33B 파라미터 모델로 **84점**의 성능을 달성하며, 대규모 모델(100B~685B)과 경쟁력 있는 결과를 보입니다. 토큰 수준 제어 방식으로 유연성과 계산 효율성을 확보한 것이 특징입니다.
