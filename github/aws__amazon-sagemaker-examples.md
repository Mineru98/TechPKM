이 프로젝트는 Amazon SageMaker를 활용한 머신러닝 모델 구축, 훈련, 배포를 위한 Jupyter 노트북 예제들을 제공합니다. SageMaker-Core라는 새로운 Python SDK를 소개하여 객체 지향 인터페이스와 리소스 체이닝 기능을 통해 SageMaker 리소스 관리를 간소화합니다. 머신러닝 워크플로우 전반에 걸친 엔드투엔드 예제와 데이터 준비, 모델 훈련, 배포 및 모니터링, 생성형 AI, MLOps, 책임 있는 AI 등 다양한 주제를 다룹니다. AWS 계정, IAM 역할, S3 버킷 설정 후 SageMaker 노트북 인스턴스에서 실행 가능하며, 커뮤니티 기여를 통해 확장됩니다.
