이 프로젝트는 Amazon SageMaker를 활용한 머신러닝 모델 구축, 학습, 배포 방법을 보여주는 예제 Jupyter 노트북을 제공합니다. SageMaker-Core라는 새로운 Python SDK를 통해 객체 지향 인터페이스와 리소스 체이닝 기능을 제공하여 SageMaker 리소스 관리를 간소화합니다. 다양한 카테고리(엔드투엔드 ML 라이프사이클, 데이터 준비, 모델 빌드 및 학습, 배포 및 모니터링 등)의 예제를 포함하며, 공식 SageMaker 팀에서 유지보수하는 공식 저장소입니다. Apache 2.0 라이선스로 배포됩니다.
