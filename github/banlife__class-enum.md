"class-enum"은 TypeScript의 기본 열거형을 대체하기 위해 설계된 라이브러리로, Java 스타일의 생성자 기반 열거형을 구현합니다. 클래스 상수와 유사한 방식으로 사용 가능하며, 각 열거형 값에 추가 속성(타이틀, 색상 등)을 정의할 수 있는 확장성을 제공합니다. Vue.js와의 통합 예시를 포함하여 다양한 사용 사례를 지원합니다.  

이 프로젝트는 NPM을 통해 설치 가능하며, `values()`, `valueOf()`, `name()`, `equals()` 등의 API를 통해 열거형의 배열 조회, 값 검색, 이름 확인, 동일성 비교 기능을 제공합니다. TypeScript enum의 한계를 극복하기 위해 개발되었습니다.
