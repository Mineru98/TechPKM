이 프로젝트는 영어와 한국어 기반 Gemma 7B 모델의 셀프 어텐션 레이어를 병합하여 다국어 처리 능력이 향상된 모델을 생성합니다. 두 모델의 강점을 결합해 어텐션 헤드를 2배로 확장함으로써 영어-한국어 혼합 작업에 최적화된 언어 모델을 구축할 수 있습니다. Python 스크립트로 간단하게 모델 병합이 가능하며, 병합된 모델은 단일 언어 모델 대비 다국어 이해 성능이 개선됩니다.
