제공된 입력 정보에는 `packages/ai/README.md` 파일 경로에 대한 구체적인 README 내용이 포함되어 있지 않아, 프로젝트의 목적, 기술 스택, 특징 등을 분석할 수 없습니다. Obsidian용 메타데이터 문서를 생성하려면 README 파일의 실제 내용이 필요합니다.

필요한 정보가 포함된 README 내용을 제공해 주시면, 다음 형식에 맞춰 메타데이터 문서를 생성해 드리겠습니다:

```
---
Language: {main_code_language}
tags:
 - {tag_name_1}
 - {tag_name_2}
aliases:
 - {alias_1}
 - {alias_2}
url: {github_url}
---
{summary} (한국어 요약)
```

예시 (가상 데이터 기준):
```
---
Language: Python
tags:
 - 인공지능
 - 머신러닝
 - 자연어 처리
aliases:
 - AI 패키지
 - 자연어 모델
url: https://github.com/example/repo/tree/main/packages/ai
---
이 프로젝트는 자연어 처리를 위한 AI 모델을 구현하는 Python 패키지입니다. BERT 기반 아키텍처를 사용하여 텍스트 분류 및 생성 작업을 지원합니다. 연구 및 상용화 환경 모두에 적용 가능한 모듈식 설계입니다.
```
