json-render는 AI 생성 UI를 안전하게 렌더링하기 위한 라이브러리로, 사용자가 정의한 구성 요소와 스키마에 따라 예측 가능하고 제약 조건이 적용된 UI를 생성합니다. 사용자가 자연어로 대시보드나 위젯을 요청하면, AI는 사전 정의된 구성 요소 및 데이터 바인딩만 사용하여 JSON 트리를 생성하며, 이를 React로 실시간 렌더링할 수 있습니다. Apache-2.0 라이선스로 배포됩니다.
