json-render는 사용자가 프롬프트로 대시보드, 위젯, 앱, 데이터 시각화를 생성할 수 있게 하는 프로젝트입니다. AI 출력을 사전 정의된 컴포넌트로 제한해 예측 가능하고 안전한 렌더링을 보장하며, React 기반 실시간 스트리밍 렌더링을 지원합니다. 주요 특징으로는 구성 요소 카탈로그 정의, 조건부 가시성, 액션 처리, 데이터 바인딩 검증 등이 있습니다. TypeScript와 Zod 스키마를 사용해 타입 안전성을 강화한 것이 핵심 특징입니다. Apache-2.0 라이선스로 배포됩니다.
