backtrader는 파이썬 기반 알고리즘 트레이딩 및 백테스팅 플랫폼입니다. 다양한 데이터 피드 지원(CSV, 야후 파이낸스, IB, Oanda 등), 실시간 거래 기능, 122개 내장 기술 지표, 브로커 시뮬레이션, 리스크 분석 등을 제공하며, matplotlib을 활용한 시각화 기능도 포함되어 있습니다. Python 3.2 이상 및 PyPy 환경에서 작동하며, 커스텀 전략 개발이 용이한 것이 특징입니다.
