이 프로젝트는 Cursor AI와 Figma 간의 모델 컨텍스트 프로토콜(MCP) 통합을 구현하여, Cursor가 Figma 디자인을 읽고 프로그래밍 방식으로 수정할 수 있도록 합니다. TypeScript 기반 MCP 서버, WebSocket 통신, Figma 플러그인 구성요소를 포함하며, 대량의 텍스트 교체, 컴포넌트 인스턴스 속성 전파, 프로토타입 반응 시각화 등 디자인 작업을 자동화하는 다양한 도구를 제공합니다. MIT 라이선스로 배포되며, 커서와 Figma 간의 실시간 디자인 협업을 가능하게 합니다.
