Oracle은 코드 파일과 프롬프트를 묶어 GPT-5, Gemini 등 다양한 AI 모델에 컨텍스트를 제공하는 CLI 도구입니다. API 또는 브라우저 모드(실험적)로 ChatGPT/Gemini와 연동해 코드 분석, 문서 생성, 테스트 검토 등을 수행할 수 있습니다. 다중 모델 비교, 세션 관리, 파일 토큰 사용량 보고 등 개발 워크플로우 최적화를 위한 고급 기능을 제공합니다. Node.js 22+ 기반으로 macOS/Linux/Windows에서 사용 가능합니다.
