Sparrow는 문서 데이터를 구조화된 JSON 형식으로 추출하는 오픈소스 프레임워크입니다. Vision LLM, MLX, Ollama 등의 다양한 백엔드를 지원하며, 은행 명세서, 송장, 금융 테이블 등 다양한 문서에서 데이터를 자동으로 추출할 수 있습니다. 플러그형 아키텍처, 실시간 처리, 스키마 검증, API 우선 설계 등의 기능을 제공하며, macOS MLX 백엔드, NVIDIA/AMD GPU 및 CPU 환경에서 동작합니다. CLI, API, 웹 UI를 통해 대화형 문서 처리가 가능하며, 복잡한 워크플로우를 위한 에이전트 시스템을 내장하고 있습니다. GPL v3 라이선스로 배포되며, 상용 라이선스는 별도로 제공됩니다.
