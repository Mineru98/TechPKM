Sparrow는 Python 기반의 문서 처리 오픈소스 프로젝트로, ML/LLM 및 Vision LLM을 활용해 구조화된 데이터 추출과 지시 호출 기능을 제공합니다. 은행 명세서, 송장, 재무 보고서 등 다양한 문서에서 JSON 형식의 구조화된 데이터를 추출할 수 있으며, MLX, Ollama, Hugging Face 등 다양한 백엔드를 지원합니다. CLI, API, 웹 인터페이스(UI)를 통해 사용자 친화적인 문서 처리 환경을 제공하며, 테이블 인식, 실시간 처리, 결과 검증 등의 고급 기능을 포함합니다. GPL 3.0 라이선스로 배포되며, 기업용 라이선스도 제공됩니다.
