이 프로젝트는 React Native 환경에서 고성능 인터랙티브 바텀 시트를 제공하는 라이브러리입니다. 완전히 구성 가능한 옵션과 다양한 스크롤 가능한 컴포넌트(FlatList, ScrollView 등)와의 통합, 키보드 처리, 모달 표현 기능 등을 지원합니다. React Native Web 호환성과 TypeScript 기반 개발, Reanimated v1-3 버전 호환성을 갖추었으며, Expo 환경에서도 사용 가능합니다. 주요 활용 사례로는 메뉴 선택, 폼 입력, 컨텐츠 확장 UI 등이 있습니다.
