이 프로젝트는 완전히 구성 가능한 옵션을 가진 고성능 인터랙티브 하단 시트를 제공하는 React Native 라이브러리입니다. React Native Web 지원, 동적 사이징, 키보드 처리, 스크롤 가능 컴포넌트 통합 등 다양한 기능을 제공하며 TypeScript로 구현되었습니다. Reanimated v1-3과 호환되며 Expo에서도 사용할 수 있습니다. 주요 목적은 모바일 애플리케이션에서 유연하고 매끄러운 하단 시트 인터랙션을 구현하는 것입니다.
