Axolotl은 대규모 언어 모델(LLM)의 미세 조정(fine-tuning)을 간소화하는 무료 오픈소스 프레임워크입니다. GPT-OSS, LLaMA, Mistral 등 다양한 모델 지원, LoRA/QLoRA/전체 미세 조정, 멀티모달 훈련 기능을 제공하며, 단일 YAML 설정 파일로 데이터셋 전처리부터 추론까지 전체 파이프라인을 관리할 수 있습니다. 멀티GPU/노드 훈련, 성능 최적화(플래시 어텐션, 멀티패킹 등)를 통해 효율적인 LLM 훈련을 가능하게 합니다.
