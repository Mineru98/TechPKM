OLMo는 Allen Institute for AI(AI2)에서 개발한 오픈소스 언어 모델입니다. 웹 기반 데이터와 고품질 데이터를 활용한 2단계 훈련 방식을 통해 1B~32B 파라미터 규모의 다양한 모델 버전을 제공하며, 과학 연구자들을 위해 설계되었습니다. PyTorch와 Hugging Face 변환기를 지원하며, 추론, 양자화, 평가 도구 등을 포함합니다. 최신 업데이트는 별도 저장소에서 관리됩니다.
