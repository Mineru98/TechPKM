WAVE(Web Application for Video Editing)는 브라우저 기반 동영상 편집 도구로, 별도의 프로그램 설치 없이 웹에서 영상 편집이 가능합니다. 서버 부하 없이 클라이언트 측에서 영상 처리(회전/반전, 확대/축소, 자르기, 필터 적용 등)를 수행하며, 편집된 영상을 로컬 다운로드 또는 서버 업로드할 수 있습니다. HTML5 \<video> 요소와 Canvas를 활용해 프레임 단위 편집을 구현하고, 사용자 친화적인 인터페이스를 제공합니다.  

> 핵심 기능: 클라이언트 측 미디어 처리, 실시간 편집 미리보기, 해상도 설정, 편집 히스토리 관리(최대 20단계)
