WAVE는 별도 프로그램 설치 없이 브라우저만으로 동영상 편집을 할 수 있는 클라이언트 기반 웹 애플리케이션입니다. HTML5 \<video> 요소를 활용해 프레임 단위로 편집 효과를 적용하고, 서버 부하 없이 로컬에서 처리 후 MP4로 인코딩하는 것이 특징입니다. 회전/반전, 확대/축소, 자르기, 필터 적용, 서명 추가 등 다양한 기능을 제공하며, 편집 내역을 최대 20개까지 관리할 수 있습니다. 핸드폰으로 찍은 영상을 간편하게 편집할 때 유용한 도구로 설계되었습니다.
