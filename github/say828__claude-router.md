Claude Router는 Claude Code를 로컬 Ollama 설치 및 오픈소스 모델과 연결하는 범용 프록시 서버입니다. Anthropic API와 호환되며, 프로젝트 구조에 맞춰 로컬 LLM을 지능적으로 활용할 수 있도록 지원합니다. Python, JavaScript 등 다양한 환경에서 별도 설정 없이 즉시 사용 가능한 원클릭 실행 기능을 제공합니다.  

이 프로젝트는 개발자들이 Claude Code의 강력한 기능을 로컬 환경(`gpt-oss:20b` 등)에서 자유롭게 사용할 수 있게 하는 것이 핵심 목적입니다. Docker 지원 및 테스트 케이스를 포함한 실용적인 개발 생태계를 갖추고 있습니다.
