이 프로젝트는 이미지나 비디오에서 픽셀 수를 시각화하는 Python 패키지입니다. 컴퓨터 비전 분석 단계에서 픽셀 강도를 실시간으로 관찰할 수 있으며, 비디오 처리 시 슬라이더를 통해 프레임별 픽셀 변화를 모니터링할 수 있습니다. OpenCV와 Matplotlib를 기반으로 구현되었으며, 이미지와 시각화 도구를 병렬로 표시하는 기능을 제공합니다. 연구와 개발 목적의 시각적 분석에 최적화된 도구입니다.
