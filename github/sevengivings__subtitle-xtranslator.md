subtitle-xtranslator는 동영상/오디오 파일에서 음성을 인식하고 Google Cloud/DeepL API로 자막을 번역하는 파이썬 자동화 스크립트입니다. Whisper, stable-ts, faster-whisper 기반의 음성 인식 기능과 번역 API 연동을 통해 자막 생성 및 변환 프로세스를 간소화합니다. 짧은 자막 필터링, 음성 분리 처리, 실시간 모니터링 등 부가 기능을 지원하며, Windows/Linux/Colab 환경에서 활용 가능합니다. 전문 번역보다는 편의성 중심의 개인용 도구로의 사용에 적합합니다.
