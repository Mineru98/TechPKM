Bolt.new은 브라우저 내에서 AI 기반 풀스택 웹 개발을 지원하는 도구로, 로컬 설정 없이 애플리케이션 생성, 실행, 편집, 배포가 가능합니다. StackBlitz의 WebContainers 기술을 활용해 npm 패키지 설치, Node.js 서버 실행, API 연동 등 실제 개발 환경을 제공하며, AI 모델이 파일 시스템부터 배포까지 전체 환경을 제어할 수 있습니다. 개발자, PM, 디자이너 등 다양한 사용자가 프로덕션 수준의 애플리케이션을 쉽게 구축할 수 있도록 설계되었습니다.
