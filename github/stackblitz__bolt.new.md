Bolt.new은 브라우저에서 AI 기반 풀스택 웹 개발을 지원하는 도구입니다. StackBlitz의 WebContainers 기술을 활용해 로컬 설정 없이 패키지 설치, 백엔드 실행, 코드 편집, 배포까지 전 과정을 브라우저에서 수행할 수 있습니다. AI 모델이 파일 시스템, 터미널, 서버 등을 직접 제어하여 애플리케이션 전체 라이프사이클을 관리하는 것이 특징입니다. 개발자, PM, 디자이너 등 다양한 사용자가 복잡한 웹 애플리케이션을 쉽게 구축할 수 있도록 지원합니다.
