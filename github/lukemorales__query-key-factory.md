Query Key Factory는 TanStack Query(React Query)를 위한 타입 안전한 쿼리 키 관리 도구입니다. 이 라이브러리는 쿼리 키 생성, 자동 완성, 캐시 무효화 등을 간편하게 처리할 수 있도록 설계되었습니다. 주요 기능은 표준화된 키 생성, 컨텍스트 기반 쿼리 지원, 단일 소스 관리, 타입 추론 및 자동 완성 기능 등을 포함합니다. 프로젝트 전반에서 일관된 쿼리 키 관리를 통해 개발 효율성을 높이고 실수를 줄일 수 있습니다.
