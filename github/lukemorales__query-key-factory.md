Query Key Factory는 TanStack Query(React Query)와 호환되는 타입세이프 쿼리 키 관리 라이브러리입니다. 이 도구는 쿼리 키 구성을 간편하게 하고 자동완성 기능을 지원하며, 캐시 무효화 및 컨텍스트 기반 쿼리 생성을 단순화합니다. 단일 파일 또는 기능별 분산 방식으로 쿼리 키를 정의하고, 타입 추론과 스마트 자동완성을 통해 개발 생산성을 향상시킵니다.  

### 주요 기능
- 표준화된 쿼리 키 생성 (배열 구조 지원)
- 컨텍스트 기반 하위 쿼리 관리
- 캐시 스코프 정의 및 무효화
- 타입세이프한 쿼리 함수 컨텍스트 타입 추론
- 단일/분산 방식의 쿼리 키 저장소 정의 및 병합
