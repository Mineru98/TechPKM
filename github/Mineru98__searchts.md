tsql은 TypeScript 객체나 객체 배열을 쿼리하기 위한 JSON 기반 쿼리 언어입니다. SQL 문법과 유사한 구조로, 필드 일치, 범위 검색, 부정 조건, 정규식, 중첩 객체/배열 검색 등 다양한 조건을 지원합니다. 주로 데이터 필터링 및 검색 시나리오에 활용되며, `matchObject`, `matchArray` 등의 API로 객체/배열과의 일치 여부를 확인합니다. 깊은 중첩 구조와 텍스트 검색(부분 일치, 단어 일치 등)을 지원하는 것이 특징입니다.
