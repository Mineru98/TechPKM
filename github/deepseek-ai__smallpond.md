smallpond는 DuckDB와 3FS를 기반으로 구축된 경량 데이터 처리 프레임워크입니다. PB 규모의 데이터셋 처리 성능을 제공하면서도 장기 실행 서비스 없이 간편한 운영이 가능한 것이 특징입니다. 분산 환경에서 SQL 기반 데이터 처리 파이프라인을 구축할 수 있는 오픈소스 프로젝트입니다.

이 프로젝트는 금융 분석 같은 고성능 데이터 처리 시나리오에 최적화되어 있으며, 파티션 관리 및 병렬 쿼리 실행 기능을 기본 제공합니다. MIT 라이선스로 배포되는 Python 패키지로, PyPI를 통해 간단히 설치할 수 있습니다.
