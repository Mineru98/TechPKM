pytube는 YouTube 동영상을 다운로드하기 위한 가볍고 독립적인 Python 라이브러리 및 명령줄 도구입니다. 종속성 없이 신뢰성 높은 다운로드를 지원하며, 플레이리스트 다운로드, 다운로드 상태 콜백, 자막 추출 등 다양한 기능을 제공합니다. Python 스크립트와 CLI에서 모두 사용할 수 있어 유연성이 뛰어납니다. 공식 문서에서 자세한 사용법을 확인할 수 있습니다.
