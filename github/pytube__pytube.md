pytube는 YouTube 동영상을 다운로드하기 위한 가볍고 종속성이 없는 Python 라이브러리이자 명령줄 도구입니다. 이 프로젝트는 비디오 스트리밍 처리, 다운로드 진행 상태 콜백, 플레이리스트 다운로드 및 자막 추출 기능을 제공합니다. Python 스크립트와 CLI 인터페이스를 통해 쉽게 사용할 수 있으며, 다양한 해상도와 형식의 스트림을 지원합니다.  

주요 특징으로는 DASH 및 점진적 스트림 지원, .srt 형식 자막 출력, 썸네일 URL 캡처 기능 등이 있습니다. 문서에 따르면 PyPI를 통해 쉽게 설치할 수 있으며, 최신 기능은 GitHub 저장소에서 직접 설치할 수 있습니다.
