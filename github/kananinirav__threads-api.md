메타의 Threads(threads.net)를 위한 비공식 루비 클라이언트 라이브러리입니다. 사용자 조회, 스레드/댓글 조회, 게시물 검색 등의 기능을 제공하며, 소셜 미디어 데이터 분석에 활용할 수 있습니다. 현재 페이지네이션 지원, 테스트 케이스 추가 등 개선 예정입니다.
