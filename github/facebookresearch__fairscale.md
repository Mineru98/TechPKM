FairScale은 고성능 및 대규모 훈련을 위한 PyTorch 확장 라이브러리입니다. 제한된 자원으로도 모델을 확장하려는 연구자들을 위해 최신 분산 훈련 기술을 조합 가능한 모듈과 사용하기 쉬운 API 형태로 제공합니다. 주요 설계 원칙은 사용성, 모듈성, 성능이며, 특히 FSDP(Fully Sharded Data Parallel)를 통해 대규모 신경망 모델 훈련을 최적화합니다. 리서치 및 프로덕션 환경 모두에서 확장 가능한 딥러닝 훈련을 지원하는 것이 핵심 목적입니다.
