NVTOP은 리눅스 환경에서 GPU 및 가속기를 모니터링하는 CLI 도구로, htop과 유사한 인터페이스를 제공합니다. AMD, 인텔, NVIDIA, 퀄컴 Adreno 등 다양한 GPU 벤더를 지원하며, 실시간 사용률, 온도, 메모리 정보 등을 표시합니다. 커널 드라이버의 fdinfo 인터페이스를 활용하여 프로세스별 GPU 사용 현황을 분석할 수 있으며, 커스터마이징 가능한 설정 메뉴를 통해 사용자 정의가 가능합니다. 오픈소스 GPLv3 라이선스로 배포됩니다.
