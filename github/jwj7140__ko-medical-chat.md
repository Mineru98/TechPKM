의료 분야 한국어 대화 특화 소형 언어모델 프로젝트입니다. QLoRA 기법으로 학습된 5.8B 파라미터 모델로, 환자-의사 대화 시뮬레이션을 위한 데이터셋과 학습 코드, 웹 UI를 포함합니다. MedText/ChatDoctor 데이터를 GPT-3.5로 변환해 구축한 의료 대화 데이터셋 기반으로 개발되었습니다. 현재 모델 한계점으로 실제 진료 대체 불가능함을 명시하고 있으며, 향후 RLHF 강화학습을 계획 중입니다.
