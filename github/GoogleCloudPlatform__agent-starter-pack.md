이 프로젝트는 Google Cloud에서 GenAI 에이전트를 개발하기 위한 **프로덕션-레디 템플릿 패키지**입니다. 인프라, CI/CD, 관측성, 보안 등 부차적인 요소를 대신 처리해 주며, 사용자는 에이전트 로직에 집중할 수 있습니다. ReAct, RAG, 멀티-에이전트, 실시간 API 등 다양한 사전 구성된 템플릿을 제공하며, Cloud Run 또는 Agent Engine에 배포할 수 있습니다. Gemini CLI 통합 및 Vertex AI 평가 도구로 개발 생산성을 높이는 것이 핵심 특징입니다.
