FlexLLMGen은 제한된 GPU 메모리에서 대규모 언어 모델(LLM)을 고처리량으로 실행하기 위한 생성 엔진입니다. IO 효율적인 오프로딩, 압축, 대규모 배치 처리를 통해 처리량을 극대화하며, 단일 저비용 GPU에서 벤치마크, 데이터 처리 등의 처리량 중심 작업에 최적화되었습니다. CPU/디스크 오프로딩과 압축 기술을 결합해 모델 크기와 리소스 제약 사이의 균형을 유연하게 조정합니다.
