Grounded-SAM은 텍스트 프롬프트를 사용하여 이미지 내 객체를 감지하고 세그멘테이션하는 통합 파이프라인 프로젝트입니다. Grounding DINO와 Segment Anything Model(SAM)을 결합하여 텍스트 기반 객체 인식 및 분할 기능을 제공합니다. 다양한 데모(예: 자동 라벨링, 인페인팅, 3D 메시 복원)와 모델 조합 기능을 지원하며, 컴퓨터 비전 작업에서 사용자 지정 가능성을 확장합니다. 주요 응용 분야는 객체 감지, 세그멘테이션, 이미지 편집, 자동 라벨링 등입니다.  

이 프로젝트는 오픈월드 시나리오에서 다양한 시각 작업을 수행하기 위해 여러 강력한 모델을 조립하는 방식을 제시합니다. 사용자는 텍스트 프롬프트로 객체를 지정하거나 자동 라벨링 기능(RAM/Tag2Text 통합)을 활용할 수 있으며, 인페인팅, 3D 복원, 대화형 편집 등 확장 가능한 데모를 제공합니다. GitHub, Hugging Face, Replicate 등에서 다양한 방식으로 접근 가능합니다.
