이 프로젝트는 TensorFlow 2.0 기반으로 구현된 Wide and Deep 모델을 다루며, 근로자 수입 데이터를 활용한 이진분류 예제입니다. 기존 Keras 구현을 데이터 처리/모델 구조로 분리하여 모듈화했으며, Wide 모델의 선형 특성과 Deep 모델의 비선형 특성을 결합한 하이브리드 접근법을 구현했습니다. UCI 머신러닝 레포지토리의 공개 데이터를 사용하며, 실행 가능한 Python 스크립트 형태로 제공됩니다.
