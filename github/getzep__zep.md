Zep은 다중 데이터 소스(채팅 기록, 비즈니스 데이터, 문서, 앱 이벤트)에서 관계 인식형 컨텍스트를 조립하여 AI 에이전트의 정확한 의사결정을 지원하는 엔드투엔드 컨텍스트 엔지니어링 플랫폼입니다. 200ms 미만의 레이턴시로 실시간 컨텍스트를 제공하며, Graphiti 기반 시간 인식 지식 그래프를 활용합니다. 주요 기능은 컨텍스트 수집, 관계 인식 검색(Retrieval), LLM 최적화를 포함합니다. Python, TypeScript, Go SDK를 지원하며 LangChain, AutoGen 등 주요 프레임워크와의 통합 예제를 제공합니다. 커뮤니티 에디션은 더 이상 지원되지 않습니다.
