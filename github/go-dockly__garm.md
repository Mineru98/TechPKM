`gARM彡`는 Go의 간결성과 ARM64 아키텍처의 고유한 기능을 결합한 고성능 컴파일러 프로젝트입니다. ARM64의 산술 명령어 최적화, 조건부 실행, NEON SIMD 연산, 하드웨어 연산 기능 등을 활용하여 Go 코드를 최적화합니다. Apple M 시리즈, AWS Graviton, 임베디드 시스템 등 ARM 기반 환경에서 5배 이상의 성능 향상을 목표로 합니다. 문서에는 ARM 어셈블리 기초, 최적화 기법, 다양한 사용 사례 등이 포함되어 있습니다.
