랭그래프 빌더는 LangGraph 애플리케이션의 인지 아키텍처를 시각적으로 설계할 수 있는 캔버스를 제공하는 도구입니다. 사용자는 디자인한 아키텍처를 바탕으로 Python 및 Typescript용 기본 코드를 생성할 수 있으며, 이는 [langgraph-gen](https://github.com/langchain-ai/langgraph-gen-py) 기술을 기반으로 합니다. 웹 애플리케이션 형태로 제공되며, 현재 데스크톱 환경에 최적화되어 있습니다.
