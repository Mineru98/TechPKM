LangGraph Builder는 LangGraph 애플리케이션의 인지 아키텍처를 설계하고 Python 및 Typescript용 보일러플레이트 코드를 생성하는 강력한 캔버스를 제공하는 웹 기반 도구입니다. langgraph-gen 기술을 기반으로 하며, 데스크톱 환경에서 최적화된 인터페이스를 통해 시각적 설계 및 코드 생성 기능을 지원합니다. 현재 병렬 노드 실행 기능은 지원되지 않습니다.  

(요약: 3문장 구성 - 목적/기능/제한사항)
