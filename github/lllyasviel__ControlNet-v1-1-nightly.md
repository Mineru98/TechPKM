ControlNet 1.1은 스테이블 디퓨전 이미지 생성을 정밀하게 제어하기 위한 딥러닝 모델 솔루션입니다. 14개의 모델(11개 프로덕션 레디, 3개 실험 모델)을 포함하며, 깊이 맵, 윤곽선(Canny), 포즈 추정(OpenPose) 등 다양한 전처리 기법을 지원합니다. 기존 1.0 버전 대비 데이터 품질 개선, 강건성 향상, 새로운 모델 추가 등의 업그레이드가 적용되었습니다. Automatic1111 웹 UI 플러그인을 통해 다중 ControlNet 조합 및 LoRA 통합이 가능하며, 연구 및 실험적 사용을 주목적으로 합니다.
