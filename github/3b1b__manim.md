ManimGL은 설명용 수학 영상 제작을 위해 설계된 정밀한 프로그램 기반 애니메이션 엔진입니다. 3Blue1Brown 채널의 영상 제작에 활용되었으며, LaTeX 통합, 다양한 애니메이션/객체 유형을 지원합니다. Python 3.7+ 기반으로 FFmpeg, OpenGL 등 시스템 요구사항이 있으며, pip를 통해 쉽게 설치할 수 있습니다. 커뮤니티 버전과 별도로 관리되는 3b1b 공식 리포지토리입니다.
