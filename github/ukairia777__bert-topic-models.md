이 프로젝트는 BERT와 SBERT를 결합한 CombinedTM 방식을 활용해 한국어의 문맥적 특성을 반영한 토픽 모델링을 제공합니다. 형태소 분석기 Mecab과 다국어 SBERT 모델을 기반으로 구축되었으며, 50개의 토픽 구조를 가집니다. 별도의 불용어 제거 없이도 기본적인 토픽 분석이 가능하며, 추가 전처리 시 성능 향상이 기대됩니다.  

(참고: 원본 Repo는 영어용이며, 이 저장소는 한국어 적용을 위해 토크나이저와 임베딩을 수정한 버전입니다.)
