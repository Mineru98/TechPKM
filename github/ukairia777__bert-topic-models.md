BERT 기반 문맥 반영 한국어 토픽 모델 프로젝트입니다. CombinedTM 구현에 한국어 Mecab 토크나이저와 SBERT 임베딩을 결합하여 문맥 정보를 반영한 토픽 추출이 가능합니다. 원본 모델을 한국어에 맞게 최적화했으며, 50개 토픽으로 구성된 기본 설정을 제공합니다. 시각적 분석과 함께 불용어 제거 등 추가 전처리 가능성을 제시하며, 한국어 토픽 모델링 연구를 위한 참고 자료로 활용됩니다.
