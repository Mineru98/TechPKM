LizzieYzy는 바둑 엔진(GTP 호환)을 위한 그래픽 인터페이스 도구로, 기존 Lizzie를 확장하여 다양한 분석 기능을 추가한 프로젝트입니다. 주요 기능으로는 실시간 승률 분석(호크아이), 병렬 기보 분석(플래시 분석), 다중 엔진 비교(더블 엔진 모드), 자동 형세 판단, 바둑판 동기화(C#/Java 버전), 사활 문제 분석 등이 포함됩니다. 고해상도 디스플레이를 완벽히 지원하며, Katago·LeelaZero 등 주요 엔진을 통합할 수 있는 것이 특징입니다. Windows 전용으로 개발된 C# 기반 동기화 도구와 Java 기반 도구를 함께 제공합니다.
