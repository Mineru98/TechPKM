LizzieYzy는 바둑 엔진(GTP 프로토콜 지원)을 위한 그래픽 인터페이스로, Lizzie를 기반으로 개발되었습니다. Katago, LeelaZero, ZenGTP 등 다양한 엔진을 지원하며, **승률 분석, 일괄 기보 처리, 실시간 형세 판단, 바둑판 동기화, 다중 엔진 대결** 등 고급 기능을 제공합니다. 고해상도 디스플레이와 시스템 스케일링에 최적화되어 있으며, Windows 전용 C# 기반 동기화 도구와 Java 기반 범용 도구를 포함합니다.  

> 🔍 주요 기능:  
> - **AI 분석 시각화**(승률 차트, 실수 탐지)  
> - **자동 바둑판 동기화**(온라인 플랫폼/이미지 지원)  
> - **분산 학습 모니터링**(KataGo 훈련 시각화)  
> - **죽활 문제 추출** 및 프레임 생성  
> - **이중 엔진 모드**로 분석 비교 가능
