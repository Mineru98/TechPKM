sharp는 Node-API 기반의 고성능 이미지 처리 라이브러리로, 대형 이미지를 웹 친화적인 JPEG, PNG, WebP, GIF, AVIF 형식으로 빠르게 변환하고 리사이징하는 데 특화되어 있습니다. libvips를 기반으로 하며, Node.js, Deno, Bun과 같은 JavaScript 런타임 환경에서 사용할 수 있으며, 회전, 크롭, 합성, 감마 보정 등 다양한 이미지 조작 기능을 제공합니다. 대부분의 macOS, Windows, Linux 시스템에서 추가 의존성 없이 사용할 수 있습니다.
