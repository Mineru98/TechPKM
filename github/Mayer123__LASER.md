LASER는 웹 네비게이션을 위한 LLM 기반 에이전트로, 상태 공간 탐색을 통해 효율적인 웹 상호작용을 구현합니다. 이 프로젝트는 논문 "LASER: LLM Agent with State-Space Exploration for Web Navigation"의 공식 코드로, 재현 가능한 실험 환경을 위해 결정적 시드를 적용했습니다. WebShop 환경을 기반으로 하며, OpenAI API 호출을 최적화하기 위해 캐시 시스템을 도입했습니다.  

요약: 웹 탐색 작업을 위한 LLM 에이전트로, 상태 공간 탐색 전략과 결정적 실행 환경을 특징으로 합니다. 연구 논문 및 재현 가능한 실험 설정을 제공합니다.
