LASER은 웹 네비게이션을 위한 상태 공간 탐색 기능을 갖춘 LLM 기반 에이전트입니다. WebShop 환경을 활용하여 제품 검색 및 구매 작업을 수행하며, 결정적 실행을 통해 OpenAI API 호출 비용을 최적화합니다. 10개의 기본 실행 예시 외에 사용자 정의 에피소드 수로 평가가 가능합니다.
