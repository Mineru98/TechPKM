이 프로젝트는 React 기반의 마크다운 에디터로, 실시간 미리보기 기능과 사용자 친화적인 인터페이스를 제공합니다. 웹 애플리케이션에서 마크다운 콘텐츠를 쉽게 작성 및 편집할 수 있도록 설계되었으며, 다양한 커스터마이징 옵션을 지원합니다. 주요 기능으로는 코드 블록, 이미지, 테이블 등의 마크다운 요소 지원이 포함됩니다. 간결한 구성과 높은 확장성이 특징입니다.
