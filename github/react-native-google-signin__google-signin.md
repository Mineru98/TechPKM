React Native 애플리케이션에서 Google 로그인을 구현하기 위한 공식 라이브러리입니다. MIT 라이선스로 배포되며, 모바일 환경에서 Google OAuth 인증을 간편하게 통합할 수 있도록 설계되었습니다. NPM 패키지(@react-native-google-signin/google-signin)로 배포되며, 최신 문서는 전용 웹사이트에서 제공됩니다.  

(참고: README 본문에 명시적인 프로젝트 목적 설명이 부재하여, 패키지 이름과 컨텍스트를 기반으로 핵심 기능을 추론하여 요약 작성함)
