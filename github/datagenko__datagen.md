DataGen은 한국형 데이터를 쉽게 생성할 수 있도록 지원하는 비영리 오픈소스 프로젝트입니다. 한국어 데이터 생성에 필요한 도구를 제공하며, 조건 없이 자유롭게 사용할 수 있습니다. 다양한 분야의 기여자들이 참여하여 한국어 데이터 생성 인프라를 구축하는 것이 핵심 목적입니다.  

(요약: 한국어 데이터 생성 도구 제공, 비영리 오픈소스, 다분야 협업 프로젝트)
