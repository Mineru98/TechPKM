Pear Desktop은 YouTube Music을 데스크톱 애플리케이션으로 사용할 수 있게 하는 오픈소스 프로젝트입니다. 네이티브 디자인과 플러그인 시스템을 지원하며, macOS/Linux/Windows에서 작동합니다. 사용자 정의 테마 적용 및 확장 기능 개발이 가능하며, 커뮤니티 기반 번역과 지속적인 업데이트가 특징입니다. 비공식 프로젝트로, YouTube/Google과는 무관합니다.
