PyMuPDF4LLM은 PyMuPDF의 확장 라이브러리로, PDF 문서를 대규모 언어 모델(LLM) 처리에 최적화된 Markdown 형식으로 변환하는 데 특화된 프로젝트입니다. 문서 구조(제목, 목록, 표)를 보존하며 텍스트 추출 및 이미지 처리를 지원합니다. RAG(검색 증강 생성) 시나리오에서 특히 유용하며, 다중 열 페이지 및 이미지 임베딩도 처리 가능합니다.  

> 핵심 특징:  
> - PDF를 구조화된 Markdown으로 변환  
> - 표, 이미지, 다단 레이아웃 자동 감지  
> - PyPI를 통한 쉬운 설치(`pip install pymupdf4llm`)  
> - XPS, MOBI 등 다양한 문서 형식 지원
