Open Source Routing Machine(OSRM)은 OpenStreetMap 데이터를 기반으로 동작하는 고성능 라우팅 엔진입니다. HTTP API, C++ 라이브러리, Node.js 래퍼를 통해 최단 경로 탐색, 좌표 매칭, 거리 행렬 계산 등 다양한 라우팅 서비스를 제공합니다. C++로 작성되었으며, 대규모 데이터 처리 및 실시간 경로 탐색에 최적화되어 있습니다.  

주요 기능으로는 Nearest, Route, Table, Match, Trip, Tile 서비스가 있으며, Docker를 통한 빠른 설정과 커뮤니티 지원(문서, 데모 서버, Discord 등)이 특징입니다. 과학 논문에서의 사용을 위한 공식 인용 정보도 제공됩니다.
