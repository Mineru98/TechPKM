Open Source Routing Machine(OSRM)은 OpenStreetMap 데이터를 기반으로 동작하는 고성능 라우팅 엔진입니다. C++로 작성되어 있으며, HTTP API, C++ 라이브러리 인터페이스, Node.js 래퍼를 통해 Nearest, Route, Table, Match, Trip, Tile 등의 서비스를 제공합니다. 교통 네트워크 분석, 최적 경로 탐색, 실시간 내비게이션 등 다양한 용도로 활용되며 Docker를 통한 쉬운 배포가 가능합니다. 관련 프로젝트로는 프론트엔드 및 텍스트 안내 생성 도구가 포함됩니다.
