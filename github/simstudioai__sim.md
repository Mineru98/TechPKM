Sim은 AI 에이전트 워크플로우를 시각적으로 구축하고 배포할 수 있는 오픈소스 플랫폼입니다. 사용자는 드래그 앤 드롭 방식으로 에이전트, 도구, 블록을 연결하여 복잡한 작업을 자동화할 수 있으며, Copilot 기능을 통해 자연어로 워크플로우를 생성/수정할 수 있습니다. 벡터 데이터베이스 연동으로 문서 기반 질의응답도 지원합니다. 클라우드 호스팅(sim.ai) 또는 Docker/Self-hosted 환경에서 실행 가능합니다. Tech Stack은 Next.js, PostgreSQL, Bun, Shadcn UI 등으로 구성되었습니다.
