YOLO-World는 대규모 데이터셋(객체 탐지, 그라운드링, 이미지-텍스트 데이터 포함)으로 사전 훈련된 실시간 오픈-보커불러리 객체 탐지 모델입니다. CVPR 2024에 채택된 이 프로젝트는 사용자 정의 단어를 효율적으로 탐지하는 "프롬프트-then-탐지" 패러다임을 도입했으며, 재매개변수화 기술로 추론 속도와 미세 조정 효율성을 개선했습니다.  
주요 기능으로는 제로샷/오픈-보커불러리 객체 탐지 및 세그멘테이션, 다양한 모델 크기(S/M/L/X), 그리고 ONNX/TFLite 배포 기능을 포함합니다. HuggingFace Spaces에서 온라인 데모를 체험할 수 있으며, PyTorch 기반의 오픈소스 구현체입니다.
