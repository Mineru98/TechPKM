SAM-GPT-4V는 Grounding DINO, Segment Anything(SAM), GPT-4V 모델을 결합하여 이미지에 세그멘테이션 마스크를 자동으로 생성하는 프로젝트입니다. 컴퓨터 비전 분야에서 소량의 세밀한 모델 학습을 위한 데이터 라벨링에 활용되며, 자동차 제조사 식별과 같은 객체 인식 작업에 적용 가능합니다. 오픈소스 MIT 라이선스로 배포되는 이미지 처리 파이프라인 도구입니다.
