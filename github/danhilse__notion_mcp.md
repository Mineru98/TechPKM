이 프로젝트는 Notion API와 Model Context Protocol(MCP)을 통합하여 Claude를 통해 개인 할 일 목록을 관리하는 간단한 서버 구현입니다. 특정 구조(태스크 제목, "오늘/나중에" 선택, 완료 체크박스)를 가진 Notion 데이터베이스와 연동되며, 할 일 추가, 전체 목록 조회, 오늘 할 일 필터링, 완료 처리 기능을 제공합니다.  

MIT 라이선스로 배포되며, 복잡한 데이터베이스 구조나 추가 기능(태깅, 우선순위 등)은 지원하지 않습니다. 사용자는 자신의 데이터베이스 구조에 맞춰 코드를 수정해야 할 수 있습니다.
