이 프로젝트는 Notion API와 Model Context Protocol(MCP)을 통합하여 개인 할 일 목록을 관리하는 간단한 서버 구현입니다. "Task", "When(오늘/나중에)", "Checkbox" 세 가지 속성만 가진 최소한의 Notion 데이터베이스에서 Claude AI를 통해 할 일 추가, 조회, 완료 처리 등의 기능을 제공합니다. 특정 사용 사례에 맞춰 설계되었으며, 다른 데이터베이스 구조 적용 시 코드 수정이 필요합니다.
