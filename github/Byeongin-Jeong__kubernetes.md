이 프로젝트는 쿠버네티스 환경에서 사용할 수 있는 다양한 서비스용 배포 YAML 파일 모음입니다. ELK 스택(elasticsearch, logstash, kibana)과 래빗MQ의 기본 배포 구성을 포함하며, 지속적으로 추가될 예정입니다. 도커 기반의 컨테이너 런타임을 사용하여 인프라를 관리하는 데 목적을 두고 있습니다. 각 구성 요소의 참조 자료도 함께 제공됩니다.
