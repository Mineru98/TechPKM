이 프로젝트는 n8n 워크플로우 자동화 플랫폼에서 Upstage의 Solar LLM 및 임베딩 모델을 활용할 수 있는 커뮤니티노드 패키지를 제공합니다. Solar-mini, solar-pro, solar-pro2 등 다양한 대화 모델과 임베딩 모델을 지원하며, 배치 처리, 스트리밍 응답, API 키 기반 인증 등의 기능을 통해 효율적인 AI 통합을 가능하게 합니다.  

주요 기능으로는 다국어 대화 완료, 고차원 임베딩 벡터 생성, 온도/최대 토큰/모델 선택 등 세부 파라미터 조정이 포함됩니다. MIT 라이선스로 배포되며 n8n UI 또는 npm을 통해 쉽게 설치할 수 있습니다.
