이 프로젝트는 n8n 워크플로우 자동화 플랫폼에 Upstage의 Solar LLM 및 임베딩 모델을 통합하는 커뮤니티 노드 패키지를 제공합니다. Solar 미니/프로/프로2 모델을 활용한 채팅 완성 기능과 검색 쿼리/문서 최적화 임베딩 생성 기능을 지원하며, n8n 환경에서 효율적인 AI 통합을 위한 인증 및 배치 처리 기능을 포함합니다. API 키 기반 설정으로 쉽게 사용할 수 있습니다.
