Marp 프레젠테이션 도구의 핵심 변환 엔진으로, Marpit 프레임워크를 확장한 Markdown 기반 슬라이드 생성기입니다. 커스텀 Markdown 문법, 공식 테마, 수학 식 렌더링, 자동 크기 조정 등의 고급 기능을 지원합니다. 16:9 및 4:3 슬라이드 비율 옵션을 제공하며, KaTeX/MathJax 수학 렌더링과 이모지 지원도 내장되어 있습니다.
