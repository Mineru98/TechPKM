YOLO-Label은 객체 감지를 위한 이미지 라벨링 도구로, 기존 도구보다 직관적인 **"더블 클릭"** 방식의 라벨링 방식을 채택하여 작업 효율성을 높였습니다. 사용자는 키보드 단축키와 마우스 이벤트를 활용해 빠르고 간편하게 바운딩 박스를 생성·수정할 수 있으며, Windows/macOS/Linux 환경에서 사용 가능합니다. 주요 기능은 이미지 탐색, 라벨 전환, 바운딩 박스 삭제 등이며, YOLO 모델 학습에 필요한 데이터셋 준비를 지원합니다.  

프로젝트 요약: 기존 라벨링 도구의 불편함을 해소한 **민감한(Sensitive)** 인터페이스 설계, 단축키 중심의 작업 흐름, 크로스 플랫폼 지원.
