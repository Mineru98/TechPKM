이 프로젝트는 React Native 0.71 이전 버전에서 TypeScript를 사용하기 위한 최소한의 템플릿을 제공합니다. React Native 0.71+부터 공식 TypeScript 지원이 추가되면서 더 이상 업데이트되지 않는 레거시 템플릿입니다. 기존 React Native CLI와 호환되며, 최소한의 의존성만 포함하는 것이 특징입니다. 레거시 버전 프로젝트에서만 사용할 것을 권장합니다.
