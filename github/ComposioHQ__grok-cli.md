Grok 4 CLI는 xAI의 Grok-4 모델을 API로 연동한 터미널 기반 LLM(대형 언어 모델) 도구입니다. 사용자는 간단한 명령어로 대화형 AI 기능을 활용할 수 있으며, 대화 기록이 유지되는 것이 특징입니다.  
Composio 플랫폼을 기반으로 동작하며, API 키 인증 후 대화형 인터페이스로 프롬프트를 입력하거나 "exit" 명령으로 종료할 수 있습니다. Python 환경에서 개발되었으며 CLI 도구와 LLM 통합 기능을 중점으로 설계되었습니다.
