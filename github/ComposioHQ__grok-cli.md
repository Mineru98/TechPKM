Grok 4 CLI는 xAI의 Grok-4 모델을 API로 활용하는 터미널 기반 LLM 도구입니다. 사용자는 프롬프트를 입력하고 대화형 AI 응답을 받을 수 있으며, 대화 기록이 유지됩니다. Python으로 개발되었으며 API 키 설정을 통해 간단한 설치와 실행이 가능합니다. CLI 환경에서 LLM 기능을 활용하는 것이 핵심 목적입니다.
