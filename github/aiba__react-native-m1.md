이 프로젝트는 Apple Silicon(M1) 기반 환경에서 React Native 0.64.2가 Rosetta 없이 컴파일되도록 구성하는 방법을 설명합니다. Xcode 12.5에서 발생하는 호환성 문제를 해결하기 위해 캐시 정리, 프로젝트 설정, 빌드 환경 조정, Swift 브리징 파일 생성 등의 단계를 제시합니다. 주요 변경 사항에는 iOS 배포 타겟 설정, 아키텍처 제외, Podfile 수정 등이 포함됩니다.
