Juman++은 일본어 형태소 분석기로, RNNLM(순환 신경망 언어 모델)을 활용해 단어 시퀀스의 의미적 타당성을 고려합니다. 버전 2는 기존 대비 정확도가 향상되고 분석 속도가 250배 이상 빨라졌으며, utf-8 인코딩 텍스트를 입력으로 처리합니다. Linux, MacOS, Windows에서 C++14 호환 컴파일러를 사용해 빌드 가능하며, 사전 훈련된 모델을 포함해 패키지 또는 GitHub 저장소를 통해 설치할 수 있습니다. 학술 연구에서 사용할 경우 EMNLP 2015 및 EMNLP 2018 논문을 인용해야 합니다.
