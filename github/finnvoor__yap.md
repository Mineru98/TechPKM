yap는 macOS 26에서 Speech.framework를 활용한 온디바이스 음성 변환 CLI 도구입니다. 오디오/비디오 파일에서 직접 음성을 텍스트로 변환하며, 텍스트 또는 자막(SRT) 형식으로 출력을 지원합니다. 로컬에서 동작하는 개인 정보 보호형 도구로, 필터링 및 다양한 형식 변환 기능을 제공합니다. yt-dlp, LLM 도구와의 연동으로 확장된 워크플로우를 구성할 수 있습니다.
