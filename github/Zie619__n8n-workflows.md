이 프로젝트는 n8n 플랫폼에서 사용할 수 있는 4,343개의 프로덕션 준비 완료 워크플로우를 제공하는 컬렉션입니다. 365개 이상의 통합 서비스와 15개의 카테고리로 구성되어 있으며, SQLite FTS5 기반의 고성능 검색 시스템을 통해 빠른 검색과 필터링이 가능합니다. 로컬 설치 또는 Docker를 통해 배포할 수 있으며, MIT 라이선스로 배포됩니다. GitHub Pages를 통해 온라인에서도 접근 가능한 사용자 친화적인 인터페이스를 제공합니다.
