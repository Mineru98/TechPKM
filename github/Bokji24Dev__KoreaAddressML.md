한국 주소 텍스트에서 지역 정보를 추출하는 머신러닝 기반 추론 모델 프로젝트입니다. '서울시 테헤란로25길 13'에서 '서울특별시'를 반환하거나, '역삼동 644-10'에서 '서울특별시 강남구'를 추출하는 등 계층적 주소 예측이 가능합니다. CPU/GPU/M1(Mac) 환경을 모두 지원하며, 자연어 처리 기반의 주소 파싱 기능을 제공합니다. 공공 데이터 기반으로 학습된 한국어 주소 특화 모델입니다.
