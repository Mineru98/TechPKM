이 프로젝트는 한국 주소를 예측하거나 세부 주소를 추출하는 머신러닝 추론 모델을 제공합니다. '서울시 테헤란로25길 13'과 같은 입력에 대해 '서울특별시' 또는 '서울특별시 강남구'와 같은 계층적 주소 정보를 반환할 수 있습니다. spaCy 기반의 모델로 CPU/GPU/Apple Silicon 환경에서 모두 동작하며, 공공 데이터 기반의 주소 데이터셋을 활용합니다.
