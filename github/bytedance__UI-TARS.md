UI-TARS는 GUI 및 게임 환경에서 자동화된 상호작용을 수행하는 오픈소스 멀티모달 에이전트 프로젝트입니다. Qwen 2.5vl 기반의 강력한 시각-언어 모델을 활용하여 데스크탑, 모바일, 브라우저 작업을 처리하며, 강화 학습을 통해 추론 능력을 향상시켰습니다. OSWorld, ScreenSpot 등에서 SOTA 성능을 달성했으며, 게임 및 실제 작업에서 100% 성공률을 보이는 등 복잡한 태스크에 특화된 "올인원" 에이전트 모델입니다.
