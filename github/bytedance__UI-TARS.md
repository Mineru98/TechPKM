UI-TARS는 데스크톱 및 모바일 환경의 GUI 자동화를 위한 오픈소스 멀티모달 에이전트 프로젝트입니다. 강화학습 기반 추론 능력과 화면 좌표 처리를 통해 복잡한 작업을 수행하며, 게임, 코드 실행, 도구 활용 등 "올인원" 에이전트 모델을 지향합니다. OSWorld, Windows Agent Arena 등 벤치마크에서 SOTA 성능을 달성했으며, Qwen 2.5vl 기반 모델을 활용해 GUI 요소 인식 및 작업 실행이 가능합니다. 데스크톱 버전 및 웹 자동화 프로젝트(Midscene)와의 통합도 지원합니다.
