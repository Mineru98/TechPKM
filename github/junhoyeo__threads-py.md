이 프로젝트는 Meta의 Threads 플랫폼을 위한 비공식 Python API 구현체로, 연구 및 교육 목적으로 개발되었습니다. 리버스 엔지니어링을 통해 제작되었으나, Meta의 요청으로 개발이 중단되었으며 더 이상 업데이트되지 않습니다. 현재는 법적 문제로 인해 사용이 권장되지 않으며, 관련 문서는 보존 문서로 이동되었습니다. 주요 기능은 Threads 플랫폼의 비공식 연동을 지원하는 것이었습니다.
