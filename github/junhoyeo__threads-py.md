threads-py는 Meta의 Threads 플랫폼을 위한 비공식 Python API 래퍼입니다. 리버스 엔지니어링으로 개발된 이 프로젝트는 원래 교육 및 연구 목적으로 제작되었으나, Meta의 요청으로 2023년 9월 8일 이후 개발 및 유지보수가 중단되었습니다. 현재는 보관 상태로, 이전 문서는 PRESERVED.md에 보관되어 있습니다. API 사용은 Meta 서비스 약관 위반 가능성이 있으므로 주의가 필요합니다.
