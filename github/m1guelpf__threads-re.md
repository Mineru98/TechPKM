Threads 앱의 API와 인증 메커니즘을 리버스 엔지니어링한 연구 노트 프로젝트입니다. README는 웹(threads.net)의 GraphQL API 호출 방법과 모바일 앱의 Bloks 프레임워크 기반 인증 방식을 상세히 설명하며, 프로필 데이터 조회, 게시물 조회, 좋아요 목록 가져오기 등의 엔드포인트와 예제 요청을 제공합니다. Threads와 Instagram의 ID 시스템 연동 정보도 포함되어 있습니다.  

> **핵심 기능**:  
> - Threads 웹/모바일 API 구조 분석  
> - GraphQL 쿼리 예제 (프로필, 게시물, 댓글, 좋아요)  
> - Bloks 기반 모바일 인증 절차  
> - Instagram과의 ID 시스템 연동 정보  

이 문서는 API 클라이언트 개발이나 플랫폼 연구 목적으로 활용될 수 있습니다.
