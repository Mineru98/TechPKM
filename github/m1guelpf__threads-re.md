이 프로젝트는 Threads 앱의 역공학을 통해 수집된 노트와 발견을 정리한 저장소입니다. 웹 및 모바일 버전의 GraphQL API 사용법과 인증 메커니즘을 문서화했으며, 프로필 조회, 게시물/댓글 조회, 좋아요 목록 조회 등의 기능을 포함합니다. 인스타그램 ID 시스템과 연동되는 점과 메타의 Bloks 프레임워크를 활용한 모바일 인증 방식을 분석하는 자료가 포함되어 있습니다. 사용자는 이 정보를 바탕으로 커스텀 클라이언트를 개발하거나 새로운 발견을 PR할 수 있습니다.
