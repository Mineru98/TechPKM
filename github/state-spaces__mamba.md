Mamba는 정보 밀도가 높은 데이터(예: 언어 모델링)에서 트랜스포머의 한계를 극복하기 위한 새로운 상태공간 모델 아키텍처입니다. 선택적 상태공간(Selective SSM)을 기반으로 효율적인 하드웨어 설계와 구현 방식을 채택하였으며, Mamba-2는 상태공간 이중성(State Space Duality)을 통해 일반화된 모델과 알고리즘을 제공합니다. 주요 특징은 선형 시간 복잡도, PyTorch 기반 구현, 다양한 사전 학습 모델 지원입니다.
