Hermes는 React Native 앱의 빠른 시작을 위해 최적화된 JavaScript 엔진으로, AOT(정적 최적화) 및 컴팩트 바이트코드를 특징으로 합니다. 이 프로젝트는 성능 향상과 효율성을 지속적으로 개선하는 것을 목표로 하며, 커뮤니티 기여를 환영합니다. React Native와의 통합을 위한 문서 및 빌드 가이드를 제공합니다. 주요 사용 사례는 React Native 앱의 자바스크립트 실행 환경 개선이며, MIT 라이선스로 배포됩니다.
