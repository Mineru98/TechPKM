Hermes는 React Native 앱의 빠른 시작을 위해 최적화된 JavaScript 엔진입니다. 정적 AOT(사전 컴파일) 최적화와 컴팩트 바이트코드를 특징으로 하며, React Native 앱의 성능을 개선하기 위해 설계되었습니다. 페이스북 주도로 개발되었으며 MIT 라이선스로 배포됩니다. 소스 빌드 및 커스터마이징이 가능하지만, 일반적으로 사전 빌드된 바이너리를 React Native 프로젝트에 통합해 사용합니다.
