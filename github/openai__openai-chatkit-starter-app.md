이 프로젝트는 OpenAI ChatKit의 최소 통합 구현 예시를 제공하는 두 가지 스타터 앱을 포함합니다. 자체 호스팅 ChatKit 통합(예시)과 호스팅 워크플로우가 포함된 관리형 ChatKit 통합(예시)을 실행할 수 있습니다. 챗봇 통합 개발을 위한 참조 구현체로 기술 스택 선택, 아키텍처 설계, 빠른 프로토타이핑에 유용합니다.
