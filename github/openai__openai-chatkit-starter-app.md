이 프로젝트는 OpenAI ChatKit의 최소 통합 구현을 보여주는 두 가지 스타터 앱(자기 호스팅형과 관리형 통합)을 제공하는 참조 구현 저장소입니다. 챗봇 통합을 위한 기본 템플릿으로, 개발자에게 ChatKit 통합 방식을 쉽게 시작할 수 있는 샘플을 제공합니다. 주요 기술 스택은 JavaScript 기반입니다.
