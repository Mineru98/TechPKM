이 프로젝트는 OpenAI의 CUA 모델과 Playwright를 활용하여 프론트엔드 테스트를 자동화하는 방법을 보여주는 데모입니다. Next.js로 구현된 프론트엔드 UI, CUA 서버, 샘플 테스트 애플리케이션으로 구성된 모노레포로, 웹 애플리케이션 테스트를 위한 테스트 케이스 실행 및 자동화 기능을 제공합니다. 테스트 에이전트의 핵심 로직은 CUA 서버에 구현되어 있으며, 사용자가 지정한 테스트 케이스에 따라 브라우저에서 동작을 실행합니다.  

> ⚠️ 주의사항: 컴퓨터 사용 기능은 미리보기 단계이며, 인증된 환경이나 고위험 작업에는 사용을 권장하지 않습니다.
