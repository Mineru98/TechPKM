이 프로젝트는 OpenAI의 CUA 모델과 Playwright를 활용해 프론트엔드 테스트 자동화를 구현하는 데모입니다. Next.js 기반 프론트엔드, CUA 서버, 샘플 테스트 앱으로 구성된 모노레포 구조로, 웹 애플리케이션 테스트 시나리오를 자동 실행하는 에이전트 솔루션을 보여줍니다. MIT 라이선스로 배포되며 테스트 환경 전용으로 설계되었습니다.
