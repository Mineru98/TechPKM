Chronos 프로젝트는 시계열 예측을 위한 프리트레인 모델 패밀리를 제공하는 오픈소스 패키지입니다. Chronos-2, Chronos-Bolt, Chronos 모델을 포함하며, 단변량/다변량 예측 및 외생 변수 지원 기능을 통해 다양한 벤치마크에서 최첨단 성능을 달성합니다. 특히 Chronos-2는 제로샷 예측 성능과 90% 이상의 승률을 보이며, Chronos-Bolt는 기존 대비 250배 빠른 속도와 20배 메모리 효율성을 제공합니다. AWS SageMaker 배포 및 Hugging Face 통합으로 프로덕션 환경에 최적화되었습니다.
