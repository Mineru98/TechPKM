HWnow는 Go + React + TypeScript + Wails v2로 구축된 크로스플랫폼 데스크톱 애플리케이션으로, CPU, 메모리, 디스크, 네트워크, GPU 등 시스템 리소스를 실시간으로 모니터링하는 도구입니다. 3초 간격으로 데이터를 수집하며, 드래그 앤 드롭으로 위젯을 배치/조정할 수 있는 커스터마이징 가능한 대시보드와 라이트/다크 모드 지원을 특징으로 합니다. NVIDIA GPU 모니터링 및 시스템 정보 조회 기능도 포함되어 있습니다.
