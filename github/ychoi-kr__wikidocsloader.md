이 프로젝트는 [wikidocs.net](https://wikidocs.net)의 콘텐츠를 LangChain 문서 형태로 로드하기 위한 Python 문서 로더입니다. 한국어 학습 자료 처리에 특화된 도구로, 위키독스의 기술/교육 콘텐츠를 LangChain 기반 애플리케이션과 연동이 가능합니다. LangChain 통합 및 한국어 자료 추출에 최적화된 것이 핵심 특징입니다.
