Large World Model(LWM)은 긴 비디오와 텍스트 시퀀스를 처리할 수 있는 대규모 멀티모달 오토리그레시브 모델입니다. 4K에서 1M 토큰까지 확장 가능한 RingAttention 기술을 활용해 복잡한 시공간 정보를 이해하며, 장문 텍스트 분석, 비디오 질문 응답, 이미지/비디오 생성 등 다양한 작업을 지원합니다. 7B 파라미터의 오픈소스 모델 패밀리로, 1M 토큰 이상의 긴 컨텍스트에서 높은 정확도를 보여줍니다.
