이 프로젝트는 React 프론트엔드와 LangGraph 기반 백엔드 에이전트를 활용한 풀스택 애플리케이션을 보여줍니다. 사용자의 질의에 대해 동적 검색어 생성, 웹 검색, 결과 분석, 반복적 개선을 통해 근거 있는 답변과 인용을 제공하는 연구 강화형 대화형 AI 구현 예시입니다. Gemini 모델을 활용한 쿼리 생성 및 반영적 추론 기능을 포함합니다.

주요 기술 스택은 LangGraph(에이전트 로직), Google Gemini(LLM), React(Vite 기반 프론트엔드), Tailwind CSS(스타일링)이며, 개발 시 핫 리로딩과 Docker 기반 배포를 지원합니다. Apache 2.0 라이선스로 배포됩니다.
