OpenAI Harmony는 gpt-oss 시리즈 모델의 응답 포맷을 정의하는 프로젝트입니다. 대화 구조 정의, 추론 출력 생성, 함수 호출 구조화를 지원하며, 오픈소스 모델과의 호환성을 위해 설계되었습니다. Rust로 핵심 성능을 구현하고 Python 바인딩을 제공하여 API 개발자에게 효율적인 통합 환경을 제공합니다.  

> 핵심 기능:  
> - 다중 채널 출력(사고 과정, 도구 호출, 일반 응답) 지원  
> - 도구 네임스페이스 및 구조화된 출력 정의  
> - OpenAI Responses API와 유사한 친숙한 포맷  
> - Rust 기반 고속 처리와 Python 편의성 결합
