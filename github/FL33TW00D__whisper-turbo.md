Whisper Turbo는 브라우저나 Electron 앱에서 클라이언트 측 실행용으로 설계된 빠르고 크로스플랫폼 Whisper 구현체입니다. WebGPU 기반으로 Windows 및 MacOS의 Chromium 기반 브라우저에서 공식 지원되며, [Ratchet](https://github.com/FL33TW00D/ratchet) Rust 라이브러리를 백엔드로 사용합니다. 데모 사이트, 문서, 로드맵을 통해 프로젝트 세부 정보를 확인할 수 있습니다. GPU 최적화 및 Rust 고급 기술(HRTB 등) 참여가 필요한 오픈소스 프로젝트입니다.
