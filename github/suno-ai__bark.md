Bark는 Suno AI에서 개발한 트랜스포머 기반 생성형 텍스트-오디오 모델입니다. 텍스트 입력을 기반으로 현실적인 다국어 음성, 음악, 효과음 등을 생성할 수 있으며, MIT 라이선스로 상용 가능합니다. 연구 및 상업적 활용을 위해 사전 학습된 체크포인트를 제공하며, 100개 이상의 음성 프리셋과 다양한 언어(한국어, 영어, 스페인어 등)를 지원합니다.  

이 모델은 전통적인 TTS와 달리 완전한 생성 방식으로 작동하여 예상치 못한 창의적 출력이 가능하지만, 프롬프트에서 벗어날 수 있음을 유의해야 합니다. 간단한 Python API 또는 명령줄 도구로 활용 가능하며, 저사양 GPU(4GB 이하)에서도 실행하도록 최적화되었습니다.
