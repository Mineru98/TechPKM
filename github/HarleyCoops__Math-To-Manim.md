Math-To-Manim은 복잡한 수학 개념을 시각적으로 표현하는 애니메이션을 자동으로 생성하기 위한 프로젝트입니다. 세 가지 AI 파이프라인(Gemini 3, Claude Sonnet 4.5, Kimi K2)을 통해 LaTeX 기반의 상세한 프롬프트와 수학적 의존성을 분석하여 정확한 Manim 코드를 생성합니다. 물리학, 금융, 컴퓨터 과학 등 다양한 분야의 55개 이상의 예제 애니메이션을 제공하며, FFmpeg과 Manim 커뮤니티 v0.19.0을 기반으로 합니다.  

MIT 라이선스로 배포되며, API 키와 Python 3.10 이상, 8GB 이상의 RAM이 필요합니다. 기여는 예제 추가, 에이전트 개선, 문서 작성 등의 방식으로 가능합니다.
