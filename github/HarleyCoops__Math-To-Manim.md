Math-To-Manim은 수학적 개념을 Manim 애니메이션으로 자동 변환하는 프로젝트입니다. 세 가지 AI 파이프라인(Google Gemini 3, Claude Sonnet 4.5, Kimi K2)을 제공하여 사용자가 API 접근성과 선호도에 따라 선택할 수 있습니다. LaTeX 기반의 상세한 프롬프트 생성, 재귀적 지식 트리 탐색, 정확한 수학 표기 지원을 통해 복잡한 개념을 시각적으로 표현합니다. 물리, 금융, 컴퓨터 과학 등 다양한 분야의 55개 이상의 예제가 포함되어 있으며, MIT 라이선스로 배포됩니다.
