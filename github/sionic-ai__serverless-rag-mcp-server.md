Storm MCP 서버는 LLM 애플리케이션과 RAG 데이터 소스/도구 간의 통합을 위한 개방형 프로토콜 구현체입니다. Anthropic의 Model Context Protocol(MCP)을 기반으로 하며, Sionic AI의 Storm Platform과 연동해 임베딩 모델·벡터DB를 활용할 수 있습니다. 파일 업로드, 도구 호출, API 통합 기능을 제공하며, Claude Desktop 등에서 RAG 솔루션 개발에 활용됩니다. Python으로 작성된 이 프로젝트는 파일 시스템 관리 및 REST API 연동 모듈을 포함합니다.
