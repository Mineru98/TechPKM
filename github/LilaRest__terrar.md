Terrar.js는 iframe의 효율적이고 투과적인 대안으로, HTML, CSS 및 JavaScript 격리를 제공하는 프로토타입 프로젝트입니다. SES 라이브러리를 활용한 권한 없는 JavaScript 컴파트먼트 생성, Shadow DOM API를 통한 HTML/CSS 격리, Esprima 라이브러리를 사용하여 제한된 객체 접근을 방지하는 기술을 특징으로 합니다. 서버 측 렌더링 지원과 함께 사용자 제공 콘텐츠나 신뢰할 수 없는 위젯의 안전한 삽입에 적합합니다.  

> ⚠️ **주의**: 현재 프로덕션 사용 불가(프로토타입 단계, 철저한 테스트 및 감사 미진행)
