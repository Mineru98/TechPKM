Terrar.js는 iframe의 효율적이고 투과적인 대안으로, HTML/CSS/JS 격리 환경을 제공하는 프로토타입 라이브러리입니다. SES(Sealed Compartment)와 Shadow DOM API를 활용해 권한 없는 JS 환경과 HTML/CSS 격리 영역을 생성하며, 사용자 제공 콘텐츠나 제3자 위젯의 안전한 렌더링에 특화되었습니다. 서버 측 렌더링을 지원하며, 현재 개발 중이며 프로덕션 사용을 권장하지 않습니다.
