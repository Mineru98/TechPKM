스테이블 디퓨전을 위한 웹 인터페이스로, Gradio 라이브러리를 사용해 구현된 사용자 친화적인 도구. 텍스트/이미지 생성, 인페인팅, 아웃페인팅, 다양한 확장 기능(GFPGAN, RealESRGAN 등)을 지원하며, 커스텀 스크립트 및 하이퍼네트워크 통합이 가능합니다. Python 설치 후 간단한 실행 스크립트로 사용 가능하며, 다양한 하드웨어 환경(NVIDIA/AMD/Intel/NPU)을 지원합니다.
