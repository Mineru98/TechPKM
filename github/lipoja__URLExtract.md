URLExtract는 주어진 텍스트에서 TLD(최상위 도메인)를 기반으로 URL을 추출하기 위한 Python 라이브러리입니다. 텍스트의 TLD를 식별하고, 양쪽 경계를 확장하여 유효한 URL을 찾으며, DNS 검증 옵션을 통해 잘못된 도메인 이름을 필터링할 수 있습니다. 새로운 TLD를 지원하기 위해 IANA.org에서 최신 TLD 목록을 다운로드하여 사용합니다. 텍스트 분석, 웹 스크래핑, 콘텐츠 필터링 등 다양한 용도로 활용할 수 있습니다.
