PyAlgoTrade는 이벤트 기반 알고리즘 트레이딩을 위한 파이썬 라이브러리입니다. 백테스팅, 페이퍼 트레이딩, 실시간 트레이딩(비트스탬프 비트코인)을 지원하며, 다양한 기술 지표와 성능 분석 기능을 제공합니다. 2023년 현재 이 프로젝트는 더 이상 유지보수되지 않으며, 대체 프로젝트로 Basana를 추천합니다. CSV 형식의 시계열 데이터, 트위터 이벤트 처리, TA-Lib 통합 등의 기능을 포함합니다.
