플렁크는 AWS SES 기반의 오픈소스 이메일 플랫폼으로, 애플리케이션에서 이메일을 쉽게 발송할 수 있도록 합니다. 트랜잭션 이메일, 캠페인 관리, 자동화 워크플로우, 연락처 관리, 실시간 분석 등의 기능을 제공하며, SendGrid나 Mailgun과 같은 서비스의 자체 호스팅 대안입니다. Docker를 통한 간편한 배포와 커스텀 도메인 지원이 가능합니다.
