Buck2는 Meta에서 개발한 빠르고 허미틱(hermetic)한 멀티언어 빌드 시스템입니다. 기존 Buck1의 후속 버전으로, 빌드 속도 최적화, 정확한 의존성 추적, 다양한 언어 간 통합 빌드 기능을 제공합니다. 원격 실행 시 완벽한 허미틱 빌드가 가능하며, Bazel 등 다른 빌드 시스템과 호환되는 원격 실행 API를 지원합니다. 현재 메인 브랜치는 Meta 내부에서 광범위하게 사용 중이지만, 외부 사용자에게는 일부 기능이 미완성 상태입니다.  

> 📌 주의: 현재 공식 안정 릴리스가 없으며, 최신 커밋 버전을 사용해야 합니다. 빌드는 빠르지만(특히 메타 내부 기준 Buck1 대비 최대 2배 빠름), 일부 툴체인 및 기능이 아직 개발 중입니다.  

Rust로 작성된 핵심 시스템은 언어 독립적인 설계로 확장성이 뛰어나며, BXL(BXL Extension Language)을 통해 빌드 그래프 분석 및 자동화가 가능합니다. 대규모 저장소 지원, 분산 컴파일, 빌드 시스템의 현대적 이론을 반영한 설계가 특징입니다. MIT/Apache-2.0 듀얼 라이선스로 배포됩니다.
