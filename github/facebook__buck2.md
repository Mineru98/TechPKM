Buck2는 메타에서 개발한 빠르고 헤르메틱하며 멀티언어를 지원하는 빌드 시스템입니다. 기존 Buck 시스템(Buck1)의 후속 버전으로, 대규모 코드베이스를 위한 빌드 최적화를 목표로 설계되었습니다. 주요 특징으로는 초고속 빌드 성능, 원격 실행 시 완전 헤르메틱 보장, 다양한 언어 간 의존성 추적, 분산 컴파일 지원 등이 있습니다. 현재는 개발 중이지만 메타 내부에서 광범위하게 사용 중이며, 빌드 자동화 및 확장성을 위한 BXL 스크립팅 기능도 제공합니다.  

> 🚧 참고: 현재 안정적인 릴리스 태그가 없으며, 최신 커밋 버전을 사용할 것을 권장합니다.  

빌드 시스템 비교 측면에서 Bazel, Pants와 유사한 개념을 가지지만, 파일시스템 가상화, 언어 불가지론적 코어 설계, 빌드 그래프 분석 기능 등에서 차별화됩니다. MIT/Apache-2.0 듀얼 라이선스로 배포됩니다.
