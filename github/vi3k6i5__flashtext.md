FlashText는 문장 내 키워드 추출 및 대체 작업을 고속으로 수행하는 Python 라이브러리입니다. Aho-Corasick 알고리즘과 트라이 구조를 기반으로 하며, 정규표현식 대비 수십 배 빠른 성능을 제공합니다. 주요 기능으로는 대소문자 구분 처리, 키워드 스팬 정보 추출, 다중 키워드 일괄 추가/삭제, 비단어 경계 문자 커스터마이징 등이 포함됩니다.  

이 프로젝트는 대량의 문서에서 키워드 처리 효율성을 극대화하기 위해 설계되었으며, 오픈소스 MIT 라이선스로 배포됩니다.
