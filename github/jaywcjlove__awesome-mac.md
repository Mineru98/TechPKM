# 🍎 Awesome Mac - macOS 추천 앱/소프트웨어/도구 목록

## 📌 요약

이 문서는 macOS 사용자를 위한 **추천 앱, 소프트웨어, 도구, 유틸리티**를 정리한 목록입니다.  
**개발자, 디자이너, 보안 전문가, 일반 사용자**까지 모두에게 유용한 도구들을 카테고리별로 분류하여 제공합니다.  
각 항목은 **공식 웹사이트, GitHub 저장소, App Store 링크**를 포함하며, **오픈소스 여부, 무료 제공 상태**도 표시되어 있습니다.

---

## 🧠 핵심 원칙

- **신뢰성**: 사용자가 제공한 README 내용을 기준으로 정보를 정리합니다.
- **정확성**: 프로젝트명, 설명, 기능, 사용 사례를 README에서 직접 추출합니다.
- **검색 최적화**: 태그와 별칭은 검색 효율성을 고려해 영어나 한국어 중 적절한 것을 선택합니다.
- **언어**: 모든 출력은 **한국어**로 작성됩니다.

## 📚 주요 카테고리 및 추천 도구

### ✍️ 문서 편집 도구
- **Visual Studio Code** (오픈소스, 무료)  
  - Microsoft의 크로스플랫폼 코드 편집기
- **Obsidian**  
  - 로컬 노트 관리 및 지식 베이스를 위한 강력한 도구
- **Typora**  
  - 마크다운 작성에 최적화된 실시간 미리보기 편집기

### 🧰 개발 도구
- **Xcode** (App Store)  
  - Apple 공식 iOS/macOS 개발 IDE
- **Docker** (오픈소스, 무료)  
  - 컨테이너 기반 애플리케이션 개발 및 배포
- **Postman**  
  - API 개발 및 테스트를 위한 GUI 도구

### 🎨 디자인 도구
- **Figma** (무료)  
  - 웹 기반 UI/UX 디자인 및 프로토타이핑
- **Affinity Designer**  
  - 벡터 그래픽 디자인 전문 도구
- **Sketch**  
  - macOS 전용 UI/UX 디자인 소프트웨어

### 🔐 보안/암호화
- **1Password**  
  - 크로스플랫폼 비밀번호 관리 도구
- **MalwareBytes** (무료)  
  - 맥용 악성코드 방지 소프트웨어
- **Lutris** (오픈소스)  
  - 리눅스 애플리케이션 윈도우/맥OS에서 실행

### 📱 원격 접속
- **TeamViewer** (무료)  
  - 원격 제어 및 파일 전송
- **RustDesk** (오픈소스, 무료)  
  - 자체 호스팅 가능한 원격 데스크톱 소프트웨어

### 🎮 게임/에뮬레이션
- **OpenEmu** (오픈소스, 무료)  
  - 다양한 콘솔 게임 에뮬레이터 통합 도구
- **PlayCover** (오픈소스)  
  - Apple Silicon 맥에서 iOS 앱 실행

### 🛠 유틸리티
- **Alfred**  
  - 키보드 기반 앱 실행 및 작업 자동화
- **CleanMyMac X** (App Store)  
  - 맥 시스템 정리 및 최적화
- **Karabiner-Elements** (오픈소스)  
  - 키보드 리매핑 및 고급 입력 설정

## 📝 주요 특징

- **오픈소스 도구**는 GitHub 링크와 함께 제공
