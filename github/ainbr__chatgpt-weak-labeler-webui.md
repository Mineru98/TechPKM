이 프로젝트는 ChatGPT 언어 모델을 활용한 간단한 개체명 인식(NER) 도구로, 약라벨링 작업에 특화되어 있습니다. OpenAI API를 통해 텍스트 내 개체명(사람, 조직, 위치 등)을 자동으로 식별하고 분류하는 기능을 제공하며, 규칙 기반 모델 대비 우수한 성능을 보일 수 있음을 입증합니다. Gradio 기반의 웹 인터페이스를 통해 API 키 입력만으로 쉽게 사용 가능합니다. 프롬프트 최적화가 모델 성능에 큰 영향을 미치는 점이 특징입니다.
