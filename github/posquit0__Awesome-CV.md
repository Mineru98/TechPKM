**Awesome CV**는 뛰어난 구직 신청서를 위한 LaTeX 템플릿입니다. 이력서(CV), 레쥬메(Resume), 커버 레터를 직관적으로 작성 및 커스터마이징할 수 있으며, 깔끔한 시맨틱 마크업으로 구성되어 있습니다. 학술 및 개발자 포트폴리오에 적합한 전문적이고 현대적인 디자인으로 제공되며, PDF 출력과 Docker 기반 빌드 환경을 지원합니다.  

이 프로젝트는 [Fancy CV](https://www.sharelatex.com/templates/cv-or-resume/fancy-cv)에서 영감을 받아 개선되었으며, Roboto 및 Source Sans Pro 폰트를 기본 지원합니다. GitHub 액션을 통한 자동화된 빌드 시스템과 실시간 편집 환경(OverLeaf)을 제공합니다.
