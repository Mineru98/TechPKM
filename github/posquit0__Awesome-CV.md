Awesome CV는 LaTeX 기반의 전문적인 이력서(CV), 레쥬메(Resume), 커버레터 템플릿을 제공하는 프로젝트입니다. 깔끔한 시맨틱 마크업으로 사용자 정의가 용이하며, 학술/기술 분야에 적합한 디자인으로 구성되었습니다. PDF 샘플과 Docker, TeX Live 환경에서의 빌드 가이드를 포함하여 실용적인 문서 작성을 지원합니다.  

**핵심 특징**:  
- Fancy CV에서 영감을 받은 모던한 디자인  
- FontAwesome6 아이콘, Roboto/Source Sans Pro 폰트 통합  
- OverLeaf 온라인 편집기 호환  
- TeX 배포판 또는 Docker 환경에서 빌드 가능
