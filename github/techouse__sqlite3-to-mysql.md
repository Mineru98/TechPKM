이 프로젝트는 SQLite3 데이터베이스에서 MySQL 또는 MariaDB로 데이터를 변환하는 Python 기반 도구입니다. CLI 인터페이스를 통해 특정 테이블 선택, 데이터 타입 매핑, 외래 키 처리 등 세부 옵션을 제공하며, Docker 및 Homebrew 설치도 지원합니다. 주요 기능은 테이블 구조 변환, 데이터 마이그레이션, 다양한 MySQL 버전 호환성입니다.
