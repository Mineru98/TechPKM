이 프로젝트는 n8n 커뮤니티에서 검증된 우수 워크플로우 템플릿을 한국어로 번역하여 공유하는 교육 목적의 저장소입니다. RAG 챗봇, Airtable 통합, 이메일 자동화, 마케팅 분석 등 다양한 비즈니스 시나리오에 적용 가능한 JSON 기반 워크플로우를 제공하며, 초보자들이 n8n을 쉽게 이해하고 활용할 수 있도록 구성되었습니다. 커뮤니티 기여를 통해 지속적으로 업데이트되는 오픈소스 템플릿 모음입니다.
