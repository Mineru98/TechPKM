이 프로젝트는 해외에서 수집된 우수한 n8n 워크플로우 템플릿을 한글로 번역·공유하여 한국 커뮤니티 초기 사용자들의 교육을 지원하는 오픈소스 컬렉션입니다. RAG 챗봇, Airtable 통합, AI 데이터 추출, 마케팅 분석 등 다양한 비즈니스 시나리오에 적용 가능한 검증 완료된 템플릿들을 카테고리별로 체계적으로 분류하여 제공합니다. 각 워크플로우는 실제 비즈니스 환경에서 검증된 사례를 기반으로 구축되었으며, 교육 및 실무 활용을 위해 번역·최적화되었습니다.
