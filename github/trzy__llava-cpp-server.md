LLaVA C++ 서버는 llama.cpp 기반의 LLaVA 모델을 위한 간단한 API 서버 구현입니다. 로컬에서 멀티모달 AI 서비스를 제공하며, 이미지 분석과 텍스트 생성을 결합한 기능을 제공합니다. macOS에서 테스트되었으며, 다른 플랫폼에서도 llama.cpp 호환 환경에서 빌드 가능합니다.  

프로젝트 목적: 이미지 기반 질문 응답을 위한 HTTP 서버 제공  
핵심 특징: LLaVA-v1.5-13b 지원, 바이너리 이미지 업로드 처리, 시스템 프롬프트 커스터마이징 가능  
사용 컨텍스트: 로컬 AI 서버 구현, 멀티모달 애플리케이션 연동, llama.cpp 모델 활용
