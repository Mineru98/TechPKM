이 프로젝트는 Ollama와 연동된 분산 태스크 관리 시스템 Swarm의 포크 버전입니다. 기본 OpenAI Swarm 아키텍처를 유지하면서 Ollama와 같은 로컬 LLM 서버와 호환되도록 환경변수를 추가했습니다. Qwen2.5:72b와 같은 대규모 언어 모델을 로컬 환경에서 효율적으로 활용할 수 있는 것이 핵심 특징입니다. 분산 환경에서의 태스크 생성과 실행을 자동화하는 오픈소스 도구입니다.
