**ALMA**는 LLM(대규모 언어 모델) 기반의 다국어 번역 모델로, 단방향 데이터 미세 조정과 고품질 병렬 데이터 최적화를 결합한 새로운 번역 패러다임을 제시합니다. ALMA-R은 Contrastive Preference Optimization(CPO)을 적용한 업그레이드 버전으로, GPT-4 수준의 번역 성능을 달성했습니다. 7B/13B 모델 크기, LoRA 미세 조정, 다양한 GPU 지원 등을 특징으로 하며, WMT 번역 평가에서 우수한 성능을 입증했습니다. 주요 지원 언어 쌍은 영어↔독일어, 영어↔체코어, 영어↔아이슬란드어, 영어↔중어, 영어↔러시아어입니다.
