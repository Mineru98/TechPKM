이 프로젝트는 최상위 기술 기업의 표준을 충족하는 코드 생성 및 관리를 위한 포괄적인 가이드라인과 워크플로우를 제공합니다. 시스템 지시, 품질 원칙, 개발 단계(설계 → 테스트 → 구현)를 체계적으로 정의하여 생산 준비 완료 상태의 코드를 보장합니다. 보안, 성능, 가독성, 오류 처리와 같은 비기능적 요구사항을 강조하며, 테스트 주도 개발(TDD) 및 안전한 리팩토링 절차를 포함합니다.  

> 📌 **요약**: 최고 수준의 코드 품질, 보안, 확장성을 위한 AI 엔지니어링 프레임워크로, 개발 전 단계를 표준화하고 테스트 검증을 강조하는 규칙집입니다.
