Microsoft PowerToys는 Windows 사용자 환경을 커스터마이징하고 일상적인 작업을 효율화하는 유틸리티 모음입니다. 25개 이상의 도구(Command Palette, FancyZones, PowerRename 등)를 제공하며, 설치 방법(GitHub, Microsoft Store, WinGet)과 주요 기능(클립보드 관리, 창 정렬, 파일 잠금 해제 등)을 상세히 설명합니다. 최신 버전(0.97)에서는 CLI 지원 확장, 커맨드 팔레트 개선, 커서 래핑 기능 추가 등 다양한 업데이트가 포함되었습니다.
