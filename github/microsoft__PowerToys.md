Microsoft PowerToys는 Windows 사용자 경험을 맞춤화하고 일상적인 작업을 효율화하는 25개 이상의 유틸리티 모음입니다. 고급 붙여넣기, 환경 변수 관리, 화면 영역 분할(FancyZones), 커맨드 팔레트 등 다양한 도구로 구성되어 있으며, CLI 지원과 확장 가능한 기능을 통해 사용자 정의 확장이 가능합니다. Windows 11 최적화 및 멀티모니터 작업 편의성 향상에 중점을 둔 오픈소스 프로젝트입니다.
