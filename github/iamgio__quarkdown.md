콰크다운(Quarkdown)은 마크다운 기반의 현대적인 문서 조판 시스템으로, 단일 프로젝트에서 인쇄용 도서나 대화형 프레젠테이션으로 원활하게 컴파일할 수 있습니다. 튜링 완전 확장 마크다운을 기반으로 함수와 표준 라이브러리(레이아웃, I/O, 수학, 조건문, 반복문 등)를 제공하며, 사용자 정의 함수/변수 정의 및 라이브러리 생성이 가능합니다. HTML(평문, 페이지드, 슬라이드) 및 PDF 출력을 지원하며, 실시간 미리보기, 빠른 컴파일 속도, 강력한 VS Code 확장이 특징입니다. 학술 논문, 도서, 지식 베이스, 프레젠테이션 등 다양한 용도로 활용 가능합니다.
