Quarkdown은 확장 가능한 마크다운 기반 문서 생성 시스템으로, 단일 프로젝트에서 인쇄용 문서나 인터랙티브 프레젠테이션으로 변환이 가능합니다. 함수 호출, 조건문, 반복문과 같은 프로그래밍 기능을 내장하여 동적 콘텐츠를 생성하며, HTML/PDF 출력을 지원합니다. 라이브 미리보기, 빠른 컴파일 속도, VS Code 확장 기능을 통해 학술 논문, 책, 지식 베이스 등 다양한 용도로 활용할 수 있습니다.  

<p> 요약: 확장 마크다운으로 복잡한 문서 구조를 생성하고, 정적/동적 출력물을 제작하는 도구. 학습 곡선은 완만하며 LaTeX보다 접근성이 높음.
