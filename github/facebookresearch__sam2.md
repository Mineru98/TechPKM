SAM 2는 이미지와 비디오에서 프롬프트 기반 시각적 분할을 해결하기 위한 파운데이션 모델입니다. 단일 프레임을 가진 비디오로 이미지를 처리하며, 실시간 비디오 처리를 위한 스트리밍 메모리가 포함된 트랜스포머 아키텍처를 사용합니다. 사용자 상호작용을 통해 모델과 데이터를 개선하는 모델-인더루프 데이터 엔진을 구축하여 SA-V 데이터셋을 수집했으며, 다양한 작업 및 시각적 도메인에서 강력한 성능을 제공합니다. SAM 2.1 체크포인트와 훈련 코드, 웹 데모도 제공됩니다.
