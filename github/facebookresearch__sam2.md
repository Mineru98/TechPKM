SAM 2(Segment Anything Model 2)는 이미지와 비디오에서 프롬프트 기반 시각적 분할을 위한 파운데이션 모델입니다. 단일 프레임 이미지를 비디오로 간주하여 SAM을 확장했으며, 실시간 비디오 처리를 위한 스트리밍 메모리가 포함된 간단한 트랜스포머 아키텍처를 채택했습니다. 사용자 상호작용을 통해 모델과 데이터를 개선하는 모델-인-더-루프 데이터 엔진을 구축하여 현재까지 가장 큰 비디오 분할 데이터셋인 SA-V를 수집했습니다. 이 모델은 다양한 작업과 시각 영역에서 강력한 성능을 제공합니다.
