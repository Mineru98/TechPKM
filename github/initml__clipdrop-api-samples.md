이 프로젝트는 다양한 애플리케이션과 플랫폼에서 ClipDrop API를 활용하는 방법을 보여주는 오픈소스 샘플 모음입니다. iOS(SwiftUI), Android(Kotlin), 웹(React/Next.js) 플랫폼에서 이미지 배경 제거, 객체 감지, 레이어 분해 등 다양한 컴퓨터 비전 기능을 구현한 데모를 포함합니다. 각 샘플은 실시간으로 API를 호출하여 이미지 편집 작업을 처리하는 방법을 보여줍니다. 주요 기술 스택은 Swift, Kotlin, JavaScript(React/Next.js)이며, Tensorflow.js와 TailwindCSS도 일부 활용됩니다.
