**D3(데이터 기반 문서)**는 데이터를 시각화하기 위한 무료 오픈소스 JavaScript 라이브러리입니다. 웹 표준 기반의 저수준 접근 방식으로 동적 데이터 기반 그래픽 제작에 뛰어난 유연성을 제공하며, 10년 이상 혁신적인 시각화 구현과 고급 차트 라이브러리의 기반 기술로 활용되어 왔습니다. 전 세계 데이터 실무자 커뮤니티의 활발한 활동과 주요 차트 라이브러리들의 핵심 구성 요소로 자리매김했습니다.
