D3.js는 데이터를 시각화하기 위한 무료 오픈소스 JavaScript 라이브러리입니다. 웹 표준 기반의 저수준 접근 방식으로 동적이고 데이터 기반의 그래픽을 유연하게 제작할 수 있으며, 10년 이상 혁신적인 시각화 사례를 지원해 왔습니다. 고급 차트 라이브러리의 기반이 되며 전 세계 데이터 실무자 커뮤니티를 활성화하고 있습니다.
