서울대 계산언어학연구실에서 개발한 한국어 특화 Sentence-BERT 모델로, 문장 간 유사도 분석 및 의미적 관계 추출에 최적화되었습니다. KLUE-NLI와 증강된 KorSTS 데이터셋으로 미세 조정되었으며, KR-BERT 기반 모델을 활용해 문맥적 문장 표현을 생성합니다. 문서 분류, 동의어 판별 등 다양한 NLP 작업에 적용 가능하며 Python 환경에서 쉽게 사용할 수 있습니다.
