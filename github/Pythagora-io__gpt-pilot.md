GPT Pilot은 LLM(대형 언어 모델)을 활용해 개발자의 감독 하에 완전한 기능을 갖춘 프로덕션 레디 애플리케이션을 생성하기 위한 오픈소스 프로젝트입니다. VS Code 확장 도구와 CLI 도구로 제공되며, "프로덕트 오너", "아키텍트", "개발자" 등 다양한 AI 에이전트가 협업하는 방식으로 앱을 단계적으로 개발합니다. 현재 유지보수는 중단되었지만 Pythagora.ai에서 최신 정보를 확인할 수 있습니다. 주요 차별점은 대규모 코드베이스를 한 번에 생성하는 대신 단계별 개발/디버깅이 가능하다는 점입니다.
