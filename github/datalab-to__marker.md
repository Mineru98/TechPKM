Marker는 다양한 형식의 문서를 빠르고 정확하게 Markdown, JSON, 청크, HTML 등으로 변환하는 오픈소스 프로젝트입니다. PDF, 이미지, PPTX, DOCX, XLSX, EPUB 파일을 지원하며, 표, 수식, 코드 블록, 이미지 추출 및 레이아웃 정리를 자동으로 처리합니다. LLM(대형 언어 모델) 통합으로 정확도를 향상시킬 수 있으며, GPU/CPU/MPS 환경에서 작동합니다. 커스텀 포맷팅 및 JSON 스키마 기반 구조화된 추출 기능도 제공합니다.  

- **핵심 기능**: 다국어 OCR, 표/수식/코드 포맷팅, 이미지 추출, LLM 통합  
- **사용 사례**: 연구 논문, 교재, 금융 문서, 프레젠테이션 변환  
- **확장성**: 사용자 정의 프로세서/렌더러 추가 가능, API 서버 배포 지원
