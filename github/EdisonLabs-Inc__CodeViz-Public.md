CodeViz는 복잡한 코드베이스를 인터랙티브한 시각화 맵으로 변환하는 도구입니다. Y Combinator의 지원을 받는 이 프로젝트는 코드 구조 탐색, 자연어 검색, LLM 기반 분석을 통해 개발자의 코드 이해 속도를 향상시킵니다. 모든 분석은 로컬에서 수행되며, 코드 저장이나 외부 전송 없이 프라이버시를 보호합니다.

주요 기능으로는 계층적 코드 맵 생성, 클릭 탐색, 영어 질의 응답, 스마트 추천, 다국어 지원, LLM 프롬프트 생성, 다이어그램 공유 및 내보내기 기능이 포함됩니다. VS Code 확장으로 제공되며, 1.60.0 이상 버전에서 사용 가능합니다.
