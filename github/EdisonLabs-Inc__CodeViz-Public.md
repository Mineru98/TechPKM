CodeViz는 복잡한 코드베이스를 대화형 시각 맵으로 변환해주는 VS Code 확장 도구입니다. 주요 기능은 코드 구조 시각화, 자연어 검색, LLM 기반 코드 분석 및 다이어그램 공유 기능 등을 포함합니다. 모든 분석은 로컬에서 처리되며 코드를 저장하지 않는 프라이버시 우선 설계를 채택했습니다.

코드베이스를 계층적 구조로 시각화하고 노드 클릭 시 해당 코드로 직접 이동할 수 있으며, 자연어로 질문하면 관련 다이어그램을 생성합니다. GCP/AWS/Anthropic와의 통합에서 제로 데이터 보존 정책을 유지하며, 분석 결과는 로컬에 저장됩니다. VS Code 1.60.0 이상에서 사용 가능합니다.
