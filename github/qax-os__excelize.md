Excelize는 Go 언어로 작성된 순수 라이브러리로, XLAM/XLSM/XLSX/XLTM/XLTX 형식의 엑셀 파일을 읽고 쓸 수 있는 기능을 제공합니다. Microsoft Excel 2007 이후 버전에서 생성된 스프레드시트 문서를 지원하며, 대용량 데이터를 위한 스트리밍 API와 차트/이미지 삽입 기능을 포함하여 높은 호환성을 자랑합니다. Go 1.24.0 이상에서 사용할 수 있으며 BSD 3-Clause 라이선스로 배포됩니다.
