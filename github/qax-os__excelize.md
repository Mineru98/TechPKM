Excelize는 Go 언어로 작성된 라이브러리로, XLAM/XLSM/XLSX/XLTM/XLTX 파일을 읽고 쓸 수 있는 기능을 제공합니다. Microsoft Excel 2007 이상에서 생성된 스프레드시트 문서를 지원하며, 대용량 데이터 처리에 최적화된 스트리밍 API와 차트/이미지 삽입 기능을 포함합니다. Go 1.24.0 이상에서 동작하며, BSD 3-Clause 라이선스를 따릅니다. 주요 기능은 셀 데이터 조작, 차트 생성, 이미지 삽입 등이며, 복잡한 스프레드시트 문서 처리에 높은 호환성을 제공합니다.
