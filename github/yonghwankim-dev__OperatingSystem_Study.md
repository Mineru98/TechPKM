이 프로젝트는 운영체제 핵심 개념을 체계적으로 정리한 학습 노트입니다. 공룡책(Operating System Concepts) 10판과 관련 강의를 기반으로 프로세스, 동기화, 메모리 관리, 스케줄링, 교착상태, 보안 등 주요 주제를 다루고 있습니다. 각 장별로 개념 설명과 실제 사례를 통해 운영체제의 구조와 동작 원리를 이해할 수 있도록 구성되었습니다. 개발자가 운영체제 이론을 체계적으로 학습하거나 복습하기 위한 참고 자료로 적합합니다.
