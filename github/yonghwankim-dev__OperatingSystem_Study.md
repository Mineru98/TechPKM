이 프로젝트는 운영체제 개념과 구조를 체계적으로 정리한 학습 자료입니다. 공룡책(Operating System Concepts, 10th Ed.)과 인프런 강의를 기반으로 프로세스, 스레드, 동기화, 메모리 관리, 스케줄링, 교착 상태, 저장 장치 등 핵심 주제를 코드 예제와 요약으로 설명합니다. 각 장별 블로그 링크와 함께 이론과 실제 사례를 결합한 실용적인 학습 가이드 역할을 합니다.  

> 📌 주제 범위: CPU 스케줄링/동기화/Deadlock/가상 메모리/파일 시스템/보안 등 OS 전반
