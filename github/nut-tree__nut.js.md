`nut.js`(Native UI Toolkit)는 크로스 플랫폼 네이티브 UI 자동화 및 테스트 도구로, 키보드와 마우스를 통한 상호작용과 이미지 매칭 기반의 화면 탐색을 제공합니다. Windows, macOS, Linux(X11) 환경을 지원하며, 클립보드 제어부터 창 관리, 화면 요소 검출 등 다양한 기능을 모듈 형태로 제공합니다. 최근 NPM 공개 패키지는 제거되었지만, 오픈 소스 프로젝트로서 소스 코드를 직접 빌드하여 사용할 수 있습니다.
