React-Native-WebRTC는 React Native 애플리케이션을 위한 WebRTC 모듈을 제공하는 오픈소스 프로젝트입니다. Android, iOS, tvOS, Web 및 Expo 환경에서 오디오/비디오 스트리밍, 데이터 채널, 화면 캡처 등의 기능을 지원하며, Unified Plan과 Simulcast 같은 최신 WebRTC 기능도 활용할 수 있습니다. 이 모듈은 Jitsi의 WebRTC 구현(M124)을 기반으로 하며, 다양한 아키텍처(ARM, x86 등)에 대한 지원을 포함합니다.  

> 핵심 기능:  
> - 크로스 플랫폼 WebRTC 구현  
> - 실시간 미디어 전송 및 데이터 채널 통신  
> - Expo 및 react-native-web 확장 지원  
> - 커뮤니티 기반의 지속적인 개선 및 문서화
