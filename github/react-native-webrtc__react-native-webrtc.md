React Native 환경에서 WebRTC 기능을 구현한 모듈로, 모바일 및 웹 플랫폼에서 실시간 오디오/비디오 통신과 데이터 채널 지원을 제공합니다. Android, iOS, tvOS, 웹 및 Expo를 포함한 다양한 환경을 지원하며, 통합 플랜과 시뮬캐스트를 통한 고급 미디어 전송 기능을 포함합니다. 네이티브 코드 기반의 실시간 통신 애플리케이션 개발을 위한 핵심 라이브러리로 설계되었습니다.
