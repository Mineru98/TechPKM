Crawl4AI는 웹 페이지를 LLM(대형 언어 모델)과 호환되는 잘 정제된 마크다운 형식으로 변환하는 오픈소스 웹 크롤러 및 스크레이퍼입니다. RAG(검색 증강 생성), 에이전트 시스템, 데이터 파이프라인에 최적화된 구조화된 데이터를 제공하며, Python 기반의 비동기 및 동기 크롤링, Docker 배포, LLM 기반 추출 전략을 지원합니다. 주요 기능으로는 실시간 모니터링 대시보드, 브라우저 풀 관리, CSS/XPath 선택기 기반 추출, 지능형 컨텐츠 필터링(BM25) 등이 포함됩니다. 커뮤니티 주도로 개발되었으며 GitHub에서 가장 인기 있는 크롤러 프로젝트 중 하나입니다.
