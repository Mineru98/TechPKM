Crawl4AI는 웹 콘텐츠를 LLM(대형 언어 모델)에 최적화된 마크다운 형식으로 변환하는 오픈소스 크롤러 및 스크레이퍼입니다. 빠른 속도, 완전한 제어, 전투 검증된 안정성을 제공하며, RAG(검색 증강 생성), AI 에이전트, 데이터 파이프라인 구축을 지원합니다. BFS/DFS 전략 기반의 심층 크롤링, LLM 기반 구조화된 데이터 추출, 브라우저 세션 관리, 프록시 지원, 캐싱 등의 기능을 포함하여 복잡한 웹 데이터 수집 및 처리를 간소화합니다. Docker 배포와 CLI 도구도 제공되어 다양한 환경에서 활용할 수 있습니다.
