p5.js는 웹 기반 크리에이티브 코딩을 위한 오픈소스 JavaScript 라이브러리입니다. 예술가, 디자이너, 교육자, 초보자 등 다양한 사용자를 위해 접근성과 창의성을 중점으로 설계된 프레임워크를 제공하며, 시각적 표현, 인터랙티브 아트, 웹 기반 실험 작품 제작에 최적화되어 있습니다. 커뮤니티 주도 개발과 다국어 문서화를 통해 학습 및 협업이 용이합니다.
