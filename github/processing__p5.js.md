p5.js는 웹 기반 창의적 코딩을 위한 무료 오픈소스 JavaScript 라이브러리입니다. 예술가, 디자이너, 교육자, 초보자 등을 위해 접근성과 포용성을 중시하며, 웹에서 인터랙티브한 시각·청각 작품 제작을 지원합니다. 스케치 방식의 직관적인 API를 제공하며, 확장 가능한 라이브러리와 다국어 문서화로 전 세계 커뮤니티가 활발히 활동하고 있습니다. 웹 편집기, 튜토리얼, 레퍼런스 자료를 통해 누구나 쉽게 시작할 수 있습니다.
