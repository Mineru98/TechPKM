manim-kor는 Manim 라이브러리를 활용한 한국어 교육 프로젝트입니다. 알고리즘, 데이터 구조, 딥러닝 개념 등을 시각적으로 설명하는 영상 제작을 위한 코드와 튜토리얼을 제공합니다. raenim 유틸리티 툴을 통해 애니메이션 구현을 간소화하며, GitHub 저장소에 다양한 주제별 예제(소팅, 어텐션 메커니즘, 해시 테이블 등)가 포함되어 있습니다. 각 영상은 `manim` 명령어로 실행되며, 주제별 폴더 구조를 통해 체계적으로 관리됩니다.
