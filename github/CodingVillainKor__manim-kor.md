Manim-Kor은 3Blue1Brown 스타일의 애니메이션 라이브러리인 Manim을 기반으로 다양한 컴퓨터 과학 개념(알고리즘, 데이터 구조, 딥러닝 등)을 시각적으로 설명하는 한국어 프로젝트입니다. `raenim` 유틸리티 도구를 사용해 애니메이션 제작을 간소화하며, 각 주제별로 독립적인 파이썬 스크립트를 제공합니다. 생성된 영상은 `media/` 폴더에 저장되며, GitHub 저장소에 포함된 YouTube 링크를 통해 실제 동영상 예제를 확인할 수 있습니다.  

이 프로젝트는 정렬 알고리즘, 신경망 구조, 메모리 관리, 트랜스포머 아키텍처 등 60개 이상의 다양한 주제를 다루며, 각 스크립트는 `manim` 명령어로 독립적으로 실행 가능합니다. 특히 딥러닝(VAE, Diffusion, DiT 등), 파이썬 기초(브로드캐스팅, 얕은 복사), 버전 관리(Git 명령어) 등 실용적인 내용을 중점적으로 다룹니다.
