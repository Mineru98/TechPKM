LlamaDuo는 클라우드 기반 상용 대형 언어 모델(LLM)을 소규모 로컬 관리 가능 모델로 원활하게 마이그레이션하기 위한 LLMOps 파이프라인입니다. 서비스 LLM의 지식과 능력을 소규모 모델에 이전하기 위해 합성 데이터셋을 생성하고, 추가 파인튜닝을 통해 성능을 개선하는 다단계 프로세스를 제공합니다. 주요 적용 시나리오는 서비스 장애, 데이터 프라이버시 보호, 오프라인 환경 요구사항 등 제약 조건 하의 AI 배포입니다. Hugging Face 오픈소스 생태계를 기반으로 구현되었으며, 다양한 서비스 LLM과 로컬 LLM 간의 마이그레이션을 지원합니다.
