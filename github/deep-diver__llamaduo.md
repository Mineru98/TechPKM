LlamaDuo는 서비스형 LLM(예: GPT-4o, Claude 3)의 지식과 기능을 소규모 로컬 LLM(Gemma, Mistral, LLaMA3 등)로 원활하게 이전하기 위한 LLMOps 파이프라인입니다. ACL 2025에 채택된 이 프로젝트는 합성 데이터 생성, 평가, 추가 미세 조정을 통해 서비스 LLM과 동등하거나 우수한 성능을 달성하며, 오프라인/프라이버시 제약 환경에서 실용적인 솔루션을 제공합니다. Hugging Face 생태계와 통합된 도구셋을 통해 배치 추론, 평가, 데이터 생성을 지원합니다.
