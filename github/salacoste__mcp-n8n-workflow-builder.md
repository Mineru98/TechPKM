**n8n 워크플로우 빌더 MCP 서버**는 자연어와 Claude AI/Cursor IDE를 활용해 n8n 워크플로우를 생성·관리·모니터링하는 프로젝트입니다. 복잡한 JSON 편집이나 UI 전환 없이 대화형으로 워크플로우를 구축할 수 있으며, 개발/스테이징/운영 환경을 단일 서버에서 관리하는 멀티 인스턴스 아키텍처를 지원합니다. 17가지 도구로 워크플로우 라이프사이클 전반을 관리하고, 보안 설계 및 상세한 문서를 제공하는 것이 특징입니다.
