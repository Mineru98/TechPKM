이 프로젝트는 HTML을 깨끗하고 가독성 있는 마크다운으로 변환하는 강력한 도구입니다. 웹 페이지 전체를 변환할 수 있으며, 복잡한 서식 처리, 사용자 정의 옵션, 플러그인 확장 기능을 지원합니다. Go 언어 기반의 라이브러리와 CLI 명령어를 제공하며, 온라인 데모와 REST API도 활용할 수 있습니다.

핵심 기능으로는 굵게/기울임꼴, 목록, 인용문, 코드 블록, 링크/이미지 처리, 스마트 이스케이프, HTML 태그 제어, 사용자 정의 플러그인(특히 표 변환 플러그인) 등이 포함됩니다. 웹 스크랩핑 후 AI를 위한 마크다운 생성이 필요한 경우에도 유용하게 사용할 수 있습니다.
