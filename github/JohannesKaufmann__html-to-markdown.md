html-to-markdown은 HTML을 깔끔하고 가독성 있는 마크다운으로 변환하는 강력한 도구입니다. 복잡한 포맷팅, 사용자 정의 옵션, 플러그인 시스템을 지원하며, Go 라이브러리, CLI, 온라인 데모, REST API 형태로 활용할 수 있습니다. 주요 기능으로는 굵은 글씨/기울임꼴, 목록, 인용문, 코드 블록, 링크/이미지 처리, 스마트한 이스케이핑, HTML 태그 필터링 등이 포함됩니다. 웹 사이트 전체 변환도 지원하며, 확장 가능한 플러그인 아키텍처로 사용자 정의가 자유롭습니다.
