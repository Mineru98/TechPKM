DuckGPT는 JSON 및 CSV 파일에 대한 질문에 답변하기 위해 ChatGPT로 SQL 쿼리를 생성하고 DuckDB로 실행하는 프로젝트입니다. 주요 목적은 데이터 분석 시 자연어 질문을 SQL로 변환하여 처리하는 것이며, 오류 발생 시 자동으로 쿼리를 재작성하는 기능을 포함합니다. 간단한 설치와 사용법, 실제 데모를 통해 기능을 직관적으로 이해할 수 있도록 구성되었습니다. 주로 데이터 분석가와 개발자를 대상으로 합니다.
