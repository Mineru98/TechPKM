DuckGPT는 JSON 및 CSV 파일에 대한 질문을 입력하면 ChatGPT를 활용해 SQL 쿼리를 생성하고 DuckDB로 실행하는 프로젝트입니다. 파일 내 데이터를 자연어로 질의할 수 있으며, 생성된 쿼리의 오류를 자동으로 수정·재시도하는 기능을 갖추고 있습니다. 기술적 실험성 프로젝트이지만 데이터 분석에 직관적으로 접근할 수 있는 흥미로운 도구입니다.  

> 요약: DuckGPT는 자연어 질의 → ChatGPT 기반 SQL 생성 → DuckDB 실행 파이프라인을 구현한 데이터 분석 도구입니다. JSON/CSV 파일을 대상으로 동작하며 쿼리 오류 자동 복구 기능을 제공합니다.
