**Talk-To-ChatGPT**는 사용자가 음성으로 ChatGPT AI와 상호작용할 수 있도록 지원하는 Google Chrome 및 Microsoft Edge 확장 프로그램입니다. 음성 인식과 텍스트 음성 변환(TTS) 기능을 통해 자연스러운 대화형 경험을 제공하며, ElevenLabs API를 통한 맞춤형 음성 생성이 가능합니다. 주로 장애인이나 고령자 접근성 향상을 목적으로 개발되었으나, OpenAI의 공식 데스크톱 애플리케이션 출시로 유지보수가 중단된 프로젝트입니다. 확장 프로그램은 쿠키 기반 설정 저장, 음성 속도/언어/목소리 조정, 키보드 단축키 등의 기능을 지원합니다.
