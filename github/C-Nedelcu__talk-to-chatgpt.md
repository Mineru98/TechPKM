이 프로젝트는 중단된 Google Chrome 및 Microsoft Edge 확장 프로그램으로, 사용자가 음성으로 ChatGPT AI와 대화하고 텍스트 음성 변환(TTS)을 통해 답변을 들을 수 있게 합니다. 특히 ElevenLabs API 통합을 지원하여 사용자 정의 음성 생성이 가능하며, 장애인 및 시각 장애인의 접근성을 향상시키는 것을 목표로 했습니다. 현재는 OpenAI의 공식 데스크톱 앱 출시로 인해 개발이 중단되었으나, 오픈소스로 공개되어 있어 포크 및 개선이 가능합니다. 주요 기능으로는 실시간 음성 대화, 음성 속도/언어/목소리 설정, 설정 쿠키 저장 등이 있습니다.
