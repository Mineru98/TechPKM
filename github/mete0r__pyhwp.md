제공된 입력 정보에 README 내용이 포함되어 있지 않아, GitHub 프로젝트에 대한 메타데이터를 생성할 수 없습니다. README 내용을 확인한 후 다시 요청해 주시기 바랍니다.  

예시 입력 형식:  
```  
# 프로젝트 제목  
[README 본문 내용]  
```  

필요한 정보가 제공되면 다음과 같은 형식으로 생성해 드리겠습니다:  
```markdown  
---  
Language: {main_code_language}  
tags:  
 - {tag_name_1}  
 - {tag_name_2}  
aliases:  
 - {alias_1}  
 - {alias_2}  
url: {github_url}  
---  
{summary}  
```
