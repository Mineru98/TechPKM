WeSQL은 MySQL의 전통적인 디스크 스토리지를 완전히 S3(및 S3 호환 시스템)로 대체한 혁신적인 MySQL 배포판입니다. 모든 MySQL 데이터(바이너리 로그, 스키마, 스토리지 엔진 메타데이터 등)는 S3 객체 저장소에 완전히 저장되며, 99.9999999%의 내구성으로 데이터 신뢰성을 크게 향상시킵니다. 클라우드 벤더 종속 없이 모든 클라우드 환경에서 실행 가능하며, 서버리스 및 BYOC(Bring Your Own Cloud) 시나리오를 지원하는 개발자 친화적인 솔루션입니다. 깨끗한 인스턴스에서 바로 S3 데이터를 로드해 즉시 서비스를 시작할 수 있어 관리가 간편합니다.
