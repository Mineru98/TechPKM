WeSQL은 컴퓨팅-스토리지 분리 아키텍처를 채택한 혁신적인 MySQL 배포판이다. 모든 MySQL 데이터(스키마, 바이너리 로그, WAL 등)를 S3에 완전 저장하며, 높은 내구성과 벤더 종속성 없는 멀티클라우드 호환성을 제공한다. 개발자와 테스트 환경에서 즉시 사용 가능한 서버리스 친화적 데이터베이스 솔루션으로, BYOC(Bring Your Own Cloud) 시나리오에 최적화되었다.  

### 핵심 특징  
- **완전한 S3 스토리지 통합**: 기존 디스크 기반 저장 방식 완전 대체  
- **11나인 내구성**: S3의 고가용성 기반 데이터 신뢰성 보장  
- **즉시 가동**: 클린 인스턴스에서 S3 연결만으로 즉시 서비스 시작  
- **멀티클라우드 지원**: 모든 클라우드 환경에서 실행 가능  

### 대상 사용자  
- 비용 효율적이며 관리가 쉬운 MySQL 솔루션이 필요한 사용자  
- 서버리스 및 BYOC(자체 클라우드) 환경을 요구하는 사용자
