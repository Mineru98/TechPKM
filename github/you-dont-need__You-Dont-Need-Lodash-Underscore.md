이 프로젝트는 Lodash/Underscore의 많은 기능들이 최신 자바스크립트 엔진의 네이티브 기능들로 대체될 수 있음을 보여주는 가이드 문서입니다. 주요 브라우저에서 지원되는 ECMAScript5/ECMAScript2015 기능을 활용하여 외부 라이브러리 의존도를 줄이는 방법을 제시합니다. 프로젝트 종속성 최소화를 원하는 개발자들에게 유용하며, 타겟 브라우저의 기능 지원 여부를 명확히 알고 있을 때 적용할 수 있는 실용적인 대안들을 제시합니다. Lodash/Underscore의 1:1 비교는 아니지만, 각 기능별로 네이티브 구현 예제와 브라우저 호환성 정보를 상세히 제공합니다. ESLint 플러그인을 통해 코드베이스에서 불필요한 Lodash/Underscore 사용을 탐지할 수 있는 기능도 포함되어 있습니다.
