Lodash/Underscore는 널리 사용되는 JavaScript 유틸리티 라이브러리지만, ECMAScript5/6에서 지원하는 네이티브 기능으로 대체할 수 있는 경우가 많습니다. 이 프로젝트는 Lodash/Underscore 메서드를 네이티브 JavaScript로 대체하는 방법을 제공합니다. 주요 목적은 현대 브라우저에서 의존성 없이 코드를 작성할 수 있도록 돕는 것입니다. 각 메서드별로 네이티브 구현 예제와 브라우저 호환성 정보를 제공하여, 프로젝트 요구사항에 맞는 최적의 선택을 할 수 있도록 지원합니다. Lodash/Underscore의 전체 기능을 1:1로 대체하는 것은 아니지만, 자주 사용하는 핵심 기능들을 네이티브 방식으로 처리할 수 있습니다.

### 요약
- **목적**: Lodash/Underscore의 네이티브 대체 방법 제공
- **주요 내용**: 배열, 컬렉션, 함수, 객체, 문자열, 숫자 관련 메서드
- **특징**: 
  - 네이티브 JavaScript 사용
  - 브라우저 호환성 정보 포함
  - Lodash/Underscore와 1:1 매핑되지 않음
- **장점**: 
  - 의존성 감소
  - 최신 브라우저 최적화
  - 가벼운 코드베이스
