Cua는 컴퓨터가 사용할 수 있는 에이전트를 구축, 벤치마킹, 배포하기 위한 오픈소스 플랫폼입니다. 데스크톱 환경과의 상호작용, 코드 실행 격리, macOS/Linux 가상화, 표준화된 벤치마크 테스트 등을 지원하며, Docker, QEMU, Apple Vz 기반의 샌드박스 환경을 제공합니다. AI 코딩 어시스턴트, 강화학습 훈련, CI/CD 테스트 등에 활용 가능한 종합적인 도구입니다. 주요 구성 요소로는 에이전트 프레임워크, 컴퓨터 제어 SDK, 벤치마크 도구, macOS 가상화 관리자 등이 포함됩니다.
