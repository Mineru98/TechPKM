Cua는 컴퓨터를 사용하는 AI 에이전트를 구축, 벤치마킹, 배포하기 위한 오픈소스 플랫폼입니다. Docker, QEMU, Apple Vz 기반의 격리된 샌드박스 환경에서 코드 실행 및 UI 자동화를 지원하며, macOS/Linux VM 관리 및 강화학습 환경 제공으로 다양한 컴퓨터 사용 작업에 최적화된 솔루션을 제공합니다. 주요 구성 요소인 Cua, Cua-Bench, Lume를 통해 에이전트 개발, 성능 평가, 가상화 인프라 운영이 가능합니다.
