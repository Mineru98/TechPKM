Stats는 macOS 시스템 모니터링 애플리케이션으로, 메뉴바에서 CPU/GPU 사용량, 메모리 사용량, 디스크/네트워크 사용량, 배터리 상태, 센서 정보 등을 실시간으로 확인할 수 있습니다. macOS 10.15(Catalina) 이상에서 동작하며, Homebrew를 통한 설치와 수동 설치 방식을 모두 지원합니다. 팬 제어 및 다중 시간대 시계 기능도 포함되어 있으며, 커뮤니티 주도로 다양한 언어를 지원합니다. MIT 라이선스로 배포되는 오픈소스 프로젝트입니다.
