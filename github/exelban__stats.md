Stats는 macOS 시스템 모니터링 애플리케이션으로 메뉴바에서 CPU, GPU, 메모리, 디스크, 네트워크 사용량, 배터리 상태 등을 실시간으로 확인할 수 있습니다. macOS 10.15 이상에서 동작하며 센서 정보, 블루투스 장치 연결, 다중 시간대 시계 등의 기능을 제공합니다. 사용자는 모듈 비활성화를 통해 CPU 사용량과 에너지 소비를 최적화할 수 있으며, Homebrew 또는 DMG 파일을 통해 쉽게 설치할 수 있습니다. MIT 라이선스로 배포되며 다국어(전 세계 40개 이상 언어) 지원을 특징으로 합니다.
