Dalai는 LLaMA와 Alpaca 모델을 로컬 컴퓨터에서 실행할 수 있게 해주는 오픈소스 프로젝트입니다. 크로스 플랫폼(Windows, macOS, Linux) 지원을 제공하며, 웹 기반 UI와 JavaScript/Socket.io API를 통해 모델을 쉽게 사용할 수 있습니다. 모델 설치 시 양자화(Quantization)를 통해 저장 공간을 최적화하며, Docker를 통한 배포도 지원합니다.  

### 핵심 기능  
1. **크로스 플랫폼 지원**: 모든 주요 운영체제에서 동작  
2. **저장 공간 최적화**: 양자화된 모델 제공(예: 7B 모델 4.21GB)  
3. **API 및 웹 UI**: Socket.io와 JavaScript API를 통한 통합  
4. **유연한 설치**: 단일/다중 모델 선택적 설치 가능  

> 📌 프로젝트 목적: 고성능 LLM을 로컬에서 실행하는 간소화된 환경 구축
