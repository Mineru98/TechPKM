이 프로젝트는 Cursor에서 프로젝트를 시작하기 위한 필수 문서 체계를 제공하는 템플릿입니다. 핵심 비전 정의, 기능적/비기능적 요구사항, 시스템 아키텍처, 기술 스택 선택 근거, 개별 기능 상세, 개발 접근법 및 사용자 흐름을 체계적으로 문서화하여 프로젝트 전반을 안내하는 것을 목표로 합니다. 각 문서는 특정 목적에 맞게 구성되어 개발 팀의 일관성과 방향성을 유지하는 "북극성" 역할을 수행합니다.
