LLaDA는 확산 모델 기반의 80억 규모 대규모 언어 모델로, 기존 LLaMA3 8B 모델과 필적하는 성능을 갖추며 비자기회귀적 접근이 가능한 생성형 AI 프로젝트입니다. 텍스트 생성 시 마스킹과 예측 방식을 결합하여 전통적인 자동회귀 모델과 차별화된 이론적 완성도를 추구하며, 다단계 추론 및 지시 수행 능력이 특징입니다. Hugging Face를 통해 모델 가중치와 데모 환경을 공개하고 있으며, 연구 논문(arXiv)과 함께 평가 코드 및 추론 가이드를 제공합니다.
