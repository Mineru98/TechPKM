이 프로젝트는 EMNLP 2023에 발표된 논문에서 제안된 의사코드 기반 프롬프트 기법을 구현한 데이터셋 및 코드 저장소입니다. Super-Natural Instructions 데이터셋의 다양한 태스크에 대해 인간이 작성한 의사코드 지시어를 포함하며, 자연어 프롬프트 대비 모호성 감소와 성능 향상을 목적으로 합니다. BLOOM 및 CodeGen 모델에서의 실험을 통해 분류 및 생성 태스크에서 평균 7-16% F1 점수 향상과 12-38% ROUGE-L 점수 개선이 입증되었습니다. 데이터셋은 Apache-2.0 라이선스로 배포되며, 각 태스크 인스턴스는 원본 데이터셋의 라이선스를 따릅니다.
