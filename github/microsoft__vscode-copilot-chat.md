GitHub Copilot은 AI 기반 페어 프로그래밍 도구로, 코드 작성 속도와 효율성을 향상시킵니다. Visual Studio Code에서 제공하는 이 도구는 인라인 코딩 제안, 자연어 기반 코드 수정, 에이전트 모드와 같은 기능을 통해 개발자의 워크플로우에 맞춤화된 지원을 제공합니다. 주로 Python, JavaScript, Java 등 다양한 프로그래밍 언어를 지원하며, 코드베이스에 특화된 AI 채팅 및 편집 기능을 활용할 수 있습니다.

> 📌 **요약**: GitHub Copilot은 VS Code 확장을 통해 인라인 코드 제안, 자연어 기반 코드 수정, 에이전트 모드 등 다양한 AI 기능을 제공하는 페어 프로그래밍 도구입니다. 개발자의 코딩 스타일과 코드베이스에 맞춰 동작하며, MIT 라이선스로 배포되는 오픈소스 프로젝트입니다.
