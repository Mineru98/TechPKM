이 프로젝트는 Visual Studio Code용 GitHub Copilot Chat 확장으로, AI 기반의 대화형 코딩 어시스턴트를 제공합니다. 코드 편집 중에 자연어 채팅으로 AI 지원을 받고, 에이전트 모드를 통해 다중 단계 코딩 작업을 자동화할 수 있으며, 인라인 제안 기능으로 코드 작성 효율성을 높이는 것이 핵심입니다. 프로젝트 요구사항이나 코드 변경 사항에 대한 대화를 통해 실시간으로 코드베이스에 적용할 수 있는 솔루션을 제안하는 것이 특징입니다. Python/JavaScript 등 주요 언어 및 프레임워크를 지원하며, VS Code와의 깊은 통합을 통해 원활한 사용자 경험을 제공합니다.
