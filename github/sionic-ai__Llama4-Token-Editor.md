LLaMA Token Editor는 대형 언어 모델(LLM)의 토크나이저를 분석하고 토큰 가중치를 조정하는 도구입니다. Llama 및 Qwen 계열 모델의 토큰화 방식을 분석하며, 한글 표현 가능성 토큰과 완성형 한글 토큰을 식별하고 가중치를 조정할 수 있습니다. 이 도구는 모델의 한국어 처리 성능을 최적화하는 데 목적을 두며, UTF-8 인코딩과 BPE 알고리즘을 역공학적으로 분석합니다. 주요 기능으로는 토큰 카테고리 분류, 가중치 조정, 그리고 다양한 모델 지원이 포함됩니다.
