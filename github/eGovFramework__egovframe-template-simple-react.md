표준프레임워크 심플홈페이지를 위한 React 기반 프론트엔드 템플릿입니다. 메인 페이지, 사용자/공지사항/게시판 관리 기능을 제공하는 경량화된 웹사이트 구조로, 백엔드와 분리된 프론트엔드 개발 환경을 지원합니다. 소셜 로그인(네이버/카카오) 통합과 관리자 기능 커스터마이징이 가능하며, 정부기관 또는 공공기관에 적합한 기본 UI 구조를 제공합니다. Vite 기반의 빠른 개발 환경을 지원합니다.
