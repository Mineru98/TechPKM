# 표준프레임워크 심플홈페이지 FrontEnd

이 프로젝트는 React 기반의 경량화된 프론트엔드 템플릿으로, 정부 포털 또는 공식 홈페이지 구축에 필요한 기본 기능을 제공합니다. 메인 페이지, 사용자/공지사항/게시판 관리, 소셜 로그인(네이버/카카오) 기능을 포함하며, 백엔드와 분리된 프론트엔드 개발 환경을 지원합니다. 샘플 페이지와 공통 컴포넌트를 활용하여 실무 맞춤형 커스터마이징이 가능합니다.  

> 📌 기술 스택: React, Vite, Node.js, npm  
> 🔑 특징: 소셜 로그인 연동, 관리자/사용자 권한 분리, 경량화된 템플릿 구조  
> 🛠 활용: 공공기관 홈페이지, 웹 포털 시스템 개발에 최적화된 기본 프레임워크 제공
