Zerox OCR은 PDF, DOCX, 이미지 등 다양한 형식의 문서를 마크다운으로 변환하는 도구입니다. 문서를 이미지로 변환한 후 GPT 모델에 전달하여 구조화된 출력을 생성하며, OpenAI, Azure, AWS Bedrock, Google Gemini 등 다양한 AI 모델을 지원합니다. Node.js 및 Python SDK를 제공하며, 레이아웃 유지, 페이지 단위 추출, 데이터 스키마 추출 등의 고급 기능을 포함합니다.  

Zerox는 복잡한 표, 차트, 다단 레이아웃이 포함된 문서에도 효과적으로 작동하며, 비동기 처리, 동시성 제어, 임시 파일 관리 등의 기능을 통해 대규모 문서 처리에 최적화되어 있습니다. Python 버전에서는 Vertex AI 지원도 추가로 제공됩니다.
