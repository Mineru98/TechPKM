이 프로젝트는 NestJS와 Websockets를 활용한 실시간 채팅 애플리케이션 보일러플레이트입니다. JWT 인증, 방 관리, 사용자 차단/강퇴 기능을 포함하며 Docker 환경을 지원합니다. 서버 기반 실시간 통신 시스템 구현을 위한 기본 템플릿으로 제작되었습니다. PostgreSQL 데이터베이스와 NestJS의 모듈화된 아키텍처를 기반으로 합니다.
