이 프로젝트는 Nest.js 프레임워크를 기반으로 실시간 채팅 애플리케이션을 구현한 예시입니다. 웹소켓 기술을 활용해 사용자 인증(PassportJS/JWT), 채팅방 생성, 사용자 차단/강퇴 등의 핵심 기능을 제공합니다. PostgreSQL과 Docker 환경을 사용하여 확장 가능한 서버 아키텍처를 구현한 것이 특징입니다. 실시간 통신이 필요한 애플리케이션 개발에 참고할 수 있는 예제 코드입니다.
