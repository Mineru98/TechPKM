Databricks의 Dolly-v2-12b는 상용 라이선스로 제공되는 인스트럭션 기반 대형 언어 모델입니다. Pythia-12b를 기반으로 Databricks 직원들이 생성한 15,000개의 인스트럭션/응답 데이터로 미세 조정되었으며, 브레인스토밍, 분류, 요약 등 다양한 작업에 활용 가능합니다. SOTA 모델은 아니지만, 창의적인 인스트럭션 수행 능력을 보입니다. Hugging Face에서 모델 가중치와 학습 데이터를 공개하며, CC-BY-SA 라이선스로 배포됩니다.  

> **참고**: 성능은 복잡한 구문, 수학 연산, 사실적 정확성 등에서 한계가 있습니다. 학습 데이터의 편향성과 제한사항이 모델에 반영될 수 있음에 주의가 필요합니다.
