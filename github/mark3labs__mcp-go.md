MCP-Go는 LLM 애플리케이션과 외부 데이터 소스/도구 간의 원활한 통합을 가능하게 하는 Model Context Protocol(MCP)의 Go 구현체입니다. 서버 생성부터 도구/리소스 관리까지 최소한의 코드로 프로토콜을 구현할 수 있도록 설계되었습니다. 주요 기능으로는 간편한 도구 등록, 리소스 제공, 세션 관리, 자동 완성 지원 등이 있으며, 현재는 개발 중입니다. MCP 사양 버전 2025-11-25를 기준으로 구현되었습니다.
