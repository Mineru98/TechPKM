ScrapeGraphAI는 LLM(대형 언어 모델)과 직접 그래프 로직을 활용해 웹사이트 및 로컬 문서(XML, HTML, JSON, 마크다운 등)에서 데이터를 추출하는 파이썬 라이브러리입니다. 사용자 프롬프트만으로 단일/다중 페이지 스크랩핑, 파이썬 스크립트 생성, 오디오 파일 변환 등의 파이프라인을 실행할 수 있으며, OpenAI, Groq, Azure, Gemini API 또는 Ollama 기반 로컬 모델을 지원합니다. 다양한 프레임워크 및 노코드 플랫폼과 통합 가능한 확장성이 특징입니다.
