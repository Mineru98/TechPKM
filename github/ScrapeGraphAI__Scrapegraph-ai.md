ScrapeGraphAI는 LLM과 그래프 로직을 활용해 웹사이트 및 로컬 문서(XML, HTML, JSON 등)에서 데이터를 추출하는 Python 라이브러리입니다. 사용자 프롬프트만으로 필요한 정보를 자동으로 추출하며, 단일/다중 페이지 스크래핑, 스크립트 생성, 음성 파일 생성 등 다양한 파이프라인을 지원합니다. OpenAI, Ollama 등 다양한 LLM과 통합 가능하며, PyPI를 통해 설치할 수 있습니다.  

> 📌 핵심 기능:  
> - 프롬프트 기반 자동 데이터 추출  
> - 다양한 LLM 모델(Gemini, Groq, Ollama 등) 지원  
> - 단일/다중 페이지 스크래핑 및 스크립트 생성  
> - Python/Node.js SDK 및 API 제공  
> - MIT 라이선스 기반의 오픈소스 프로젝트
