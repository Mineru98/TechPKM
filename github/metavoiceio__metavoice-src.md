MetaVoice-1B는 100K 시간의 음성 데이터로 학습된 1.2B 파라미터 기반 모델로, 영어 감정 표현, 미국식/영국식 목소리 제로샷 클로닝, 파인튜닝을 통한 크로스-링구얼 음성 클로닝, 임의의 길이 텍스트 합성 등을 지원합니다. Apache 2.0 라이선스로 공개되어 제한 없이 사용할 수 있으며, Docker를 통한 웹 UI/서버 배포 및 파인튜닝 기능을 제공합니다. Hugging Face, Google Colab 데모, 로컬 실행 등 다양한 사용 환경을 지원합니다.
