`mcp-ui`는 Model Context Protocol(MCP) 기반의 대화형 웹 컴포넌트를 제공하는 오픈소스 프로젝트입니다. 서버와 클라이언트 간 UI 리소스를 생성하고 렌더링하는 SDK를 포함하며, TypeScript, Ruby, Python 환경에서 사용 가능합니다. HTML, 외부 URL, RemoteDOM 등의 다양한 유형의 UI를 지원하며, 호스트 환경과의 상호작용을 위한 액션 처리 기능을 제공합니다. AI 에이전트와의 통합을 통해 사용자 인터페이스를 확장할 수 있는 실험적 커뮤니티입니다. 보안은 샌드박스된 iframe을 통해 보장됩니다.
