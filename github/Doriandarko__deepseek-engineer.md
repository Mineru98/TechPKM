DeepSeek Engineer v2는 대화형 터미널 인터페이스를 통해 자연어와 함수 호출을 지원하는 AI 기반 코딩 어시스턴트입니다. DeepSeek의 고급 추론 모델과 통합되어 파일 읽기/작성, 코드 분석, 프로젝트 자동화 등의 기능을 제공하며, 체인 오브 사고(CoT)를 통해 추론 과정을 가시화합니다. 사용자는 파일 경로 언급만으로 자동 파일 읽기, 다중 파일 생성, 코드 편집 등의 작업을 수행할 수 있으며, 실시간 스트리밍과 구조화된 피드백을 지원합니다.  

> 핵심 기능:  
- **자동 파일 읽기/작성** (예: `read_file`, `create_file`)  
- **코드 분석** 및 최적화 제안  
- **함수 호출 기반** 프로젝트 생성 (예: Flask API 자동 생성)  
- **보안 강화** (디렉토리 탐색 방지, 파일 크기 제한 등)  
- **터미널 인터페이스** (컬러 코드, 진행 상태 표시, 테이블 형식 출력)  

MIT 라이선스로 배포되며, 실험적인 기능 탐색을 위한 오픈소스 프로젝트입니다.
