DeepSeek Engineer v2는 대화형 터미널 인터페이스를 제공하는 AI 기반 코딩 보조 도구입니다. 딥시크의 추론 모델과 통합되어 자연스러운 대화와 함수 호출을 통해 파일 조작, 코드 분석 및 개발 지원을 가능하게 합니다. 자동 파일 읽기/쓰기, 실시간 추론 시각화, 보안 강화 기능을 통해 효율적인 코딩 환경을 구축합니다.  

### 핵심 기능  
- **AI 기능**: 체인의 사고(CoT) 추론을 통한 전문가 수준의 코드 분석 및 문제 해결  
- **함수 호출**: 파일 읽기/생성/편집 등 5가지 자동 도구 지원  
- **보안**: 경로 정규화, 디렉토리 탐색 방지, 파일 크기 제한  
- **인터페이스**: 실시간 스트리밍, 색상 피드백, 구조화된 테이블 제공  

이 프로젝트는 MIT 라이선스로 배포되며, 딥시크 API 키와 Python 3.11 이상이 필요합니다.
