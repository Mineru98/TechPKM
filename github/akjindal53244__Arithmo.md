Arithmo 프로젝트는 수학 문제를 추론하고 해결하는 데 특화된 LLM 모델입니다. Mistral-7B 기반에 QLoRA를 사용해 단일 GPU에서 파인튜닝되었으며, GSM8K 및 MATH 벤치마크에서 경쟁력 있는 성능을 보입니다. Arithmo2-Mistral-7B 모델은 이전 버전 대비 GSM8K +1.7%, MATH +1.9% 성능 향상이 있으며, Chain-of-Thought(CoT)와 Python 코드 생성(PoT) 방식을 지원합니다. Apache-2.0 라이선스로 공개되어 있습니다.
