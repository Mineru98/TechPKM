이 프로젝트는 React 컴포넌트를 사용한 A/B 테스트를 위한 라이브러리로, `<Experiment>`과 `<Variant>` 컴포넌트를 통해 UI 실험을 구현합니다. 각 변형은 무작위로 선택되며 로컬 스토리지에 저장되어 실험 일관성을 유지합니다. Mixpanel 및 Segment.com과 같은 분석 도구와의 통합을 지원하며, 서버 측 렌더링과 디버깅 기능도 제공합니다. 사용자는 실험 참여자 및 승리 이벤트를 추적하여 데이터 기반 의사결정을 할 수 있습니다.
