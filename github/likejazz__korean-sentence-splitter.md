한국어 텍스트를 휴리스틱 알고리즘으로 문장 단위로 분할하는 오픈소스 라이브러리입니다. 카카오 NLP 리더의 연구에서 영감을 받았으며, 구어체와 문어체 모두에서 뛰어난 성능을 보입니다. 현재는 Python으로 포팅되어 유지보수되며, `pip install kss`로 간단히 설치할 수 있습니다.  

주요 특징으로는 테스트 세트에서 최적의 성능을 달성했으며, Python 3+ 환경에서 사용할 수 있습니다. C++ 기반에서 시작되었으나 현재는 Python 포팅 버전이 활발히 개선 중입니다.
