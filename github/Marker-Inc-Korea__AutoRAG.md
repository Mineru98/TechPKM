AutoRAG는 사용자의 데이터와 사용 사례에 최적화된 RAG(Retrieval-Augmented Generation) 파이프라인을 자동으로 탐색·평가하는 도구입니다. 다양한 RAG 모듈 조합을 자동으로 실행하여 최적의 파이프라인을 찾을 수 있으며, 데이터 생성부터 파이프라인 배포까지 종합적인 워크플로우를 지원합니다. 주요 기능으로는 파싱, 청킹, QA 데이터 생성, RAG 모듈 최적화, 평가 지표 분석 및 배포 기능이 포함됩니다.  

AutoRAG는 Python 3.10 이상에서 동작하며, GPU 지원 모델과 사용자 정의 모델 통합도 가능합니다. 사용자는 YAML 설정 파일을 통해 파이프라인 구성을 정의하고, 자동으로 평가된 결과를 대시보드에서 시각화할 수 있습니다. 또한 Hugging Face Space와 Colab 튜토리얼을 통해 쉽게 시작할 수 있습니다.
