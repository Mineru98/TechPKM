AutoRAG는 사용자 데이터와 사용 사례에 최적화된 RAG(Retrieval-Augmented Generation) 파이프라인을 자동으로 탐색하고 평가하는 AutoML 도구입니다. 다양한 RAG 모듈 조합을 자동으로 테스트하여 최적의 파이프라인을 찾아주며, 데이터 준비부터 최적화, 배포까지 전 과정을 지원합니다. 사용자 정의 LLM 및 임베딩 모델 통합이 가능하며, 대시보드와 API 서버를 통한 결과 시각화 및 배포 기능을 제공합니다.  

이 프로젝트는 Python 3.10 이상에서 실행되며, GPU 지원 설치 옵션과 데이터 생성, 파이프라인 최적화, 웹 인터페이스 등 다양한 기능을 포함합니다. Hugging Face 스페이스와 Colab 튜토리얼을 통해 쉽게 시작할 수 있습니다.
