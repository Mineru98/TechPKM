**Stackflow**는 모바일 기기(iOS/Android 등)에서 사용되는 스택 네비게이션 UX를 JavaScript 환경에서 구현한 프로젝트입니다. 하이브리드 앱과 웹뷰 개발을 용이하게 하기 위해 화면 스택 관리, 스크롤 상태 유지, 전환 효과, iOS 스타일 스와이프 백 기능 등을 지원합니다. 기존 내비게이션 라이브러리와 달리 UI와 상태 관리를 분리하고 플러그인 시스템을 통해 확장성이 뛰어나며, 다양한 프론트엔드 프레임워크(현재는 React 지원)와 통합될 수 있습니다. 서버사이드 렌더링(SSR)도 지원합니다.  

문서의 전체 설명과 데모는 [https://stackflow.so](https://stackflow.so)에서 확인할 수 있습니다.
