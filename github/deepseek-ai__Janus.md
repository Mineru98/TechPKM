Janus-Series는 멀티모달 이해와 생성 작업을 통합하는 딥러닝 모델 시리즈입니다. Janus는 시각 인코딩을 분리하여 단일 아키텍처로 이해 및 생성 작업을 처리하며, JanusFlow는 자동회귀 모델과 정류 흐름(Rectified Flow)을 결합해 이미지 생성 성능을 향상시켰습니다. Janus-Pro는 데이터 확장 및 모델 크기 증가를 통해 멀티모달 이해와 텍스트-이미지 생성 능력을 개선한 고급 버전입니다. 이 프로젝트는 연구 및 상업적 사용을 지원하며 MIT 라이선스와 모델 협약을 따릅니다.
