이 프로젝트는 Telegram API와 AI 어시스턴트 간의 브리지 역할을 하는 MCP(Model Context Protocol) 기반 서버입니다. 텔레그램 대화 목록 및 메시지 읽기 기능을 제공하며, Claude Desktop과 같은 AI 도구와의 연동을 통해 미확인 메시지 요약 등의 실용적인 사용 사례를 지원합니다. 현재는 읽기 전용 기능만 구현되어 있으며, 향후 메시지 확인 및 미디어 다운로드 등의 기능이 추가될 예정입니다. Python으로 개발되었으며 MCP 프로토콜을 통해 로컬 서비스와 안전하게 통신합니다.
