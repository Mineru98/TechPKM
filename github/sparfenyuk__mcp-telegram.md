이 프로젝트는 Telegram API와 AI 어시스턴트를 연결하는 MCP(Model Context Protocol) 기반 서버입니다. 현재 대화 목록 조회, 읽지 않은 메시지 확인 등 읽기 전용 기능을 제공하며, Claude Desktop과 같은 AI 도구와의 통합을 목표로 합니다. 사용자는 텔레그램 API를 통해 메시지 요약 생성, 특정 주제 논의 분석 등의 실용적인 작업을 수행할 수 있습니다. Python으로 구현되었으며 `uv` 도구를 통해 설치 및 관리가 가능합니다.
