제공된 입력이 비어 있으므로, README 내용으로 추정할 수 있는 프로젝트 정보가 없습니다.  
빈 README 또는 누락된 입력을 기반으로 메타데이터를 생성할 수 없습니다.  

프로젝트 정보를 포함한 README 내용을 제공해 주시면,  
Obsidian 관리용 메타데이터 문서를 정확하게 생성해 드리겠습니다.  

예시 형식:  
```markdown
---
Language: [언어]
tags:
 - [태그1]
 - [태그2]
aliases:
 - [별칭1]
 - [별칭2]
url: [GitHub_URL]
---
[요약문: 프로젝트 목적과 특징을 2~4문장으로 설명]
```
