Celery는 Python으로 작성된 분산 작업 큐 시스템으로, 비동기 작업 처리, 작업 예약 및 메시지 기반 통신을 지원합니다. RabbitMQ, Redis 등의 브로커를 사용하여 클라이언트와 워커 간의 작업을 분산 처리하며, 다양한 메시지 전송 방식, 동시성 모델, 결과 저장소를 지원합니다. Django, Flask 등 주요 웹 프레임워크와 통합되어 확장성과 유연성을 제공합니다. Python 3.8+ 호환되며, Tidelift 및 Open Collective를 통해 기업 지원 및 커뮤니티 후원이 가능합니다.
