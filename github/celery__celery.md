Celery는 Python 기반의 분산 작업 큐 시스템으로, 비동기 작업 처리를 위한 메시지 브로커(RabbitMQ, Redis 등)와 연동하여 작업을 생성하고 관리합니다. 단일 서버 또는 다중 서버 환경에서 사용할 수 있으며, 언어 간 호환성과 확장 가능한 아키텍처를 지원합니다. 웹 프레임워크(Django, Flask 등)와의 통합이 용이하며, 다양한 메시지 전송, 결과 저장, 직렬화 방식을 제공합니다.  

핵심 특징으로는 간단한 사용법, 고가용성, 빠른 처리 속도, 유연성을 갖추고 있으며, 이벤트 기반 처리, 메시지 서명과 압축, 다양한 백엔드 지원 등의 기능을 포함합니다. Python 3.8 이상 버전에서 공식적으로 지원됩니다.
