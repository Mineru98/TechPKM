이 프로젝트는 Kotlin/Wasm을 활용한 다양한 예제를 제공하는 저장소입니다. 각 예제는 소스 코드와 빌드/실행 방법을 설명하는 README.md 파일을 포함하며, 웹 애플리케이션부터 WASI API 사용 사례, C 언어와의 상호 운용성까지 다루습니다. Compose Multiplatform을 이용한 웹 애플리케이션 템플릿과 DOM API를 활용한 브라우저 예제 등이 포함되어 Kotlin/Wasm의 다양한 활용 방식을 보여줍니다. Kotlin 공식 문서에서 WebAssembly 관련 정보도 참조할 수 있습니다.
