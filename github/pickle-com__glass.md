Glass는 픽클에서 개발한 데스크톱 기반 AI 도구로, 사용자의 화면과 음성을 실시간으로 분석해 구조화된 지식으로 변환합니다. 회의 중 액션 아이템 추출, 요약 생성, 즉각적인 답변 제공 등 업무 효율성을 높이는 기능을 제공하며, macOS/Windows에서 실행 가능합니다. 오픈소스 기반으로 OpenAI/Gemini API 및 로컬 LLM 모델을 지원하며, 개발 기여는 GitHub을 통해 활발히 진행 중입니다.  

> 핵심 기능: 실시간 음성 인식, 회의 요약, AI 기반 질문 응답, 로컬 LLM 통합  
> 기술 스택: JavaScript/Node.js, Python, Rust(오디오 처리), OpenAI/Gemini API  
> 특징: 화면 녹화/캡처 방지, 키보드 단축키 지원, 모듈식 아키텍처 재설계 진행 중  

이 프로젝트는 [CheatingDaddy](https://github.com/sohzm/cheating-daddy)를 포크한 버전으로, 픽클의 미션인 "모든 사람을 위한 디지털 클론" 구현의 첫 단계입니다.
