Glass는 데스크탑에서 작동하는 AI 기반 디지털 마인드 확장 도구로, 화면과 오디오를 실시간으로 분석해 구조화된 지식으로 변환합니다. 회의 중 액션 아이템 추출, 요약 생성, 즉각적인 답변 제공 등 프로액티브한 기능을 제공하며, 화면 녹화/캡처 시 완전히 사라지는 "투명성"을 특징으로 합니다. OpenAI, Gemini, Ollama 등 다양한 LLM과 STT 모델 호환이 가능하며, macOS 및 Windows(베타) 버전을 지원합니다. 

프로젝트 공식 사이트: [https://pickle.com/glass](https://pickle.com/glass)  
핵심 기능: 실시간 화면 분석, AI 기반 지식 추출, 회의 보조, 로컬 LLM 지원, 키보드 단축키 기반 제어
