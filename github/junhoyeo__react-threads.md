이 프로젝트는 React/Next.js 애플리케이션에 정적 스레드(Threads) UI를 임베드하는 컴포넌트를 제공했으며, 메타(Meta)의 스레드(Threads) 플랫폼용 UI 요소를 구현했습니다. junhoyeo/threads-api 프로젝트를 기반으로 동작했으나, 2023년 9월 8일 이후로 Meta의 법적 통지로 인해 개발이 중단되고 아카이브 처리되었습니다. 관련 문서는 PRESERVED.md로 이동되었습니다. 프로젝트는 교육 및 연구 목적으로만 개발되었으나, 현재 유지보수되지 않습니다.
