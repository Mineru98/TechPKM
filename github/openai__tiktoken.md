이 프로젝트는 OpenAI 모델의 빠른 BPE(Byte Pair Encoding) 토크나이저를 제공하는 Python 라이브러리입니다. 텍스트를 토큰으로 인코딩/디코딩할 수 있으며, GPT-4o 등의 특정 모델에 맞는 토크나이저를 지원합니다. 오픈소스 버전으로 PyPI에서 설치 가능하며, 기존 토크나이저 대비 3-6배 빠른 성능을 제공합니다. 토큰 카운팅, 텍스트 처리, NLP 작업에 유용하게 사용될 수 있습니다.
