tus-js-client는 tus 프로토콜을 기반으로 한 재개 가능한 파일 업로드를 구현하는 순수 JavaScript 클라이언트입니다. 브라우저, Node.js, React Native 및 Cordova 환경에서 사용할 수 있으며, 네트워크 문제나 서버 중단 시에도 이전 데이터를 재전송하지 않고 업로드를 재개할 수 있는 기능을 제공합니다. 주요 기능으로는 업로드 상태 추적, 자동 재시도, 메타데이터 지원 등이 포함됩니다. MIT 라이선스로 배포되는 오픈소스 프로젝트입니다.
