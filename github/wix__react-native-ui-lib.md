이 프로젝트는 React Native를 위한 UI 툴셋 및 컴포넌트 라이브러리입니다. Wix에서 개발한 오픈소스로, 디자인 시스템을 쉽게 구축할 수 있는 모듈식 구성 요소를 제공합니다. 커스터마이징 가능한 테마 시스템과 다양한 UI 컴포넌트를 통해 일관된 디자인 구현이 가능하며, React Native 0.73 이상을 지원합니다. 현재 새로운 React Native 아키텍처 지원을 준비 중입니다.
