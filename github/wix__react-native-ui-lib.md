React Native를 위한 UI 툴셋 및 컴포넌트 라이브러리입니다. Wix에서 개발한 이 프로젝트는 커스터마이징 가능한 디자인 시스템과 다양한 UI 컴포넌트를 제공합니다. 애플리케이션의 시각적 일관성과 개발 효율성을 높이기 위해 컬러, 타이포그래피, 스페이싱 등 디자인 시스템을 쉽게 적용할 수 있으며, React Native 0.73 이상 버전과 호환됩니다. 현재는 새로운 React Native 아키텍처(New Arc) 지원을 준비 중입니다.
