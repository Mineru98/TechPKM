Rembg는 이미지 배경을 제거하는 도구로 CLI, Python 라이브러리, HTTP 서버, Docker 컨테이너 형태로 사용 가능합니다. ONNX 런타임 기반의 다양한 사전 학습 모델을 지원하며 CPU/GPU 환경에서 동작합니다. 사용자는 단일 이미지 처리부터 배치 작업, 실시간 비디오 스트림 처리까지 유연하게 활용할 수 있습니다.
