이 프로젝트는 자연어 처리 경험이 없는 사용자를 위해 텍스트 파일을 입력하면 자동으로 전처리, 모델 학습/추론까지 수행하는 도구를 제공합니다. 주요 기능으로는 TF-IDF, Word2Vec, Doc2Vec 임베딩 생성 및 BERT 기반 문장 분류와 텍스트 요약이 포함됩니다. 사용자는 샘플 데이터셋으로 실습한 후 자체 데이터에 적용할 수 있으며, 초보자를 위한 6시간 분량의 유튜브 강의도 제공됩니다. Colab 노트북 링크를 통해 즉시 실행 가능한 환경을 지원합니다.
