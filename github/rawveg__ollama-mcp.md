ollama-mcp는 Ollama SDK를 MCP(Model Context Protocol) 도구로 노출시켜 로컬 LLM 모델과 Claude Desktop, Cline 등의 MCP 호환 애플리케이션 간 원활한 통합을 가능하게 하는 서버입니다. 타입세이프 TypeScript 구현, 14개의 포괄적인 툴, Ollama 클라우드 지원, 실시간 웹 검색/페치 기능을 특징으로 하며, 하이브리드 모드(로컬+클라우드)를 통해 유연한 AI 통합 경험을 제공합니다. GNU Affero GPL v3.0 라이선스로 배포됩니다.
