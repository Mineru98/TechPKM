LocalGPT는 사용자의 데이터를 로컬에서 완전히 비공개로 처리하는 문서 지능 플랫폼입니다. PDF, DOCX 등 다양한 문서에서 질문에 답변하고 요약하며 인사이트를 추출하는 기능을 제공합니다. 하이브리드 검색 엔진, 스마트 라우터, 문맥 풍부화, 문장 수준 컨텍스트 가지치기 등의 고급 기술을 활용해 정확한 결과를 생성하며, Ollama를 통해 다양한 오픈소스 모델을 지원합니다.  

PyTorch, Transformers, LanceDB 등의 핵심 라이브러리를 기반으로 구축되었으며, Docker 또는 직접 개발 환경에서 실행 가능합니다. 웹 UI를 통해 인덱스를 생성하고 채팅할 수 있으며, RESTful API를 통해 RAG 애플리케이션에 통합할 수 있습니다. CPU, GPU, HPU 등 다양한 하드웨어를 지원합니다.
