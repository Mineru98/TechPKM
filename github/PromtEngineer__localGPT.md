LocalGPT는 **완전히 프라이빗한 온프레미스 문서 인텔리전스 플랫폼**입니다. 사용자의 데이터를 로컬에서 처리하여 최첨단 AI로 질문에 답변하고 요약하며 인사이트를 도출할 수 있습니다. 전통적인 RAG 도구보다 진보된 하이브리드 검색 엔진(의미적 유사도, 키워드 매칭, Late Chunking), 스마트 라우터(RAG/직접 LLM 응답 선택), 문맥 풍부화(contextual enrichment), 문장 수준 컨텍스트 프루닝(pruning), 독립적 검증(verification) 기능을 제공합니다. 모듈식 경량 아키텍처로 필요 컴포넌트만 활성화 가능하며, 순수 파이썬 기반 최소 의존성 설계로 다양한 인프라에 쉽게 배포 및 유지보수할 수 있습니다.  

GPU/CPU/HPU/MPS 지원, Ollama 통합, RESTful API, 실시간 진행 상태 표시, 다양한 모델/임베딩 선택 기능 등을 제공합니다. PDF/DOCX/TXT 등 다중 포맷 문서 처리 가능하며, RAG 애플리케이션 구축을 위한 API를 포함합니다.
