KO-Platypus는 영어로 구성된 Open-Platypus 데이터셋을 DeepL Pro API로 번역한 후 수작업 검증을 거쳐 제작한 한국어 데이터셋(KOpen-Platypus)으로 llama-2-ko 모델을 파인튜닝한 오픈소스 LLM 프로젝트입니다. 다양한 벤치마크에서 기존 모델 대비 향상된 성능을 보이며, 특히 zero-shot 환경에서 우수한 결과를 제공합니다. 여러 크기의 모델(7B/13B) 버전을 제공하며, KoT-Platypus2 및 CoTy-platypus-ko 같은 체인 오브 사고(CoT) 기반 모델도 포함되어 있습니다. Hugging Face를 통해 모델과 데이터셋을 공개하여 한국어 NLP 연구 발전에 기여합니다.
