한국어 기반 오픈 플라티푸스 데이터셋을 활용해 **llama-2-ko** 모델을 파인튜닝한 한국어 특화 LLM 프로젝트입니다. 영문 Open-Platypus 데이터셋을 DeepL Pro로 번역하고 수작업 검증을 거쳐 **KOpen-Platypus** 데이터셋을 구축했으며, 이를 기반으로 다양한 한국어 모델(예: Ko-Platypus2-7B, Ko-Platypus2-13B)을 개발했습니다. KO-LLM 벤치마크에서 향상된 성능을 입증했으며, HuggingFace에서 모델 및 데이터셋을 공개하고 있습니다. 주요 기술로는 번역 데이터셋 구축, 파인튜닝, 한국어 LLM 평가가 포함됩니다.
