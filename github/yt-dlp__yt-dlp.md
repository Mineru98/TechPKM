yt-dlp는 수천 개의 사이트에서 고해상도 오디오/비디오 다운로드를 지원하는 명령줄 도구입니다. youtube-dl과 youtube-dlc를 기반으로 한 포크로, 다양한 기능과 개선 사항을 제공합니다. 주요 특징으로는 SponsorBlock 통합, 포맷 정렬 기능, 브라우저 쿠키 자동 추출, 부분 다운로드, 포맷 병합 기능 등이 있습니다. 

yt-dlp는 Python 3.10+ 환경에서 작동하며, Windows, macOS, Linux 등 다양한 플랫폼을 지원합니다. 사용자는 CLI 옵션이나 설정 파일을 통해 다운로드 동작을 세부적으로 제어할 수 있으며, 플러그인 시스템을 통해 확장성도 제공합니다. 

이 프로젝트는 활발히 유지보수되며, 공식 웹사이트와 위키에서 추가 정보를 확인할 수 있습니다. 주요 기능으로는:
- YouTube, BiliBili 등 다양한 사이트 지원
- 포맷 선택 및 병합 기능
- 자막/썸네일 다운로드
- 병렬 다운로드 및 외부 다운로더 지원
- 메타데이터 임베딩 기능

yt-dlp는 youtube-dl과 호환되는 옵션을 제공하며, `--compat-options` 플래그로 기존 동작을 유지할 수 있습니다. 프로젝트는 MIT 라이선스로 배포되며, 커뮤니티 기여를 환영합니다.
