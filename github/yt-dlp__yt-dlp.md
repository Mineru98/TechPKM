yt-dlp는 다중 사이트에서 오디오/비디오를 다운로드할 수 있는 강력한 명령줄 도구입니다. 다양한 사용자 정의 옵션과 고급 포맷 선택 기능을 제공하며, 1000개 이상의 사이트를 지원합니다. 주요 기능으로는 스폰서블록 통합, 다중 형식 병합, 다운로드 섹션 지정, 챕터 기반 분할 등이 있습니다. 사용자는 포맷 정렬, 필터 적용, 메타데이터 수정 등을 통해 다운로드 결과를 세부적으로 제어할 수 있습니다. 공식 빌드 및 야간/마스터 빌드 업데이트 채널을 지원하며, Python 3.10+ 환경이 필요합니다. 다양한 태그를 통해 분류 및 검색이 용이합니다.
