LabelImg는 Python과 Qt 기반의 그래픽 이미지 어노테이션 도구입니다. PASCAL VOC, YOLO, CreateML 포맷으로 XML 형식의 어노테이션을 저장하며, 주로 컴퓨터 비전 모델 학습을 위한 데이터셋 제작에 사용됩니다. 다양한 플랫폼에서 설치 가능하고, 단축키 지원 및 다중 이미지 처리 기능을 제공합니다. 현재는 Label Studio 커뮤니티에 통합되어 더 포괄적인 라벨링 솔루션을 제공하는 Label Studio 프로젝트를 추천합니다.
