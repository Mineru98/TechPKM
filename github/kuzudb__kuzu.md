Kuzu는 복잡한 분석 워크로드를 위해 설계된 임베디드 그래프 데이터베이스로, 대규모 데이터베이스 처리 속도와 확장성을 최적화했습니다. 속성 그래프 모델과 Cypher 쿼리 언어를 지원하며, 애플리케이션에 임베딩 가능한 서버리스 아키텍처, 네이티브 풀텍스트 검색 및 벡터 인덱스 기능을 제공합니다. 멀티코어 병렬 처리와 ACID 트랜잭션을 통해 고성능 그래프 데이터 관리를 가능하게 합니다.  

Kuzu는 0.11.3 버전에서 `algo`, `fts`, `json`, `vector` 확장 기능을 기본 포함하며, 이전 버전 사용자는 로컬 확장 서버를 통해 추가 기능을 설치할 수 있습니다. MIT 라이선스로 배포되는 오픈소스 프로젝트입니다.
