MuPDF.js는 MuPDF를 JavaScript 및 WebAssembly로 포팅한 라이브러리로, PDF 문서를 PNG, SVG, HTML 형식으로 빠르게 렌더링할 수 있습니다. 웹 브라우저와 서버를 포함한 모든 WebAssembly 지원 플랫폼에서 동작하며, 간결한 API로 유연한 PDF 처리가 가능합니다. 공식 패키지로의 마이그레이션을 권장하며, 레거시 시스템에서만 이전 버전(1.1.1)을 사용해야 합니다. 텍스트 추출 및 검색 기능도 제공합니다.
