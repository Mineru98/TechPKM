이 프로젝트는 MuPDF를 JavaScript와 WebAssembly로 포팅한 라이브러리로, PDF 문서를 PNG, SVG, HTML 형식으로 변환하는 기능을 제공합니다. 브라우저와 서버 환경에서 작동하며, 빠른 렌더링 속도와 간단한 API를 특징으로 합니다. 공식 유지보수 패키지로의 전환이 권장되나, 레거시 코드 호환을 위해 여전히 사용 가능합니다. 텍스트 추출 및 검색 기능도 포함되어 있습니다.
