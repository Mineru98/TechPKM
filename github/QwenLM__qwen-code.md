Qwen Code는 터미널에서 작동하는 오픈소스 AI 에이전트로, 대규모 코드베이스 분석과 반복 작업 자동화를 지원합니다. Qwen3-Coder 모델과 최적화되어 있으며, OpenAI 호환 API 및 OAuth 인증 방식을 통해 개발자들이 코드 이해, 리팩토링, 테스트 생성 등 다양한 작업에 활용할 수 있습니다. 터미널 중심의 인터페이스를 제공하며 VS Code, Zed, JetBrains IDE와의 통합도 지원합니다.
