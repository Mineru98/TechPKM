Qwen Code는 터미널에서 동작하는 오픈소스 AI 에이전트로, Qwen3-Coder 모델에 최적화되었습니다. 대규모 코드베이스 분석, 반복 작업 자동화, 개발 생산성 향상을 목적으로 설계되었으며, OpenAI 호환 API 또는 Qwen OAuth를 통한 인증 방식을 지원합니다. 터미널 기반 대화형 인터페이스, IDE 연동, 헤드리스 모드 등 다양한 사용 방식을 제공합니다.  

이 프로젝트는 Google Gemini CLI를 기반으로 파서 수준의 변경사항을 적용하여 Qwen-Coder 모델 지원을 강화했습니다. NPM, Homebrew를 통한 설치와 TypeScript SDK를 활용한 확장 기능을 제공하며, VS Code, Zed, JetBrains IDE와의 통합도 가능합니다.
