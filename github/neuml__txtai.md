txtai는 의미 검색, LLM 오케스트레이션 및 언어 모델 워크플로우를 위한 올인원 AI 프레임워크입니다. 임베딩 데이터베이스, 벡터 검색, 그래프 분석, 멀티모달 인덱싱을 기반으로 LLM 기반 애플리케이션의 지식 소스 역할을 하며, RAG(검색 증강 생성), 자동 에이전트, 워크플로우 구축 등을 지원합니다. Python 3.10+와 Hugging Face Transformers, Sentence Transformers, FastAPI를 사용하여 개발되었으며 Apache 2.0 라이선스로 오픈소스로 제공됩니다.
