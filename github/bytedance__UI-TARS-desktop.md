TARS는 멀티모달 AI 에이전트 스택으로, CLI 및 웹 UI를 통해 GUI 에이전트와 비전 기능을 터미널, 컴퓨터, 브라우저 및 제품에 통합하는 것을 목표로 합니다. 주요 프로젝트인 [Agent TARS]와 [UI-TARS Desktop]을 제공하며, 최첨단 멀티모달 LLM과 다양한 실제 도구(MCP)와의 원활한 통합을 통해 인간과 유사한 작업 완료 워크플로우를 구현합니다. UI-TARS Desktop은 로컬 및 원격 컴퓨터/브라우저 제어를 지원하는 네이티브 GUI 에이전트입니다.
