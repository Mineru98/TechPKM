TARS는 터미널, 컴퓨터, 브라우저 및 제품에 GUI 에이전트 및 비전 기능을 제공하는 멀티모달 AI 에이전트 스택입니다. 현재 CLI와 웹 UI를 제공하는 "Agent TARS"와 로컬/원격 컴퓨터 및 브라우저 운영자를 지원하는 데스크탑 애플리케이션 "UI-TARS Desktop" 두 가지 프로젝트를 포함합니다. 최첨단 멀티모달 LLM과 실시간 도구 통합을 통해 인간과 유사한 작업 완료 워크플로우를 제공하는 것이 목표입니다.  

UI-TARS Desktop은 UI-TARS 및 Seed-1.5-VL/1.6 시리즈 모델 기반의 네이티브 GUI 에이전트로, 화면 인식, 정확한 마우스/키보드 제어, 실시간 피드백 등의 기능을 지원합니다. 윈도우, macOS, 브라우저 등 크로스플랫폼을 지원하며 완전한 로컬 처리로 보안성을 유지합니다.
