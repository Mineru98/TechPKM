이 프로젝트는 Flutter, Supabase, Firebase를 연동하여 실시간 채팅 애플리케이션에 푸시 알림 기능을 구현하는 방법을 설명합니다. Supabase 데이터베이스에서 Firebase FCM 키를 관리하고, SQL 함수를 통해 푸시 알림을 전송하는 로직을 구현합니다. 채팅 메시지 전송 시 수신자에게 알림이 전달되도록 트리거 함수를 설정하며, 관련 YouTube 튜토리얼 링크가 제공됩니다. 주요 기술 스택으로 Flutter, Supabase, Firebase를 사용합니다.
