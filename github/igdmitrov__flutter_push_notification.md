이 프로젝트는 플러터(Flutter), 서페이스(Supabase), 파이어베이스(Firebase)를 통합하여 푸시 알림 기능을 구현한 샘플 애플리케이션입니다. Part I에서는 채팅 앱 구조를 다루고 Part II에서는 푸시 알림 전송 시스템을 설명하며, SQL 함수를 통해 FCM 토큰 관리와 실시간 알림을 구현합니다. 주요 기능은 사용자의 Firebase 키 업데이트, 푸시 메시지 전송, 새 메시지 트리거 시 알림 발송입니다.
