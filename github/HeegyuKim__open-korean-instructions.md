이 프로젝트는 한국어 언어모델 학습을 위한 공개 인스트럭션 데이터셋을 정리한 저장소입니다. Alpaca 번역, ShareGPT 번역, 지식 기반 대화 데이터, RLHF 데이터 등 다양한 출처와 형식의 한국어 데이터를 포함하며, 각 데이터셋의 특징, 크기, 링크를 체계적으로 분류하고 있습니다. 한국어 LLM 개발 시 학습 또는 평가 데이터로 활용할 수 있으며, 지속적인 커뮤니티 기여를 통해 확장되고 있습니다.
