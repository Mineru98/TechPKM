이 프로젝트는 무료 오픈소스 HTML 이메일 디자이너로, 드래그 앤 드롭 방식, HTML 코드 편집, 순수 텍스트 이메일 생성을 지원합니다. 모든 이메일 클라이언트에서 호환되는 템플릿을 빠르게 제작할 수 있으며, Angular 기반의 Angular 15 호환 라이브러리로 제공됩니다. NPM/Yarn 설치 가능하며, 다양한 클라우드 스토리지 연동과 UI 커스터마이징 기능을 포함합니다.  

> 핵심 기능:  
> - 진정한 오픈소스/무료  
> - 중간 변환 단계 없는 순수 HTML 기반  
> - 모든 이메일 서비스 제공업체와 호환 가능한 템플릿 저장  
> - 최소한의 인터페이스로 직관적인 디자인 가능
