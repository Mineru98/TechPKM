이 프로젝트는 Cursor AI 환경에서 Jira 이슈 설명과 댓글을 직접 읽을 수 있는 Model Context Protocol(MCP) 도구를 제공합니다. 개발 환경에서 Jira 데이터 연동을 통해 작업 효율성을 높이는 것이 핵심 목적이며, 자연어 질의로 특정 이슈 정보를 조회할 수 있습니다. Jira 인증 설정 및 Cursor AI와의 통합을 위한 단계별 가이드와 예제 상호작용을 포함합니다. 주요 기능으로는 이슈 설명/댓글 조회 명령어가 있으며, 문제 해결을 위한 기본적인 트러블슈팅 절차를 제공합니다.
