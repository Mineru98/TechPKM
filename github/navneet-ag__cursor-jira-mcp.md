이 프로젝트는 Cursor AI 코딩 환경에서 Jira 이슈 설명과 댓글을 직접 읽을 수 있도록 하는 Model Context Protocol(MCP) 도구입니다. Jira 인증 설정, 패키지 빌드, Cursor 도구 등록 과정을 통해 개발 환경 내에서 자연어 명령으로 Jira 이슈 정보를 조회할 수 있습니다. 주요 기능으로 특정 이슈 설명 조회 및 해당 이슈의 모든 댓글 조회 기능을 제공하며, Obsidian 관리용 메타데이터 문서입니다.
