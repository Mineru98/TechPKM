이 프로젝트는 네트워크 패킷 분석 도구인 Scapy를 Docker 컨테이너로 패키징한 것입니다. 매일 자동으로 빌드되며, 권한 있는 네트워크와 호스트 모드에서 실행되는 컨테이너를 제공합니다. 보안 테스트 및 네트워크 분석에 편리하게 사용할 수 있는 환경을 구축하는 것이 목적입니다. DockerHub에서 쉽게 가져와 실행할 수 있습니다.
