이 프로젝트는 텍스트를 음성으로 변환하는 BARK AI에 사용자 지정 오디오 샘플을 이용한 보이스 클론 기능을 추가한 오픈소스 도구입니다. 5-12초 분량의 음성 샘플로 개인화된 음성 생성이 가능하며, RVC 모델과 연동해 사용자 정의 음성 모델을 훈련시킬 수 있습니다. 영어, 독일어, 스페인어 등 다국어 지원과 음악/효과음 생성 기능도 포함되어 있습니다. 로컬에서 실행 가능한 데이터 프라이버시 보호형 AI 음성 복제 솔루션을 지향합니다.
