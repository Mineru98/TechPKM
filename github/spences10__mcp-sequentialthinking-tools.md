이 프로젝트는 복잡한 문제를 단계별로 분해하고 각 단계에 적합한 MCP 도구를 추천하는 서버입니다. 동적 사고 과정을 지원하며, 신뢰도 점수와 근거를 바탕으로 도구 추천을 제공하여 문제 해결을 체계적으로 안내합니다. LLM 분석과 도구 추천을 결합하여 사고 흐름의 유연성과 적응성을 확보했습니다.

핵심 기능으로는 분기 가능한 사고 프로세스, 도구 추천 우선순위 시스템, 메모리 관리(히스토리 제한/수동 정리), 단계별 진행 상태 추적 등이 있습니다. MCP 생태계와 연동되어 환경별 도구 추천이 가능하며, 노드 기반 개발과 WSL/데스크톱 환경 호환성을 갖춘 오픈소스 프로젝트입니다.
