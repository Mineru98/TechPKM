이 프로젝트는 TypeScript 7의 실험적 네이티브 컴파일러 구현체로, 현재 기능 완성도가 진행 중인 미리보기 버전입니다. 기존 TypeScript 5.9와의 호환성을 목표로 하며, CLI 도구 및 VS Code 확장을 통해 기본적인 빌드/타입 검사 기능을 제공합니다. 장기적으로는 공식 TypeScript 저장소로 통합될 예정이며, 현재 주요 기능 중 일부는 개발 중이거나 프로토타입 단계입니다.
