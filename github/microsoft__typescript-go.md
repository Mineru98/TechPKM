이 프로젝트는 TypeScript의 네이티브 포트 구현체로, 기존 TypeScript 5.9과 유사한 기능을 제공하는 미리보기 버전입니다. `@typescript/native-preview` 패키지를 통해 사용할 수 있으며, 타입 체킹, JSX 지원, 선언 파일 생성 등의 기능이 개발 중입니다. 장기적으로는 공식 TypeScript 저장소로 통합될 예정입니다.  

주요 기능으로는 `tsgo` 명령어로 컴파일 가능하며, VS Code 확장을 통한 통합 개발이 지원됩니다. 현재 일부 기능은 완성되지 않았으며, 버그 리포트 시 기능 상태를 고려해야 합니다. TypeScript 5.9 대비 변경 사항은 CHANGES.md에서 확인할 수 있습니다.
