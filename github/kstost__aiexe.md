이 프로젝트는 원래 `kstost/aiexe`로 시작되었으나 현재 `kstost/aiexeauto-gui`로 이전되었습니다. 새 리포지토리는 Python 기반의 GUI 자동화 도구를 개발하며, 기존 프로젝트는 더 이상 유지되지 않습니다. 최신 기능과 업데이트는 새로운 리포지토리에서 확인할 수 있습니다. 자동화 스크립트 및 GUI 관련 기술 스택이 핵심입니다.
