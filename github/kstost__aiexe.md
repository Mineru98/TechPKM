이 프로젝트는 원래 `kstost/aiexe` 리포지토리에서 운영되었으나, 현재 `kstost/aiexeauto-gui`로 이전되었습니다. 최신 업데이트와 개발 현황은 새 리포지토리에서 확인할 수 있으며, 기존 리포지토리는 더 이상 유지되지 않습니다. 새 리포지토리는 자동 실행 기능을 갖춘 GUI 도구로 발전 중이며, 모든 작업이 이전되었습니다.
