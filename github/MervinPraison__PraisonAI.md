PraisonAI는 프로덕션 환경에서 사용 가능한 멀티 에이전트 프레임워크로, 간단한 작업부터 복잡한 과제까지 자동화할 수 있는 AI 에이전트를 생성하기 위한 낮은 코드 솔루션을 제공합니다. 이 프로젝트는 단순성, 사용자 정의, 효과적인 인간-에이전트 협업을 강조하며, 다양한 사용 사례와 통합 옵션을 지원합니다. 100개 이상의 LLM 제공업체를 지원하며, Python과 JavaScript SDK를 통해 접근할 수 있습니다. 주요 기능으로는 자체 반영 기능, 계획 모드, RAG(검색 증강 생성), 다양한 워크플로우 패턴(순차적, 계층적, 병렬 실행), 지속적 메모리 관리, 100개 이상의 사용자 정의 도구 통합 등이 포함됩니다. CLI와 프로그래밍 방식 모두 지원하며, YAML 기반 구성 파일을 통해 복잡한 멀티 에이전트 시스템을 쉽게 정의할 수 있습니다.
