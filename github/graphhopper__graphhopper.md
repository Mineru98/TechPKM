GraphHopper는 Apache License 2.0으로 배포되는 빠르고 메모리 효율적인 라우팅 엔진입니다. Java 라이브러리 또는 독립형 웹 서버로 사용할 수 있으며, 여러 지점 간의 경로 거리, 시간, 이동 경로 및 도로 속성 정보를 계산합니다. OpenStreetMap 및 GTFS 데이터를 기본으로 지원하며, 맵 매칭, 등시권 계산, 모바일 내비게이션 등의 기능을 제공합니다. 다양한 사용자 정의 및 확장 가능한 프로파일과 효율적인 데이터 구조를 통해 유연한 라우팅 솔루션을 구현합니다.
