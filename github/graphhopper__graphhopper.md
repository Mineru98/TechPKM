GraphHopper는 Apache License 2.0으로 배포되는 빠르고 메모리 효율적인 오픈소스 라우팅 엔진입니다. Java 라이브러리 또는 독립형 웹 서버로 사용할 수 있으며, A에서 B까지의 경로 계산, 등시권 분석, 지도 매칭, 모바일 내비게이션 등 다양한 기능을 제공합니다. OpenStreetMap 및 GTFS 데이터를 기본으로 지원하며, 커스텀 데이터 소스 통합이 가능합니다. 차량, 자전거, 보행자 등 다양한 이동 수단에 대한 사전 정의된 프로필과 실시간 교통 데이터 처리 기능을 갖추고 있습니다.
