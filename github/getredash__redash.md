Redash는 모든 사용자가 대규모 및 소규모 데이터를 활용할 수 있도록 설계된 오픈소스 비즈니스 인텔리전스 도구입니다. SQL 사용자는 브라우저에서 데이터를 탐색, 쿼리, 시각화 및 공유할 수 있으며, 이를 통해 조직 내 모든 구성원이 데이터에 기반한 의사 결정을 할 수 있습니다. 전 세계 수천 개 조직의 수백만 사용자가 매일 Redash를 통해 인사이트를 개발하고 있습니다. 주요 기능으로는 브라우저 기반 작업, 직관적인 사용성, 다양한 데이터 소스 지원, 자동화된 데이터 새로 고침, 경고 시스템, REST API 등이 있습니다. 35개 이상의 SQL/NoSQL 데이터 소스를 기본 지원하며 확장 가능한 아키텍처를 갖추고 있습니다.
