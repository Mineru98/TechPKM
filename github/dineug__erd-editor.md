이 프로젝트는 엔티티-관계 다이어그램(ERD)을 생성하고 편집할 수 있는 오픈소스 도구입니다. 웹 애플리케이션 및 VSCode/IntelliJ 확장 형태로 제공되며, PWA 지원, 실시간 협업, 브라우저 자동 저장, 탭 간 동기화 등의 기능을 제공합니다. 암호화 기반의 로컬 퍼스트 설계로 오프라인 환경에서도 작업 가능합니다.  

(요약: 오픈소스 ERD 편집 도구, 실시간 협업/오프라인 지원, 다중 플랫폼 확장 제공)
