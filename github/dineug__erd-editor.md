erd-editor는 웹 기반의 Entity-Relationship Diagram(ERD) 편집 도구로, 오프라인 지원이 가능한 PWA 기능과 실시간 협업, 종단간 암호화를 제공합니다. 브라우저 자동 저장 기능과 탭 간 실시간 동기화 기능을 갖추고 있으며, 웹 앱, VSCode 확장, IntelliJ 플러그인으로 제공됩니다. 문서 가이드와 API를 통해 사용자 및 개발자 친화적으로 설계되었습니다.
