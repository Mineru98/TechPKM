AutoShape는 ICCV 2021 논문에서 소개된 단안 카메라를 이용한 실시간 형태 인식 3D 객체 검출 프로젝트입니다. KITTI 데이터셋에 자동 레이블링된 자동차 3D 모델 데이터를 제공하며, PaddlePaddle 및 PyTorch 기반으로 구현되었습니다. 주요 특징은 3000개 이상의 3D 키포인트를 활용한 차량 형태 인식 및 효율적인 모델 학습입니다. 자율주행 시스템의 객체 인식에 적용 가능한 기술입니다.
