Mobile ALOHA는 저비용 전신 원격 조작을 통해 양팔 모바일 매니퓰레이션 학습을 구현하는 프로젝트입니다. 이 코드베이스는 ALOHA 저장소를 포크한 것으로, Mobile ALOHA 하드웨어용 원격 조작 및 데이터 수집 기능을 포함합니다. ROS 기반 제어 시스템과 카메라 통합을 통해 4개의 로봇 팔과 3개의 카메라를 동시에 관리하며, 강화 학습 훈련 및 데이터 수집에 최적화되어 있습니다. 하드웨어 설정 가이드와 소프트웨어 설치 절차를 상세히 제공합니다.
