Deep Java Library(DJL)는 자바 개발자를 위한 오픈소스 딥러닝 프레임워크로, 엔진-애그노스틱 설계되어 TensorFlow, PyTorch 등 다양한 백엔드와 호환됩니다. 기존 자바 개발 경험을 활용해 쉽게 학습하고 모델을 배포할 수 있으며, 자동 CPU/GPU 선택으로 최적화된 성능을 제공합니다. 코드 예제를 통해 추론 및 훈련 과정을 직관적으로 구현할 수 있는 것이 특징입니다.
