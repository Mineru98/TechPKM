Autodistill GPT-4V 모듈은 GPT-4V 멀티모달 모델을 활용해 이미지를 자연어로 분류하고 라벨을 생성하는 프로젝트입니다. OpenAI의 GPT-4V API를 기반으로 한 이 모듈은 이미지 분석을 자동화하며, 사용자 정의 온톨로지를 통해 클래스별 캡션을 매핑할 수 있습니다. 이미지당 1회의 API 호출이 발생하며, 유료 과금 정책에 따라 비용이 청구됩니다. Autodistill 생태계와 통합되어 컴퓨터 비전 작업 흐름을 간소화합니다.
