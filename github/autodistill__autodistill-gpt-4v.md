이 프로젝트는 [Autodistill](https://github.com/autodistill/autodistill) 프레임워크와 연동하여 GPT-4V 모델을 활용한 이미지 분류 기능을 제공합니다. OpenAI의 GPT-4V API를 기반으로 하며, 자연어 프롬프트를 통해 이미지를 분석하고 사전 정의된 온톨로지에 따라 분류 라벨을 생성합니다. 사용자는 API 키를 통해 직접 이미지를 라벨링할 수 있으며, OpenAI의 요금제에 따라 API 호출 비용이 발생합니다.  

> 참고: `pip install autodistill-gpt-4v`로 설치 가능하며, Python 3 환경에서 동작합니다.
