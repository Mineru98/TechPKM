DeLorenzo Claude Skills Marketplace는 AI 개발, 워크플로우 자동화 및 전문 도구링을 위한 Claude Code 플러그인 컬렉션입니다. 이 프로젝트는 AI 에이전트 프레임워크, 이미지 생성 도구, 애자일 개발 워크플로우, 디버깅 솔루션 등 다양한 도구를 제공하며, 개발 생태계와의 통합을 중점으로 설계되었습니다. 사용자는 GitHub 기반 마켓플레이스를 통해 플러그인을 쉽게 설치 및 관리할 수 있으며, 기술 스택으로는 Python, Rust, Docker 등이 활용됩니다.  

이 프로젝트는 AI 기반 문제 해결과 효율적인 개발 환경 구축을 목표로 하며, 오픈소스 라이선스로 제공됩니다. 주요 특징으로는 다양한 기술 카테고리별 플러그인 분류, 플러그인 생성 가이드라인, 그리고 iMi, Bloodbank, mise 등과의 생태계 통합이 있습니다. 사용자는 필요에 따라 AI 도구, 디버깅 솔루션, 또는 프레임워크 특화 플러그인을 선택하여 활용할 수 있습니다.
