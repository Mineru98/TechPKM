이 프로젝트는 스트리밍 서비스(넷플릭스, 유튜브 등)에서 다운로드한 XML 또는 VTT 형식의 자막을 SRT 형식으로 변환하는 도구입니다. 웹 브라우저에서 직접 변환하거나 Python 스크립트를 사용해 대량의 자막을 처리할 수 있으며, 타임스탬프 조정 기능도 제공합니다. 주로 동영상 재생 소프트웨어에서 호환되지 않는 자막 형식을 변환하는 데 목적을 두고 있습니다.
