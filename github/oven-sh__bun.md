Bun은 JavaScript와 TypeScript 애플리케이션을 위한 통합 도구 세트로, Node.js 대체 가능한 빠른 런타임을 제공합니다. Zig로 작성되었으며, JavaScriptCore 기반으로 동작하여 시작 시간과 메모리 사용량을 최적화했습니다. 번들러, 패키지 관리자, 테스트 러너 등의 기능을 단일 실행 파일(`bun`)로 제공하며, 기존 Node.js 프로젝트와의 호환성을 유지합니다.  

Bun은 `bun run`, `bun install`, `bun test`, `bunx` 등의 명령어를 통해 개발 워크플로를 간소화하며, Linux, macOS, Windows 환경에서 설치 및 사용이 가능합니다. 또한, HTTP 서버, WebSocket, 데이터베이스 연동, 스트리밍 등의 기능을 내장 API로 지원합니다.
