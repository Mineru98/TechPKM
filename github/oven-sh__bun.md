Bun은 JavaScript와 TypeScript 애플리케이션을 위한 올인원 툴킷입니다. Node.js를 대체할 수 있는 빠른 런타임으로, Zig로 작성되고 JavaScriptCore를 기반으로 하여 시작 시간과 메모리 사용량을 크게 줄였습니다. 단일 실행 파일(`bun`)로 제공되며, 패키지 관리, 테스트 실행, 스크립트 실행 등 다양한 기능을 통합했습니다. TypeScript, JSX, Web API를 기본 지원하며, 기존 Node.js 프로젝트와의 호환성을 유지하면서 성능을 개선했습니다. Linux, macOS, Windows를 지원하며, Docker, npm, Homebrew 등을 통한 설치가 가능합니다.
