PhoneAgent는 OpenAI 모델을 활용하여 아이폰에서 여러 앱을 넘나들며 인간과 유사한 작업을 수행하는 자동화 에이전트입니다. iOS의 접근성 트리를 분석하여 앱 제어, 텍스트 입력, 앱 전환 등의 기능을 제공하며, 음성 명령과 후속 작업 처리를 지원합니다. 이 프로젝트는 OpenAI 해커톤에서 개발되었으며, 실험적 목적으로 제한된 환경에서 사용하기 위한 개인 프로젝트입니다. Xcode UI 테스트 프레임워크를 기반으로 하며, 탈옥 없이 작동합니다. 주요 기능은 토크 모드 및 음성 명령 지원, API 키 보안 저장 등입니다.
