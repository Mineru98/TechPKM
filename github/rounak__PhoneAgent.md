PhoneAgent는 OpenAI 모델을 활용해 아이폰에서 인간과 유사한 작업을 수행하는 자동화 솔루션입니다. 접근성 트리 분석을 통해 앱 간 상호작용(탭, 스와이프, 텍스트 입력 등)을 구현하며, 음성 명령과 API 키 보안 저장 기능을 지원합니다. 실험적인 프로젝트로, UI 테스트 환경을 통해 시스템 전체 제어가 가능합니다. 주요 활용 사례는 사진 전송, 앱 다운로드, 메시지 발송 등 일상적인 아이폰 작업입니다.
