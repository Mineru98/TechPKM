CTransformers는 GGML 라이브러리를 기반으로 C/C++로 구현된 트랜스포머 모델에 대한 Python 바인딩 프로젝트입니다. LLaMA, GPT-2, Falcon 등 다양한 오픈소스 언어 모델을 로컬에서 효율적으로 실행할 수 있도록 지원하며, GPU 가속 및 Hugging Face Transformers, LangChain과의 통합 기능을 제공합니다. 특히 GPTQ 양자화 모델과 Metal/CUDA/ROCm 백엔드를 통해 하드웨어 최적화를 가능하게 합니다. MIT 라이선스로 배포되는 오픈소스 프로젝트입니다.
