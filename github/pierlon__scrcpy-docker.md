이 프로젝트는 scrcpy를 도커 컨테이너로 실행할 수 있도록 구성된 이미지입니다. AMD, Intel, Nvidia 그래픽 하드웨어별로 최적화된 이미지를 제공하며, USB를 통해 연결된 안드로이드 기기를 PC에서 미러링하거나 제어할 수 있게 합니다. 호스트 시스템의 X 서버 및 USB 장치 접근을 위한 설정이 포함되어 있습니다.  

(요약: scrcpy의 도커 이미지 제공, 그래픽 하드웨어별 최적화, USB 기반 안드로이드 기기 미러링/제어 지원)
