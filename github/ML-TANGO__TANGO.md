TANGO(**T**arget **A**ware **N**o-code neural network **G**eneration and **O**peration framework)는 코드 작성 없이 타겟 장치에 최적화된 딥러닝 모델을 자동 생성·배포하는 통합 머신러닝 프레임워크입니다. 주요 특징으로는 라벨링 도구, 베이스 모델 선택기, NAS/HPO 기반 자동 신경망 생성, 다양한 타겟 환경(클라우드, K8S, 온디바이스) 지원 등이 있으며, MSA와 컨테이너 기술을 기반으로 서비스 간 REST API 연동을 구현합니다. 사용자는 데이터셋과 타겟 장치 정보를 제공하면 TANGO가 모델 생성부터 배포까지 전 과정을 자동 처리합니다.
