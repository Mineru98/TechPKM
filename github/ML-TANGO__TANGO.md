TANGO(Target Aware No-code neural network Generation and Operation)는 비전문가도 코드 작성 없이 신경망 모델을 자동 생성·배포할 수 있는 통합 머신러닝 프레임워크입니다. 라벨링 도구, 베이스 모델 선택, AutoNN(자동 신경망 생성), 다양한 타겟 장치 지원을 통해 데이터 준비부터 배포까지 전 과정을 자동화하며, MSA와 컨테이너 기술로 유연성을 확보했습니다. 분류, 객체 감지, 인스턴스 세분화 작업을 지원하는 것이 핵심 특징입니다.
