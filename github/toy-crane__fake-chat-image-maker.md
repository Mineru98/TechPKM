이 프로젝트는 Next.js를 기반으로 채팅 인터페이스를 모방한 이미지를 생성하는 웹 애플리케이션입니다. 사용자는 실제 채팅 UI와 유사한 이미지를 생성하여 프로토타입이나 시각적 자료로 활용할 수 있습니다. Vercel 배포를 지원하며 Geist 폰트를 활용한 반응형 디자인이 특징입니다. 로컬 개발 서버를 통해 빠르게 테스트하고 편집할 수 있는 환경을 제공합니다.
