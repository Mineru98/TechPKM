DeepWiki는 GitHub, GitLab, BitBucket 저장소를 분석하여 자동으로 아름다운 인터랙티브 위키를 생성하는 오픈소스 프로젝트입니다. AI 기반의 코드 분석, 문서 생성, Mermaid 다이어그램 시각화 기능을 제공하며, RAG 기술을 활용한 "Ask" 기능과 "DeepResearch" 다단계 연구 기능을 통해 저장소와 대화형으로 상호작용할 수 있습니다. Google Gemini, OpenAI, OpenRouter, Azure OpenAI, Ollama 등 다양한 LLM 제공업체를 지원하며, Docker와 수동 설치 옵션 모두를 제공합니다.  

핵심 기능:  
- 코드 구조 분석 및 임베딩 생성  
- AI 기반 문서 생성 및 다이어그램 시각화  
- RAG 기반 저장소 Q&A 기능  
- 멀티턴 심층 연구(DeepResearch)  
- Google/OpenAI 호환 임베딩 모델 선택  
- 프라이빗 저장소 지원 및 토큰 인증  

사용 목적: 개발자는 저장소 URL을 입력하면 AI가 자동으로 문서를 생성하고, 대화형 Q&A로 복잡한 주제를 조사할 수 있습니다.
