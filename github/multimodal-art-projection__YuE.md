**YuE(乐)**는 가사로 풀송을 생성하는 오픈소스 파운데이션 모델 시리즈입니다. 영어, 일본어, 한국어, 중국어 등 다양한 언어와 장르를 지원하며, 보컬 및 반주 트랙을 포함한 몇 분 길이의 완전한 곡을 생성할 수 있습니다. Apache 2.0 라이선스로 자유롭게 사용 가능하며, 아티스트로 하여금 생성 결과를 상업적 작업에 활용하도록 권장합니다. 주요 기능으로는 음악 스타일 변환(ICL 모드), 증분 생성, LoRA 파인튜닝 등을 지원하며, Gradio 기반 인터페이스와 다양한 GPU 환경에서 실행 가능합니다.  

> 핵심 특징:  
> - 가사 기반 음악 생성(lyrics2song)  
> - 멀티트랙(듀얼/싱글) 음악 컨텍스트 학습  
> - Apache 2.0 오픈소스 라이선스  
> - 7B/1B 파라미터 모델 시리즈  
> - 저사양 GPU 최적화(8GB VRAM 지원)
