LLM은 OpenAI, Anthropic의 Claude, Google의 Gemini, Meta의 Llama 등 수십 개의 대규모 언어 모델(Large Language Model)과 상호작용하기 위한 명령줄 인터페이스(CLI) 도구 및 Python 라이브러리입니다. 원격 API를 통한 모델 사용과 로컬 머신에서 실행 가능한 모델 모두를 지원하며, SQLite에 프롬프트 및 응답 로깅, 임베딩 생성, 텍스트와 이미지에서 구조화된 콘텐츠 추출, 도구 실행 기능 등의 기능을 제공합니다. 다양한 설치 방법(pip, Homebrew, pipx, uv)을 지원하며, 모델 플러그인을 통해 확장 가능한 아키텍처를 갖추고 있습니다.
