LLM은 OpenAI, Anthropic의 Claude, Google의 Gemini, Meta의 Llama 등 다양한 대규모 언어 모델(Large Language Models)과 상호작용할 수 있는 CLI 도구 및 Python 라이브러리입니다. 원격 API 호출과 로컬 설치 모델 실행을 모두 지원하며, 프롬프트 실행, SQLite 로깅, 임베딩 생성, 구조화된 데이터 추출, 도구 통합 등의 기능을 제공합니다. 사용자는 다양한 설치 방법(Homebrew, pip, pipx 등)을 통해 쉽게 설치하고 CLI 명령어로 간편하게 모델을 활용할 수 있습니다.
