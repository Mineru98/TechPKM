이 프로젝트는 Firecrawl의 웹 스크래핑 기능과 Anthropic Claude의 고급 AI 추론 능력을 결합한 연구 도구입니다. 웹 콘텐츠 검색, 분석, 이해를 위해 설계된 이 도구는 실시간 스크래핑, AI의 사고 과정 표시, 자동 인용 생성 등의 기능을 제공합니다. Next.js, TypeScript, Tailwind CSS로 구현된 이 애플리케이션은 연구자들이 웹 기반 정보를 보다 효과적으로 활용할 수 있도록 지원합니다. Split View 인터페이스를 통해 채팅과 검색 결과를 동시에 확인할 수 있습니다.
