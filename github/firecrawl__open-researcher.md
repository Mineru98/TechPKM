Open Researcher는 Firecrawl의 웹 스크래핑 기술과 Anthropic의 AI 추론 능력을 결합한 연구 도구입니다. 웹 콘텐츠를 검색, 분석, 이해하는 기능으로 AI 기반 검색, 실시간 스크래핑, 사고 과정 표시, 스마트 인용 생성, 분할 화면 인터페이스를 제공합니다. Node.js 18+와 API 키(ANTHROPIC_API_KEY 필수)가 필요하며, Next.js 15, TypeScript, Tailwind CSS 기술로 구축되었습니다. 연구팀은 웹 분석 결과를 기반으로 후속 질문을 하거나 출처를 확인할 수 있습니다.
