Polyglot은 영어 외 언어의 성능을 강화한 다국어 대규모 언어 모델 프로젝트입니다. 기존 다국어 모델의 비영어 성능 한계를 극복하고, 한국어(Polyglot-Ko)와 일본어(Japanese StableLM) 특화 모델을 개발하며, 1.3B~12.8B 파라미터 규모의 다양한 버전을 제공합니다. 한국어 모델은 TUNiB의 1.2TB 데이터셋으로 학습되었으며, 일본어 모델은 StabilityAI와 협력으로 구축되었습니다. 연구 및 상업적 활용을 목표로 하는 오픈소스 프로젝트입니다.
