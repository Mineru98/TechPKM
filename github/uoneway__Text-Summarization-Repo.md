# 텍스트 요약 분야 자료 저장소

텍스트 요약(Text Summarization) 분야의 핵심 이론과 실무 자료를 체계적으로 정리한 저장소입니다. 요약 기반 논문, 데이터셋, 코드, 프리트레인 모델 등을 종합적으로 제공하며, NLP 초보자부터 연구자까지 폭넓게 활용할 수 있습니다. 주요 연구 주제(추출/생성 요약, 장문 요약, 지식 기반 생성 등)를 체계적으로 분류하고, 한국어 및 영문 자료를 병행하여 한국어 요약 기술 발전에 중점을 둔 특징을 가집니다. BERT, BART, PEGASUS 등 최신 모델 적용 사례와 함께 실용적인 코드 예시 및 학습 자료를 포함합니다. 

요약 기술 이해를 위한 핵심 개념(정의, 평가 지표, 주요 논문) 설명과 함께 실제 구현에 필요한 리소스(코드/데이터/모델)를 정리하여 연구 및 개발 과정을 체계적으로 지원합니다.
