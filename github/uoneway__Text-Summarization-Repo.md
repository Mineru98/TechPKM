텍스트 요약 분야의 이론과 실습을 위한 종합 자료 저장소입니다. 텍스트 요약의 정의, 주요 논문, 코드, 데이터셋, 프리트레인 모델을 체계적으로 정리하여 NLP 초보자부터 전문가까지 활용할 수 있도록 구성되었습니다. 추출 요약(Extractive)과 생성 요약(Abstractive)의 기본 개념부터 최신 연구 동향(예: PEGASUS, BART)까지 폭넓은 정보를 제공하며, 한국어 데이터셋 및 모델 사용법에 대한 실용적인 가이드를 포함합니다. 요약 평가 방법, 컨트롤 가능한 생성, 지식 증강 기법 등 핵심 주제를 다룹니다.
