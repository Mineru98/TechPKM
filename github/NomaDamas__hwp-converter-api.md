본 프로젝트는 HWP 파일을 텍스트로 변환하는 API 서버로, hwplib 라이브러리를 기반으로 구축되었습니다. Javalin 프레임워크와 Kotlin을 사용하여 개발되었으며, Docker를 통해 쉽게 배포할 수 있습니다. 표 포함 전체 내용 또는 본문만 선택하여 변환할 수 있는 옵션을 제공합니다. (단, hwpx 파일과 표 형식 보존 기능은 지원하지 않음)
