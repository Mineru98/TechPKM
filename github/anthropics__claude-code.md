Claude Code는 터미널에서 동작하는 에이전트 기반 코딩 도구로, 코드베이스를 이해하고 자연어 명령을 통해 루틴 작업 실행, 복잡한 코드 설명, Git 워크플로우 처리 등을 지원하는 도구입니다. IDE나 GitHub에서도 사용할 수 있으며, 플러그인 시스템을 통해 기능을 확장할 수 있습니다. 공식 문서에서 자세한 설치 방법과 사용법을 확인할 수 있습니다.
