Claude Code는 터미널, IDE 또는 GitHub에서 자연어 명령으로 코드베이스를 이해하고 작업 효율성을 높이는 에이전트 기반 코딩 도구입니다. 코드 설명, Git 워크플로우 처리, 반복 작업 실행 등 다양한 기능을 제공하며 플러그인 시스템을 통해 확장 가능합니다. 사용자가 자연어로 명령하면 AI 에이전트가 코드 변경을 제안하거나 실행하며, npm, Homebrew, PowerShell 등 다양한 설치 방법을 지원합니다.  

> 주요 특징: 코드베이스 분석, 자연어 명령 실행, Git 통합, 플러그인 확장성, 크로스 플랫폼 지원
