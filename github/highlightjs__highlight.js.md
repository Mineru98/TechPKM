Highlight.js는 JavaScript로 작성된 문법 강조 라이브러리입니다. 브라우저와 서버에서 모두 작동하며, 다양한 마크업과 호환되고 프레임워크에 의존하지 않으며 자동 언어 감지 기능을 제공합니다. 180개 이상의 언어를 지원하며, 사용자 정의가 가능하고 웹 워커, Vue.js 통합 등 다양한 환경에서 활용할 수 있습니다. CDN, NPM, 직접 다운로드 등의 방식으로 쉽게 도입할 수 있습니다.
