Highlight.js는 JavaScript로 작성된 코드 문법 강조 라이브러리입니다. 브라우저와 서버에서 모두 작동하며, 모든 마크업과 호환되고 프레임워크 의존성이 없으며 자동으로 언어를 감지할 수 있습니다. 180개 이상의 언어를 기본 지원하며, CDN 또는 NPM 설치 방식으로 쉽게 사용할 수 있습니다. `<pre><code>` 태그를 기반으로 코드 블록을 강조하거나, 특정 언어를 수동으로 지정할 수 있으며, 웹 워커나 Vue.js 통합 등 다양한 사용 시나리오를 지원합니다.
