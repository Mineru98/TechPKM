SongGeneration Studio는 Tencent AI Lab의 LeVo 모델을 활용하여 가창, 가사, 악기 트랙이 포함된 완전한 AI 생성 음악을 제작하는 도구입니다. 사용자는 가사 입력과 장르/분위기/보컬 스타일/템포 등의 설정을 통해 맞춤형 곡을 생성할 수 있으며, 레퍼런스 오디오를 업로드하거나 개별 트랙(전체 믹스/보컬만/악기만)을 분리하는 기능도 제공합니다. Pinokio 플랫폼을 통한 간편한 설치와 Web UI 기반의 직관적인 인터페이스로 음악 제작이 가능합니다.
