Omnisearch for Obsidian은 Obsidian 환경에서 노트, 문서, PDF 및 이미지를 즉시 검색하는 고성능 검색 엔진 플러그인입니다. BM25 알고리즘 기반의 스마트 가중치 시스템으로 검색 결과의 관련성을 실시간으로 평가하며, 키보드 중심의 효율적인 검색 워크플로우를 제공합니다. 로컬 HTTP 서버 지원, 오타 허용, 필터링 및 링크 삽입 등의 기능으로 확장 가능하며, 텍스트 추출기와 연동해 다양한 파일 형식을 인덱싱합니다. GPL-3 라이선스로 배포되는 오픈소스 프로젝트입니다.
