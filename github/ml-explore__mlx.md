MLX는 Apple Silicon에서 머신러닝 작업을 위한 배열 프레임워크로, NumPy와 유사한 Python API를 제공하며 C++/C/Swift API도 지원합니다. 효율적인 모델 학습 및 배포를 목표로 하며, 자동 미분/벡터화, 지연 계산, 동적 그래프 생성, 멀티디바이스 연산, 통합 메모리 모델 등의 특징을 갖추고 있습니다. 연구자들을 위해 설계되었으며 PyTorch와 유사한 `mlx.nn` 및 `mlx.optimizers`를 통해 복잡한 모델 구축을 간소화합니다. Transformer, LLaMA, Stable Diffusion 등의 예제가 제공됩니다.
