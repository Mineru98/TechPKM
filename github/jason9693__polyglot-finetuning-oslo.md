이 프로젝트는 OSLO 라이브러리의 TensorParallel 기능을 활용하여 EleutherAI Polyglot-ko 1.3B 모델을 KorQuADv1 데이터셋으로 분산 학습하는 방법을 보여주는 예제입니다. 싱글노드와 멀티노드 설정에서의 학습 방법을 비교하며, 대규모 모델 병렬 처리 구현에 초점을 맞추고 있습니다. OSLO 설치 방법과 함께 다양한 GPU 구성에서의 실행 명령어를 제공합니다. 분산 학습 환경 구성에 필요한 핵심 파라미터 설명도 포함되어 있습니다.
