Webpack 5 기반의 프론트엔드 보일러플레이트 프로젝트로, TypeScript와 React를 중심으로 다양한 개발 도구를 통합 지원합니다. Babel, SASS/LESS, CSS 모듈, SVG 컴포넌트 변환, ESLint/Prettier, Husky/Git 훅 등 현대적인 웹 개발 워크플로우를 구현합니다. React Refresh와 DevServer 프록시 기능을 통해 효율적인 개발 환경을 제공합니다.
