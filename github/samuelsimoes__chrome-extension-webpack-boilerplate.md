이 프로젝트는 Webpack을 기반으로 한 Chrome 확장 개발을 위한 기본 보일러플레이트입니다. 모던한 JavaScript 코드와 CSS 관리를 용이하게 하며, 코드 변경 시 자동 브라우저 리로딩 기능을 제공합니다. 팝업, 옵션 페이지, 백그라운드 페이지 등 확장 구성 요소를 쉽게 커스터마이즈할 수 있도록 설계되었습니다. ES6 모듈 시스템을 활용한 코드 조직과 개발 효율성을 극대화하는 워크플로우를 지원합니다.
