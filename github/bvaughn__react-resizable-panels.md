`react-resizable-panels`는 리액트 애플리케이션에서 수평/수직으로 리사이즈 가능한 패널 그룹 레이아웃을 구현하기 위한 UI 컴포넌트 라이브러리입니다. `Group`, `Panel`, `Separator` 컴포넌트를 통해 유연한 레이아웃 구성이 가능하며, TypeScript 지원, 테스트 ID 속성, 키보드 접근성 등을 제공합니다. 패널 크기 제한(min/max), 기본 레이아웃 설정, 레이아웃 변경 이벤트 처리 등 다양한 기능을 포함합니다.  

**핵심 특징**  
- 수평/수직 리사이즈 지원  
- 패널 크기 제한 및 접을 수 있는 기능  
- 레이아웃 상태 저장 및 복원  
- 키보드 접근성 준수  
- TypeScript 타입 정의 제공
