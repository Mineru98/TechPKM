`react-resizable-panels`는 리액트 애플리케이션에서 수평/수직 방향으로 크기 조절이 가능한 패널 그룹을 구현하기 위한 컴포넌트 라이브러리입니다. 그룹(Group), 패널(Panel), 분리기(Separator) 컴포넌트로 구성되어 유연한 레이아웃 관리를 지원하며, 반응형 크기 조정, 최소/최대 크기 제약, 접을 수 있는 패널 기능 등을 제공합니다. TypeScript와 호환되며, 테스트 ID와 접근성 기능을 내장하여 개발 생산성을 높였습니다. Vercel에서 호스팅되는 공식 문서에서 사용법과 API를 상세히 확인할 수 있습니다.
