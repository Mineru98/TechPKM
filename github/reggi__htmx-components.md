HTMX 컴포넌트는 HTMX를 TypeScript의 타입 안전한 JSX 환경에서 사용할 수 있도록 구현한 프레임워크입니다. 비동기 서버 컴포넌트, HTMX 속성의 `hx` 접두사 제거, 파일/폴더 기반 라우팅 없이 동적 라우팅 및 중첩을 지원합니다. `deno task` 명령어로 예제를 실행할 수 있으며, HTMX 공식 문서의 주요 예제들과 1:1 대응이 가능한 구조를 제공합니다. 서버 마크업과 URL 라우트를 직접 결합하는 철학을 기반으로 HTMX의 HTML 기반 통신 방식을 보완합니다.
