HTMX 컴포넌트는 HTMX 속성을 타입이 정의된 JSX로 사용할 수 있는 비동기 서버 컴포넌트 라이브러리입니다. URL 패턴과 마크업을 결합하여 동적 라우팅/중첩을 구현하며, 파일/폴더 구조 없이도 컴포넌트 기반 웹 개발을 지원합니다. HTMX의 HTML over the wire 접근 방식과 자연스럽게 통합되며, 타입 안전한 경로 생성 및 다양한 요청 메서드 지원이 특징입니다. Deno 환경에서 동작하는 예제를 포함하며, 기존 HTMX 예제와의 1:1 대응 구현체를 제공합니다.
