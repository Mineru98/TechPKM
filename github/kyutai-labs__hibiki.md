Hibiki는 **실시간 음성-음성 번역**을 위한 디코더 전용 모델입니다. 사용자가 말하는 동안 소스 음성을 지속적으로 처리하여 목표 언어의 자연스러운 음성과 텍스트 번역을 생성합니다. Moshi의 멀티스트림 아키텍처를 활용해 입력 스트림을 처리하며, 12.5Hz의 일정한 프레임률로 오디오 및 텍스트 토큰을 출력합니다. 현재 프랑스어-영어 번역을 지원하며, 소규모 버전(Hibiki-M)은 스마트폰에서도 실행 가능합니다. PyTorch, Rust, MLX 등의 백엔드를 통해 다양한 환경에서 사용할 수 있습니다.
