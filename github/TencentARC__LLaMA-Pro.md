LLaMA Pro는 블록 확장 기술을 활용해 점진적으로 대규모 언어 모델을 강화하는 프로젝트입니다. 해당 리포지토리는 LLaMA 및 Mistral 기반 모델의 훈련 코드와 평가 도구를 공유하며, 특히 수학/코드 성능 향상에 중점을 둡니다. ACL 2024에 채택된 기술을 기반으로 오픈소스 모델과 데모를 제공하며, MetaMath-Mistral-Pro는 GSM8k 78.4%, MATH 30.3% 패스율을 기록해 기존 모델 대비 우수한 성능을 입증했습니다. Hugging Face를 통해 모델과 데모 접속이 가능합니다.
