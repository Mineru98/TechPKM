Strix는 AI를 활용해 애플리케이션 취약점을 자동으로 탐지하고 검증하는 오픈소스 도구입니다. 실제 해커처럼 동작하며 동적 분석, 실시간 PoC 생성, CI/CD 통합을 지원합니다. 개발자 및 보안 팀이 정적 분석 도구의 오탐 없이 정확하고 신속한 보안 테스트를 수행할 수 있도록 설계되었습니다.  

주요 기능으로는 풀스택 보안 테스트 도구, 에이전트 협업 시스템, 개발자 친화적 CLI, 자동화된 보고서 생성 등이 포함됩니다. 웹 애플리케이션, GitHub 리포지토리, 로컬 코드베이스 등 다양한 대상에 적용 가능하며, OpenAI, Anthropic, Gemini 등 LLM과 연동됩니다.
