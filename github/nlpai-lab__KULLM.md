고려대학교 NLP&AI 연구실에서 개발한 한국어 대형 언어 모델(KULLM) 프로젝트입니다. KULLM3은 8×A100 GPU로 학습된 instruction-tuning 모델로, upstage/SOLAR-10.7B-v1.0을 기반으로 구축되었으며 대화 및 일반 질의응답에 특화되었습니다. 한국어 평가와 환각 현상 감소를 위한 시스템 프롬프트가 포함되어 있으며, 학습 데이터와 코드 재현성을 공개했습니다. Apache-2.0 라이선스로 배포됩니다.
