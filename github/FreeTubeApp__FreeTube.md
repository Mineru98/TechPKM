FreeTube은 프라이버시를 중시하는 오픈소스 데스크톱 유튜브 플레이어입니다. 광고 없이 유튜브를 시청하며 구글의 추적(쿠키/자바스크립트)을 방지합니다. Electron 기반으로 개발되어 Windows, macOS, Linux에서 사용 가능하며, 내장된 추출기 또는 인비디우스 API를 통해 데이터를 가져옵니다. 구독, 재생목록, 시청 기록은 로컬에 저장되어 외부 유출이 없으며, VPN 또는 Tor 사용을 권장합니다.  

기능으로는 광고 차단, 다중 추출기 선택, 프리미엄 기능(프로필별 구독 관리, 스포서블록 통합 등), 다양한 다운로드 옵션(공식/커뮤니티 빌드), 브라우저 확장 연동(LibRedirect/RediTube) 등을 제공합니다. 기여 및 로컬라이제이션에 개방적이며, GNU AGPLv3 라이선스로 배포됩니다.
