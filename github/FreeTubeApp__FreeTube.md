FreeTube는 프라이버시 보호를 염두에 두고 개발된 오픈소스 데스크톱 YouTube 플레이어입니다. 광고 없이 YouTube를 시청하고 Google의 쿠키 및 JavaScript 추적을 방지할 수 있으며, Windows, macOS, Linux에서 Electron 기반으로 실행됩니다. 구독, 재생목록, 시청 기록을 로컬에 저장하며 Invidious API 또는 내장 추출기를 선택적으로 사용할 수 있습니다. 베타 버전으로 지속적인 개선이 진행 중입니다.
