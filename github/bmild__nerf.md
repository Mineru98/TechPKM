이 프로젝트는 단일 장면을 신경 래디언스 필드(NeRF)로 표현하고 새로운 뷰를 합성하는 신경망 기반의 3D 렌더링 솔루션입니다. 텐서플로우를 사용하여 5D 입력(공간 위치 + 시야 방향)을 4D 출력(색상 + 불투명도)으로 매핑하는 완전 연결 네트워크를 학습시킵니다. 합성 및 실제 장면 데이터를 활용한 뷰 생성, 메시 추출, 사용자 정의 포즈 생성 기능을 제공하며 ECCV 2020 최우수 논문 후보로 선정되었습니다.
