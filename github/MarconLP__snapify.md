Snapify3는 자체 호스팅이 가능한 Loom 대체 웹 애플리케이션으로, 화면 녹화 및 비디오 공유를 위한 플랫폼입니다. 사용자는 탭, 데스크톱, 애플리케이션의 무제한 녹화, 공개 링크를 통한 공유, 특정 시간 후 녹화 삭제/목록 해제 기능을 사용할 수 있으며, 기존 비디오 업로드 및 공유도 지원합니다. Next.js 기반의 오픈소스 프로젝트로 MIT 라이선스 기반으로 배포됩니다. 주요 기술 스택으로는 Prisma ORM, GitHub OAuth, AWS S3(Backblaze B2) 스토리지 연동이 포함됩니다.
