ScreenSuite는 그래픽 사용자 인터페이스(GUI) 에이전트의 인식, 단일 단계 및 다중 단계 행동 능력을 종합적으로 평가하기 위한 벤치마크 도구입니다. 웹, 모바일, 데스크톱 플랫폼에서 다양한 MLLM(Multimodal Large Language Model)을 테스트하며, smolagents 기반의 간단한 에이전트 구현을 사용합니다. 주요 평가 항목에는 화면 인식 정확도, 웹/모바일 제어, 다중 단계 작업 수행 능력이 포함됩니다. Docker, Python 3.11+, KVM 가상화 환경이 필요하며 Apache 2.0 라이선스로 배포됩니다.
