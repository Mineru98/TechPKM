Excalidraw는 오픈소스 기반의 가상 손그림 스타일 화이트보드 도구로, 협업 및 엔드투엔드 암호화를 지원합니다. 무한 캔버스, 다양한 도형/툴, 이미지/도형 라이브러리, 다국어 지원, PNG/SVG/JSON 파일 내보내기 등의 기능을 제공하며, 로컬 우선 저장과 오프라인 PWA 기능으로도 활용할 수 있습니다. React 기반의 npm 패키지로 자체 앱에 통합 가능합니다. Google Cloud, Meta, Obsidian 등 다양한 기업에서 활용 중입니다.
