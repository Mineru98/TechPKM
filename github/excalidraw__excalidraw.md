Excalidraw는 오픈소스 기반의 가상 핸드드로우 스타일 화이트보드 도구입니다. 실시간 협업과 종단 간 암호화 기능을 지원하며, 다이어그램 및 와이어프레임 작성에 최적화되어 있습니다. 로컬 저장, 다양한 내보내기 형식, 사용자 정의 가능한 도구 세트를 제공하는 React 기반의 웹 애플리케이션으로, 개발자 및 디자이너가 널리 활용합니다. 주요 사용자들은 Google Cloud, Meta, CodeSandbox 등이 포함됩니다.
