"Docker SDK for Python" 프로젝트는 Docker Engine API를 Python 애플리케이션에서 활용할 수 있게 해주는 라이브러리입니다. 이 라이브러리를 통해 컨테이너 실행, 관리, Swarm 클러스터 제어 등 CLI 명령어(`docker`)로 수행하는 모든 작업을 Python 코드 내에서 프로그래밍 방식으로 처리할 수 있습니다. PyPI를 통해 설치 가능하며, 로컬 도커 환경 또는 원격 환경에 대한 연결 기능을 기본 제공합니다.  

주요 기능으로는 컨테이너/이미지 생성·조회·제어, 로그 스트리밍, 네트워크/볼륨 관리 등이 포함됩니다. 공식 문서에서는 보다 상세한 사용법과 예제 코드를 확인할 수 있습니다.
