이 프로젝트는 Docker Engine API를 위한 Python 라이브러리로, Python 애플리케이션 내에서 컨테이너 실행, 컨테이너 및 Swarm 관리 등 `docker` CLI 명령어의 모든 기능을 사용할 수 있게 합니다. Docker CLI의 기능을 Python 코드로 구현할 수 있어 DevOps 통합 및 자동화에 유용합니다. 최신 버전은 PyPI에서 설치할 수 있으며, 환경과 기본 소켓을 통해 Docker에 연결하는 간결한 API를 제공합니다. 전체 기능은 공식 문서를 참조하여 확인할 수 있습니다.
