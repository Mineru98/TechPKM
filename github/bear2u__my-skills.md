**My Skills Hub**는 Claude Code를 위한 다양한 자동화 스킬을 제공하는 프로젝트입니다. 개발 생산성 향상을 위해 Flutter/Next.js 프로젝트 생성, 코드 변경사항 문서화, AI 기반 프롬프트 최적화, 웹 콘텐츠 변환, 카드 뉴스 생성 등 18가지 핵심 스킬을 포함합니다. 각 스킬은 마켓플레이스를 통해 쉽게 설치/관리할 수 있으며, Clean Architecture/타입 안전성/디자인 시스템 등 분야별 최적화된 템플릿을 지원합니다. Claude Code와 Codex/Cursor AI 툴을 결합한 다중 검증 루프로 고품질 코드 품질을 보장하는 것이 특징입니다.
