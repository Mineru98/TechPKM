My Skills Hub는 Claude Code를 위한 커스텀 자동화 스킬 모음입니다. 개발 생산성 향상을 위해 Flutter/Next.js 프로젝트 생성, 코드 변경사항 자동화, 프롬프트 최적화 등 17가지 스킬을 제공하며, 웹 콘텐츠 변환, 카드 뉴스 생성, 문서 자동화 등 다양한 개발 워크플로우를 지원합니다. Python 기반으로 제작된 이 도구들은 코드 품질 검증, 디자인 시스템 구축, AI 엔지니어링 루프 등 고도화된 개발 환경을 구현합니다.  

### 주요 특징:
1. **프로젝트 생성**: Clean Architecture 기반의 Flutter/Next.js 템플릿 자동 생성
2. **코드 관리**: AI 기반 변경사항 문서화 및 버전 추적
3. **프롬프트 엔지니어링**: 컨텍스트 인식 프롬프트 향상 및 메타 프롬프트 생성
4. **콘텐츠 제작**: 인스타그램 카드 뉴스, 랜딩페이지 디자인 가이드 포함
5. **AI 통합**: Codex/Cursor/Claude 연계를 통한 3중 검증 시스템 지원

### 사용 시나리오:
- 신규 프로젝트 빠른 시작 (Flutter/Next.js)
- 코드 리뷰 자동화 및 문서화
- 소셜 미디어 콘텐츠 제작
- AI 기반 웹 개발 워크플로우 구축
