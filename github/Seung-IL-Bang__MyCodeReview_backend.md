코딩 테스트 문제 풀이 공유 및 복습을 위한 웹 사이트의 백엔드 시스템입니다. 2023년 4월부터 11월까지 7개월간 개인 프로젝트로 프론트엔드, 백엔드, CI/CD를 모두 구현했으며, Kafka 로그 모니터링, DB 복제, Redis 캐싱, JWT 인증 등 다양한 기술을 적용해 확장성과 성능을 최적화했습니다. 게시글/댓글/리뷰 관리, 좋아요, 동적 쿼리, 동시성 제어 등의 핵심 기능을 포함합니다.
