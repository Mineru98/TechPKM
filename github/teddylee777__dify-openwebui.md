이 프로젝트는 Docker, Ollama, Git을 활용해 Dify와 OpenWebUI를 로컬 환경에 통합하는 설치 가이드 문서입니다. Ubuntu/Mac/Windows 시스템별 환경 구성과 컨테이너 실행, API 연결, 모델 설정 등 전체 워크플로우를 단계별로 설명하며, 로컬 LLM 기반 AI 워크플로우 구축을 목표로 합니다. 주요 기술 스택으로 Docker Compose, Ollama 모델 관리, OpenWebUI 파이프라인 설정을 다룹니다.
