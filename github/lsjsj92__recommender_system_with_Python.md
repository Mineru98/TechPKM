이 프로젝트는 파이썬을 활용한 다양한 추천 시스템 구현 방법을 다루는 리포지토리입니다. 컨텐츠 기반 필터링, 협업 필터링, 행렬 분해와 같은 전통적 방법부터 딥러닝 기반 추천 시스템, Wide & Deep 모델, LLM(Large Language Model)을 활용한 설명 가능한 추천 시스템까지 폭넓은 접근법을 포함합니다. 각 구현은 Kaggle 데이터셋과 실제 네이버 뉴스 데이터를 활용했습니다. 프로젝트는 이론과 실무 구현의 균형을 유지하며, 블로그 링크를 통해 상세한 설명을 제공합니다.
