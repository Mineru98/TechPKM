이 프로젝트는 파이썬을 활용한 다양한 추천 시스템 구현 방법을 다룹니다. 컨텐츠 기반 필터링, 협업 필터링, 행렬 분해, 딥러닝 기반 추천 시스템 등 핵심 추천 알고리즘 이론과 실제 구현 사례를 포함합니다. 네이버 뉴스 추천, Keras를 활용한 모델, ChatGPT/LLM 기반 설명 가능 추천 시스템 등 최신 기술 적용 사례도 제공합니다. Kaggle 데이터셋을 주로 활용하며 머신러닝부터 딥러닝까지 폭넓게 다룹니다.
