이미지에서 텍스트를 추출하는 OCR 기능을 제공하는 Flask 기반 웹 애플리케이션으로, Tesseract와 OpenCV를 활용하여 이미지 노이즈를 제거하고 정확도를 향상시킵니다. Docker 컨테이너로 배포되어 로컬 또는 클라우드 환경에서 쉽게 실행할 수 있으며, 테스트용 이미지 링크도 포함되어 있습니다. Python 3.6으로 업데이트되어 호환성을 개선했습니다.
