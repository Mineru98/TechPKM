html2pdf.js는 웹 페이지나 HTML 요소를 클라이언트 사이드에서 이미지로 변환한 후 PDF로 생성하는 JavaScript 라이브러리입니다. html2canvas와 jsPDF를 기반으로 하며, CSS 페이지 분할, 이미지 품질 조정, 다양한 설치 방식 등을 지원합니다. 사용자는 웹 콘텐츠를 쉽게 PDF로 내보낼 수 있으며, Promise 기반 API를 통해 변환 과정을 세부적으로 제어할 수 있습니다.  

주요 기능으로는 CDN/NPM/Bower 설치 지원, 페이지 분할 설정, 이미지 형식/품질 선택, 진행 상태 추적 등이 포함됩니다. Node.js 환경에서는 작동하지 않으며, 브라우저 기반으로만 사용됩니다.
