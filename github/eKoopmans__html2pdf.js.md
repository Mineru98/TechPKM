html2pdf.js는 웹페이지 또는 특정 요소를 클라이언트 사이드에서 PDF로 변환하는 JavaScript 라이브러리입니다. html2canvas와 jsPDF를 활용해 이미지 기반 PDF 생성을 지원하며, 다양한 설정 옵션과 체이닝 가능한 Promise 기반 API를 제공합니다. 웹 콘텐츠를 검색 불가능한 이미지로 렌더링하는 방식으로 동작하며, 페이지 분할, 이미지 품질 조정 등 고급 기능을 포함합니다. MIT 라이선스로 배포되는 오픈소스 프로젝트입니다.
