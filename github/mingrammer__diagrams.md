Diagrams는 Python 코드로 클라우드 시스템 아키텍처를 시각화하는 도구입니다. 설계 도구 없이 새로운 시스템 아키텍처를 프로토타이핑하거나 기존 시스템을 문서화할 때 사용되며, AWS, Azure, GCP, Kubernetes 등 주요 클라우드 제공업체와 온프레미스/SaaS 환경을 지원합니다. 버전 관리 시스템과 연동해 아키텍처 변경 사항을 추적할 수 있지만, 실제 클라우드 리소스나 인프라 코드를 생성하지는 않습니다.  

> 핵심 기능:  
> - 코드 기반 다이어그램 생성  
> - 10개 이상 클라우드 제공업체 지원  
> - Python 3.9+ 호환  
> - Graphviz 렌더링 통합  
> - Apache Airflow 등 실제 프로젝트 적용 사례 보유
