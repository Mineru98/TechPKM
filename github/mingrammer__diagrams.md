**Diagrams**는 Python 코드로 클라우드 시스템 아키텍처를 그리는 도구입니다. AWS, Azure, GCP, Kubernetes 등 주요 클라우드 제공업체와 온프레미스, SaaS, 프로그래밍 프레임워크를 지원하며, 설계 도구 없이도 시스템 아키텍처를 프로토타이핑하거나 기존 시스템을 시각화할 수 있습니다. 버전 관리 시스템과 연동해 아키텍처 변경 사항을 추적할 수 있으며, Graphviz를 기반으로 다이어그램을 렌더링합니다. 실제 클라우드 리소스 제어 또는 인프라 코드 생성 기능은 제공하지 않습니다.
