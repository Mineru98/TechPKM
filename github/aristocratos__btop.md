btop++는 Linux, macOS, FreeBSD, NetBSD, OpenBSD를 지원하는 크로스 플랫폼 시스템 리소스 모니터링 도구입니다. C++23로 개발되었으며, CPU, 메모리, 디스크, 네트워크 사용량 및 프로세스 정보를 실시간으로 시각화합니다. 게임 스타일의 메뉴 시스템, 마우스 지원, GPU 모니터링, 사용자 정의 테마, 프로세스 필터링/정렬 기능을 제공합니다. 터미널 기반 인터페이스에서 동작하며, braille/block/tty 그래프 심볼과 사용자 정의 프리셋으로 유연한 레이아웃을 지원합니다.
