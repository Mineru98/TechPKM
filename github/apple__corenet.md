CoreNet은 다양한 작업(CLIP, LLM, 분류, 검출, 세그멘테이션 등)에 대해 표준 및 새로운 소규모/대규모 모델을 훈련하기 위한 딥러닝 툴킷입니다. 애플에서 개발한 이 라이브러리는 재현 가능한 훈련 레시피, 사전 훈련된 모델 가중치, 튜토리얼을 제공하며, 컴퓨터 비전을 넘어 파운데이션 모델 훈련까지 지원합니다. PyTorch 기반으로 Linux/macOS에서 사용 가능하며, MLX를 통한 Apple Silicon 최적화 예제도 포함되어 있습니다.
