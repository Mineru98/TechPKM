이 프로젝트는 노션 테이블의 "발행 요청" 또는 "수정 요청" 상태 레코드를 감지하여 자동으로 티스토리 블로그에 게시물을 업로드하는 자동화 도구입니다. 노션 테이블 구조와 티스토리 API 연동을 통해 콘텐츠 발행 프로세스를 간소화하며, 업로드 완료 시 상태를 "발행 완료"로 자동 업데이트합니다. Selenium과 티스토리 Open API를 활용하며, 선택적으로 Gmail 알림 기능도 제공합니다.  

핵심 특징:  
1. 노션 테이블 컬럼 매핑 시스템 (제목/카테고리/태그/상태/링크)  
2. HTML 코드 블록 테마비 지원  
3. 보안 토큰과 API 키를 통한 안전한 인증 시스템  
4. 확장 가능한 설정 구조 (config.py)
