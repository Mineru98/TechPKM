Guidance는 언어 모델 출력을 효율적으로 제어하기 위한 프로그래밍 패러다임입니다. Python 기반의 직관적인 인터페이스를 제공하여 정규표현식, CFG(문맥 자유 문법), JSON 스키마 등을 활용한 제약 생성을 지원하며, Jupyter 노트북 위젯과의 통합으로 풍부한 사용 경험을 제공합니다. 설치 및 사용이 간편하며, 다양한 백엔드(Transformers, llama.cpp, OpenAI 등)와 호환됩니다.  

핵심 특징으로는 모델 출력 형식 보장, 사용자 정의 Guidance 함수 생성, HTML/XML 생성, JSON 스키마 기반 생성 등이 포함됩니다. 특히 문법 기반 생성과 토큰 패스트 포워딩 기능으로 계산 비용을 절감하면서도 구조적 출력을 보장합니다.
