Guidance는 언어 모델의 출력을 효율적으로 제어하기 위한 프로그래밍 패러다임으로, 구조화된 출력을 보장하며 지연 시간과 비용을 줄입니다. 정규 표현식, 문맥 자유 문법(CFG), 조건문, 루프 등을 활용해 생성 과정을 세밀하게 제어할 수 있으며, Jupyter 노트북과 같은 환경에서 직관적인 위젯을 지원합니다. PyPI를 통해 설치 가능하며 다양한 백엔드(Transformers, llama.cpp, OpenAI 등)와 호환됩니다. 주요 기능으로 JSON 스키마 기반 생성, 사용자 정의 Guidance 함수 작성, HTML과 같은 구조적 텍스트 생성이 포함됩니다.
