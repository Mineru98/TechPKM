PyMySQL은 순수 Python으로 구현된 MySQL과 MariaDB 클라이언트 라이브러리로, PEP 249 표준을 준수합니다. Python 애플리케이션에서 MySQL/MariaDB 데이터베이스 연결, 쿼리 실행, 트랜잭션 관리 등을 쉽게 수행할 수 있도록 설계되었습니다. CPython 3.9 이상과 PyPy 최신 버전을 지원하며, 다양한 인증 방식(sha256_password, ed25519 등)을 선택적으로 사용할 수 있습니다. MIT 라이선스로 배포되며, PyPI를 통해 쉽게 설치할 수 있습니다.
